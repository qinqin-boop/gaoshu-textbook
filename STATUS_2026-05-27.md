## 七、技术问题与解决

### 1. Sender 卡顶栏 OCR

- **问题**: sender 重启后顶栏 OCR 读"湫桃" 实际读空, 留队列重试
- **诊断**: 手机端通知栏拉下来挡了顶栏标题, ChattingUI focus 但 OCR 看不到
- **修复**: adb shell input keyevent 4 关通知栏 → sender 恢复发送
- **教训**: sender 的"拒盲打"是好行为, 不能改

### 2. screencap /sdcard 路径在 Git Bash 转义

- **问题**: adb shell screencap -p /sdcard/x.png 被 Git Bash 转成 C:/Program Files/Git/sdcard
- **修复**: MSYS_NO_PATHCONV=1 前缀 或 用 PowerShell

### 3. PIL 读 PNG 经 stdout 损坏

- **问题**: adb shell screencap -p > file.png 在 Git Bash 把 \n 转 \r\n, PIL 读不了
- **修复**: 用 adb shell screencap -p /sdcard/tmp.png + adb pull
