## 8.0 整章在干啥?

上册微积分研究**一元函数** $y=f(x)$,自变量在数轴上。本章把数学几何化:**向量**取代数,**空间曲面**取代曲线。学完本章你能用代数语言描述三维空间里的任何线、面、立体——这是后续所有多元微积分(§9-12)与所有工科 3D 应用(图形学 / 机器人 / CAD / 物理仿真)的几何源。

### 8.0.1 全章地图

```
§8.1 向量代数 ──→ 线性运算 + 标准基 + 投影 ─→ §8.2 数量积/向量积/混合积
                                                  │
§8.3 曲面方程 (球/旋转/柱/二次5兄弟) ←─────────┘
                                                  │
§8.4 平面方程 (点法式 + 一般式 + 距离公式) ←─── 数量积 + 法向量
                                                  │
§8.5 空间直线 (参数式 + 对称式 + 一般式) ←──── 向量积 (求方向)
                                                  │
                       综合: 直线/平面/曲面位置关系 + 距离计算
                                                  │
                       应用: §9 切平面 / §10 球柱坐标 / §11 曲面积分
```

### 8.0.2 核心代数对象

| 代数对象 | 几何角色 | 后续章用途 |
| --- | --- | --- |
| 向量 $\vec{a}$ | 方向 + 大小 | 全章基础 |
| 数量积 $\vec{a}\cdot\vec{b}$ | 投影 / 角度 / 功 | §9.6 方向导数 / §11.2 力做功 |
| 向量积 $\vec{a}\times\vec{b}$ | 平行四边形面积 / 平面法向 | §9.7 切平面 / §11.4 $dS$ |
| 混合积 $[\vec{a},\vec{b},\vec{c}]$ | 平行六面体体积 / 共面 | §10.3 三重积分 / §11.5 通量 |
| 平面 $\vec{n}\cdot(\vec{x}-\vec{x}_0)=0$ | 切平面骨架 | §9.7 / §11.4 |
| 二次曲面 | 椭球 / 抛物面 / 双曲面 | §10.3 球坐标 / §11.4 法向 |

### 8.0.3 学这章的意义

- **几何代数化**:用代数式表达几何对象,告别"画图证明"——这是 Descartes 在 17 世纪发起的革命;
- **多元微积分的语言**:§9-12 全靠向量记号才能简洁;
- **AI / ML**:词向量 / 图像特征 / 神经网络权重都是高维向量,数量积 = 余弦相似度,向量积无直接 3D 推广(改用张量 / 外代数);
- **工程 / 物理**:刚体力学 / 电磁学 / 流体力学的微分方程全用向量场写。
