## 11.0 整章在干啥?

之前的积分都建在**简单几何对象**上:
- 一元定积分 $\int_a^b f\,dx$ — 在直线段上;
- 二重积分 $\iint_D f\,d\sigma$ — 在平面区域上;
- 三重积分 $\iiint_\Omega f\,dV$ — 在空间立体上。

但物理世界里大量"积分量"住在**弯曲对象**上:弯钢丝的质量、力沿曲线的功、电场穿过弯曲面的通量、流体穿过控制面的流量。本章把积分搬到**一维流形**(曲线)与**二维流形**(曲面)上,并用三大公式 Green / Gauss / Stokes 把这些"新"积分还原回"旧"重积分。学完本章,你能直接看懂 Maxwell 电磁场方程组的积分形式 / 流体动力学控制体方程 / 计算机图形学光照渲染——这是经典数学物理的最高峰之一。

### 11.0.1 全章地图:四类积分 + 三大公式

::intuition
**四类积分**(按几何对象 × 方向感知交叉分):

| 几何对象 | 不带方向(标量化) | 带方向(向量化) |
| --- | --- | --- |
| **曲线** $L$ | 第一类 $\int_L f\,ds$(对弧长) | 第二类 $\int_L\vec{F}\cdot d\vec{r}$(对坐标) |
| **曲面** $\Sigma$ | 第一类 $\iint_\Sigma f\,dS$(对面积) | 第二类 $\iint_\Sigma\vec{F}\cdot d\vec{S}$(对坐标) |

- 第一类积分 $\Leftrightarrow$ 弧长 / 面积元 $ds,dS$ **永远正**,$f$ 是标量(密度);
- 第二类积分 $\Leftrightarrow$ 微元 $d\vec{r}=\vec{\tau}\,ds$、$d\vec{S}=\vec{n}\,dS$ 带方向,$\vec{F}$ 是向量(力 / 流速 / 电场);
- 两类积分通过 $d\vec{r}=\vec{\tau}\,ds$、$d\vec{S}=\vec{n}\,dS$ 互转。

**三大公式**(把曲线 / 曲面积分还原到内部重积分):

| 公式 | 边界 | 内部 |
| --- | --- | --- |
| **Green**(§11.3) | 平面闭曲线 $\oint_{\partial D}P\,dx+Q\,dy$ | 平面区域 $\iint_D(\partial_xQ-\partial_y P)\,d\sigma$ |
| **Gauss**(§11.5) | 闭曲面通量 $\oiint_{\partial V}\vec{F}\cdot d\vec{S}$ | 立体散度 $\iiint_V\nabla\cdot\vec{F}\,dV$ |
| **Stokes**(§11.6) | 空间闭曲线环流 $\oint_{\partial\Sigma}\vec{F}\cdot d\vec{r}$ | 曲面旋度通量 $\iint_\Sigma(\nabla\times\vec{F})\cdot d\vec{S}$ |
::end

```

                              ├──→ §11.3 Green 公式 (平面闭路 ↔ 区域)
§11.2 第二类曲线积分(对坐标)──┘                  │
                                                  │
§11.4 第一类曲面积分(对面积)──┐                  │
                              ├──→ §11.5 Gauss 公式 (闭曲面 ↔ 立体)
§11.5 第二类曲面积分(对坐标)──┘                  │
                                                  │
                              §11.6 Stokes 公式 (曲线 ↔ 曲面)
                                                  │
                              统一表达: ∫_∂M ω = ∫_M dω  (广义 Stokes)
                                                  │
                              应用: Maxwell 方程组 / 流体 N-S / de Rham
```

### 11.0.3 与 §9-10 的承接

- §9.6 **梯度** $\nabla f$、§9.7 **切向 / 法向**:本章 $d\vec{r}=\vec{\tau}\,ds$、$d\vec{S}=\vec{n}\,dS$ 的代数源;
- §9 偏导**算子** $\nabla=(\partial_x,\partial_y,\partial_z)$ 在本章具体体现为**散度** $\nabla\cdot\vec{F}$、**旋度** $\nabla\times\vec{F}$、**梯度** $\nabla\phi$;
- §10 重积分提供本章三大公式的"内部"积分工具(Fubini + 极/柱/球坐标换元);
- 微分形式视角下,**所有积分都是 $\int_M\omega$**,定理统一为 $\int_{\partial M}\omega=\int_M d\omega$——这是数学物理统一观的入口。
