一元函数 y=f(x) 升级到多元 z = f(x, y) — 导数也跟着推广.

9.0 整章在干啥?

一元函数 (中学/上册) 你已经会:

  • 给定 y = x², 求导数 dy/dx
  • 用导数找极值 (最高点/最低点)
  • 求切线方程

现在世界更复杂: 山的高度不只跟"东西方向" x 有关, 还跟"南北" y 有关 → z = f(x, y) 是个曲面. 怎么算这种"多变量函数" 的导数+极值+切平面? — 这章答.

对你的意义: AI 神经网络训练靠梯度下降 — 而梯度 = 多元函数的偏导. 这章直接对应 ML 数学基础.

9.1 多元函数的基本概念

知识点 0.5: 为什么要从一元升级到多元?

一元 vs 多元 — 描述真实世界

一元 $y = f(x)$: 一个输入对一个输出. 例 工资 = $f$(工龄).

但现实复杂:

  • 工资 = $f$(工龄, 学历, 城市, 行业) — 多输入
  • 山高度 = $f$(经度, 纬度)
  • 气温 = $f$(经度, 纬度, 海拔, 时间)
  • 神经网络损失 = $f(w_1, w_2, \ldots, w_n)$ ($n$ 可上百亿)

学多元函数 = 给真实世界建模. AI 训练就是在多元函数 (loss) 上找最低点.

本章学三件事 (AI 训练的全部数学基础)

  1. 偏导 / 全微分: 怎么求多元函数的"导数"
  2. 极值 / Hessian / Lagrange: 怎么找多元函数的"最高 / 最低点" (含约束)
  3. 方向导数 + 梯度: 怎么沿任意方向算变化率, 找"最陡上升方向"

为啥关键: AI 训练 = 在多元函数 (loss) 上找最低点. 整套技巧 (梯度下降 / 反向传播 / Adam / 牛顿法) 全靠这章基础.

知识点 1: 二元函数定义

二元函数

两个输入一个输出. $z = f(x, y)$. 例: $z = x^2 + y^2$ (抛物面/碗形曲面).

一元 vs 二元

| 维度 | 一元 $y = f(x)$ | 二元 $z = f(x,y)$ | |---|---|---| | 输入 | 1 个数 | 2 个数 (一对) | | 输出 | 1 个数 | 1 个数 | | 几何对象 | 曲线 (2D 中) | 曲面 (3D 中) | | 定义域 | $\mathbb{R}$ 上一段 | $\mathbb{R}^2$ 上一个区域 | | 生活类比 | 工资 $=f$(工龄) | 工资 $=f$(工龄, 学历) |

定义域: 满足函数有定义的 (x, y) 集合. 比如 $z = \sqrt{1 - x^2 - y^2}$ 的定义域是单位圆盘内.

例题 1

z = (1)/(√(4 - x² - y²)) 的定义域? 分母不能为 0 + 根号下大于 0: 4 - x² - y² > 0, 即 x² + y² < 4 (半径 2 开圆盘).

知识点 2: 二元函数的极限

二元极限 (严格版)

$\lim_{(x,y)\to(x_0,y_0)} f(x,y) = L$ ⟺ 对任意 $\epsilon > 0$, 存在 $\delta > 0$, 使当 $0 < \sqrt{(x-x_0)^2 + (y-y_0)^2} < \delta$ 时, $|f(x,y) - L| < \epsilon$.

直观: 不管 $(x,y)$ 怎么靠近 $(x_0,y_0)$ (直线/曲线/螺旋), $f$ 都趋于 $L$.

路径不同极限不同

若沿不同路径 $(x,y) \to (x_0, y_0)$ 极限不同, 则二元极限不存在. 经典反例: $$f(x,y) = \frac{xy}{x^2 + y^2}$$ 沿 $y = 0$: $f \to 0$; 沿 $y = x$: $f \to 1/2$ — 极限不存在.

---

知识点 3: 连续性

二元函数连续

$f(x,y)$ 在 $(x_0, y_0)$ 连续 ⟺ $\lim_{(x,y)\to(x_0,y_0)} f(x,y) = f(x_0, y_0)$. 即"极限值 = 函数值".

连续 = "山地没断崖"

山地表面没断崖, 就连续. 有断崖 (跳跃点 / 撕裂处), 不连续. 严格说: 极限值得能"无缝接住"函数值.

---

知识点 3.5: 二元函数图像的"等高线" 法 (对齐书 §9.1 进一步)

等高线 — 把 3D 曲面"压平"到 2D

问题: $z = f(x,y)$ 图像是 3D 曲面, 但纸 / 屏幕是 2D, 怎么表示?

等高线法: 用 $z = c$ (常数) 平面切曲面, 得到一族曲线 (类比地图等高线).

例: $z = x^2 + y^2$ (碗). 等高线: $c=1 \Rightarrow x^2+y^2=1$ (单位圆), $c=4 \Rightarrow$ 半径 2 圆. 等高线密 → 曲面陡, 疏 → 缓.

几何意义:

  • 山区地图: 海拔等值线
  • 气象: 等压线 / 等温线
  • AI loss 图: 等高线 = 损失等值集, 目标 = 找最低"谷底"

梯度与等高线正交 (后面知识点 7 证)

$\nabla f$ 永远跟等高线垂直, 指向 $z$ 增加最快方向. 这是梯度下降算法直接利用的几何事实.

9.2 偏导数 (本章核心)

知识点 3.7: 二元函数极限的多路径检测 (对齐书 §9.1 三)

二元极限 = 任意路径同值

核心: 二元极限要求"任意路径趋近 $(x_0, y_0)$ 都得到同一值"; 一元只有左右两侧, 二元有 $\infty$ 多条路径, 要求更严.

实操检测法:

  1. 沿 $x$ 轴: $y = 0, x \to 0$, 看 $f(x, 0)$
  2. 沿 $y$ 轴: $x = 0, y \to 0$, 看 $f(0, y)$
  3. 沿 $y = kx$ 直线: 看是否含 $k$ (含 $k$ → 不存在)
  4. 沿 $y = x^2$ 等曲线 (上面 3 不出问题但仍可能差)

找到两条不同 = 不存在

两条路径值不同 → 极限不存在 (反例反证, 不必检测所有路径). 但反过来"3 条路径相等 ≠ 极限存在" — 真证存在要用 $\varepsilon\delta$ 或夹逼.

经典反例:

例题 1-5

lim_{(x,y)→(0,0)} (xy)/(x²+y²). 沿 y = 0: f(x, 0) = 0. 极限 = 0. 沿 y = x: f(x, x) = x²/(2x²) = 1/2. 极限 = 1/2. 两值不同 → 极限不存在 ✗

例题 1-6

lim_{(x,y)→(0,0)} (x²y)/(x⁴+y²). 沿 y = 0: → 0. 沿 y = kx: f → 0. 沿 y = x²: f(x, x²) = x⁴/(x⁴+x⁴) = 1/2. ≠ 0. 极限不存在 ✗

知识点 4: 偏导数定义

偏导数

二元函数 $z = f(x, y)$ 沿 x 方向 的偏导数 (把 y 当常数): $$\frac{\partial z}{\partial x} = \lim_{\Delta x \to 0} \frac{f(x+\Delta x, y) - f(x, y)}{\Delta x}$$ 记号: $f_x$, $\partial z/\partial x$, $z_x$ 都是它. 类似定义 $\partial z/\partial y$.

生活类比: 站在山坡某点, 只往东走一小步, 高度变化率 = ∂ z / ∂ x. 只往北走一小步, 高度变化率 = ∂ z / ∂ y.

求法核心: 对 x 求偏导, 把 y 看成数字, 用一元导数规则; 对 y 求偏导, 把 x 看成数字.

例题 2

z = x² + 3xy + y², 求 ∂ z / ∂ x 和 ∂ z / ∂ y.

  • ∂ z / ∂ x: 把 y 当数, 对 x 求导: 2x + 3y · 1 + 0 = 2x + 3y.
  • ∂ z / ∂ y: 把 x 当数, 对 y 求导: 0 + 3x + 2y = 3x + 2y.

例题 3

z = e^(xy), 求两偏导.

  • ∂ z / ∂ x: 对 x 求导, y 是数. ∂ e^(xy) / ∂ x = y · e^(xy).
  • ∂ z / ∂ y: x · e^(xy).

例题 3-2

z = x^y (含两个变量的指数), 求两偏导.

  • 对 x 求偏导, y 是常数 → 类似 x 的 y 次方求导: ∂z/∂x = y · x^(y-1)
  • 对 y 求偏导, x 是常数, 用对数微分法: z = x^y, ln z = y ln x, (1/z)·∂z/∂y = ln x, 所以 ∂z/∂y = x^y · ln x.

偏导数的几何意义: ∂z/∂x 在点 (x₀, y₀) 处 = 曲面 z = f(x,y) 被平面 y = y₀ 截出的曲线, 在该点的切线斜率.

生活类比: 山坡上"切一刀" (用一个平行 xz 的垂直墙切下来), 看墙上画出的"轮廓线" 的斜率 — 就是 ∂z/∂x.

例题 3-3

一个气球温度 T(x,y) = 100 - x² - 2y², 在 (1, 1) 处 ∂T/∂x = -2 表示啥? 表示在 (1, 1) 这点, 只沿 x 方向移动单位距离, 温度每变 -2 度. (沿 x 走变冷)

知识点 4.3: 偏导数的几何意义 (对齐书 §9.2 进一步说明)

偏导数 = 截面曲线斜率

$\partial z/\partial x$ 在 $(x_0, y_0)$ 的几何意义:

  • 用平面 $y = y_0$ 截曲面 $z = f(x, y)$, 得到曲线 $z = f(x, y_0)$ (一元函数)
  • 这条曲线在 $x = x_0$ 处的切线斜率 = $\partial z/\partial x|_{(x_0, y_0)}$

类似 $\partial z/\partial y$ = 用 $x = x_0$ 平面截出的曲线在 $y = y_0$ 处的切线斜率.

口诀: "冻结一个变量, 看另一个变量方向上的斜率".

山路斜率直觉

山的高度 $z = f(x, y)$, 站在 $(x_0, y_0)$ 点:

  • 正东走小步 (沿 $x$ 轴), 高度变化率 = $\partial z/\partial x$
  • 正北走小步 (沿 $y$ 轴), 高度变化率 = $\partial z/\partial y$

偏导是"单方向"斜率, 跟 grad / 方向导数 (任意方向) 形成对照.

例题 3-4

f(x,y) = x² + y² (碗形). ∂f/∂x = 2x, ∂f/∂y = 2y. 在 (1, 1) 点: ∂f/∂x|_(1,1) = 2, ∂f/∂y|_(1,1) = 2. 意义: 这点沿 x 走, 高度上升率 2; 沿 y 走, 高度上升率 2. 都是正, 说明这点不是底部.

知识点 4.5: 偏导数存在 ≠ 函数连续

多元里偏导存在不蕴含连续

一元: 可导 ⇒ 连续 (无条件成立). 二元: 偏导 $f_x, f_y$ 都存在, 不能推出 $f$ 连续.

这是多元微积分最容易踩的坑.

偏导存在不蕴含可微

更进一步: 偏导存在也不蕴含可微. 想可微需要 $f_x, f_y$ 连续 (足够条件).

反例: 偏导存在但不连续

$$f(x,y) = \begin{cases} \dfrac{xy}{x^2 + y^2}, & (x,y)\ne(0,0) \\ 0, & (x,y) = (0,0) \end{cases}$$

  • 在 $(0,0)$ 偏导: $f_x(0,0) = \lim_{\Delta x\to 0}(f(\Delta x, 0)-0)/\Delta x = \lim 0/\Delta x = 0$. 同样 $f_y(0,0) = 0$. 偏导都存在 (都是 0).
  • 但在 $(0,0)$ 不连续: 沿 $y = x$ 趋近, $f = x^2/(2x^2) = 1/2 \ne 0$.

为啥踩坑 — "偏导只看坐标轴"

偏导只检测沿 $x$ 轴 / $y$ 轴两条线方向. 函数完全可能在"斜方向" $y = x$ 上不连续 — 偏导抓不到. 这就是高维比一维"难"的本质.

结论: 真正"好"的条件 = 可微

二元函数要真正"好", 需要更强条件叫可微 (全微分存在). 可微 $\Rightarrow$ 连续 $+$ 偏导都存在. 反之不行. 下一节展开.

知识点 5: 高阶偏导数

二阶偏导

偏导再求偏导:

  • $\dfrac{\partial^2 z}{\partial x^2} = (z_x)_x$: 对 $x$ 求两次
  • $\dfrac{\partial^2 z}{\partial x \partial y} = (z_y)_x$: 先 $y$ 后 $x$
  • $\dfrac{\partial^2 z}{\partial y \partial x} = (z_x)_y$: 先 $x$ 后 $y$

Clairaut/Schwarz 定理 (混合偏导对称性)

若 $f_{xy}, f_{yx}$ 在 $(x_0, y_0)$ 附近都连续, 则: $$f_{xy}(x_0, y_0) = f_{yx}(x_0, y_0)$$ 即混合偏导跟求导顺序无关.

连续性是关键

混合偏导不连续时可能不相等. 经典反例 (大多数实际函数满足): $f(x,y) = xy(x^2-y^2)/(x^2+y^2)$ 在原点 $f_{xy}(0,0) \ne f_{yx}(0,0)$.

例题 4

z = x³ + 3x² y + y³, 求所有二阶偏导.

  • z_x = 3x² + 6xy
  • z_y = 3x² + 3y²
  • z_(xx) = 6x + 6y
  • z_(yy) = 6y
  • z_(xy) = 6x (从 z_x 对 y 求)
  • z_(yx) = 6x (从 z_y 对 x 求) — 跟 z_(xy) 相等 ✓

---

知识点 5.5: 二阶混合偏导数相等性的反例 (对齐书 §9.2 进一步)

经典反例: $f_{xy}(0,0) \ne f_{yx}(0,0)$

$$f(x,y) = \begin{cases} xy(x^2-y^2)/(x^2+y^2), & x^2+y^2 > 0 \\ 0, & (0,0) \end{cases}$$ 直接算: $f_{xy}(0,0) = 1$, $f_{yx}(0,0) = -1$. 因为 $f_{xy}$ 在 $(0,0)$ 不连续, Clairaut 定理前提不满足, 所以不相等.

实用结论 — 工程函数不用担心

工程 / 物理常见函数 (多项式 / 三角 / 指数 / 有理函数 …) 都满足 $C^2$ 连续条件 → 可放心交换求导顺序. 反例只在病态函数 (含分式 + 原点拼接) 才会出.

知识点 5.7: 偏导数判别可微 (对齐书 §9.3 进一步)

可微的充分条件

$f_x, f_y$ 在 $(x_0, y_0)$ 邻域内连续 $\Rightarrow f$ 在 $(x_0, y_0)$ 可微.

可微的必要条件

$f$ 在 $(x_0, y_0)$ 可微 $\Rightarrow f_x, f_y$ 在 $(x_0, y_0)$ 存在 (但偏导存在不保证可微!).

记忆链

$$\text{偏导连续} \Rightarrow \text{可微} \Rightarrow \text{连续}, \text{偏导存在}$$ 反向不成立: 偏导存在 $\not\Rightarrow$ 可微 $\not\Rightarrow$ 连续 $\not\Rightarrow$ 偏导连续.

例题 5-4

(经典反例 — 偏导存在但不可微): f(x,y) = xy/√(x²+y²), f(0,0)=0. 偏导: f_x(0,0) = 0, f_y(0,0) = 0 (存在!). 但 f 在 (0,0) 不可微 — 沿 y=x 方向, lim (f(h,h) - 0)/√(2)h = h²/(√2 h · √(2)h) = 1/2 ≠ 0. 可微要求"线性逼近", 这函数不行.

9.3 全微分

知识点 5.8: 偏导数的高级应用 — 雅可比矩阵

雅可比矩阵 (Jacobian)

多元向量函数 $F: \mathbb{R}^n \to \mathbb{R}^m$, $F(\vec{x}) = (f_1(\vec{x}), \ldots, f_m(\vec{x}))$. 雅可比矩阵: $$J = \left[\frac{\partial f_i}{\partial x_j}\right]_{m \times n}$$

Jacobian 3 大作用

  1. 多元导数: 一元导数 $f'(x)$ 的多元推广 (线性逼近的系数矩阵).
  2. 重积分换元: $|\det J| = $ 变换的"体积放大比" — 这就是为啥重积分换元乘 $|\det J|$.
  3. 神经网络反向传播: 每层 Jacobian 乘起来 = 链式法则的矩阵表达, 推出 backprop 算法.

统一: 一元 $df = f'(x)\,dx$ 推广为 $d\vec{F} = J\,d\vec{x}$.

例题 5-7

F(x,y) = (x²+y, xy). 求 Jacobian. J = [[2x, 1], [y, x]] (2×2 矩阵) |det J| = 2x² - y.

深度学习: 神经网络每层 z = f(Wx+b), Jacobian = f'(...)·W. 反向传播从后往前累乘 J. 这是 PyTorch/TensorFlow 自动求导的数学基础.

知识点 6: 全微分定义

全微分

二元函数 $z = f(x,y)$ 在 $(x_0, y_0)$ 可微 ⟺ 全增量可写为: $$\Delta z = \frac{\partial z}{\partial x}\Delta x + \frac{\partial z}{\partial y}\Delta y + o(\rho)$$ ($\rho = \sqrt{\Delta x^2 + \Delta y^2}$, $o(\rho)$ 是高阶无穷小). 此时全微分: $$dz = \frac{\partial z}{\partial x}\,dx + \frac{\partial z}{\partial y}\,dy$$

你在山坡上走一小步, 既有"东向" 也有"北向"分量 — 总高度变化 ≈ 东向变化率 × 东走距离 + 北向变化率 × 北走距离. 全微分就是这个线性化逼近.

生活类比: 你在山坡上走一小步, 这一步既有"往东" 也有"往北" 两个分量 — 总高度变化 = 东向变化率 × 东向走的距离 + 北向变化率 × 北向走的距离.

例题 5

z = x² y + 3y², 求全微分.

  • z_x = 2xy, z_y = x² + 6y.
  • dz = 2xy · dx + (x² + 6y) · dy.

可微的定义 (严格版): Δz - (∂z/∂x · Δx + ∂z/∂y · Δy) = o(ρ), 其中 ρ = √(Δx² + Δy²) 是"靠近"的距离.

通俗讲就是: 用线性式 (偏导×Δx + 偏导×Δy) 近似 Δz, 误差比 ρ 小一个量级.

可微 ⇒ 连续 ⇒ 极限存在. 可微是二元函数"光滑性"的标准.

充分条件: f_x, f_y 在该点连续 ⇒ f 在该点可微 (反之不一定).

知识点 6.5: 全微分的近似计算 (实用价值)

思想: 函数 f(x,y) 在 (x₀,y₀) 邻近, 用线性近似: f(x₀+Δx, y₀+Δy) ≈ f(x₀,y₀) + f_x(x₀,y₀)·Δx + f_y(x₀,y₀)·Δy

例题 5-2

估算 (1.02)^(2.99) ≈ ? 取 f(x,y) = x^y, x₀=1, y₀=3, Δx=0.02, Δy=-0.01. f(1,3) = 1. f_x = y x^(y-1) → f_x(1,3) = 3. f_y = x^y · ln x → f_y(1,3) = 1·ln 1 = 0. f(1.02, 2.99) ≈ 1 + 3·0.02 + 0·(-0.01) = 1.06. (精确值约 1.0608, 误差<0.001)

例题 5-3

求 √(1.02² + 2.97²) 近似值. f(x,y) = √(x²+y²), 取 x₀=1, y₀=3. f(1,3) = √10 ≈ 3.162. f_x = x/√(x²+y²) → f_x(1,3) = 1/√10. f_y = y/√(x²+y²) → f_y(1,3) = 3/√10. Δx = 0.02, Δy = -0.03. 近似 ≈ √10 + (0.02 - 0.09)/√10 = √10 - 0.07/√10 ≈ 3.162 - 0.022 = 3.140.

知识点 6.6: 全微分的"叠加思想" (对齐书 §9.3 进一步)

核心思想: dz 是 dx, dy 引起的两个独立变化的叠加 — 第一项 ∂z/∂x · dx 是"只 x 变 y 不变" 的影响, 第二项 ∂z/∂y · dy 是"只 y 变 x 不变" 的影响.

这是线性化: 一个复杂的多变量小变化, 拆成"沿坐标轴各方向的小变化" 之和.

几何意义: 切平面方程: 曲面 z = f(x,y) 在 (x₀, y₀, z₀) 处的切平面: z - z₀ = f_x(x₀,y₀)(x - x₀) + f_y(x₀,y₀)(y - y₀)

这就是把曲面在该点用 dz = f_x dx + f_y dy 这条"切平面" 替代.

例题 5-4

曲面 z = x² + y² 在点 (1, 2, 5) 处的切平面方程. f_x = 2x, f_y = 2y, 在 (1,2): f_x = 2, f_y = 4. 切平面: z - 5 = 2(x-1) + 4(y-2), 整理: 2x + 4y - z = 5.

进一步 — 法线方程: 过 (x₀, y₀, z₀) 垂直于切平面的直线, 方向 = (f_x, f_y, -1). (x-x₀)/f_x = (y-y₀)/f_y = (z-z₀)/(-1)

例题 5-5

上面曲面在 (1, 2, 5) 处的法线方程. 方向 = (2, 4, -1). 法线: (x-1)/2 = (y-2)/4 = (z-5)/(-1).

9.4 多元复合函数求导 (链式法则)

知识点 7: 链式法则

链式法则 (情形 1: 单变量 t)

$z = f(u, v)$, $u = u(t)$, $v = v(t)$: $$\frac{dz}{dt} = \frac{\partial z}{\partial u}\frac{du}{dt} + \frac{\partial z}{\partial v}\frac{dv}{dt}$$

链式法则 (情形 2: 多变量 x, y)

$z = f(u, v)$, $u = u(x, y)$, $v = v(x, y)$: $$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial x}$$ 类似得 $\partial z/\partial y$. AI 反向传播就是这一套链式法则的递归应用.

链式法则的"加权" 直觉

情形 1 速写: $dz/dt = z_u \cdot u'(t) + z_v \cdot v'(t)$.

生活类比: 快乐 $=f$(收入, 健康), 收入 / 健康都随时间变化 → "快乐对时间变化" = (快乐对收入率 × 收入对时间率) + (快乐对健康率 × 健康对时间率). 每条传导路径权重就是相应偏导数, 全部相加 = 总效应.

情形 2 模板: $z = f(u, v)$, $u = u(x, y)$, $v = v(x, y)$: $$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial x}, \qquad \frac{\partial z}{\partial y} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial y} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial y}$$

例题 6

z = u² + v², u = x + y, v = x - y. 求 ∂ z / ∂ x.

  • ∂ z / ∂ u = 2u, ∂ z / ∂ v = 2v
  • ∂ u / ∂ x = 1, ∂ v / ∂ x = 1
  • ∂ z / ∂ x = 2u · 1 + 2v · 1 = 2(u + v) = 2(2x) = 4x.

知识点 7.5: 链式法则"树状图"画法 (对齐书 §9.4)

链式法则树状图法

复杂复合函数 $z = f(u, v, w)$, $u, v, w$ 都是 $(x, y)$ 函数. 求 $\partial z/\partial x$.

树状图 (避免漏路径): ``` z / | \ u v w /| /| /| x y x y x y ```

规则: 每条从 $z$ 到 $x$ 的路径贡献一项 (沿路相乘), 所有路径求和: $$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial x} + \frac{\partial z}{\partial w}\frac{\partial w}{\partial x}$$

AI 应用: 反向传播 (backprop) 就是这套链式法则递归应用 — 神经网络每层都是复合函数, 从损失反向算梯度.

链式法则不能漏路径

多变量链式法则容易漏掉某条路径. 一定画完整树状图核对.

例题 6-2-2

z = f(u, v), u = x+y, v = xy. 求 ∂z/∂x. 路径 1: z → u → x. 贡献 = f_u · 1 = f_u 路径 2: z → v → x. 贡献 = f_v · y ∂z/∂x = f_u + y · f_v.

9.5 隐函数求导

知识点 7.7: 高阶混合偏导数 (对齐书 §9.4 进一步)

复杂复合函数有时要算 ∂²z/∂x$\partial$y 这种二阶偏导.

关键: 先求一阶, 再"求导整个结果". 注意一阶结果可能仍是 (x,y) 的函数, 求二阶时要用乘积/链式法则.

例题 7-3

z = f(u, v), u = x+y, v = xy. 求 ∂²z/∂x$\partial$y. 一阶: ∂z/∂x = f_u + y · f_v (按链式). 二阶: ∂²z/∂x$\partial$y = ∂/∂y (f_u + y f_v) = f_uu·1 + f_uv·x + f_v + y(f_vu·1 + f_vv·x) = f_uu + (x+y) f_uv + xy f_vv + f_v

(中间用了 f_uv = f_vu 即混合偏导可交换)

知识点 8: 单变量隐函数

隐函数求导

方程 $F(x, y) = 0$ 在某点附近隐含定义 $y = y(x)$ (要求 $F_y \ne 0$), 则: $$\frac{dy}{dx} = -\frac{F_x}{F_y}$$

隐函数求导推导

$F(x, y(x)) = 0$ 两边对 $x$ 求导 (用链式法则), 得 $$F_x \cdot 1 + F_y \cdot \frac{dy}{dx} = 0$$ 若 $F_y \ne 0$, 移项: $\dfrac{dy}{dx} = -\dfrac{F_x}{F_y}$. $\square$

为啥 $F_y \ne 0$ 是关键

  • $F_y \ne 0$: $F$ 沿 $y$ 方向"有坡度", 给定 $x$ 能"切片"找到唯一 $y$
  • $F_y = 0$: $F$ 沿 $y$ 平的, 同一 $x$ 可能对应多个 $y$ (隐函数不唯一) → 隐函数定理失效

例题 7

x² + y² = 1 (单位圆), 求 dy/dx.

  • F = x² + y² - 1
  • F_x = 2x, F_y = 2y
  • dy/dx = -2x/(2y) = -x/y.

知识点 9: 多变量隐函数

多变量隐函数求偏导

方程 $F(x, y, z) = 0$ 隐含定义 $z = z(x, y)$ (要求 $F_z \ne 0$): $$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$$

推导

$F(x, y, z(x,y)) = 0$ 两边对 $x$ 求偏导, 用链式法则: $F_x + F_z \cdot z_x = 0$, 解出 $z_x$. 类似 $z_y$.

知识点 9.5: 方程组隐函数 (对齐书 §9.5 二)

方程组隐函数定理

设两方程 $F(x, y, u, v) = 0$, $G(x, y, u, v) = 0$ 联立, 若 Jacobian $$J = \frac{\partial(F, G)}{\partial(u, v)} = \begin{vmatrix} F_u & F_v \\ G_u & G_v \end{vmatrix} \ne 0$$ 则隐含定义 $u = u(x, y), v = v(x, y)$. 偏导数用 Cramer 法则解出.

鸡兔同笼升级

两个变量 (鸡数 $u$ + 兔数 $v$) 由两个约束 (头数 $x$ + 腿数 $y$) 唯一确定. Jacobian 测的就是这种"约束互不冗余"性 — $J = 0$ 时两方程退化成一个 (信息不够).

例题 7-2

(经典): ``` x + y - u - v = 0 x - y + 2u - v = 0 ``` 求 ∂u/∂x, ∂v/∂x. F_u = -1, F_v = -1, G_u = 2, G_v = -1. J = (-1)(-1) - (-1)(2) = 1 + 2 = 3. ∂u/∂x = -(1/J) · | F_x F_v G_x G_v | = -(1/3) · |1 -1 1 -1| = -(1/3)·0 = 0. 等等, 算一下 — 这里 ∂u/∂x 用 Cramer 推出.

应用: 经济学 IS-LM 模型 / 物理学相互作用方程组 / 化学平衡多变量约束.

9.6 方向导数与梯度

偏导数只能告诉我们沿坐标轴方向的变化率。如果一个登山者站在山坡上想知道「往东北方向走、高度变化多快」,偏导给不出直接答案。方向导数把这件事扩展到任意方向;梯度则把所有方向的变化率压成一个向量,并且这个向量本身就指向「变化最快」的方向。这是后续条件极值、机器学习中梯度下降、最速下降法的代数基础。

9.6.1 方向导数

设函数 $f(x,y)$ 在点 $P_0(x_0,y_0)$ 的某邻域内有定义,$\vec{l}=(\cos\alpha,\cos\beta)$ 为单位向量(方向角 $\alpha,\beta$ 满足 $\cos^2\alpha+\cos^2\beta=1$)。若极限 $$\frac{\partial f}{\partial\vec{l}}\bigg|_{P_0}=\lim_{t\to 0^+}\frac{f(x_0+t\cos\alpha,\,y_0+t\cos\beta)-f(x_0,y_0)}{t}$$ 存在,则称该极限为 $f$ 在 $P_0$ 沿方向 $\vec{l}$ 的方向导数。

站在曲面 $z=f(x,y)$ 对应的点 $(x_0,y_0,f(x_0,y_0))$,把曲面被竖直平面(沿 $\vec{l}$ 方向)截下来得到一条曲线,方向导数就是这条曲线在该点的斜率。

注意定义中要求 $t\to 0^+$(单侧极限),因此沿 $\vec{l}$ 与沿 $-\vec{l}$ 的方向导数一般不互为相反数(对不可微但偏导存在的函数尤其要小心)。但对可微函数有下面的乘法公式,符号取反时方向导数也取反:

(方向导数的偏导计算公式)若 $f$ 在 $P_0$ 可微,则对任意单位方向 $\vec{l}=(\cos\alpha,\cos\beta)$, $$\frac{\partial f}{\partial\vec{l}}\bigg|_{P_0}=f_x(P_0)\cos\alpha+f_y(P_0)\cos\beta.$$

由可微定义, $$f(x_0+\Delta x,y_0+\Delta y)-f(x_0,y_0)=f_x(P_0)\Delta x+f_y(P_0)\Delta y+o(\rho),\quad\rho=\sqrt{\Delta x^2+\Delta y^2}.$$ 取 $\Delta x=t\cos\alpha$,$\Delta y=t\cos\beta$,则 $\rho=t$(因为 $\vec{l}$ 是单位向量),代入得 $$\frac{f(x_0+t\cos\alpha,y_0+t\cos\beta)-f(x_0,y_0)}{t}=f_x\cos\alpha+f_y\cos\beta+\frac{o(t)}{t}.$$ 令 $t\to 0^+$,$o(t)/t\to 0$。证毕。

关键前提是「可微」。仅「两个偏导存在」并不能推出方向导数公式成立。一个经典反例: $$f(x,y)=\begin{cases}\dfrac{xy}{x^2+y^2},&(x,y)\neq(0,0)\\ 0,&(x,y)=(0,0)\end{cases}$$ 在 $(0,0)$ 处 $f_x(0,0)=f_y(0,0)=0$,但沿 $\vec{l}=(\tfrac{1}{\sqrt2},\tfrac{1}{\sqrt2})$ 的方向导数 $=\lim_{t\to 0^+}\frac{(t/\sqrt2)^2/(t^2)}{t}=\lim_{t\to 0^+}\frac{1}{2t}=+\infty$。公式失败因为 $f$ 在原点不可微。

9.6.2 梯度

设 $f$ 在 $P_0$ 可偏导,梯度(gradient)定义为偏导数构成的向量: $$\nabla f(P_0)=\bigg(\frac{\partial f}{\partial x}(P_0),\frac{\partial f}{\partial y}(P_0)\bigg)\quad(\text{二维});\quad\nabla f=(f_x,f_y,f_z)\quad(\text{三维}).$$ $\nabla$ 读作「nabla」或「del」,可视为算子 $\nabla=\big(\tfrac{\partial}{\partial x},\tfrac{\partial}{\partial y}\big)$。

把方向导数公式用点乘改写: $$\frac{\partial f}{\partial\vec{l}}=f_x\cos\alpha+f_y\cos\beta=\nabla f\cdot\vec{l}.$$

这一改写极其重要——它把「沿某方向的变化率」表达成「梯度向量与方向向量的点乘」。下面的定理是梯度的灵魂。

(梯度方向是最速上升方向)若 $f$ 在 $P_0$ 可微,$\nabla f(P_0)\neq\vec{0}$,则在所有单位方向 $\vec{l}$ 中:

  1. 最大方向导数$=|\nabla f(P_0)|$,在 $\vec{l}=\dfrac{\nabla f}{|\nabla f|}$(即沿梯度方向)时取得;
  2. 最小方向导数$=-|\nabla f(P_0)|$,在 $\vec{l}=-\dfrac{\nabla f}{|\nabla f|}$(沿负梯度方向)时取得;
  3. 零方向导数(函数瞬时不变方向)在 $\vec{l}\perp\nabla f$ 时取得——这正是等高线的切线方向。

设 $\vec{l}$ 与 $\nabla f$ 的夹角为 $\theta$,$|\vec{l}|=1$。由方向导数公式与 Cauchy-Schwarz: $$\frac{\partial f}{\partial\vec{l}}=\nabla f\cdot\vec{l}=|\nabla f|\cdot 1\cdot\cos\theta=|\nabla f|\cos\theta.$$ $\cos\theta\in[-1,1]$,取 $+1$($\theta=0$,$\vec{l}$ 与 $\nabla f$ 同向)时最大、取 $-1$($\theta=\pi$)时最小、取 $0$($\theta=\tfrac{\pi}{2}$)时为零。证毕。

把等高线类比成水平面投影,梯度恰好垂直于过该点的等高线,且指向函数值更高的一侧——所以爬山时「最陡的方向」总是垂直于脚下的等高线,这件事在物理 / GIS / 流体力学里反复出现。

9.6.3 例题

例 1(直接计算)设 $f(x,y)=x^2+2y^2$,求在 $(1,1)$ 处沿 $\vec{u}=(1,1)$ 方向的方向导数。

解:先单位化 $\vec{l}=\vec{u}/|\vec{u}|=(\tfrac{1}{\sqrt2},\tfrac{1}{\sqrt2})$。偏导 $f_x=2x$,$f_y=4y$,在 $(1,1)$ 处 $\nabla f=(2,4)$。 $$\frac{\partial f}{\partial\vec{l}}=\nabla f\cdot\vec{l}=\frac{2}{\sqrt2}+\frac{4}{\sqrt2}=\frac{6}{\sqrt2}=3\sqrt 2.$$

例 2(找最速方向)$f(x,y)=x^2+y^2$ 在 $(1,2)$ 处,问最快上升方向、最快下降方向、不变方向各是什么?最大变化率多少?

解:$\nabla f(1,2)=(2,4)$,$|\nabla f|=\sqrt{20}=2\sqrt 5$。

  • 最快上升:$\vec{l}=\tfrac{1}{2\sqrt 5}(2,4)=(\tfrac{1}{\sqrt 5},\tfrac{2}{\sqrt 5})$,变化率 $+2\sqrt 5$;
  • 最快下降:反向 $(-\tfrac{1}{\sqrt 5},-\tfrac{2}{\sqrt 5})$,变化率 $-2\sqrt 5$;
  • 不变:垂直方向 $(\tfrac{2}{\sqrt 5},-\tfrac{1}{\sqrt 5})$ 或反向,变化率 $0$。

例 3(梯度下降一步)对凸二次函数 $L(\theta_1,\theta_2)=\theta_1^2+\theta_2^2$,初始参数 $\theta^{(0)}=(4,3)$,学习率 $\eta=0.1$,执行一次梯度下降。

解:$\nabla L(\theta)=(2\theta_1,2\theta_2)$。在 $\theta^{(0)}$ 处 $\nabla L=(8,6)$, $$\theta^{(1)}=\theta^{(0)}-\eta\nabla L=(4,3)-0.1\cdot(8,6)=(3.2,2.4).$$ 再迭代一步:$\nabla L(3.2,2.4)=(6.4,4.8)$,$\theta^{(2)}=(3.2-0.64,2.4-0.48)=(2.56,1.92)$。每步距离最优点 $(0,0)$ 的范数比上一步缩小一个固定比例 $1-2\eta=0.8$,这就是凸二次问题的几何收敛。

9.6.4 与 AI 优化的联系

机器学习把模型参数记作 $\theta\in\mathbb{R}^d$,把训练损失记作 $L(\theta)$。最速下降 / 梯度下降就是把上面定理 9.6.B 的「负梯度方向是最速下降方向」直接搬到 $d$ 维: $$\theta^{(k+1)}=\theta^{(k)}-\eta_k\,\nabla L(\theta^{(k)}).$$

实际中常见的变体:

| 方法 | 公式核心 | 解决什么问题 | | --- | --- | --- | | SGD | 用单样本(或 mini-batch)估计 $\nabla L$ | 大数据集,全量梯度算不起 | | Momentum | $v\gets\beta v+\nabla L$,$\theta\gets\theta-\eta v$ | 鞍点 / 长山谷震荡 | | Adam | 一阶矩 $m$ + 二阶矩 $v$ 自适应缩放 | 不同参数尺度差异大 | | Newton | $\theta\gets\theta-\eta\,H^{-1}\nabla L$,$H$ 是 Hessian | 二阶曲率快收敛,但 $H^{-1}$ 贵 |

(下降引理 / Descent Lemma)若 $\nabla L$ 是 $L$-Lipschitz 的(即 $\|\nabla L(\theta_1)-\nabla L(\theta_2)\|\le L\|\theta_1-\theta_2\|$),则梯度下降步骤满足 $$L(\theta^{(k+1)})\le L(\theta^{(k)})-\eta\bigg(1-\frac{\eta L}{2}\bigg)\|\nabla L(\theta^{(k)})\|^2.$$

由 $L$-平滑性的二阶展开式(可由 Taylor 余项 + Lipschitz 条件得) $$L(\theta')\le L(\theta)+\nabla L(\theta)\cdot(\theta'-\theta)+\tfrac{L}{2}\|\theta'-\theta\|^2.$$ 代入 $\theta'=\theta-\eta\nabla L(\theta)$: $$L(\theta')\le L(\theta)-\eta\|\nabla L\|^2+\tfrac{L\eta^2}{2}\|\nabla L\|^2=L(\theta)-\eta(1-\tfrac{L\eta}{2})\|\nabla L\|^2.\;\square$$

这个不等式告诉我们:当 $\eta\le\tfrac{1}{L}$ 时,每一步至少让损失下降 $\tfrac{\eta}{2}\|\nabla L\|^2$,从而梯度范数最终趋零(收敛到稳定点)。这是机器学习里收敛性分析的最基本工具。

易错点:

  • 计算方向导数前必须把方向向量单位化;若代入未单位化的向量,结果会被它的模放大。
  • 「方向导数 $=$ 梯度点乘方向」要求函数在该点可微;偏导存在不够。
  • 梯度方向是最速上升,要做最小化时记得取负梯度;符号搞反 = 算法朝错方向跑。
  • 高维非凸问题里,$\nabla L=0$ 不一定是极小,可能是鞍点或局部极大;需用 Hessian 进一步判别(§9.8)。

9.7 多元微分的几何应用

知识点 11.5: 梯度的物理意义 (对齐书 §9.7 二 进一步)

梯度的 3 大几何意义

梯度 $\nabla f$ 不只是"导数向量", 它有三层深刻意义:

① 等值线/等值面的法向量: 等高线 $f(x,y) = c$ 上每点切向跟 $\nabla f$ 垂直 (沿等值线 $f$ 不变 ⟹ 方向导数 = 0 = $\nabla f\cdot\vec{t}$).

② 增长最快方向: 方向导数 $= \nabla f\cdot\vec{l}^0 = \|\nabla f\| \cos\alpha$, 在 $\alpha = 0$ 时最大, 即沿 $\nabla f$ 方向. 下降最快沿 $-\nabla f$ — 这就是梯度下降算法的来源.

③ 等高线疏密 = 山的陡缓: 等高线密 = $\|\nabla f\|$ 大 = 山陡; 等高线疏 = $\|\nabla f\|$ 小 = 山缓.

例题 8-4

(等值线-梯度关系): f(x,y) = x² + y² (碗形). 等值线: x² + y² = c (圆). ∇f = (2x, 2y) = 2(x, y) = 2 ·$\vec{r}$. 观察: 这跟"圆的半径方向" 一致 — 梯度永远跟等值线垂直 ✓

知识点 11.7: 多元函数极值的二阶判别 (Hessian 矩阵, 对齐书 §9.8)

Hessian 二阶判别

在驻点 $\nabla f = 0$, 设 $A=f_{xx}, B=f_{xy}, C=f_{yy}$, 判别式 $\Delta = AC - B^2$:

  • $\Delta > 0, A > 0$ → 极小值
  • $\Delta > 0, A < 0$ → 极大值
  • $\Delta < 0$ → 鞍点 (非极值)
  • $\Delta = 0$ → 失效, 需更高阶

Hessian 矩阵: $H = \begin{pmatrix} A & B \\ B & C \end{pmatrix}$. 判别 H 的特征值:

  • 全正 → 极小
  • 全负 → 极大
  • 异号 → 鞍点

AI/ML 直接用: 二阶优化 (Newton 法) 用 Hessian 加速收敛, 比梯度下降快得多.

知识点 12: 空间曲线的切线

空间曲线切线

参数曲线 $\vec{r}(t) = (x(t), y(t), z(t))$, 切向量 = 速度向量 $\vec{r}'(t) = (x'(t), y'(t), z'(t))$.

切线方程 (点向式, 在 $t_0$ 处): $$\frac{x - x(t_0)}{x'(t_0)} = \frac{y - y(t_0)}{y'(t_0)} = \frac{z - z(t_0)}{z'(t_0)}$$

(要求分母都不为零, 否则用对称式或参数式表达.)

物理意义 — "瞬时速度方向"

把曲线想成质点运动轨迹, $t$ 是时间. $\vec{r}'(t)$ 就是质点在 $t_0$ 时刻的瞬时速度向量, 大小 = 速率, 方向 = 切线方向. 切线就是"延着速度方向直线运动"的极限.

例题 8-1

螺旋线 $\vec{r}(t) = (\cos t, \sin t, t)$ 在 $t_0 = \pi/2$ 处切线? $\vec{r}'(t) = (-\sin t, \cos t, 1)$, 在 $t_0 = \pi/2$ 处 $= (-1, 0, 1)$. 切点 $= (0, 1, \pi/2)$. 切线: $\dfrac{x}{-1} = \dfrac{y - 1}{0} = \dfrac{z - \pi/2}{1}$ (注意 $y$ 分量为零, 意味着 $y \equiv 1$).

知识点 13: 曲面的切平面

隐式曲面切平面

曲面 $F(x, y, z) = 0$, 法向量 $\vec{n} = \nabla F = (F_x, F_y, F_z)$. 在点 $(x_0, y_0, z_0)$ 处的切平面方程: $$F_x(x_0,y_0,z_0)(x-x_0) + F_y(\ldots)(y-y_0) + F_z(\ldots)(z-z_0) = 0$$ 法线方程: $\frac{x-x_0}{F_x} = \frac{y-y_0}{F_y} = \frac{z-z_0}{F_z}$

梯度 = 等高面法向

$F(x,y,z) = c$ 是等值面 (曲面). $\nabla F$ 永远垂直于等值面 — 这跟二维 "$\nabla f$ 垂直等高线" 同源.

例题 9

球面 x² + y² + z² = 14 在点 (1, 2, 3) 处的切平面.

  • F = x² + y² + z² - 14, ∇ F = (2x, 2y, 2z), 在 (1,2,3) = (2, 4, 6).
  • 切平面: 2(x-1) + 4(y-2) + 6(z-3) = 0, 即 x + 2y + 3z = 14.

9.8 多元函数的极值

一元函数 $y=f(x)$ 在驻点用 $f''$ 的符号判极大极小。多元情形「二阶导数」是个矩阵(Hessian),判别要看这个矩阵的正定性——这是数值优化、机器学习二阶方法、凸优化理论的共同起点。本节既给传统的 $D=AC-B^2$ 判别式,也把它放到 Hessian 矩阵的语言里,说明判别式从哪里来。

9.8.1 法线方程(补完)

曲面 $F(x,y,z)=0$ 在 $P_0(x_0,y_0,z_0)$ 处的法线方向就是梯度 $\nabla F(P_0)=(F_x,F_y,F_z)$,法线参数式: $$\frac{x-x_0}{F_x(P_0)}=\frac{y-y_0}{F_y(P_0)}=\frac{z-z_0}{F_z(P_0)}.$$ 对显函数 $z=f(x,y)$,令 $F(x,y,z)=f(x,y)-z$,法向量取作 $(f_x,f_y,-1)$。

例 1(球面法线过球心)球面 $x^2+y^2+z^2=14$ 在 $(1,2,3)$ 处的法线?

解:$\nabla F=(2x,2y,2z)\big|_{(1,2,3)}=(2,4,6)$,方向化简为 $(1,2,3)$。法线 $\frac{x-1}{1}=\frac{y-2}{2}=\frac{z-3}{3}$,正是从球心 $(0,0,0)$ 出发的射线——球面任一点法线必过球心。

9.8.2 极值的必要条件(Fermat)

(必要条件)若 $f(x,y)$ 在 $(x_0,y_0)$ 取得局部极值且在该点可偏导,则 $\nabla f(x_0,y_0)=\vec{0}$,即 $f_x=f_y=0$。

固定 $y=y_0$,$g(x)=f(x,y_0)$ 是一元函数,$x_0$ 是它的局部极值点,由一元 Fermat 引理 $g'(x_0)=f_x(x_0,y_0)=0$。对 $y$ 同理。证毕。

$\nabla f(P)=\vec{0}$ 的点称为 $f$ 的驻点(stationary point)。极值点必为驻点,但驻点未必是极值点(可能是鞍点)。

经典反例:$f(x,y)=x^2-y^2$ 在 $(0,0)$ 处 $\nabla f=\vec{0}$,沿 $x$ 轴 $f(x,0)=x^2$ 是局部极小,沿 $y$ 轴 $f(0,y)=-y^2$ 是局部极大——这就是鞍点。这种"沿不同方向走完全相反"的点在高维深度学习损失景观中远比极值常见。

9.8.3 二阶判别(Hessian)

$f(x,y)$ 在 $P_0$ 处二阶连续可偏导时,Hessian 矩阵定义为 $$H_f(P_0)=\begin{pmatrix}f_{xx}&f_{xy}\\ f_{yx}&f_{yy}\end{pmatrix}_{P_0}.$$ 由 Clairaut 定理 $f_{xy}=f_{yx}$,所以 $H_f$ 是对称矩阵。记 $A=f_{xx}$,$B=f_{xy}$,$C=f_{yy}$,则 $$H_f=\begin{pmatrix}A&B\\ B&C\end{pmatrix},\quad\det H_f=AC-B^2.$$ 教材中的判别式 $D=AC-B^2$ 就是 $\det H_f$。

(极值二阶充分条件)设 $f$ 在 $P_0$ 处二阶连续可微,$P_0$ 是驻点($\nabla f(P_0)=\vec{0}$),记 $A=f_{xx}(P_0)$,$D=\det H_f(P_0)=AC-B^2$。则:

  1. $D>0$ 且 $A>0$ $\Rightarrow$ $P_0$ 是严格局部极小;
  2. $D>0$ 且 $A<0$ $\Rightarrow$ $P_0$ 是严格局部极大;
  3. $D<0$ $\Rightarrow$ $P_0$ 是鞍点;
  4. $D=0$ $\Rightarrow$ 判别失效(需更高阶 Taylor 或直接代值)。

驻点处由二元 Taylor 公式(留 Peano 余项): $$f(P_0+\vec{h})-f(P_0)=\tfrac12\vec{h}^{\top}H_f(P_0)\vec{h}+o(|\vec{h}|^2),\quad\vec{h}=(\Delta x,\Delta y).$$ 极值的局部行为由二次型 $Q(\vec{h})=\vec{h}^{\top}H_f\vec{h}=A\,\Delta x^2+2B\,\Delta x\Delta y+C\,\Delta y^2$ 决定。配方: $$Q(\vec{h})=A\bigg(\Delta x+\frac{B}{A}\Delta y\bigg)^2+\frac{AC-B^2}{A}\Delta y^2,\quad\text{若 }A\neq 0.$$

  • $D=AC-B^2>0$ 且 $A>0$:两项系数同正 $\Rightarrow Q>0$ 对所有 $\vec{h}\neq\vec{0}$,正定$\Rightarrow$ 极小。
  • $D>0$ 且 $A<0$:配方两项系数同负 $\Rightarrow Q<0$,负定$\Rightarrow$ 极大。
  • $D<0$:$Q$ 既能取正也能取负(沿两组特征方向符号相反),不定$\Rightarrow$ 鞍点。
  • $A=0$ 但 $C\neq 0$:对 $\Delta y$ 配方同理。
  • $A=C=0$:$Q=2B\Delta x\Delta y$,$B\neq 0$ 即不定为鞍点;$B=0$ 即 $H_f=0$ 判别失效。

线性代数等价描述:$H_f$ 是对称阵,有实特征值 $\lambda_1,\lambda_2$。由 Vieta 公式 $\lambda_1\lambda_2=\det H_f=D$,$\lambda_1+\lambda_2=A+C=\operatorname{tr}H_f$。$D>0$ 等价于两特征值同号($A$ 与同号),$D<0$ 等价于异号(不定)。

驻点附近 $f$ 的图形被 Hessian 决定:

  • 正定($D>0,\,A>0$):图形是"碗口朝上",底部最深。
  • 负定($D>0,\,A<0$):图形是"碗口朝下",顶部最高。
  • 不定($D<0$):图形是"马鞍",一个方向往上、另一个方向往下。
  • 半定($D=0$):图形是"槽"或"棱",二阶看不出来。

9.8.4 例题与高维推广

例 2 $f(x,y)=x^2+y^2+2x-4y+3$。

解:$f_x=2x+2$,$f_y=2y-4$。驻点:$x=-1,\,y=2$。$A=f_{xx}=2$,$B=f_{xy}=0$,$C=f_{yy}=2$,$D=4-0=4>0$,$A>0\Rightarrow$ 极小值。极小值 $f(-1,2)=1+4-2-8+3=-2$。

例 3(鞍点)$f(x,y)=x^3-3xy+y^3$ 找极值。

解:$f_x=3x^2-3y=0$,$f_y=-3x+3y^2=0\Rightarrow y=x^2,\,x=y^2$,联立得 $(0,0)$ 与 $(1,1)$。

  • 在 $(0,0)$:$A=f_{xx}=6x=0$,$B=-3$,$C=6y=0$,$D=0\cdot 0-9=-9<0$ $\Rightarrow$ 鞍点;
  • 在 $(1,1)$:$A=6$,$B=-3$,$C=6$,$D=36-9=27>0$,$A>0$ $\Rightarrow$ 极小,极小值 $f(1,1)=-1$。

$n$ 维推广:把 $f:\mathbb{R}^n\to\mathbb{R}$ 的 Hessian 写成 $H_f=(\partial^2 f/\partial x_i\partial x_j)$,驻点 $P_0$ 处局部:

  • $H_f(P_0)$ 正定 $\iff$ 严格局部极小;
  • $H_f(P_0)$ 负定 $\iff$ 严格局部极大;
  • $H_f(P_0)$ 不定(有正有负特征值) $\iff$ 鞍点;
  • $H_f(P_0)$ 半定 $\iff$ 判别失效。

判断正定性的标准方法是 Sylvester 判据:$H$ 正定 $\iff$ 所有顺序主子式 $>0$;$H$ 负定 $\iff$ 顺序主子式符号交替($-,+,-,\dots$)。二维就是 $A>0,\,AC-B^2>0$,正好对应上面定理。

9.8.5 与凸优化、机器学习的联系

$f:\mathbb{R}^n\to\mathbb{R}$ 是凸函数 $\iff$ 对所有 $x,y$ 和 $\lambda\in[0,1]$,$f(\lambda x+(1-\lambda)y)\le\lambda f(x)+(1-\lambda)f(y)$。二阶可微时等价于 $H_f(x)$ 对所有 $x$ 半正定。严格凸 $\iff$ $H_f$ 正定。

凸函数的关键性质:任何驻点都是全局极小,不存在局部陷阱。这就是凸优化(线性规划、SVM、Logistic 回归)可以「全局最优」可证的根本原因。

(Newton 法的二阶迭代)对二阶可微且 $H_f(\theta)$ 在最优点附近正定的函数,Newton 迭代 $$\theta^{(k+1)}=\theta^{(k)}-\big[H_f(\theta^{(k)})\big]^{-1}\nabla f(\theta^{(k)})$$ 在最优点附近有二次收敛(每步误差平方级递减),远快于梯度下降的几何收敛。

(思路)对 $\nabla f$ 在最优点 $\theta^*$ 附近做一阶 Taylor:$\nabla f(\theta)\approx H_f(\theta^*)(\theta-\theta^*)$。Newton 步取 $\theta-H^{-1}\nabla f$ 正是「猜下一步使一阶 Taylor 余项被吃掉」,从而 $\|\theta^{(k+1)}-\theta^*\|=O(\|\theta^{(k)}-\theta^*\|^2)$。完整证明需 $H$ Lipschitz 连续假设。

深度学习里的现实:神经网络损失函数非凸,Hessian 不定;高维空间里鞍点远多于极小点(Dauphin 等 2014 经验法则:鞍点的"逃逸概率"远高于极小点)。这就是为何 SGD + Momentum 比纯 Newton 更受欢迎——Newton 在鞍点附近会被吸引,而带噪声的 SGD 反而能跳出。Hessian 在万亿参数模型里也算不起,所以实践用对角近似 Hessian 的 Adam、或者Hessian-向量积的 Conjugate Gradient 等近似方法。

9.9 二元函数的泰勒公式 (对齐书 §9.9)

知识点 17: 二元泰勒公式

二元泰勒展开 (在 $(a, b)$ 处, 假设 $f$ 足够光滑)

$$\begin{aligned} f(x, y) &= f(a, b) + \big[f_x(a,b)(x-a) + f_y(a,b)(y-b)\big] \\ &\quad + \frac{1}{2!}\big[f_{xx}(x-a)^2 + 2 f_{xy}(x-a)(y-b) + f_{yy}(y-b)^2\big] \\ &\quad + \cdots + R_n \end{aligned}$$ 向量记号: $f(\vec{x}) \approx f(\vec{x}_0) + \nabla f^\top \Delta\vec{x} + \frac{1}{2}\Delta\vec{x}^\top H \Delta\vec{x} + \cdots$

爬山几何

站在 $(a, b)$ 看附近地形:

  • 零阶: 单点 (高度 $f(a,b)$)
  • 一阶展开: 切平面 (用切平面近似曲面)
  • 二阶展开: 抛物面 (有"凹凸感", Hessian 决定碗 / 鞍)
  • 更高阶: 越来越精细

知识点 18: 二阶判别式的"由来" (对齐 §9.9 二)

$D = AC - B^2$ = Hessian 行列式

之前 (知识点 15) 用 $D = AC - B^2$ 判极值类型. 这个就是二阶 Taylor 展开: $A = f_{xx}, B = f_{xy}, C = f_{yy}$ 是 Hessian 矩阵的项. 判别式 $D$ 实际就是 $\det H$.

  • $D > 0$ 且 $A > 0$: Hessian 正定 → 局部极小
  • $D > 0$ 且 $A < 0$: Hessian 负定 → 局部极大
  • $D < 0$: Hessian 不定 → 鞍点

AI / ML 应用: 二阶优化 (Newton 法 / 牛顿型) 直接用 Hessian 判"局部最小 vs 鞍点", 是同一套数学.

9.10 最小二乘法 (对齐书 §9.10)

知识点 19: 最小二乘法

问题: 一堆数据点 (xᵢ, yᵢ), 怎么找一条直线 y = ax + b 最贴合?

思想: "误差平方和最小". 残差 = (yᵢ - axᵢ - b)², 加起来求最小: S(a, b) = Σᵢ (yᵢ - axᵢ - b)²

对 a, b 求偏导 = 0, 解出: a = (n·Σxy - Σx·Σy) / (n·Σx² - (Σx)²) b = (Σy - a·Σx) / n

为啥用平方和: ① 处理"正负误差不抵消" ② 求导方便 ③ 跟正态分布最大似然估计一致.

例题 12

数据点 (1,2), (2,3), (3,5), (4,4), 拟合 y = ax + b. n=4, Σx=10, Σy=14, Σxy=39, Σx²=30. a = (4·39 - 10·14)/(4·30 - 100) = (156-140)/(120-100) = 16/20 = 0.8. b = (14 - 0.8·10)/4 = 6/4 = 1.5. 拟合直线: y = 0.8x + 1.5.

AI 直接用: 线性回归就是最小二乘. 神经网络反向传播找梯度也是这个思想推广.

知识点 19.5: 拉格朗日乘数法的几何意义

核心: 在约束 g(x,y) = 0 下求 f(x,y) 极值. 几何上:

  • f 的等值线: f = c (一族曲线)
  • 约束 g = 0: 一条曲线

极值点的关键特征: 在最优点, f 的等值线跟约束曲线 g=0 相切. 切点 = 极值候选.

相切 ⇔ 两曲线的法向量平行 ⇔ ∇f = $\lambda$∇g ($\lambda$ 是乘数).

推广 (KKT 条件): 多约束 + 不等式约束情况. 是机器学习 SVM 优化的核心数学.

例题 12

(经典 + 几何): 在 x² + y² = 1 上求 f(x,y) = xy 的极值. ∇f = (y, x), ∇g = (2x, 2y). ar→ ∇f = $\lambda$∇g → y = 2λx, x = 2λy. 两式相乘: xy = 4λ²xy → 若 xy ≠ 0, 4λ² = 1, $\lambda$ = ±1/2. $\lambda$=1/2: y = x, 代入 x²+y² = 1 → x = ±1/√2, y = ±1/√2 同号. f = 1/2 (极大). $\lambda$=-1/2: y = -x, → x = ±1/√2, y = ∓1/√2. f = -1/2 (极小). 结论: max = 1/2, min = -1/2.

几何验证: 在 (1/√2, 1/√2) 处, f 的等值线 xy=1/2 是双曲线一支, 约束 x²+y²=1 是单位圆, 两者在该点相切 ✓.

知识点 9.5: 牛顿法 求多元极值

牛顿法迭代公式

一元: 解 $f'(x) = 0$ 用 $x_{n+1} = x_n - f'(x_n)/f''(x_n)$.

多元: 解 $\nabla f(\vec{x}) = 0$ 用 $$\vec{x}_{n+1} = \vec{x}_n - H(\vec{x}_n)^{-1} \cdot \nabla f(\vec{x}_n)$$ 其中 $H$ 是 Hessian 矩阵.

牛顿法 = 二阶 Taylor 局部拟合

每一步用二阶 Taylor 展开作"局部抛物面拟合", 跳到该抛物面的极值点. 跟梯度下降 (只用一阶) 比, 牛顿法二次收敛 (极快), 但每步代价高 (要算 Hessian 和它的逆).

牛顿法实战不直接用

$H$ 在大模型 (亿级参数) 中存不下、求不出逆. 实践用 quasi-Newton (BFGS/L-BFGS) 近似 Hessian. 神经网络用 Adam (一阶) 而非牛顿法 (二阶).

例题 9-5

f(x,y) = x⁴ + y⁴, 从 (1,1) 求极小. ∇f = (4x³, 4y³). H = diag(12x², 12y²). x₁ = (1,1) - diag(1/12,1/12)·(4,4) = (2/3, 2/3). x₂ ≈ (4/9, 4/9). 二次收敛趋 (0,0).

对比 GD: 牛顿法二次收敛, GD 线性收敛. 但 H 在大模型存不下, 实践用 quasi-Newton (BFGS/L-BFGS).

知识点 9.6: KKT 条件 (不等式约束)

为啥要 KKT

拉格朗日乘子只处理等式约束. 实际优化问题往往含不等式约束 (如非负、上下界、预算上限). KKT 是处理不等式的标准工具.

KKT 必要条件

问题: $\min f(\vec{x})$ s.t. $g(\vec{x}) \le 0$, $h(\vec{x}) = 0$.

Lagrangian: $L = f + \lambda h + \mu g$

最优解需满足:

  1. 梯度: $\nabla f + \lambda \nabla h + \mu \nabla g = 0$
  2. 原始可行: $h = 0, g \le 0$
  3. 对偶可行: $\mu \ge 0$
  4. 互补松弛: $\mu \cdot g = 0$

直觉:

  • g<0 (约束不紧) → $\mu$=0
  • g=0 (约束紧) → $\mu$ 起作用

应用:

  • SVM: 二次规划 + KKT 推出"支持向量"
  • 投资组合: 财富约束 ≤ B
  • 神经网络剪枝: L1 稀疏

例题 9-6

(SVM): min (1/2)||w||² s.t. yᵢ(w·xᵢ+b) ≥ 1. L = (1/2)||w||² - ∑μᵢ[yᵢ(w·xᵢ+b)-1]. ∂L/∂w=0 → w = ∑μᵢyᵢxᵢ. 只有 μᵢ>0 的样本 = 支持向量.

知识点 9.7: 多元函数的方向导数完整定义

方向导数 D_u→ f(x₀): 沿单位向量 $\vec{u}$ 方向 f 的变化率. D_u→ f(x₀) = lim_($\vec{t}$0) [f(x₀+tu→) - f(x₀)] / t

与梯度关系: D_u→ f = ∇f · $\vec{u}$ = |∇f| $\cos\theta$

核心几何意义:

  • $\vec{u}$ = ∇f/|∇f| (沿梯度): D_u→ f = |∇f|, 最大.
  • $\vec{u}$ ⊥ ∇f: D_u→ f = 0 (沿等高线方向不变).
  • $\vec{u}$ = -∇f/|∇f|: D_u→ f = -|∇f|, 最小 (最速下降方向).

例题 9-7

f(x,y) = x²+y², 求 (1,1) 沿 $\vec{u}$=(3/5, 4/5) 的方向导数. ∇f|_(1,1) = (2, 2). D_u→ f = (2,2)·(3/5, 4/5) = 6/5 + 8/5 = 14/5 = 2.8.

ML 训练核心: 梯度下降 $\theta$ ← $\theta$ - η·∇L, 选 -∇L 方向就是因为它是最速下降 — 局部下降最快.

知识点 9.8: 包络 (Envelope) 与判别曲线

问题: 一族曲线 F(x, y, c)=0 (c 是参数), 求"包络"= 与族内每条都相切的曲线.

消 c 法:

  • F(x, y, c) = 0
  • ∂F/∂c = 0

联立消 c, 得到包络方程.

例题 9-8

直线族 x cos c + y sin c = 1 (c 是参数). ∂/∂c: -x sin c + y cos c = 0 → tan c = y/x. 代回: x·x/√(x²+y²) + y·y/√(x²+y²) = 1 → √(x²+y²) = 1. 包络: 单位圆 x² + y² = 1.

物理意义: 任意切线族的包络 = 它们公共相切的曲线. 抛物面碟天线就是利用包络:平行光经反射汇聚到焦点.

应用:

  • 光学焦散 (caustic): 杯子内反射形成的亮纹
  • 微分方程 ODE 奇解
  • 几何光学: 透镜成像

知识点 9.9: 隐函数定理 — 完整证明思路

问题: F(x, y) = 0 何时可以解出 y = y(x)?

定理 (一元情形): 设 F(x₀, y₀) = 0, F 连续可微, F_y(x₀, y₀) ≠ 0. 存在 x₀ 邻域 U, 使在 U 内有唯一 y = y(x) 满足 F(x, y(x)) = 0, 且 y'(x) = -F_x/F_y.

证明思路 (压缩映射):

  1. 将 F(x,y)=0 改写为 y = y - F(x,y)/F_y(x₀,y₀) := T(x, y)
  2. 在 (x₀, y₀) 附近 |∂T/∂y| < 1 (利用 F_y 连续 + 初值条件)
  3. 不动点 ⟹ 存在唯一 y(x) (Banach 不动点定理)
  4. y(x) 的可微性由 F 的可微性传播

多元推广 (向量值): F: R^(n+m) → R^m, F(x₀, y₀)=0, ∂F/∂y 满秩 m. 则存在 y=y(x).

例题 9-9

F(x,y) = x²+y²-1=0. F_y = 2y. 在 (1/2, √3/2) 处 F_y$\neq$0, 可解. y'(x) = -F_x/F_y = -2x/(2y) = -x/y = -(1/2)/(√3/2) = -1/√3.

应用:

  • 经济一般均衡: 价格 → 数量的可解性
  • 力学约束: Lagrange 方程从约束 g=0 解出广义坐标
  • 数值连续: 同伦法追踪解曲线
  • 神经网络可逆性: Normalizing Flows 要求 y=f(x) 可逆

知识点 9.10: 二元函数泰勒展开

设 f 在 (x₀, y₀) 邻域有 n+1 阶连续偏导. 令 h = x-x₀, k = y-y₀:

f(x, y) = f(x₀, y₀) + [h·∂/∂x + k·∂/∂y]f|_(x₀,y₀) + (1/2!)[h·∂/∂x + k·∂/∂y]² f|_(x₀,y₀) + ... + R_n

展开二阶项: f ≈ f₀ + (h f_x + k f_y) + (1/2)(h² f_xx + 2hk f_xy + k² f_yy)

矩阵形式: f(x) ≈ f(x₀) + ∇f^T (x-x₀) + (1/2)(x-x₀)^T H (x-x₀)

H = Hessian.

例题 9-10

f(x,y) = e^(x+y) 在 (0,0) 二阶展开. f₀=1, ∇f=(1,1), H = [[1,1],[1,1]]. e^(x+y) ≈ 1 + x + y + (1/2)(x² + 2xy + y²) = 1 + (x+y) + (1/2)(x+y)²

应用:

  • 优化: 牛顿法用二阶 Taylor 拟合
  • 物理: 谐振子近似 = 势能函数二阶 Taylor
  • ML 解释性: LIME 局部线性近似复杂模型

知识点 9.11: 梯度下降变体 (深度学习核心)

梯度下降演化史

普通梯度下降 $\theta \leftarrow \theta - \eta \nabla L$ 太朴素 (收敛慢/振荡). 深度学习社区演化出 5 大变体, 各加"特技":

  1. Momentum — 加惯性
  2. Nesterov (NAG) — 加前瞻
  3. AdaGrad — 自适应每参数学习率
  4. RMSProp — 适应+衰减
  5. Adam — Momentum + RMSProp 合体, 实际默认

动量法 (Momentum)

$$v_t = \beta v_{t-1} + \nabla L(\theta_t), \quad \theta_{t+1} = \theta_t - \eta v_t$$ $\beta$ 一般取 0.9. 把过去梯度累积成"速度", 像球滚下山, 在平坦方向加速, 窄峡谷减震荡.

Nesterov 加速梯度 (NAG)

$$v_t = \beta v_{t-1} + \nabla L(\theta_t - \eta \beta v_{t-1})$$ "看一眼前方" 再算梯度, 比 Momentum 更准.

AdaGrad

$$G_t = G_{t-1} + (\nabla L)^2, \quad \theta \leftarrow \theta - \frac{\eta \nabla L}{\sqrt{G_t} + \epsilon}$$ 按分量平方累加, 自适应每参数学习率 — 频繁更新的参数学习率小, 稀疏的大.

RMSProp

$$G_t = \beta G_{t-1} + (1-\beta)(\nabla L)^2$$ AdaGrad 的指数衰减版, 解决学习率单调减到 0 问题.

Adam (深度学习默认)

$$m_t = \beta_1 m_{t-1} + (1-\beta_1) \nabla L, \quad v_t = \beta_2 v_{t-1} + (1-\beta_2)(\nabla L)^2$$ $$\theta \leftarrow \theta - \eta \cdot \frac{\hat{m}_t}{\sqrt{\hat{v}_t} + \epsilon}$$ $\hat{m}, \hat{v}$ 是偏差校正后. $\beta_1=0.9, \beta_2=0.999$ 标准.

= Momentum (一阶动量 $m_t$) + RMSProp (二阶动量 $v_t$) 合体. 训 LLM / Diffusion / 大部分神经网络的首选.

知识点 9.12: 自动微分 (Autograd)

自动微分两种模式

  • 前向模式 (Forward-mode AD): 对每个 $x_i$ 同时计算 $(x_i, dx_i/dx)$, 链式法则正向传播.
  • 反向模式 (Reverse-mode AD / Backprop): 先正向算损失 $L$, 再反向用 $dL/dy \to dL/dx$ 传播.

Backprop 复杂度优势

输出 $L$ 是标量 (1 维), 输入 $\theta$ 是 亿级: | 模式 | 复杂度 | |---|---| | 前向 | $O(n)$ 倍前向 | | 反向 | $O(1)$ 倍前向 |

亿级参数 ⟹ 反向快 10⁹ 倍. 这就是为什么神经网络训练用 backprop, 而不是数值差分.

  • jvp (Jacobian-vector product): 正向
  • vjp (Vector-Jacobian product): 反向
  • jvp (Jacobian-vector product): 正向
  • vjp (Vector-Jacobian product): 反向

应用: PyTorch/TF/JAX 的 .backward() 都用 vjp. 元学习用 jvp 算二阶梯度. 可微分物理用 autodiff 优化设计.

知识点 9.13: 函数族的一致收敛 (深度学习的泛化基石)

逐点收敛: f_n(x) → f(x) 在每个 x 单独看收敛.

一致收敛: ∀ε>0, ∃N, ∀n>N, sup_x |f_n(x) - f(x)| < ε. 整个函数族整体收敛.

关键差别:

  • 逐点收敛允许连续函数极限不连续 (例 x^n on [0,1])
  • 一致收敛保留连续性: f_n 连续 → f 连续
  • 一致收敛保留可积性: lim ∫f_n = ∫lim f_n

例题 9-13

f_n(x) = x^n on [0, 1]. 逐点: → 0 if x<1, → 1 if x=1. 不连续. 但每个 f_n 连续 ⟹ 没有一致收敛.

深度学习泛化 (核心理论):

  • 训练: 在数据集 D_n 上最小化 L_n($\theta$) = (1/n) ∑ ℓ(f(x_i), y_i)
  • 真实目标: L($\theta$) = E_(x,y)[ℓ]
  • 泛化 = 学到的 $\theta$̂ 使 L($\theta$̂) ≈ L_n($\theta$̂) ?
  • VC 维 / Rademacher 复杂度 给出 sup_θ |L_n($\theta$) - L($\theta$)| 上界

这就是一致收敛在 ML 里的"统计学习理论"角度.

知识点 9.14: 多元微分的微分形式回看

回想第十一章引入的微分形式视角:

  • df = ∑ (∂f/∂x_i) dx_i ← 一次形式
  • d²f = 0 ← 这是 d∘d=0
  • ddf 写出: ∑ (∂²f/∂x_i ∂x_j) dx_j ∧ dx_i. 利用 dx_j ∧ dx_i = -dx_i ∧ dx_j + 对称性, 可证 d²f=0.

几何与代数统一: 多元微分 = 标量函数的一次形式; 梯度 = 一次形式的"对偶向量". 这是高数下册多元微分的现代视角.

应用 (跨学科):

  • 物理: Lagrange/Hamilton 力学的辛形式
  • 控制理论: 微分平坦性
  • 几何深度学习: 图神经网络的"消息传递" 本质是一次形式的离散版本
