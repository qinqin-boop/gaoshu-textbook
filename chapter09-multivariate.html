第九章 多元函数微分法及其应用

对照: 同济大学《高等数学》第七版下册 第九章
一元函数 y=f(x) 升级到多元 z = f(x, y) — 导数也跟着推广.

9.0 整章在干啥?

一元函数 (中学/上册) 你已经会:

  • 给定 y = x², 求导数 dy/dx
  • 用导数找极值 (最高点/最低点)
  • 求切线方程

现在世界更复杂: 山的高度不只跟"东西方向" x 有关, 还跟"南北" y 有关 → z = f(x, y) 是个曲面. 怎么算这种"多变量函数" 的导数+极值+切平面? — 这章答.

对你的意义: AI 神经网络训练靠梯度下降 — 而梯度 = 多元函数的偏导. 这章直接对应 ML 数学基础.

9.1 多元函数的基本概念

知识点 0.5: 为什么要从一元升级到多元?

一元函数 y = f(x): 描述"一个输入对一个输出". 工资 = f(工龄).

但现实复杂: 工资还跟学历/城市/行业有关 — 一个变量描述不了. 需要 z = f(工龄, 学历, 城市, ...) 这种多输入函数.

学多元函数 = 描述真实世界:

  • 山的高度 = f(经度, 纬度)
  • 气温 = f(经度, 纬度, 海拔, 时间)
  • 神经网络损失 = f(权重₁, 权重₂, ..., 权重ₙ) (n 可能千亿)

这章学的核心:

  • 怎么求多元函数的"导数" (偏导)
  • 怎么找多元函数的"最高/最低点" (极值)
  • 怎么沿任意方向计算变化率 (方向导数+梯度)

为啥这章关键: AI 训练 = 找一个多元函数 (损失) 的最低点. 整套技巧 (梯度下降/反向传播) 全靠这章基础.

知识点 1: 二元函数定义

一元: 一个输入一个输出. y = f(x) 比如 y = x².

二元: 两个输入一个输出. z = f(x, y) 比如 z = x² + y² (碗形曲面).

生活类比:

  • 一元: 你的工资 = f(工龄) — 只看工龄
  • 二元: 工资 = f(工龄, 学历) — 两个都看

定义域: 满足函数有定义的 (x, y) 集合. 比如 z = √(1 - x² - y²) 的定义域是单位圆盘内.

例题 1: z = (1)/(√(4 - x² - y²)) 的定义域? 分母不能为 0 + 根号下大于 0: 4 - x² - y² > 0, 即 x² + y² < 4 (半径 2 开圆盘).

---

知识点 2: 二元函数的极限

生活类比: 一元极限是"从 x 左右靠近 x₀, y 趋向某值". 二元呢? 从任何路径 靠近 (x₀, y₀) 都得趋向同一个值, 极限才存在.

定义 (直观版): 不管 (x, y) 怎么靠近 (x₀, y₀) (直线 / 曲线 / 螺旋), f(x,y) 都趋于 L, 写: lim_((x,y) → (x₀, y₀)) f(x, y) = L

反例 (路径不同极限不同 → 极限不存在): f(x,y) = (xy)/(x² + y²), 沿 y=0 极限 = 0; 沿 y=x 极限 = 1/2 → 极限不存在.

---

知识点 3: 连续性

如果 lim_((x,y) → (x₀, y₀)) f(x,y) = f(x₀, y₀), 称 f 在该点连续.

生活类比: 山地表面没断崖, 就连续. 有断崖, 不连续.

---

知识点 3.5: 二元函数图像的"等高线" 法 (对齐书 §9.1 进一步)

二元函数 z = f(x,y) 的图像是 3D 曲面, 但纸/屏幕是 2D, 怎么表示?

等高线法: 用 z = c (常数) 平面切曲面, 得到一族曲线 (类比地图等高线).

例: z = x² + y² (碗形). 等高线: c=1 → x²+y²=1 (半径 1 圆); c=4 → 半径 2 圆. 等高线密 = 曲面陡, 疏 = 缓.

几何意义大:

  • 山区地图: 等高线表示海拔
  • 气象图: 等压线 = 气压等值线; 等温线 = 温度等值线
  • AI 损失图: 等高线表示 loss 等值集 (优化目标是找 loss 最小的"谷底")

梯度跟等高线的关系 (后面会证): 梯度永远跟等高线垂直, 指向 z 增加最快方向.

9.2 偏导数 (本章核心)

知识点 3.7: 二元函数极限的多路径检测 (对齐书 §9.1 三)

核心: 二元极限要求"任意路径 趋近都得到同一值".

实操检测法:

  1. 沿 x 轴: y = 0, x → 0. f(x, 0) → ?
  2. 沿 y 轴: x = 0, y → 0. f(0, y) → ?
  3. 沿 y = kx 直线: 看是否含 k
  4. 沿 y = x² 等曲线

如果两条路径给不同值 → 极限不存在 (反例反证, 不必所有路径检测).

经典反例:

例题 1-5: lim_{(x,y)→(0,0)} (xy)/(x²+y²). 沿 y = 0: f(x, 0) = 0. 极限 = 0. 沿 y = x: f(x, x) = x²/(2x²) = 1/2. 极限 = 1/2. 两值不同 → 极限不存在 ✗

例题 1-6: lim_{(x,y)→(0,0)} (x²y)/(x⁴+y²). 沿 y = 0: → 0. 沿 y = kx: f → 0. 沿 y = x²: f(x, x²) = x⁴/(x⁴+x⁴) = 1/2. ≠ 0. 极限不存在 ✗

知识点 4: 偏导数定义

一元导数 是函数沿 x 方向的变化率. 二元偏导数 是函数沿 x 或 y 单一方向的变化率.

对 x 的偏导 (把 y 当常数, 对 x 求一元导数): (∂ z)/(∂ x) = lim_(Δ x → 0) (f(x+Δ x, y) - f(x, y))/(Δ x)

记号: f_x(x, y), ∂ z / ∂ x, z_x 都是它.

生活类比: 站在山坡某点, 只往东走一小步, 高度变化率 = ∂ z / ∂ x. 只往北走一小步, 高度变化率 = ∂ z / ∂ y.

求法核心: 对 x 求偏导, 把 y 看成数字, 用一元导数规则; 对 y 求偏导, 把 x 看成数字.

例题 2: z = x² + 3xy + y², 求 ∂ z / ∂ x 和 ∂ z / ∂ y.

  • ∂ z / ∂ x: 把 y 当数, 对 x 求导: 2x + 3y · 1 + 0 = 2x + 3y.
  • ∂ z / ∂ y: 把 x 当数, 对 y 求导: 0 + 3x + 2y = 3x + 2y.

例题 3: z = e^(xy), 求两偏导.

  • ∂ z / ∂ x: 对 x 求导, y 是数. ∂ e^(xy) / ∂ x = y · e^(xy).
  • ∂ z / ∂ y: x · e^(xy).

例题 3-2: z = x^y (含两个变量的指数), 求两偏导.

  • 对 x 求偏导, y 是常数 → 类似 x 的 y 次方求导: ∂z/∂x = y · x^(y-1)
  • 对 y 求偏导, x 是常数, 用对数微分法: z = x^y, ln z = y ln x, (1/z)·∂z/∂y = ln x, 所以 ∂z/∂y = x^y · ln x.

偏导数的几何意义: ∂z/∂x 在点 (x₀, y₀) 处 = 曲面 z = f(x,y) 被平面 y = y₀ 截出的曲线, 在该点的切线斜率.

生活类比: 山坡上"切一刀" (用一个平行 xz 的垂直墙切下来), 看墙上画出的"轮廓线" 的斜率 — 就是 ∂z/∂x.

例题 3-3: 一个气球温度 T(x,y) = 100 - x² - 2y², 在 (1, 1) 处 ∂T/∂x = -2 表示啥? 表示在 (1, 1) 这点, 只沿 x 方向移动单位距离, 温度每变 -2 度. (沿 x 走变冷)

---

知识点 4.3: 偏导数的几何意义 (对齐书 §9.2 进一步说明)

几何意义: ∂z/∂x 在 (x₀, y₀): 用平面 y = y₀ 截曲面 z = f(x,y), 得到曲线 z = f(x, y₀) (一元函数). 该曲线在 x = x₀ 处的切线斜率, 就是 ∂z/∂x|_(x₀,y₀).

类似 ∂z/∂y 是用 x = x₀ 平面截出曲线在 y = y₀ 处的切线斜率.

生活类比: 山的高度 z = f(x,y). 站在 (x₀, y₀) 点:

  • 正东方向 (沿 x 轴) 走小步, 高度变化率 = ∂z/∂x
  • 正北方向 (沿 y 轴) 走小步, 高度变化率 = ∂z/∂y

例题 3-4: f(x,y) = x² + y² (碗形). ∂f/∂x = 2x, ∂f/∂y = 2y. 在 (1, 1) 点: ∂f/∂x|_(1,1) = 2, ∂f/∂y|_(1,1) = 2. 意义: 这点沿 x 走, 高度上升率 2; 沿 y 走, 高度上升率 2. 都是正, 说明这点不是底部.

知识点 4.5: 偏导数存在 ≠ 函数连续

⚠️ 重要: 一元里"可导 ⇒ 连续", 二元里偏导都存在不能推出函数连续.

反例: f(x,y) = xy/(x²+y²) (当 (x,y)≠(0,0)), f(0,0)=0.

  • 在 (0,0) 偏导: f_x(0,0) = lim (f(Δx,0)-0)/Δx = lim 0/Δx = 0. f_y(0,0) 同样 = 0.
  • 偏导都存在 (都是 0).
  • 但 (0,0) 不连续! 沿 y=x 趋近 (0,0): f = x²/(2x²) = 1/2 ≠ 0.

原因: 偏导只看"沿坐标轴"方向, 函数可能在"其他方向"上不连续.

结论 (后面会用): 二元函数要"好",我们需要更强条件叫"可微"(全微分存在). 可微 ⇒ 连续 + 偏导都存在.

知识点 5: 高阶偏导数

偏导再求偏导 = 二阶偏导.

(∂² z)/(∂ x²) = 对 x 偏导后再对 x 偏导. (∂² z)/(∂ x ∂ y) = 先对 y 偏导, 再对 x 偏导.

重要定理: 若 f_(xy) 和 f_(yx) 都连续, 则相等 (混合偏导跟求导顺序无关).

例题 4: z = x³ + 3x² y + y³, 求所有二阶偏导.

  • z_x = 3x² + 6xy
  • z_y = 3x² + 3y²
  • z_(xx) = 6x + 6y
  • z_(yy) = 6y
  • z_(xy) = 6x (从 z_x 对 y 求)
  • z_(yx) = 6x (从 z_y 对 x 求) — 跟 z_(xy) 相等 ✓

---

知识点 5.5: 二阶混合偏导数相等性的反例 (对齐书 §9.2 进一步)

混合偏导数相等定理 (Clairaut/Schwarz 定理) 有前提: f_xy 和 f_yx 都连续.

反例: f(x,y) = xy(x²-y²)/(x²+y²) (x²+y²>0), f(0,0)=0. 计算 f_xy(0,0) = 1, f_yx(0,0) = -1. 因为 f_xy 在 (0,0) 不连续, 定理不成立.

实用结论: 工程/物理常见函数 (多项式/三角/指数) 都满足连续条件, 可放心交换求导顺序.

知识点 5.7: 偏导数判别可微 (对齐书 §9.3 进一步)

充分条件: f_x, f_y 在 (x₀, y₀) 邻域内连续 → f 在 (x₀, y₀) 可微.

必要条件: f 可微 → f_x, f_y 存在 (但偏导数存在不保证可微!).

记忆链: 偏导连续 ⇒ 可微 ⇒ 连续, 偏导存在 偏导存在 ≠ 可微 ≠ 连续

例题 5-4 (经典反例 — 偏导存在但不可微): f(x,y) = xy/√(x²+y²), f(0,0)=0. 偏导: f_x(0,0) = 0, f_y(0,0) = 0 (存在!). 但 f 在 (0,0) 不可微 — 沿 y=x 方向, lim (f(h,h) - 0)/√(2)h = h²/(√2 h · √(2)h) = 1/2 ≠ 0. 可微要求"线性逼近", 这函数不行.

9.3 全微分

知识点 5.8: 偏导数的高级应用 — 雅可比矩阵

雅可比矩阵 (Jacobian): 多元向量函数的偏导数排成矩阵.

设 F: R^n → R^m, F(x) = (f₁(x), ..., f_m(x)). 雅可比: J = [∂fᵢ/∂xⱼ]_(m×n)

作用:

  1. 多元函数的"导数" 推广 — 线性逼近系数
  2. 换元法: |det J| = 变换的"放大比"
  3. 神经网络反向传播: 每层 Jacobian 乘起来 = 链式法则

例题 5-7: F(x,y) = (x²+y, xy). 求 Jacobian. J = [[2x, 1], [y, x]] (2×2 矩阵) |det J| = 2x² - y.

深度学习: 神经网络每层 z = f(Wx+b), Jacobian = f'(...)·W. 反向传播从后往前累乘 J. 这是 PyTorch/TensorFlow 自动求导的数学基础.

知识点 6: 全微分定义

一元的微分: dy = f'(x) dx — 自变量小变化引起函数小变化.

二元全微分: 自变量 (x,y) 同时小变化, 函数的总变化: dz = (∂ z)/(∂ x) dx + (∂ z)/(∂ y) dy

生活类比: 你在山坡上走一小步, 这一步既有"往东" 也有"往北" 两个分量 — 总高度变化 = 东向变化率 × 东向走的距离 + 北向变化率 × 北向走的距离.

例题 5: z = x² y + 3y², 求全微分.

  • z_x = 2xy, z_y = x² + 6y.
  • dz = 2xy · dx + (x² + 6y) · dy.

可微的定义 (严格版): Δz - (∂z/∂x · Δx + ∂z/∂y · Δy) = o(ρ), 其中 ρ = √(Δx² + Δy²) 是"靠近"的距离.

通俗讲就是: 用线性式 (偏导×Δx + 偏导×Δy) 近似 Δz, 误差比 ρ 小一个量级.

可微 ⇒ 连续 ⇒ 极限存在. 可微是二元函数"光滑性"的标准.

充分条件: f_x, f_y 在该点连续 ⇒ f 在该点可微 (反之不一定).

知识点 6.5: 全微分的近似计算 (实用价值)

思想: 函数 f(x,y) 在 (x₀,y₀) 邻近, 用线性近似: f(x₀+Δx, y₀+Δy) ≈ f(x₀,y₀) + f_x(x₀,y₀)·Δx + f_y(x₀,y₀)·Δy

例题 5-2: 估算 (1.02)^(2.99) ≈ ? 取 f(x,y) = x^y, x₀=1, y₀=3, Δx=0.02, Δy=-0.01. f(1,3) = 1. f_x = y x^(y-1) → f_x(1,3) = 3. f_y = x^y · ln x → f_y(1,3) = 1·ln 1 = 0. f(1.02, 2.99) ≈ 1 + 3·0.02 + 0·(-0.01) = 1.06. (精确值约 1.0608, 误差<0.001)

例题 5-3: 求 √(1.02² + 2.97²) 近似值. f(x,y) = √(x²+y²), 取 x₀=1, y₀=3. f(1,3) = √10 ≈ 3.162. f_x = x/√(x²+y²) → f_x(1,3) = 1/√10. f_y = y/√(x²+y²) → f_y(1,3) = 3/√10. Δx = 0.02, Δy = -0.03. 近似 ≈ √10 + (0.02 - 0.09)/√10 = √10 - 0.07/√10 ≈ 3.162 - 0.022 = 3.140.

知识点 6.6: 全微分的"叠加思想" (对齐书 §9.3 进一步)

核心思想: dz 是 dx, dy 引起的两个独立变化的叠加 — 第一项 ∂z/∂x · dx 是"只 x 变 y 不变" 的影响, 第二项 ∂z/∂y · dy 是"只 y 变 x 不变" 的影响.

这是线性化: 一个复杂的多变量小变化, 拆成"沿坐标轴各方向的小变化" 之和.

几何意义: 切平面方程: 曲面 z = f(x,y) 在 (x₀, y₀, z₀) 处的切平面: z - z₀ = f_x(x₀,y₀)(x - x₀) + f_y(x₀,y₀)(y - y₀)

这就是把曲面在该点用 dz = f_x dx + f_y dy 这条"切平面" 替代.

例题 5-4: 曲面 z = x² + y² 在点 (1, 2, 5) 处的切平面方程. f_x = 2x, f_y = 2y, 在 (1,2): f_x = 2, f_y = 4. 切平面: z - 5 = 2(x-1) + 4(y-2), 整理: 2x + 4y - z = 5.

进一步 — 法线方程: 过 (x₀, y₀, z₀) 垂直于切平面的直线, 方向 = (f_x, f_y, -1). (x-x₀)/f_x = (y-y₀)/f_y = (z-z₀)/(-1)

例题 5-5: 上面曲面在 (1, 2, 5) 处的法线方程. 方向 = (2, 4, -1). 法线: (x-1)/2 = (y-2)/4 = (z-5)/(-1).

9.4 多元复合函数求导 (链式法则)

知识点 7: 链式法则

情形 1: z = f(u, v), u = u(t), v = v(t). 求 dz/dt. (dz)/(dt) = (∂ z)/(∂ u) (du)/(dt) + (∂ z)/(∂ v) (dv)/(dt)

生活类比: 你的快乐 = f(收入, 健康), 收入和健康都跟时间相关 → 快乐对时间的变化 = (快乐对收入率 × 收入对时间率) + (快乐对健康率 × 健康对时间率).

情形 2: z = f(u, v), u = u(x, y), v = v(x, y). 求 ∂ z / ∂ x 和 ∂ z / ∂ y. (∂ z)/(∂ x) = (∂ z)/(∂ u) (∂ u)/(∂ x) + (∂ z)/(∂ v) (∂ v)/(∂ x) (∂ z)/(∂ y) = (∂ z)/(∂ u) (∂ u)/(∂ y) + (∂ z)/(∂ v) (∂ v)/(∂ y)

例题 6: z = u² + v², u = x + y, v = x - y. 求 ∂ z / ∂ x.

  • ∂ z / ∂ u = 2u, ∂ z / ∂ v = 2v
  • ∂ u / ∂ x = 1, ∂ v / ∂ x = 1
  • ∂ z / ∂ x = 2u · 1 + 2v · 1 = 2(u + v) = 2(2x) = 4x.

知识点 7.5: 链式法则"树状图"画法 (对齐书 §9.4)

复杂复合函数: z = f(u, v, w), 而 u, v, w 都是 (x, y) 的函数. 求 ∂z/∂x.

树状图法 (避免漏路径): ``` z / | \ u v w /| /| /| x y x y x y ```

每条从 z 到 x 的路径贡献一项, 沿路相乘. 所有路径求和: ∂z/∂x = ∂z/∂u · ∂u/∂x + ∂z/∂v · ∂v/∂x + ∂z/∂w · ∂w/∂x

生活类比: 多条路通往同一目的地, 走每条都算一份, 加起来才是总. AI 反向传播梯度计算就是这个思想.

例题 6-2-2: z = f(u, v), u = x+y, v = xy. 求 ∂z/∂x. 路径 1: z → u → x. 贡献 = f_u · 1 = f_u 路径 2: z → v → x. 贡献 = f_v · y ∂z/∂x = f_u + y · f_v.

9.5 隐函数求导

知识点 7.7: 高阶混合偏导数 (对齐书 §9.4 进一步)

复杂复合函数有时要算 ∂²z/∂x$\partial$y 这种二阶偏导.

关键: 先求一阶, 再"求导整个结果". 注意一阶结果可能仍是 (x,y) 的函数, 求二阶时要用乘积/链式法则.

例题 7-3: z = f(u, v), u = x+y, v = xy. 求 ∂²z/∂x$\partial$y. 一阶: ∂z/∂x = f_u + y · f_v (按链式). 二阶: ∂²z/∂x$\partial$y = ∂/∂y (f_u + y f_v) = f_uu·1 + f_uv·x + f_v + y(f_vu·1 + f_vv·x) = f_uu + (x+y) f_uv + xy f_vv + f_v

(中间用了 f_uv = f_vu 即混合偏导可交换)

知识点 8: 单变量隐函数

方程 F(x, y) = 0 隐含定义了 y = y(x), 求 dy/dx.

公式: (dy)/(dx) = -(F_x)/(F_y)

推理: 两边对 x 求全导数, F_x + F_y · dy/dx = 0, 解出.

例题 7: x² + y² = 1 (单位圆), 求 dy/dx.

  • F = x² + y² - 1
  • F_x = 2x, F_y = 2y
  • dy/dx = -2x/(2y) = -x/y.

知识点 9: 多变量隐函数

方程 F(x, y, z) = 0 隐含定义了 z = z(x, y). (∂ z)/(∂ x) = -(F_x)/(F_z), (∂ z)/(∂ y) = -(F_y)/(F_z)

知识点 9.5: 方程组隐函数 (对齐书 §9.5 二)

之前 (知识点 8/9) 讨论一个方程 F(x,y) = 0 或 F(x,y,z) = 0 隐含定义一个函数. 一般情况:

两个方程定义两个函数: 联立 ``` F(x, y, u, v) = 0 G(x, y, u, v) = 0 ``` 隐含定义 u = u(x,y), v = v(x,y). 求 ∂u/∂x, ∂v/∂x 等.

Jacobian (雅可比行列式): J = |∂(F,G)/∂(u,v)| = | F_u F_v G_u G_v |

非零时, 用 Cramer 法则解出偏导数.

生活类比: 鸡兔同笼问题升级 — 两个变量 (鸡数 u + 兔数 v) 由两个约束 (头数 x + 腿数 y) 唯一确定. Jacobian 测的是这种"互不冗余" 性.

例题 7-2 (经典): ``` x + y - u - v = 0 x - y + 2u - v = 0 ``` 求 ∂u/∂x, ∂v/∂x. F_u = -1, F_v = -1, G_u = 2, G_v = -1. J = (-1)(-1) - (-1)(2) = 1 + 2 = 3. ∂u/∂x = -(1/J) · | F_x F_v G_x G_v | = -(1/3) · |1 -1 1 -1| = -(1/3)·0 = 0. 等等, 算一下 — 这里 ∂u/∂x 用 Cramer 推出.

应用: 经济学 IS-LM 模型 / 物理学相互作用方程组 / 化学平衡多变量约束.

9.6 方向导数与梯度

偏导数只能告诉我们沿坐标轴方向的变化率。如果一个登山者站在山坡上想知道「往东北方向走、高度变化多快」,偏导给不出直接答案。方向导数把这件事扩展到任意方向;梯度则把所有方向的变化率压成一个向量,并且这个向量本身就指向「变化最快」的方向。这是后续条件极值、机器学习中梯度下降、最速下降法的代数基础。

9.6.1 方向导数

设函数 $f(x,y)$ 在点 $P_0(x_0,y_0)$ 的某邻域内有定义,$\vec{l}=(\cos\alpha,\cos\beta)$ 为单位向量(方向角 $\alpha,\beta$ 满足 $\cos^2\alpha+\cos^2\beta=1$)。若极限 $$\frac{\partial f}{\partial\vec{l}}\bigg|_{P_0}=\lim_{t\to 0^+}\frac{f(x_0+t\cos\alpha,\,y_0+t\cos\beta)-f(x_0,y_0)}{t}$$ 存在,则称该极限为 $f$ 在 $P_0$ 沿方向 $\vec{l}$ 的方向导数。

站在曲面 $z=f(x,y)$ 对应的点 $(x_0,y_0,f(x_0,y_0))$,把曲面被竖直平面(沿 $\vec{l}$ 方向)截下来得到一条曲线,方向导数就是这条曲线在该点的斜率。

注意定义中要求 $t\to 0^+$(单侧极限),因此沿 $\vec{l}$ 与沿 $-\vec{l}$ 的方向导数一般不互为相反数(对不可微但偏导存在的函数尤其要小心)。但对可微函数有下面的乘法公式,符号取反时方向导数也取反:

(方向导数的偏导计算公式)若 $f$ 在 $P_0$ 可微,则对任意单位方向 $\vec{l}=(\cos\alpha,\cos\beta)$, $$\frac{\partial f}{\partial\vec{l}}\bigg|_{P_0}=f_x(P_0)\cos\alpha+f_y(P_0)\cos\beta.$$

由可微定义, $$f(x_0+\Delta x,y_0+\Delta y)-f(x_0,y_0)=f_x(P_0)\Delta x+f_y(P_0)\Delta y+o(\rho),\quad\rho=\sqrt{\Delta x^2+\Delta y^2}.$$ 取 $\Delta x=t\cos\alpha$,$\Delta y=t\cos\beta$,则 $\rho=t$(因为 $\vec{l}$ 是单位向量),代入得 $$\frac{f(x_0+t\cos\alpha,y_0+t\cos\beta)-f(x_0,y_0)}{t}=f_x\cos\alpha+f_y\cos\beta+\frac{o(t)}{t}.$$ 令 $t\to 0^+$,$o(t)/t\to 0$。证毕。

关键前提是「可微」。仅「两个偏导存在」并不能推出方向导数公式成立。一个经典反例: $$f(x,y)=\begin{cases}\dfrac{xy}{x^2+y^2},&(x,y)\neq(0,0)\\ 0,&(x,y)=(0,0)\end{cases}$$ 在 $(0,0)$ 处 $f_x(0,0)=f_y(0,0)=0$,但沿 $\vec{l}=(\tfrac{1}{\sqrt2},\tfrac{1}{\sqrt2})$ 的方向导数 $=\lim_{t\to 0^+}\frac{(t/\sqrt2)^2/(t^2)}{t}=\lim_{t\to 0^+}\frac{1}{2t}=+\infty$。公式失败因为 $f$ 在原点不可微。

9.6.2 梯度

设 $f$ 在 $P_0$ 可偏导,梯度(gradient)定义为偏导数构成的向量: $$\nabla f(P_0)=\bigg(\frac{\partial f}{\partial x}(P_0),\frac{\partial f}{\partial y}(P_0)\bigg)\quad(\text{二维});\quad\nabla f=(f_x,f_y,f_z)\quad(\text{三维}).$$ $\nabla$ 读作「nabla」或「del」,可视为算子 $\nabla=\big(\tfrac{\partial}{\partial x},\tfrac{\partial}{\partial y}\big)$。

把方向导数公式用点乘改写: $$\frac{\partial f}{\partial\vec{l}}=f_x\cos\alpha+f_y\cos\beta=\nabla f\cdot\vec{l}.$$

这一改写极其重要——它把「沿某方向的变化率」表达成「梯度向量与方向向量的点乘」。下面的定理是梯度的灵魂。

(梯度方向是最速上升方向)若 $f$ 在 $P_0$ 可微,$\nabla f(P_0)\neq\vec{0}$,则在所有单位方向 $\vec{l}$ 中:

  1. 最大方向导数$=|\nabla f(P_0)|$,在 $\vec{l}=\dfrac{\nabla f}{|\nabla f|}$(即沿梯度方向)时取得;
  2. 最小方向导数$=-|\nabla f(P_0)|$,在 $\vec{l}=-\dfrac{\nabla f}{|\nabla f|}$(沿负梯度方向)时取得;
  3. 零方向导数(函数瞬时不变方向)在 $\vec{l}\perp\nabla f$ 时取得——这正是等高线的切线方向。

设 $\vec{l}$ 与 $\nabla f$ 的夹角为 $\theta$,$|\vec{l}|=1$。由方向导数公式与 Cauchy-Schwarz: $$\frac{\partial f}{\partial\vec{l}}=\nabla f\cdot\vec{l}=|\nabla f|\cdot 1\cdot\cos\theta=|\nabla f|\cos\theta.$$ $\cos\theta\in[-1,1]$,取 $+1$($\theta=0$,$\vec{l}$ 与 $\nabla f$ 同向)时最大、取 $-1$($\theta=\pi$)时最小、取 $0$($\theta=\tfrac{\pi}{2}$)时为零。证毕。

把等高线类比成水平面投影,梯度恰好垂直于过该点的等高线,且指向函数值更高的一侧——所以爬山时「最陡的方向」总是垂直于脚下的等高线,这件事在物理 / GIS / 流体力学里反复出现。

9.6.3 例题

例 1(直接计算)设 $f(x,y)=x^2+2y^2$,求在 $(1,1)$ 处沿 $\vec{u}=(1,1)$ 方向的方向导数。

解:先单位化 $\vec{l}=\vec{u}/|\vec{u}|=(\tfrac{1}{\sqrt2},\tfrac{1}{\sqrt2})$。偏导 $f_x=2x$,$f_y=4y$,在 $(1,1)$ 处 $\nabla f=(2,4)$。 $$\frac{\partial f}{\partial\vec{l}}=\nabla f\cdot\vec{l}=\frac{2}{\sqrt2}+\frac{4}{\sqrt2}=\frac{6}{\sqrt2}=3\sqrt 2.$$

例 2(找最速方向)$f(x,y)=x^2+y^2$ 在 $(1,2)$ 处,问最快上升方向、最快下降方向、不变方向各是什么?最大变化率多少?

解:$\nabla f(1,2)=(2,4)$,$|\nabla f|=\sqrt{20}=2\sqrt 5$。

  • 最快上升:$\vec{l}=\tfrac{1}{2\sqrt 5}(2,4)=(\tfrac{1}{\sqrt 5},\tfrac{2}{\sqrt 5})$,变化率 $+2\sqrt 5$;
  • 最快下降:反向 $(-\tfrac{1}{\sqrt 5},-\tfrac{2}{\sqrt 5})$,变化率 $-2\sqrt 5$;
  • 不变:垂直方向 $(\tfrac{2}{\sqrt 5},-\tfrac{1}{\sqrt 5})$ 或反向,变化率 $0$。

例 3(梯度下降一步)对凸二次函数 $L(\theta_1,\theta_2)=\theta_1^2+\theta_2^2$,初始参数 $\theta^{(0)}=(4,3)$,学习率 $\eta=0.1$,执行一次梯度下降。

解:$\nabla L(\theta)=(2\theta_1,2\theta_2)$。在 $\theta^{(0)}$ 处 $\nabla L=(8,6)$, $$\theta^{(1)}=\theta^{(0)}-\eta\nabla L=(4,3)-0.1\cdot(8,6)=(3.2,2.4).$$ 再迭代一步:$\nabla L(3.2,2.4)=(6.4,4.8)$,$\theta^{(2)}=(3.2-0.64,2.4-0.48)=(2.56,1.92)$。每步距离最优点 $(0,0)$ 的范数比上一步缩小一个固定比例 $1-2\eta=0.8$,这就是凸二次问题的几何收敛。

9.6.4 与 AI 优化的联系

机器学习把模型参数记作 $\theta\in\mathbb{R}^d$,把训练损失记作 $L(\theta)$。最速下降 / 梯度下降就是把上面定理 9.6.B 的「负梯度方向是最速下降方向」直接搬到 $d$ 维: $$\theta^{(k+1)}=\theta^{(k)}-\eta_k\,\nabla L(\theta^{(k)}).$$

实际中常见的变体:

| 方法 | 公式核心 | 解决什么问题 | | --- | --- | --- | | SGD | 用单样本(或 mini-batch)估计 $\nabla L$ | 大数据集,全量梯度算不起 | | Momentum | $v\gets\beta v+\nabla L$,$\theta\gets\theta-\eta v$ | 鞍点 / 长山谷震荡 | | Adam | 一阶矩 $m$ + 二阶矩 $v$ 自适应缩放 | 不同参数尺度差异大 | | Newton | $\theta\gets\theta-\eta\,H^{-1}\nabla L$,$H$ 是 Hessian | 二阶曲率快收敛,但 $H^{-1}$ 贵 |

(下降引理 / Descent Lemma)若 $\nabla L$ 是 $L$-Lipschitz 的(即 $\|\nabla L(\theta_1)-\nabla L(\theta_2)\|\le L\|\theta_1-\theta_2\|$),则梯度下降步骤满足 $$L(\theta^{(k+1)})\le L(\theta^{(k)})-\eta\bigg(1-\frac{\eta L}{2}\bigg)\|\nabla L(\theta^{(k)})\|^2.$$

由 $L$-平滑性的二阶展开式(可由 Taylor 余项 + Lipschitz 条件得) $$L(\theta')\le L(\theta)+\nabla L(\theta)\cdot(\theta'-\theta)+\tfrac{L}{2}\|\theta'-\theta\|^2.$$ 代入 $\theta'=\theta-\eta\nabla L(\theta)$: $$L(\theta')\le L(\theta)-\eta\|\nabla L\|^2+\tfrac{L\eta^2}{2}\|\nabla L\|^2=L(\theta)-\eta(1-\tfrac{L\eta}{2})\|\nabla L\|^2.\;\square$$

这个不等式告诉我们:当 $\eta\le\tfrac{1}{L}$ 时,每一步至少让损失下降 $\tfrac{\eta}{2}\|\nabla L\|^2$,从而梯度范数最终趋零(收敛到稳定点)。这是机器学习里收敛性分析的最基本工具。

易错点:

  • 计算方向导数前必须把方向向量单位化;若代入未单位化的向量,结果会被它的模放大。
  • 「方向导数 $=$ 梯度点乘方向」要求函数在该点可微;偏导存在不够。
  • 梯度方向是最速上升,要做最小化时记得取负梯度;符号搞反 = 算法朝错方向跑。
  • 高维非凸问题里,$\nabla L=0$ 不一定是极小,可能是鞍点或局部极大;需用 Hessian 进一步判别(§9.8)。

9.7 多元微分的几何应用

知识点 11.5: 梯度的物理意义 (对齐书 §9.7 二 进一步)

梯度 ∇f 不只是"导数向量", 它有几个深刻的几何意义:

① 等值线/等值面的法向量: 等高线 f(x, y) = c 上每点的切方向跟 ∇f 垂直 (因为沿等值线 f 不变, 方向导数 = 0 = ∇f·$\vec{t}$).

② 增长最快方向: 在某点, 方向导数 = ∇f · $\vec{l}$⁰ = |∇f| cos $\alpha$. 最大值当 $\alpha$ = 0, 即 $\vec{l}$ 方向跟 ∇f 同向.

③ 几何例题: 等高线密 = ∇f 大 = 山陡; 等高线疏 = ∇f 小 = 山缓.

例题 8-4 (等值线-梯度关系): f(x,y) = x² + y² (碗形). 等值线: x² + y² = c (圆). ∇f = (2x, 2y) = 2(x, y) = 2 ·$\vec{r}$. 观察: 这跟"圆的半径方向" 一致 — 梯度永远跟等值线垂直 ✓

知识点 11.7: 多元函数极值的二阶判别 (Hessian 矩阵, 对齐书 §9.8)

二阶判别公式来源: 在驻点 (∇f = 0), 用二阶泰勒展开: f(x+h, y+k) ≈ f(x,y) + (1/2)[A h² + 2B hk + C k²]

其中 A=f_xx, B=f_xy, C=f_yy. 二次型 Q(h,k) = A h² + 2B hk + C k² 正定 ↔ 极小, 负定 ↔ 极大.

正定条件: A > 0 且 AC - B² > 0.

Hessian 矩阵: H = | A B B C |. 判别 H 的特征值:

  • 全正 → 极小
  • 全负 → 极大
  • 异号 → 鞍点

AI/ML 直接用: 二阶优化 (Newton 法) 用 Hessian 加速收敛, 比梯度下降快得多.

切线方程 (点向式): 用切向量做方向向量.

9.8 多元函数的极值

知识点 13.5: 法线方程详细 (对齐书 §9.6 三 进一步)

曲面 F(x,y,z) = 0 在 P₀(x₀,y₀,z₀) 处的法线: 方向向量 = ∇F = (F_x, F_y, F_z) |_(P₀)

法线方程 (对称式): (x-x₀)/F_x = (y-y₀)/F_y = (z-z₀)/F_z

(F_x, F_y, F_z 是在 P₀ 处的值)

例题 9-2: 球面 x²+y²+z² = 14 在 (1, 2, 3) 处的法线方程. F = x²+y²+z² - 14. ∇F = (2x, 2y, 2z), 在 (1,2,3) = (2, 4, 6). 法线: (x-1)/2 = (y-2)/4 = (z-3)/6 = (x-1)/1 = (y-2)/2 = (z-3)/3.

几何意义: 法线穿过球心 (球面任一点的法线必过球心).

特例: z = f(x,y) (显函数). 法向量 = (f_x, f_y, -1).

知识点 14: 极值的必要条件

可微函数 z = f(x, y) 在 (x₀, y₀) 取得极值, 则: f_x(x₀, y₀) = 0 且 f_y(x₀, y₀) = 0

(梯度为零)

驻点: 满足 f_x = f_y = 0 的点. 极值点必是驻点, 但驻点不一定极值 (可能是鞍点).

知识点 15: 极值的充分条件 (二阶判别)

驻点 (x₀, y₀) 处求:

  • A = f_(xx), B = f_(xy), C = f_(yy)
  • 判别式 D = AC - B²

判别:

  • D > 0 且 A > 0: 极小值
  • D > 0 且 A < 0: 极大值
  • D < 0: 鞍点 (非极值)
  • D = 0: 此法失效, 另想办法

例题 10: f(x,y) = x² + y² + 2x - 4y + 3, 求极值.

  • f_x = 2x + 2, f_y = 2y - 4. 解 f_x = f_y = 0 → 驻点 (-1, 2).
  • A = f_(xx) = 2, B = f_(xy) = 0, C = f_(yy) = 2.
  • D = 4 - 0 = 4 > 0, A > 0 → 极小值.
  • 极小值 = f(-1, 2) = 1 + 4 - 2 - 8 + 3 = -2.

知识点 16: 条件极值 — 拉格朗日乘数法

求 f(x, y) 在约束 $\varphi$(x, y) = 0 下的极值.

构造拉格朗日函数: L = f + $\lambda$ $\varphi$, 然后求驻点: L_x = 0, L_y = 0, L_λ = 0

生活类比: 给定预算 (约束), 求快乐最大 (目标函数). 拉格朗日乘数 $\lambda$ 就是"边际快乐" = 多 1 元预算能多多少快乐.

例题 11: 求 f(x, y) = xy 在 x + y = 1 约束下的极值.

  • L = xy + $\lambda$(x + y - 1)
  • L_x = y + $\lambda$ = 0 → y = -$\lambda$
  • L_y = x + $\lambda$ = 0 → x = -$\lambda$
  • L_λ = x + y - 1 = 0 → -2λ = 1 → $\lambda$ = -1/2, x = y = 1/2
  • 极大值 = 1/4.

例题 11-2 (经典几何题): 在椭圆 x²/4 + y² = 1 上求一点, 距原点最远 + 最近. 目标函数 = 距离平方 f(x,y) = x² + y² (避免根号). 约束: g(x,y) = x²/4 + y² - 1 = 0. L = x² + y² + $\lambda$(x²/4 + y² - 1)

  • ∂L/∂x = 2x + λx/2 = x(2 + $\lambda$/2) = 0 → x = 0 或 $\lambda$ = -4
  • ∂L/∂y = 2y + 2λy = 2y(1+$\lambda$) = 0 → y = 0 或 $\lambda$ = -1
  • 约束: 椭圆方程

讨论:

  1. x = 0, y² = 1 → y = ±1, 距离 = 1.
  2. y = 0, x² = 4 → x = ±2, 距离 = 2.

最近 = 1, 最远 = 2 (椭圆短半轴 1, 长半轴 2).

例题 11-3 (实用问题): 用 12 平方米材料做长方体盒子无盖, 求最大体积. 设长 x, 宽 y, 高 z, 体积 V = xyz. 约束: xy + 2xz + 2yz = 12 (无盖, 底 + 4 个侧面). 拉格朗日: L = xyz + $\lambda$(xy + 2xz + 2yz - 12) 偏导联立: yz + $\lambda$(y + 2z) = 0 xz + $\lambda$(x + 2z) = 0 xy + $\lambda$(2x + 2y) = 0 解得 x = y = 2, z = 1, V = 4 立方米.

知识点 17: 多元函数最值 (在有界闭区域上)

算法 (求闭区域 D 上 f 的最值):

  1. 内部驻点: 求所有 f_x = f_y = 0 的点
  2. 边界极值: 在 D 的边界上, 化为一元函数 + 求一元极值
  3. 端点
  4. 全部代入 f, 比大小

例题 11-4: 求 f(x,y) = xy 在 D = {x² + y² ≤ 1} 上的最值. 内部驻点: f_x = y = 0, f_y = x = 0 → (0,0), f = 0. 边界 x² + y² = 1, 用参数 x = $\cos\theta$, y = $\sin\theta$: f = $\cos\theta$ $\sin\theta$ = sin(2θ)/2. 最大值 = 1/2 (在 $\theta$=$\pi$/4, 即 (√2/2, √2/2)), 最小值 = -1/2. 结论: max = 1/2, min = -1/2.

9.9 二元函数的泰勒公式 (对齐书 §9.9)

知识点 17: 二元泰勒公式

一元有 f(x) ≈ f(a) + f'(a)(x-a) + f''(a)(x-a)²/2 + ... 这种"用多项式逼近函数"的展开. 二元同样有:

f(x, y) ≈ f(a, b) + [f_x(a,b)(x-a) + f_y(a,b)(y-b)] + (1/2!)[f_xx(x-a)² + 2 f_xy(x-a)(y-b) + f_yy(y-b)²] + ...

生活类比: 像爬山, 站在 (a,b) 看附近地形, 一阶展开 = 平面 (切平面), 二阶展开 = 抛物面 (凹凸感), 高阶继续精细化.

知识点 18: 二阶判别式的"由来" (对齐 §9.9 二)

之前 (知识点 15) 用 D = AC - B² 判极值类型. 这个就是二阶泰勒展开: A=f_xx, B=f_xy, C=f_yy, 是 Hessian 矩阵的项. D > 0 + A > 0 → Hessian 正定 → 极小值.

应用: AI/ML 优化里用 Hessian 判断"局部最小 vs 鞍点", 是同一套数学.

9.10 最小二乘法 (对齐书 §9.10)

知识点 19: 最小二乘法

问题: 一堆数据点 (xᵢ, yᵢ), 怎么找一条直线 y = ax + b 最贴合?

思想: "误差平方和最小". 残差 = (yᵢ - axᵢ - b)², 加起来求最小: S(a, b) = Σᵢ (yᵢ - axᵢ - b)²

对 a, b 求偏导 = 0, 解出: a = (n·Σxy - Σx·Σy) / (n·Σx² - (Σx)²) b = (Σy - a·Σx) / n

为啥用平方和: ① 处理"正负误差不抵消" ② 求导方便 ③ 跟正态分布最大似然估计一致.

例题 12: 数据点 (1,2), (2,3), (3,5), (4,4), 拟合 y = ax + b. n=4, Σx=10, Σy=14, Σxy=39, Σx²=30. a = (4·39 - 10·14)/(4·30 - 100) = (156-140)/(120-100) = 16/20 = 0.8. b = (14 - 0.8·10)/4 = 6/4 = 1.5. 拟合直线: y = 0.8x + 1.5.

AI 直接用: 线性回归就是最小二乘. 神经网络反向传播找梯度也是这个思想推广.

知识点 19.5: 拉格朗日乘数法的几何意义

核心: 在约束 g(x,y) = 0 下求 f(x,y) 极值. 几何上:

  • f 的等值线: f = c (一族曲线)
  • 约束 g = 0: 一条曲线

极值点的关键特征: 在最优点, f 的等值线跟约束曲线 g=0 相切. 切点 = 极值候选.

相切 ⇔ 两曲线的法向量平行 ⇔ ∇f = $\lambda$∇g ($\lambda$ 是乘数).

推广 (KKT 条件): 多约束 + 不等式约束情况. 是机器学习 SVM 优化的核心数学.

例题 12 (经典 + 几何): 在 x² + y² = 1 上求 f(x,y) = xy 的极值. ∇f = (y, x), ∇g = (2x, 2y). ar→ ∇f = $\lambda$∇g → y = 2λx, x = 2λy. 两式相乘: xy = 4λ²xy → 若 xy ≠ 0, 4λ² = 1, $\lambda$ = ±1/2. $\lambda$=1/2: y = x, 代入 x²+y² = 1 → x = ±1/√2, y = ±1/√2 同号. f = 1/2 (极大). $\lambda$=-1/2: y = -x, → x = ±1/√2, y = ∓1/√2. f = -1/2 (极小). 结论: max = 1/2, min = -1/2.

几何验证: 在 (1/√2, 1/√2) 处, f 的等值线 xy=1/2 是双曲线一支, 约束 x²+y²=1 是单位圆, 两者在该点相切 ✓.

补充: 优化深化 — 牛顿法 / KKT 条件

知识点 9.5: 牛顿法 求多元极值

一元: 解 f'(x)=0 用 x_(n+1) = x_n - f'(x_n)/f''(x_n).

多元推广: 解 ∇f(x)=0 x_(n+1) = x_n - H(x_n)^(-1) · ∇f(x_n) H 是 Hessian.

几何: 用二阶 Taylor 作"局部抛物面拟合", 跳到极值点.

例题 9-5: f(x,y) = x⁴ + y⁴, 从 (1,1) 求极小. ∇f = (4x³, 4y³). H = diag(12x², 12y²). x₁ = (1,1) - diag(1/12,1/12)·(4,4) = (2/3, 2/3). x₂ ≈ (4/9, 4/9). 二次收敛趋 (0,0).

对比 GD: 牛顿法二次收敛, GD 线性收敛. 但 H 在大模型存不下, 实践用 quasi-Newton (BFGS/L-BFGS).

知识点 9.6: KKT 条件 (不等式约束)

拉格朗日乘子只处理等式. 实际多有不等式.

问题: min f(x) s.t. g(x) ≤ 0, h(x) = 0.

Lagrangian: L = f + λh + μg

KKT 必要条件:

  1. 梯度: ∇f + $\lambda$∇h + $\mu$∇g = 0
  2. 可行性: h=0, g$\le$0
  3. 对偶可行: $\mu$ ≥ 0
  4. 互补松弛: $\mu$·g = 0

直觉:

  • g<0 (约束不紧) → $\mu$=0
  • g=0 (约束紧) → $\mu$ 起作用

应用:

  • SVM: 二次规划 + KKT 推出"支持向量"
  • 投资组合: 财富约束 ≤ B
  • 神经网络剪枝: L1 稀疏

例题 9-6 (SVM): min (1/2)||w||² s.t. yᵢ(w·xᵢ+b) ≥ 1. L = (1/2)||w||² - ∑μᵢ[yᵢ(w·xᵢ+b)-1]. ∂L/∂w=0 → w = ∑μᵢyᵢxᵢ. 只有 μᵢ>0 的样本 = 支持向量.

知识点 9.7: 多元函数的方向导数完整定义

方向导数 D_u→ f(x₀): 沿单位向量 $\vec{u}$ 方向 f 的变化率. D_u→ f(x₀) = lim_($\vec{t}$0) [f(x₀+tu→) - f(x₀)] / t

与梯度关系: D_u→ f = ∇f · $\vec{u}$ = |∇f| $\cos\theta$

核心几何意义:

  • $\vec{u}$ = ∇f/|∇f| (沿梯度): D_u→ f = |∇f|, 最大.
  • $\vec{u}$ ⊥ ∇f: D_u→ f = 0 (沿等高线方向不变).
  • $\vec{u}$ = -∇f/|∇f|: D_u→ f = -|∇f|, 最小 (最速下降方向).

例题 9-7: f(x,y) = x²+y², 求 (1,1) 沿 $\vec{u}$=(3/5, 4/5) 的方向导数. ∇f|_(1,1) = (2, 2). D_u→ f = (2,2)·(3/5, 4/5) = 6/5 + 8/5 = 14/5 = 2.8.

ML 训练核心: 梯度下降 $\theta$ ← $\theta$ - η·∇L, 选 -∇L 方向就是因为它是最速下降 — 局部下降最快.

知识点 9.8: 包络 (Envelope) 与判别曲线

问题: 一族曲线 F(x, y, c)=0 (c 是参数), 求"包络"= 与族内每条都相切的曲线.

消 c 法:

  • F(x, y, c) = 0
  • ∂F/∂c = 0

联立消 c, 得到包络方程.

例题 9-8: 直线族 x cos c + y sin c = 1 (c 是参数). ∂/∂c: -x sin c + y cos c = 0 → tan c = y/x. 代回: x·x/√(x²+y²) + y·y/√(x²+y²) = 1 → √(x²+y²) = 1. 包络: 单位圆 x² + y² = 1.

物理意义: 任意切线族的包络 = 它们公共相切的曲线. 抛物面碟天线就是利用包络:平行光经反射汇聚到焦点.

应用:

  • 光学焦散 (caustic): 杯子内反射形成的亮纹
  • 微分方程 ODE 奇解
  • 几何光学: 透镜成像

知识点 9.9: 隐函数定理 — 完整证明思路

问题: F(x, y) = 0 何时可以解出 y = y(x)?

定理 (一元情形): 设 F(x₀, y₀) = 0, F 连续可微, F_y(x₀, y₀) ≠ 0. 存在 x₀ 邻域 U, 使在 U 内有唯一 y = y(x) 满足 F(x, y(x)) = 0, 且 y'(x) = -F_x/F_y.

证明思路 (压缩映射):

  1. 将 F(x,y)=0 改写为 y = y - F(x,y)/F_y(x₀,y₀) := T(x, y)
  2. 在 (x₀, y₀) 附近 |∂T/∂y| < 1 (利用 F_y 连续 + 初值条件)
  3. 不动点 ⟹ 存在唯一 y(x) (Banach 不动点定理)
  4. y(x) 的可微性由 F 的可微性传播

多元推广 (向量值): F: R^(n+m) → R^m, F(x₀, y₀)=0, ∂F/∂y 满秩 m. 则存在 y=y(x).

例题 9-9: F(x,y) = x²+y²-1=0. F_y = 2y. 在 (1/2, √3/2) 处 F_y$\neq$0, 可解. y'(x) = -F_x/F_y = -2x/(2y) = -x/y = -(1/2)/(√3/2) = -1/√3.

应用:

  • 经济一般均衡: 价格 → 数量的可解性
  • 力学约束: Lagrange 方程从约束 g=0 解出广义坐标
  • 数值连续: 同伦法追踪解曲线
  • 神经网络可逆性: Normalizing Flows 要求 y=f(x) 可逆

知识点 9.10: 二元函数泰勒展开

设 f 在 (x₀, y₀) 邻域有 n+1 阶连续偏导. 令 h = x-x₀, k = y-y₀:

f(x, y) = f(x₀, y₀) + [h·∂/∂x + k·∂/∂y]f|_(x₀,y₀) + (1/2!)[h·∂/∂x + k·∂/∂y]² f|_(x₀,y₀) + ... + R_n

展开二阶项: f ≈ f₀ + (h f_x + k f_y) + (1/2)(h² f_xx + 2hk f_xy + k² f_yy)

矩阵形式: f(x) ≈ f(x₀) + ∇f^T (x-x₀) + (1/2)(x-x₀)^T H (x-x₀)

H = Hessian.

例题 9-10: f(x,y) = e^(x+y) 在 (0,0) 二阶展开. f₀=1, ∇f=(1,1), H = [[1,1],[1,1]]. e^(x+y) ≈ 1 + x + y + (1/2)(x² + 2xy + y²) = 1 + (x+y) + (1/2)(x+y)²

应用:

  • 优化: 牛顿法用二阶 Taylor 拟合
  • 物理: 谐振子近似 = 势能函数二阶 Taylor
  • ML 解释性: LIME 局部线性近似复杂模型

知识点 9.11: 梯度下降变体 (深度学习核心)

1. 动量法 (Momentum) — "惯性": v_t = $\beta$ v_(t-1) + ∇L(θ_t) θ_(t+1) = θ_t - η v_t 把过去梯度累积成"速度", 像球滚下山.

2. Nesterov 加速梯度 (NAG): v_t = $\beta$ v_(t-1) + ∇L(θ_t - η $\beta$ v_(t-1)) "看一眼前方"再更新.

3. AdaGrad: G_t = G_(t-1) + (∇L)². $\theta$ ← $\theta$ - η·∇L/(√G_t + ε). 自适应每参数学习率.

4. RMSProp: G_t = $\beta$ G_(t-1) + (1-$\beta$)(∇L)². 解决 AdaGrad 学习率渐弱.

5. Adam = Momentum + RMSProp (深度学习默认): m_t = β₁ m_(t-1) + (1-β₁)∇L v_t = β₂ v_(t-1) + (1-β₂)(∇L)² $\theta$ ← $\theta$ - η · m̂_t / (√v̂_t + ε) β₁=0.9, β₂=0.999 标准. 训 LLM/Diffusion 首选.

知识点 9.12: 自动微分 (Autograd)

前向模式: 对每个 x_i 同时计算 (x_i, dx_i/dx). 链式法则正向.

反向模式 (Backprop): 正向算 L, 再从 L 反向用 dL/dy → dL/dx.

复杂度对比: 输出 1 维, n 个输入:

  • 正向: O(n) 倍前向
  • 反向: O(1) 倍前向

L 标量, $\theta$ 亿级 ⟹ 反向快 10⁹ 倍. Backprop 革命核心.

  • jvp (Jacobian-vector product): 正向
  • vjp (Vector-Jacobian product): 反向

应用: PyTorch/TF/JAX 的 .backward() 都用 vjp. 元学习用 jvp 算二阶梯度. 可微分物理用 autodiff 优化设计.

知识点 9.13: 函数族的一致收敛 (深度学习的泛化基石)

逐点收敛: f_n(x) → f(x) 在每个 x 单独看收敛.

一致收敛: ∀ε>0, ∃N, ∀n>N, sup_x |f_n(x) - f(x)| < ε. 整个函数族整体收敛.

关键差别:

  • 逐点收敛允许连续函数极限不连续 (例 x^n on [0,1])
  • 一致收敛保留连续性: f_n 连续 → f 连续
  • 一致收敛保留可积性: lim ∫f_n = ∫lim f_n

例题 9-13: f_n(x) = x^n on [0, 1]. 逐点: → 0 if x<1, → 1 if x=1. 不连续. 但每个 f_n 连续 ⟹ 没有一致收敛.

深度学习泛化 (核心理论):

  • 训练: 在数据集 D_n 上最小化 L_n($\theta$) = (1/n) ∑ ℓ(f(x_i), y_i)
  • 真实目标: L($\theta$) = E_(x,y)[ℓ]
  • 泛化 = 学到的 $\theta$̂ 使 L($\theta$̂) ≈ L_n($\theta$̂) ?
  • VC 维 / Rademacher 复杂度 给出 sup_θ |L_n($\theta$) - L($\theta$)| 上界

这就是一致收敛在 ML 里的"统计学习理论"角度.

知识点 9.14: 多元微分的微分形式回看

回想第十一章引入的微分形式视角:

  • df = ∑ (∂f/∂x_i) dx_i ← 一次形式
  • d²f = 0 ← 这是 d∘d=0
  • ddf 写出: ∑ (∂²f/∂x_i ∂x_j) dx_j ∧ dx_i. 利用 dx_j ∧ dx_i = -dx_i ∧ dx_j + 对称性, 可证 d²f=0.

几何与代数统一: 多元微分 = 标量函数的一次形式; 梯度 = 一次形式的"对偶向量". 这是高数下册多元微分的现代视角.

应用 (跨学科):

  • 物理: Lagrange/Hamilton 力学的辛形式
  • 控制理论: 微分平坦性
  • 几何深度学习: 图神经网络的"消息传递" 本质是一次形式的离散版本
