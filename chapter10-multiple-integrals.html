第十章 重积分

对照: 同济大学《高等数学》第七版下册 第十章
一元定积分 ∫ f(x) dx → 推广到二/三重积分.

10.0 整章在干啥?

一元定积分 ∫_a^b f(x) dx 几何意义 = 曲线 y=f(x) 跟 x 轴在 [a,b] 之间围的面积.

二重积分 ∬_D f(x,y) dxdy 几何意义 = 曲面 z=f(x,y) 在区域 D 上方围的体积.

三重积分 ∭_V f(x,y,z) dxdydz = 在三维区域 V 内的质量/总量 (密度 f 不均匀分布时的总质量).

对你的意义: 物理(质量/重心/转动惯量)、电磁(电场总通量)、概率(联合分布积分求概率)、图形(体积渲染) 全用. ML 里贝叶斯统计的概率密度积分也是这.

10.1 二重积分概念与性质

知识点 0.5: 为啥从"一元定积分" 升级到"二/三重积分"?

中学/大一上册学的 ∫ f(x) dx 只能算"线段上的面积/总和". 现实需要算:

  • 一片田地的总产量 (面积上累加)
  • 一个房间的总气体质量 (体积上累加)
  • 一片湖的总水量

这些都需要"二维/三维 上的积分" — 重积分.

核心思想: 切块 → 求和 → 取极限. 跟一元定积分思路一样, 只是"块"从"线段" 升级到"小方块/小立方体".

这章学:

  • 二重积分: 算曲面下的体积 / 平面密度积分
  • 三重积分: 算空间区域的"总量"
  • 换元 (极坐标/柱坐标/球坐标): 不规则形状变规则

知识点 1: 定义 (思想)

把区域 D 切成 n 个小块 Δσ_i, 每块挑一点 (ξ_i, η_i), 求和: Σ_(i=1)^n f(ξ_i, η_i) Δσ_i 取极限 (块越切越细): ∬_D f(x,y) dσ = lim_(λ → 0) Σ f(ξ_i, η_i) Δσ_i

生活类比: 一片田地按高低不平 (每点产量不同 f(x,y)), 算总产量 = 切成无数小方块, 每块产量 × 块面积, 加起来.

知识点 2: 性质 (跟一元积分类似)

  • 线性: ∬ (af + bg) = a∬ f + b∬ g
  • 可加: 区域分成两部分, 积分等于两部分之和
  • 比较: f ≤ g 则 ∬ f ≤ ∬ g
  • 估值: m·σ ≤ ∬ f dσ ≤ M·σ (m, M 是 f 在 D 上的最小/最大值, σ 是 D 面积)
  • 中值定理: 存在 (ξ, η) ∈ D 使 ∬ f dσ = f(ξ, η) · σ. 几何意义: "平均值 × 区域面积 = 总积分"

例题 0-1 (估值定理应用): D = {0 ≤ x ≤ 1, 0 ≤ y ≤ 1}, 估 ∬_D e^(x²+y²) dσ 的范围. 在 D 上: e⁰ ≤ e^(x²+y²) ≤ e² 即 1 ≤ ... ≤ e². σ(D) = 1. 所以 1 ≤ ∬ ≤ e² ≈ 7.39.

10.2 二重积分的计算

知识点 3: 直角坐标 — X 型区域

D 由 a ≤ x ≤ b, φ₁(x) ≤ y ≤ φ₂(x) 定义. ∬_D f(x,y) dσ = ∫_a^b [∫_(φ₁(x))^(φ₂(x)) f(x,y) dy] dx

做法: 先内层后外层. 内层把 x 当常数对 y 积; 外层再对 x 积.

例题 1: ∬_D xy dσ, D 为 0 ≤ x ≤ 1, 0 ≤ y ≤ x (三角形). = ∫_0¹ ∫₀^x xy dy dx = ∫_0¹ [(xy²)/(2)]₀^x dx = ∫_0¹ (x³)/(2) dx = (1)/(8)

例题 1-2: ∬_D (x²+y²) dσ, D 为单位正方形 0≤x≤1, 0≤y≤1. = ∫_0¹∫_0¹ (x²+y²) dy dx = ∫_0¹ (x² + 1/3) dx = 1/3 + 1/3 = 2/3.

例题 1-3: ∬_D x dσ, D 为 y=x, y=x² 围的区域 (0≤x≤1). 描述: x² ≤ y ≤ x. = ∫_0¹ ∫_(x²)^x x dy dx = ∫_0¹ x(x - x²) dx = ∫_0¹ (x² - x³) dx = 1/3 - 1/4 = 1/12.

知识点 4: 直角坐标 — Y 型区域

D 由 c ≤ y ≤ d, ψ₁(y) ≤ x ≤ ψ₂(y) 定义. ∬_D f dσ = ∫_c^d [∫_(ψ₁(y))^(ψ₂(y)) f dx] dy

口诀: 谁的范围是常数, 谁是外层.

知识点 4.5: 交换积分次序 (经典套路)

有时一个次序难算, 换次序就秒. 步骤:

  1. 画区域 D 图
  2. 把"先 y 后 x" 重新描述为"先 x 后 y"
  3. 重写积分

例题 1-4: 计算 ∫_0¹ ∫_x^1 sin(y²) dy dx. 问题: 内层 ∫ sin(y²) dy 没有初等原函数, 这个次序算不出来! 画区域: 0 ≤ x ≤ 1, x ≤ y ≤ 1 — 即三角形 (0,0), (1,1), (0,1). 换次序: 0 ≤ y ≤ 1, 0 ≤ x ≤ y. = ∫_0¹ ∫_0^y sin(y²) dx dy = ∫_0¹ y sin(y²) dy 令 u = y², du = 2y dy: = ∫_0¹ sin u · (du/2) = (1 - cos 1)/2.

为啥换? 内层变成 ∫_0^y sin(y²) dx 这个里 y 是常数, 直接 = y · sin(y²), 然后外层 ∫ y sin(y²) dy 可以用代换法算.

知识点 5: 极坐标变换

D 复杂 (圆 / 扇形) 时, 换极坐标 x = rcosθ, y = rsinθ 算更简单.

变换公式: ∬_D f(x,y) dxdy = ∬_D f(rcosθ, rsinθ) · r drdθ

关键: 多一个 r 因子! (面积微元 dxdy = r drdθ, 几何上极坐标小块面积 = 半径 × 角度微元 × dr).

知识点 4.7: 极坐标下的常见区域 (对齐书 §10.2 二 进一步)

圆盘: 0 ≤ r ≤ R, 0 ≤ θ ≤ 2π 扇形: 0 ≤ r ≤ R, α ≤ θ ≤ β 圆环: a ≤ r ≤ b, 0 ≤ θ ≤ 2π 半圆: 0 ≤ r ≤ R, 0 ≤ θ ≤ π (上半) 心脏线: r = a(1+cosθ), 0 ≤ θ ≤ 2π (玫瑰花形) 双纽线: r² = a²cos(2θ), -π/4 ≤ θ ≤ π/4 ∪ 3π/4 ≤ θ ≤ 5π/4 (∞ 形)

例题 2 (再回顾): ∬_D e^(-(x²+y²)) dσ, D 是单位圆.

例题 2: ∬_D e^(-(x²+y²)) dσ, D 是单位圆. 极坐标: 0 ≤ r ≤ 1, 0 ≤ θ ≤ 2π. = ∫₀^(2π) ∫_0¹ e^(-r²) · r drdθ = 2π · [-(1)/(2) e^(-r²)]_0¹ = π(1 - e⁻¹)

例题 2-2 (经典: 高斯积分推广): ∬_(R²) e^(-(x²+y²)) dxdy = ? 全平面极坐标: 0 ≤ r < ∞, 0 ≤ θ ≤ 2π. = ∫₀^(2π) ∫_0^∞ e^(-r²) r drdθ = 2π · 1/2 = π. 重要推论: (∫_(-∞)^∞ e^(-x²) dx)² = π → ∫_(-∞)^∞ e^(-x²) dx = √π. (这就是著名的高斯积分, 概率论正态分布归一化用的!)

例题 2-3: ∬_D (x²+y²) dσ, D 为 1 ≤ x²+y² ≤ 4 (圆环). 极坐标: 1 ≤ r ≤ 2, 0 ≤ θ ≤ 2π. = ∫₀^(2π) ∫_1² r² · r drdθ = 2π · [r⁴/4]_1² = 2π · (4 - 1/4) = 15π/2.

例题 2-4: 用二重积分求圆 x²+y² = R² 围的面积. S = ∬_D 1 dxdy. 极坐标: = ∫₀^(2π) ∫_0^R r drdθ = 2π · R²/2 = πR² ✓.

10.3 三重积分

知识点 5.5: 关于积分次序的"右手定则"

X 型 vs Y 型 选择: 看 D 的边界曲线"哪个变量数值能用单调函数描述".

实操:

  • 边界由 y = φ(x) 这种函数描述 (上下边界): 用 X 型, 外层 x 内层 y
  • 边界由 x = ψ(y) 描述 (左右边界): 用 Y 型, 外层 y 内层 x

复杂 D 的拆分: 不是 X 型也不是 Y 型, 用直线切成几块, 每块单独按 X/Y 型算.

例题 5-5: D 是单位圆 x²+y²≤1 上半部分. 算 ∬ x dxdy. X 型: 0 ≤ x ≤ ... 不对, x 范围跟 y 有关. 准确写法: -1 ≤ x ≤ 1, 0 ≤ y ≤ √(1-x²). Y 型: 0 ≤ y ≤ 1, -√(1-y²) ≤ x ≤ √(1-y²). 两种都行. 选 X 型: ∬ x dxdy = ∫_{-1}^1 ∫_0^√(1-x²) x dy dx = ∫_{-1}^1 x√(1-x²) dx. 被积是奇函数, 对称区间: = 0.

知识点 6: 直角坐标三重积分

把空间区域 V 投影到 xy 平面得 D_(xy), V 由 z₁(x,y) ≤ z ≤ z₂(x,y) 定义: ∭_V f(x,y,z) dV = ∬_(D_(xy)) [∫_(z₁)^(z₂) f dz] dσ

口诀: 三层积分, 由内到外算. 最内层把外两个当常数.

例题 3: ∭_V z dV, V 为 0 ≤ x ≤ 1, 0 ≤ y ≤ 1, 0 ≤ z ≤ 2 (单位长方体). = ∫_0¹ ∫_0¹ ∫_0² z dz dy dx = ∫_0¹ ∫_0¹ 2 dy dx = 2

例题 3-2 (切片法): ∭_V z dV, V = 0≤z≤1, 截面是边长为 z 的正方形 (顶点 (0,0,0) 的金字塔形). 切片法: 用 z 平面切, 每片是 z×z 的正方形, 面积 = z². V_z = ∬_(D_z) z dxdy = z · z² = z³. ∭ = ∫_0¹ z³ dz = 1/4.

两种三重积分思路:

  • 投影法 (先一后二): 投到 xy 平面, 内层对 z 积, 外层对 D_xy 积
  • 切片法 (先二后一): 用平行于 xy 的平面切, 每片做二重积分, 外层对 z 积

两种都对, 看哪个更顺手.

知识点 6.5: 三重积分的"先一后二"与"先二后一" 思想 (对齐书 §10.3)

两种切法:

先一后二 (投影法): ∭_V f dV = ∬_(D_xy) [∫_(z₁(x,y))^(z₂(x,y)) f dz] dxdy

先把 z 当一元变量积 (x, y 当常数), 内层得到一个仅是 (x,y) 的函数, 然后对 (x,y) 在投影区域 D_xy 上二重积分.

先二后一 (切片法): ∭_V f dV = ∫_c^d [∬_(D_z) f(x,y,z) dxdy] dz

先固定 z 切一片 (z = c 平面与 V 的截面 D_z), 算这片上的二重积分得到 A(z), 然后对 z 在 [c, d] 上一元积分.

选哪个?

  • f 跟 z 有简单关系, 投影区域复杂 → 先一后二
  • f 是 z 的简单函数, 切片形状好算 → 先二后一

例题 3-2 (切片法): ∭_V z dV, V 是 z=x²+y² 跟 z=4 围的区域. 切片 z 固定, D_z 是 x²+y² ≤ z (半径 √z 的圆), 面积 = πz. ∭ = ∫_0⁴ z · πz dz = π · z³/3 |_0⁴ = 64π/3.

知识点 6.7: 三重积分球坐标用法 (对齐书 §10.3 三 进一步)

球坐标 (r, φ, θ) 适合球对称区域:

  • r ≥ 0: 到原点距离
  • 0 ≤ φ ≤ π: 跟 z 轴正半轴的夹角 (天顶角)
  • 0 ≤ θ ≤ 2π: 在 xy 平面投影跟 x 轴的夹角 (方位角)

坐标转换: x = r sinφ cosθ y = r sinφ sinθ z = r cosφ

Jacobian = r² sinφ

例题 6-4 (球体体积): x²+y²+z² ≤ R². V = ∭_球 1 dV = ∫_0^(2π) ∫_0^π ∫_0^R r² sinφ dr dφ dθ = 2π · 2 · R³/3 = (4/3) π R³ ✓

例题 6-5 (球体质量 + 密度随 r 变): ρ = r (中心 0 边缘大). M = ∭ r · r² sinφ dr dφ dθ = ∫_0^(2π) ∫_0^π sinφ dφ dθ · ∫_0^R r³ dr = 2π · 2 · R⁴/4 = π R⁴.

知识点 7: 柱坐标

适合轴对称 (圆柱 / 圆锥) 体. 用 (r, θ, z), 其中 x = rcosθ, y = rsinθ, z = z.

变换: ∭ f(x,y,z) dV = ∭ f(...) · r dr dθ dz

(多一个 r 因子, 跟极坐标二重一样)

例题 7-2: 求圆柱 x²+y² ≤ 1, 0 ≤ z ≤ 2 的体积. 柱坐标: 0 ≤ r ≤ 1, 0 ≤ θ ≤ 2π, 0 ≤ z ≤ 2. V = ∫₀^(2π) ∫_0¹ ∫_0² r dz dr dθ = 2π · (1/2) · 2 = 2π ✓ (πR²h = π·1·2)

例题 7-3: 求圆锥 z = √(x²+y²) 跟 z = 1 围的体积. 柱坐标: z 从 r 到 1; r 从 0 到 1; θ 从 0 到 2π. V = ∫₀^(2π) ∫_0¹ ∫_r¹ r dz dr dθ = 2π ∫_0¹ r(1-r) dr = 2π · (1/2 - 1/3) = π/3.

知识点 7.5: 二重积分换元法 (一般化) (对齐书 §10.2 三)

极坐标只是换元的特例. 一般情况:

x = x(u, v), y = y(u, v), 是从 (u, v) 平面到 (x, y) 平面的变换. ∬_D f(x,y) dxdy = ∬_(D') f(x(u,v), y(u,v)) · |J| du dv

其中 Jacobian J = ∂(x,y)/∂(u,v) = | ∂x/∂u ∂x/∂v ∂y/∂u ∂y/∂v |

生活类比: 你把方格地图扭曲变形 (比如鱼眼镜头), 每个小方格面积变了 — |J| 就是"变形比".

极坐标特例: x = r cosθ, y = r sinθ. J = | cosθ -r sinθ sinθ r cosθ | = r(cos²θ + sin²θ) = r. ✓ (跟之前的 r 因子一致)

知识点 8: 球坐标

适合球对称 (球 / 半球) 体. 用 (r, φ, θ):

  • r = 到原点距离
  • φ = 跟 z 轴正方向夹角 (0 到 π)
  • θ = 在 xy 平面投影跟 x 轴夹角 (0 到 2π)

坐标转换:

  • x = rsinφcosθ
  • y = rsinφsinθ
  • z = rcosφ

变换: ∭ f dV = ∭ f(...) · r² sinφ dr dφ dθ

(因子是 r² sinφ)

例题 4: 球体 x²+y²+z² ≤ R² 的体积 (用球坐标). V = ∫₀^(2π) ∫₀^π ∫₀^R r² sinφ dr dφ dθ = (R³)/(3) · 2 · 2π = (4)/(3)π R³ 经典!

---

知识点 8.7: 球坐标 vs 柱坐标的选择诀窍

核心: 看积分区域跟被积函数.

球坐标好用:

  • 区域含 √(x²+y²+z²) 这种球对称
  • 被积函数含 (x²+y²+z²)^n
  • 例: 求球或半球体积, 球内积分

柱坐标好用:

  • 区域含 √(x²+y²) (圆柱对称, 但 z 方向独立)
  • 被积函数含 (x²+y²)^n + z 项
  • 例: 圆柱体, 圆锥体, 旋转体

例题 8-6 (柱 vs 球 选择): ∭_V (x²+y²) dV, V 是球 x²+y²+z² ≤ R². 球坐标试: x²+y² = r² sin²φ. ∫_0^(2π) ∫_0^π ∫_0^R r² sin²φ · r² sinφ dr dφ dθ = 2π · (R^5/5) · ∫_0^π sin³φ dφ = 2π · R^5/5 · 4/3 = 8πR^5/15. ✓

(柱坐标也能算但要分段, 球坐标更直接)

10.4 重积分的应用

知识点 8.5: 三重积分使用极/柱/球坐标的"判别口诀"

直角坐标: 区域是长方体, 简单平面边界. 柱坐标 (r, θ, z): 区域绕 z 轴对称 (圆柱/圆锥/旋转体). 球坐标 (r, φ, θ): 区域球对称 (球/球壳/半球).

关键判别:

  • 边界含 x²+y² → 柱
  • 边界含 x²+y²+z² → 球
  • 边界含 z = √(x²+y²) (圆锥) → 柱 或 球
  • 没特殊对称 → 直角

例题 8-3 (柱坐标): ∭_V z dV, V 是圆柱 x²+y²≤4, 0≤z≤3. 柱坐标: 0≤r≤2, 0≤θ≤2π, 0≤z≤3. = ∫_0^(2π)∫_0²∫_0³ z·r dz dr dθ = 2π · 2 · (9/2) = 18π.

知识点 9: 求体积

z = f(x,y) ≥ 0 在 D 上的体积: V = ∬_D f(x,y) dσ

知识点 10: 求质量

平面薄板密度 μ(x,y), 质量: m = ∬_D μ(x,y) dσ 空间体密度 μ(x,y,z), 质量: m = ∭_V μ dV

知识点 11: 求重心

平面薄板重心: x̄ = (∬ xμ dσ)/(∬ μ dσ), ȳ = (∬ yμ dσ)/(∬ μ dσ)

知识点 12: 求转动惯量

绕 x 轴: I_x = ∬ y² μ dσ 绕 y 轴: I_y = ∬ x² μ dσ 绕原点: I_O = ∬ (x²+y²) μ dσ

(距离² × 质量微元 是核心思想)

例题 12: 均匀薄圆盘 半径 R, 面密度 ρ, 求绕中心 (z 轴) 的转动惯量. I = ∬ (x²+y²) ρ dσ. 极坐标: = ∫₀^(2π) ∫_0^R r² · ρ · r drdθ = 2πρ · R⁴/4 = (πρR⁴)/2. 质量 M = πR²ρ. 所以 I = M R²/2. (这就是工科常用公式: 圆盘绕中心转动惯量 = (1/2) M R².)

知识点 12.5: 重积分应用的几何例题

例题 12-1: 用二重积分求 y=x² 和 y=x 围的区域面积. 求交点: x²=x → x=0, 1. 区域 0≤x≤1, x²≤y≤x. S = ∬_D 1 dxdy = ∫_0¹∫_(x²)^x dy dx = ∫_0¹ (x-x²) dx = 1/6.

例题 12-2 (旋转体): 求 y=√x, y=x² 围的区域绕 x 轴旋转的体积. V = ∫_0¹ π(y_上²-y_下²) dx = π∫_0¹ (x-x⁴) dx = π·(1/2-1/5) = 3π/10.

例题 12-3 (重心): 半圆 x²+y²≤R², y≥0, 求重心 ȳ. 对称: x̄=0. M = πR²/2. ∬ y dA: 极坐标 = ∫_0^π∫_0^R r sinθ·r dr dθ = (R³/3)·2 = 2R³/3. ȳ = (2R³/3)/(πR²/2) = 4R/(3π).

知识点 13: 引力计算 (重积分应用)

质量 M 均匀分布在体 V 上, 它对体外一点 P 的引力 = 体内每个小质量元 ρ dV 对 P 引力的矢量和.

|F→| = G ∭_V (ρ dV)/r² (方向沿连线)

例题 13: 均匀球壳对外部一点 P 的引力 = ? (牛顿"壳定理") 结论: 球壳对外的引力 = 把全部质量集中到球心后的引力. (这就是为啥地球表面附近, 我们近似把地球当成球心质点算重力)

知识点 13.5: 重积分的概率应用 (现代加值)

联合概率密度 f(x,y) 满足 ∬_(R²) f dxdy = 1 (整体概率 1).

事件概率: P((X,Y) ∈ D) = ∬_D f(x,y) dxdy 边缘分布: f_X(x) = ∫_{-∞}^∞ f(x,y) dy 期望值: E[g(X,Y)] = ∬ g·f dxdy 协方差: Cov(X,Y) = E[XY] - E[X]·E[Y] = ∬ xy·f dxdy - μ_X·μ_Y

例题 14-3 (二维正态分布积分): ∬_(R²) (1/(2π)) e^(-(x²+y²)/2) dxdy = 1 (验证归一化) 极坐标: = (1/(2π)) ∫_0^(2π) ∫_0^∞ e^(-r²/2) · r drdθ = (1/(2π)) · 2π · 1 = 1. ✓

AI 应用: 贝叶斯网络 / 概率图模型 / 高斯混合模型 (GMM) — 都涉及多维概率分布的积分.

知识点 14: 曲面面积公式

曲面 z = f(x,y) 在区域 D 上的面积: S = ∬_D √(1 + f_x² + f_y²) dxdy

例题 14: 球面 x²+y²+z² = R² 在 z ≥ 0 部分的面积. z = √(R²-x²-y²), z_x = -x/z, z_y = -y/z. 1+z_x²+z_y² = 1 + (x²+y²)/z² = (x²+y²+z²)/z² = R²/(R²-x²-y²). √= R/√(R²-x²-y²). S = ∬_(D: x²+y²≤R²) R/√(R²-x²-y²) dxdy 极坐标: = ∫₀^(2π) ∫_0^R R r/√(R²-r²) drdθ = 2πR · [-√(R²-r²)]_0^R = 2πR · R = 2πR². ✓ (整个球表面 = 4πR², 上半球 = 一半 = 2πR² ✓ 验证)

知识点 13.5: 引力分量推导 (对齐书 §10.4 四)

问题: 一个均匀薄片 D, 面密度 μ, 对薄片外一点 P(x₀,y₀,z₀) 的引力 F→.

思想: 把薄片切成无穷小, 每小块 dA 看作质点, 质量 dm = μ dA. 它对 P 的引力大小 = G·m·dm/r², 方向沿连线. 矢量积分.

F_x = G·m ∬_D (x-x₀)/r³ · μ dA F_y = G·m ∬_D (y-y₀)/r³ · μ dA F_z = G·m ∬_D (z-z₀)/r³ · μ dA

其中 r = √((x-x₀)² + (y-y₀)² + (z-z₀)²).

物理应用:

  • 月球-地球引力 (近似当点质量, 但实际地球非球对称要用积分)
  • 行星轨道扰动 (太阳系多体问题)
  • 卫星轨道设计

知识点 13.7: 二重积分用极坐标的标准化

口诀: "圆形/扇形/环形 → 极坐标; 长方形/平行四边形 → 直角坐标".

实用判据: 看 D 的边界方程

  • 含 x²+y² → 极坐标更顺
  • 含 x², y² 分开 → 直角坐标更顺
  • 含 (x-a)² + (y-b)² → 圆 — 平移 + 极坐标

例题 2-5 (圆环带): D = {1 ≤ x²+y² ≤ 4} 上算 ∬ ln(x²+y²) dxdy. 极坐标: 1 ≤ r ≤ 2, 0 ≤ θ ≤ 2π. = ∫_0^(2π) ∫_1² ln(r²) · r drdθ = 2π · ∫_1² 2r ln(r) dr. 设 u = ln r, dv = 2r dr: = 2π · [r² ln r]_1² - 2π · ∫_1² r dr = 2π · 4 ln 2 - 2π · [r²/2]_1² = 8π ln 2 - 3π.

知识点 15: 含参变量的积分

问题: 一个积分里, 除了积分变量 x 还有"参数" t. 比如: I(t) = ∫_a^b f(x, t) dx

这个 I 是 t 的函数 — 参数 t 变, 积分值跟着变.

生活类比: 一片区域内"风强度" f(x, t), x 是位置, t 是时间. 不同时间 t 算"整片区域总风量" 就是 I(t).

关键性质:

  1. 连续性: f 连续 → I(t) 连续
  2. 可微性 (Leibniz 公式): 在合理条件下可以"积分号下求导":

I'(t) = ∫_a^b ∂f/∂t dx

例题 12-2: I(t) = ∫_0¹ ln(1+tx)/(1+x²) dx, 求 I'(t). I'(t) = ∫_0¹ x/((1+tx)(1+x²)) dx. 进一步用部分分式可解出. (繁琐略)

应用: 微分方程 / 变换积分顺序的桥梁 / Feynman 积分技巧 (沿参数微分让难积分变易).

知识点 16: 三重积分的物理应用大全

物理量在三维体 V 上的累加:

| 物理量 | 公式 | |---|---| | 体积 | V = ∭ 1 dV | | 质量 | M = ∭ ρ dV (ρ 是密度) | | 质心 (x方向) | x̄ = (∭ x·ρ dV)/M | | 绕 z 轴转动惯量 | I_z = ∭ (x²+y²) ρ dV | | 转动惯量 (一般轴) | I = ∭ d² ρ dV (d 是点到轴距离) | | 引力分量 (沿 x) | F_x = G m ∭ ρ(x-x₀)/r³ dV | | 电场分量 (沿 x) | E_x = (1/4πε₀) ∭ ρ(x-x₀)/r³ dV | | 概率 (某事件) | P = ∭ f dV (f 是联合密度) | | 总能量 (引力势) | U = -Gm ∭ ρ/r dV |

核心思想: "密度乘体积微元 dV 累计" — 这是三重积分的物理本质.

补充: 蒙特卡洛积分 + 高斯积分

知识点 13.5: 蒙特卡洛积分原理

问题: n 维积分用网格法复杂度 O(N^n), n>5 计算上不可行 — AI/物理常要几百维.

蒙特卡洛 (MC): 用随机采样代替网格.

要算 I = ∫_D f(x) dx, D 体积 V. 在 D 上均匀取 N 个独立样本 x₁,...,x_N: Î = V · (1/N) ∑ f(xᵢ)

收敛速度: 误差 ~ σ/√N, 与维数无关 — MC 战胜网格法的关键.

例题 13-5 (用 MC 算 π): 单位正方形 [0,1]² 撒 N 点, 落在 x²+y²≤1 内的比例 ≈ π/4. N=10⁶ 时, π ≈ 3.1416 ± 0.002.

重要性采样: 若 f 在某些区域取值大, 应多采那里. 引提议分布 q(x): I = ∫ [f/q] · q dx = E_q[f/q] Î = (1/N) ∑ f(xᵢ)/q(xᵢ), xᵢ ~ q

AI 关键应用:

  • 神经渲染 NeRF: 沿光线积分
  • 强化学习: 策略梯度 ∇J = E[∇log π · A]
  • 变分推断: ELBO 中的期望
  • LLM PPO/DPO: advantage estimation
  • 扩散模型: score matching 中的高维期望

知识点 13.6: 高斯积分

核心: ∫_(-∞)^∞ e^(-x²) dx = √π.

推导 (二维转极坐标): I² = ∫∫ e^(-(x²+y²)) dxdy = ∫_0^(2π) ∫_0^∞ e^(-r²) r dr dθ = 2π · (1/2) = π. ∴ I = √π.

推广: ∫_(-∞)^∞ e^(-ax²+bx) dx = √(π/a) · e^(b²/(4a))

n 维: ∫_R^n e^(-(1/2) x^T A x) dx = (2π)^(n/2) / √det(A), A 正定. 这是多元高斯归一化常数.

ML 应用:

  • 高斯过程: 协方差 Σ 正定, det(Σ) 在后验
  • 变分推断: 用高斯近似复杂后验
  • 扩散模型: 加噪 transition kernel 是高斯

知识点 13.7: Beta 函数 与 Gamma 函数

Gamma 函数: Γ(s) = ∫_0^∞ x^(s-1) e^(-x) dx, s > 0.

核心性质:

  • 阶乘推广: Γ(n+1) = n!, 因 Γ(s+1) = s·Γ(s).
  • Γ(1/2) = √π (用高斯积分 + 换元).
  • 解析延拓后定义在除 0,-1,-2,... 外整个复平面.

Beta 函数: B(p, q) = ∫_0^1 x^(p-1) (1-x)^(q-1) dx, p,q > 0.

桥: B(p, q) = Γ(p)Γ(q) / Γ(p+q).

推导草图: Γ(p)Γ(q) 用二元积分换元 (x=tu, y=t(1-u), t=x+y) 转 Beta.

例题 13-7: 求 ∫_0^(π/2) sin^4(θ) dθ. 换元 x = sin²(θ), 化为 B(5/2, 1/2)/2. = (1/2)·Γ(5/2)Γ(1/2)/Γ(3) = (1/2)·(3/4·√π)·√π/2 = 3π/16.

应用:

  • 统计: 卡方分布 χ²_k ~ Γ(k/2, 2)
  • 贝叶斯: Beta 是二项分布的共轭先验 (后验仍 Beta)
  • 物理: 第一类 Stirling 公式 Γ(n) ~ √(2π) n^(n-1/2) e^(-n)
  • 推荐系统: Thompson 采样用 Beta 分布建模点击率不确定性

知识点 13.8: Stirling 公式 — n! 的渐近估计

Stirling: n! ~ √(2π n) · (n/e)^n (n → ∞)

精确版: n! = √(2π n) · (n/e)^n · (1 + 1/(12n) + 1/(288n²) - ...)

例题 13-8: 估计 100!. ln(100!) ≈ (1/2)ln(200π) + 100·ln(100) - 100 ≈ 363.7 所以 100! ≈ 10^158.

应用:

  • 统计力学: 玻尔兹曼熵 S = k_B ln(N!/...) 化简
  • 信息论: 大数定律 + 信息量
  • 复杂度理论: 阶乘增长的算法 (TSP 暴力 = O(n!))
  • 香农定理: 编码界限

知识点 13.9: 多元积分换元雅可比详细推导

问题: 重积分换元 ∬ f(x,y) dxdy → ∬ f(x(u,v), y(u,v)) |J| du dv 中, |J| 从哪里来?

几何推导: 在 (u,v) 平面取小矩形 [u, u+du]×[v, v+dv]. 映射到 (x,y) 平面变成"准平行四边形":

  • 边 1: r→_u = (∂x/∂u, ∂y/∂u) du
  • 边 2: r→_v = (∂x/∂v, ∂y/∂v) dv

面积 = |r→_u × r→_v| = |∂x/∂u · ∂y/∂v - ∂x/∂v · ∂y/∂u| · du dv = |J| du dv.

即雅可比 = 局部"面积放大比".

例题 13-9 (极坐标): x = r cos θ, y = r sin θ. J = |∂x/∂r ∂x/∂θ| = |cos θ -r sin θ| = r cos²θ + r sin²θ = r. |∂y/∂r ∂y/∂θ| |sin θ r cos θ|

所以 dxdy = r dr dθ. 这就是极坐标多出 r 的原因.

例题 13-10 (球坐标): x = r sin φ cos θ, y = r sin φ sin θ, z = r cos φ. J = r² sin φ. dxdydz = r² sin φ dr dφ dθ.

知识点 13.10: 概率密度变换 (机器学习的核心)

设随机变量 X 密度 p_X(x), 单调变换 Y = g(X). 则 Y 的密度: p_Y(y) = p_X(g⁻¹(y)) · |dg⁻¹(y)/dy| = p_X(x) / |g'(x)|

多元: p_Y(y) = p_X(x) / |det J|, 其中 J = ∂y/∂x.

应用:

  • 标准化流 (Normalizing Flows): 用一系列可逆变换 + 雅可比把简单分布变复杂
  • 变分自编码器 VAE: 重参数化 z = μ+σ·ε, 用雅可比传梯度
  • GAN: 生成器 G 把 N(0,I) 变成数据分布
  • 扩散模型: 加噪逆过程的概率密度变换

例题 13-11 (Box-Muller): 从均匀分布生成高斯随机数. U₁, U₂ ~ Uniform(0,1). Z₁ = √(-2 ln U₁) cos(2π U₂), Z₂ = √(-2 ln U₁) sin(2π U₂). 计算雅可比可证 Z₁, Z₂ ~ N(0,1) 独立. 这是计算机生成正态随机数的经典方法.

知识点 13.11: 含参积分 (Feynman 求导技巧)

问题: 直接积分困难时, 引入参数将积分化为参数函数, 再对参数求导.

Leibniz 求导规则: d/dα [∫_a^b f(x, α) dx] = ∫_a^b ∂f/∂α dx (积分限不依赖α)

例题 13-12: 求 I = ∫_0^∞ (sin x)/x dx. 引入 I(α) = ∫_0^∞ e^(-αx) (sin x)/x dx, α≥0. ∂I/∂α = -∫_0^∞ e^(-αx) sin x dx = -1/(α²+1) I(α) = -arctan α + C. α→∞: I→0, 所以 C = π/2. I(0) = π/2. 狄利克雷积分 ✓

Feynman 技巧威力: 物理学家 Feynman 这套方法被誉为"积分黑魔法". 把一个难积分通过参数化变成 ODE 问题.

应用:

  • 物理: 路径积分计算
  • 信号: 滤波器频响推导
  • 统计: 矩生成函数求矩 E[X^n] = M^(n)(0)
  • 高维 ML: backprop 链式法则本质是含参求导

知识点 13.12: 反常重积分 与 收敛性判别

反常 (improper) 重积分: 积分区域无界或被积函数无界.

两种类型:

  1. 区域无界: ∬_(R²) f dA = lim_(D→R²) ∬_D f dA
  2. 函数无界: ∬_(D\{P}) f dA, P 是奇点

收敛判别 (二维):

  • 在原点 r→0: 若 |f| ~ 1/r^α, α < 2 则收敛 (用极坐标 r dr dθ)
  • 在无穷 r→∞: 若 |f| ~ 1/r^α, α > 2 则收敛

例题 13-13: ∬_(R²) e^(-(x²+y²)) dA = π (前面高斯积分已证).

  • 区域无界但 e^(-r²) 在 r→∞ 急剧衰减 ⟹ 收敛.

知识点 13.13: 多重积分的 Fubini 定理详解

Fubini 定理: 若 f 在矩形 [a,b]×[c,d] 上 Riemann 可积, 则: ∬ f dxdy = ∫_a^b [∫_c^d f(x,y) dy] dx = ∫_c^d [∫_a^b f(x,y) dx] dy

可以交换积分次序.

反例 (无界函数, Fubini 失效): f(x,y) = (x-y)/(x+y)³ on (0,1]². ∫_0^1 [∫_0^1 f dy] dx = 1/2. ∫_0^1 [∫_0^1 f dx] dy = -1/2.

两边不等! 因为 |f| 在原点附近不可积 (∬|f| dxdy = ∞).

Tonelli 定理 (Fubini 的非负版本): 若 f ≥ 0 且可测, 则两种顺序积分总相等 (允许 ∞).

应用:

  • 概率论: 期望的累积形式 E[X] = ∫ P(X>t) dt (Layer Cake)
  • 偏微分方程: 弱解构造常用 Fubini
  • 信号: 卷积 ∫f(x-y)g(y)dy 频域分析
  • ML: 多变量损失函数边缘化

知识点 13.14: 雅可比与微分形式的等价

回顾: 重积分换元 dxdy = |J| du dv.

微分形式视角: dx ∧ dy = (∂x/∂u du + ∂x/∂v dv) ∧ (∂y/∂u du + ∂y/∂v dv) = (∂x/∂u · ∂y/∂v) du∧dv + (∂x/∂v · ∂y/∂u) dv∧du = (∂x/∂u · ∂y/∂v - ∂x/∂v · ∂y/∂u) du∧dv = det J · du∧dv

外积反对称 dv∧du = -du∧dv 自动产生雅可比行列式. 这是微分形式优雅之处.

知识点 13.15: 立方体的"双重广义体积"

n 维球体体积: V_n(R) = π^(n/2) R^n / Γ(n/2 + 1)

例:

  • n=1: 2R
  • n=2: πR²
  • n=3: (4/3)πR³
  • n=4: (1/2)π² R⁴
  • n=10: π^5 R^10 / 120

惊人现象: V_n(1) 在 n=5 达到峰值约 5.26, 然后下降, n→∞ 时 → 0. 高维单位球"几乎是空的".

应用:

  • "维数诅咒" (Curse of Dimensionality): 高维数据点几乎全在边界, ML 算法面对挑战
  • 蒙特卡洛积分: 高维球内采样效率随 n 指数下降
  • KNN 算法: 高维下"近邻"概念失效
