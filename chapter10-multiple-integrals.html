第十章 重积分

对照: 同济大学《高等数学》第七版下册 第十章
一元定积分 ∫ f(x) dx → 推广到二/三重积分.

10.0 整章在干啥?

一元定积分 ∫_a^b f(x) dx 几何意义 = 曲线 y=f(x) 跟 x 轴在 [a,b] 之间围的面积.

二重积分 ∬_D f(x,y) dxdy 几何意义 = 曲面 z=f(x,y) 在区域 D 上方围的体积.

三重积分 ∭_V f(x,y,z) dxdydz = 在三维区域 V 内的质量/总量 (密度 f 不均匀分布时的总质量).

对你的意义: 物理(质量/重心/转动惯量)、电磁(电场总通量)、概率(联合分布积分求概率)、图形(体积渲染) 全用. ML 里贝叶斯统计的概率密度积分也是这.

10.1 二重积分概念与性质

知识点 0.5: 为啥从"一元定积分" 升级到"二/三重积分"?

升级动机

中学/上册 $\int_a^b f(x)\,dx$ 只能算"线段上的面积/总和". 现实复杂场景:

  • 一片田地总产量 (面积上累加密度)
  • 一个房间总气体质量 (体积上累加密度)
  • 一片湖的总水量 / 三维物体重心 / 联合概率

需要"二/三维上的积分" → 重积分.

统一核心思想: 切块 → 求和 → 取极限. 块从"线段"升级到"小方块/小立方体".

这章学:

  • 二重积分: 算曲面下的体积 / 平面密度积分
  • 三重积分: 算空间区域的"总量"
  • 换元 (极坐标/柱坐标/球坐标): 不规则形状变规则

知识点 1: 定义 (思想)

二重积分

把区域 D 切成 n 小块 $\Delta\sigma_i$, 每块挑点 $(\xi_i, \eta_i)$, 求和取极限: $$\iint_D f(x,y)\,d\sigma = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i, \eta_i)\,\Delta\sigma_i$$ 其中 $\lambda$ 是最大小块直径.

生活类比: 一片田地按高低不平 (每点产量不同 f(x,y)), 算总产量 = 切成无数小方块, 每块产量 × 块面积, 加起来.

知识点 2: 性质 (跟一元积分类似)

二重积分基本性质

  1. 线性: $\iint (af + bg)\,d\sigma = a\iint f\,d\sigma + b\iint g\,d\sigma$
  2. 区域可加: 若 $D = D_1 \cup D_2$ ($D_1 \cap D_2$ 是边界), 则 $\iint_D f = \iint_{D_1} f + \iint_{D_2} f$
  3. 比较: $f \le g$ on $D$ ⟹ $\iint_D f \le \iint_D g$
  4. 估值: $m \cdot \sigma \le \iint_D f\,d\sigma \le M \cdot \sigma$ ($m, M$ 是 $f$ 在 $D$ 上的最小/最大值, $\sigma$ 是 $D$ 面积)
  5. 中值定理: 存在 $(\xi, \eta) \in D$ 使 $\iint_D f\,d\sigma = f(\xi, \eta) \cdot \sigma$

中值定理 = "平均值 × 面积"

$f(\xi, \eta) = \iint_D f\,d\sigma / \sigma$ 就是 $f$ 在 $D$ 上的平均值. 跟一元中值"平均值 × 长度 = 积分"完全同构.

例题 0-1

(估值定理应用): D = {0 ≤ x ≤ 1, 0 ≤ y ≤ 1}, 估 ∬_D e^(x²+y²) dσ 的范围. 在 D 上: e⁰ ≤ e^(x²+y²) ≤ e² 即 1 ≤ ... ≤ e². σ(D) = 1. 所以 1 ≤ ∬ ≤ e² ≈ 7.39.

10.2 二重积分的计算

知识点 3: 直角坐标 — X 型区域

X 型区域积分

设 $D = \{(x,y) : a \le x \le b, \varphi_1(x) \le y \le \varphi_2(x)\}$ (上下边界是 x 的函数), 则: $$\iint_D f(x,y)\,d\sigma = \int_a^b \left[\int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y)\,dy\right] dx$$ 做法: 先内层 (对 $y$ 积, 把 $x$ 当常数), 再外层 (对 $x$ 积).

做法: 先内层后外层. 内层把 x 当常数对 y 积; 外层再对 x 积.

例题 1

∬_D xy dσ, D 为 0 ≤ x ≤ 1, 0 ≤ y ≤ x (三角形). = ∫_0¹ ∫₀^x xy dy dx = ∫_0¹ [(xy²)/(2)]₀^x dx = ∫_0¹ (x³)/(2) dx = (1)/(8)

例题 1-2

∬_D (x²+y²) dσ, D 为单位正方形 0$\le$x$\le$1, 0$\le$y$\le$1. = ∫_0¹$\int$_0¹ (x²+y²) dy dx = ∫_0¹ (x² + 1/3) dx = 1/3 + 1/3 = 2/3.

例题 1-3

∬_D x dσ, D 为 y=x, y=x² 围的区域 (0$\le$x$\le$1). 描述: x² ≤ y ≤ x. = ∫_0¹ ∫_(x²)^x x dy dx = ∫_0¹ x(x - x²) dx = ∫_0¹ (x² - x³) dx = 1/3 - 1/4 = 1/12.

知识点 4: 直角坐标 — Y 型区域

Y 型区域积分

设 $D = \{(x,y) : c \le y \le d, \psi_1(y) \le x \le \psi_2(y)\}$ (左右边界是 y 的函数), 则: $$\iint_D f(x,y)\,d\sigma = \int_c^d \left[\int_{\psi_1(y)}^{\psi_2(y)} f(x,y)\,dx\right] dy$$

X 型 vs Y 型口诀

谁的范围是常数, 谁就是外层. 外层积成 1 个数, 内层先把外层变量当常数算成函数, 再外层数值积分.

知识点 4.5: 交换积分次序 (经典套路)

换次序套路

有时一个次序难算, 换次序就秒. 步骤:

  1. 画区域 D 图 (重要!)
  2. 把"先 y 后 x" 重新描述 为"先 x 后 y" (或反之)
  3. 重写积分上下限并重算

上下限 vs 区域不要混

换次序时积分上下限要重新画图确定, 不能简单交换变量字母. 区域是 $0 \le x \le 1, x \le y \le 1$ (X 型), 换 Y 型变成 $0 \le y \le 1, 0 \le x \le y$ — 上下限完全不一样.

例题 1-4

计算 ∫_0¹ ∫_x^1 sin(y²) dy dx. 问题: 内层 ∫ sin(y²) dy 没有初等原函数, 这个次序算不出来! 画区域: 0 ≤ x ≤ 1, x ≤ y ≤ 1 — 即三角形 (0,0), (1,1), (0,1). 换次序: 0 ≤ y ≤ 1, 0 ≤ x ≤ y. = ∫_0¹ ∫_0^y sin(y²) dx dy = ∫_0¹ y sin(y²) dy 令 u = y², du = 2y dy: = ∫_0¹ sin u · (du/2) = (1 - cos 1)/2.

为啥换? 内层变成 ∫_0^y sin(y²) dx 这个里 y 是常数, 直接 = y · sin(y²), 然后外层 ∫ y sin(y²) dy 可以用代换法算.

知识点 5: 极坐标变换

极坐标换元公式

D 复杂 (圆 / 扇形) 时, 换极坐标 $x = r\cos\theta$, $y = r\sin\theta$: $$\iint_D f(x,y)\,dxdy = \iint_{D'} f(r\cos\theta, r\sin\theta) \cdot r\,drd\theta$$ 关键: 面积微元 $dA = r\,drd\theta$, 多一个 $r$ 因子.

变换公式: $$\iint_D f(x,y)\,dxdy = \iint_D f(r\cos\theta, r\sin\theta) \cdot r\,drd\theta$$

关键: 多一个 r 因子! 几何上极坐标小块面积 = 半径 × 角度微元 × dr.

知识点 4.7: 极坐标下的常见区域 (对齐书 §10.2 二 进一步)

极坐标 6 大常见区域

| 区域 | 极坐标描述 | 形状 | |---|---|---| | 圆盘 | $0 \le r \le R, 0 \le \theta \le 2\pi$ | 实心圆 | | 扇形 | $0 \le r \le R, \alpha \le \theta \le \beta$ | 扇面 | | 圆环 | $a \le r \le b, 0 \le \theta \le 2\pi$ | 甜甜圈截面 | | 半圆 | $0 \le r \le R, 0 \le \theta \le \pi$ | 上半圆 | | 心脏线 | $r = a(1+\cos\theta)$ | 心形/玫瑰花 | | 双纽线 | $r^2 = a^2\cos 2\theta$ | 8 字形 / 无穷符号 |

选区域口诀: 圆/扇/环 → 极坐标基本; 心脏/双纽 → 极坐标直接出方程, 直角坐标几乎写不出.

例题 2

(再回顾): ∬_D e^(-(x²+y²)) dσ, D 是单位圆.

例题 2

∬_D e^(-(x²+y²)) dσ, D 是单位圆. 极坐标: 0 ≤ r ≤ 1, 0 ≤ $\theta$ ≤ 2π. = ∫₀^(2π) ∫_0¹ e^(-r²) · r drdθ = 2π · [-(1)/(2) e^(-r²)]_0¹ = $\pi$(1 - e⁻¹)

例题 2-2

(经典: 高斯积分推广): ∬_(R²) e^(-(x²+y²)) dxdy = ? 全平面极坐标: 0 ≤ r < ∞, 0 ≤ $\theta$ ≤ 2π. = ∫₀^(2π) ∫_0^∞ e^(-r²) r drdθ = 2π · 1/2 = $\pi$. 重要推论: (∫_(-∞)^∞ e^(-x²) dx)² = $\pi$ → ∫_(-∞)^∞ e^(-x²) dx = √$\pi$. (这就是著名的高斯积分, 概率论正态分布归一化用的!)

例题 2-3

∬_D (x²+y²) dσ, D 为 1 ≤ x²+y² ≤ 4 (圆环). 极坐标: 1 ≤ r ≤ 2, 0 ≤ $\theta$ ≤ 2π. = ∫₀^(2π) ∫_1² r² · r drdθ = 2π · [r⁴/4]_1² = 2π · (4 - 1/4) = 15π/2.

例题 2-4

用二重积分求圆 x²+y² = R² 围的面积. S = ∬_D 1 dxdy. 极坐标: = ∫₀^(2π) ∫_0^R r drdθ = 2π · R²/2 = πR² ✓.

10.3 三重积分

三重积分

设 V 是空间闭区域, $f(x,y,z)$ 是 V 上的连续函数. 三重积分: $$\iiint_V f(x,y,z)\,dV = \lim_{\lambda\to 0} \sum_{i=1}^n f(\xi_i,\eta_i,\zeta_i)\,\Delta V_i$$ 物理意义: $f$ 是密度 ⟹ 总质量; $f=1$ ⟹ 区域体积.

知识点 5.5: 关于积分次序的"右手定则"

X 型 vs Y 型 选择

看 $D$ 的边界曲线"哪个变量能用单调函数描述":

  • 边界由 $y = \varphi(x)$ 描述 (上下边界) ⟹ X 型 (外层 $x$ 内层 $y$)
  • 边界由 $x = \psi(y)$ 描述 (左右边界) ⟹ Y 型 (外层 $y$ 内层 $x$)

复杂区域要拆分

不是 X 型也不是 Y 型 (有凸凹/边界不单调) → 用直线切成几块, 每块单独按 X/Y 型算. 强行硬上一种类型会算错.

例题 5-5

D 是单位圆 x²+y²$\le$1 上半部分. 算 ∬ x dxdy. X 型: 0 ≤ x ≤ ... 不对, x 范围跟 y 有关. 准确写法: -1 ≤ x ≤ 1, 0 ≤ y ≤ √(1-x²). Y 型: 0 ≤ y ≤ 1, -√(1-y²) ≤ x ≤ √(1-y²). 两种都行. 选 X 型: ∬ x dxdy = ∫_{-1}^1 ∫_0^√(1-x²) x dy dx = ∫_{-1}^1 x√(1-x²) dx. 被积是奇函数, 对称区间: = 0.

知识点 6: 直角坐标三重积分

投影法 (先一后二)

把 V 投影到 xy 平面得 $D_{xy}$, V 由 $z_1(x,y) \le z \le z_2(x,y)$ 定义: $$\iiint_V f(x,y,z)\,dV = \iint_{D_{xy}} \left[\int_{z_1}^{z_2} f\,dz\right] d\sigma$$

口诀: 三层积分, 由内到外算. 最内层把外两个当常数.

例题 3

∭_V z dV, V 为 0 ≤ x ≤ 1, 0 ≤ y ≤ 1, 0 ≤ z ≤ 2 (单位长方体). = ∫_0¹ ∫_0¹ ∫_0² z dz dy dx = ∫_0¹ ∫_0¹ 2 dy dx = 2

例题 3-2

(切片法): ∭_V z dV, V = 0$\le$z$\le$1, 截面是边长为 z 的正方形 (顶点 (0,0,0) 的金字塔形). 切片法: 用 z 平面切, 每片是 z×z 的正方形, 面积 = z². V_z = ∬_(D_z) z dxdy = z · z² = z³. ∭ = ∫_0¹ z³ dz = 1/4.

两种三重积分思路:

  • 投影法 (先一后二): 投到 xy 平面, 内层对 z 积, 外层对 D_xy 积
  • 切片法 (先二后一): 用平行于 xy 的平面切, 每片做二重积分, 外层对 z 积

两种都对, 看哪个更顺手.

知识点 6.5: 三重积分的"先一后二"与"先二后一" 思想 (对齐书 §10.3)

三重积分两种切法

先一后二 (投影法): $$\iiint_V f\,dV = \iint_{D_{xy}} \left[\int_{z_1(x,y)}^{z_2(x,y)} f\,dz\right] dxdy$$

先二后一 (切片法): $$\iiint_V f\,dV = \int_c^d \left[\iint_{D_z} f(x,y,z)\,dxdy\right] dz$$ ($D_z$ 是 $z$ 平面与 $V$ 的截面.)

怎么选?

  • $f$ 跟 $z$ 简单, 投影区域 $D_{xy}$ 复杂 → 先一后二
  • $f$ 是 $z$ 的简单函数, 切片 $D_z$ 好算 → 先二后一

经验: 旋转体 / 沿轴密度变化 → 切片; 复杂三维区域 → 投影.

例题 3-2

(切片法): ∭_V z dV, V 是 z=x²+y² 跟 z=4 围的区域. 切片 z 固定, D_z 是 x²+y² ≤ z (半径 √z 的圆), 面积 = πz. ∭ = ∫_0⁴ z · πz dz = $\pi$ · z³/3 |_0⁴ = 64π/3.

知识点 6.7: 三重积分球坐标用法 (对齐书 §10.3 三 进一步)

球坐标 (r, $\varphi$, $\theta$) 适合球对称区域:

  • r ≥ 0: 到原点距离
  • 0 ≤ $\varphi$ ≤ $\pi$: 跟 z 轴正半轴的夹角 (天顶角)
  • 0 ≤ $\theta$ ≤ 2π: 在 xy 平面投影跟 x 轴的夹角 (方位角)

坐标转换: x = r sinφ $\cos\theta$ y = r sinφ $\sin\theta$ z = r cosφ

Jacobian = r² sinφ

例题 6-4

(球体体积): x²+y²+z² ≤ R². V = ∭_球 1 dV = ∫_0^(2π) ∫_0^$\pi$ ∫_0^R r² sinφ dr dφ dθ = 2π · 2 · R³/3 = (4/3) $\pi$ R³ ✓

例题 6-5

(球体质量 + 密度随 r 变): ρ = r (中心 0 边缘大). M = ∭ r · r² sinφ dr dφ dθ = ∫_0^(2π) ∫_0^$\pi$ sinφ dφ dθ · ∫_0^R r³ dr = 2π · 2 · R⁴/4 = $\pi$ R⁴.

知识点 7: 柱坐标

柱坐标换元

适合轴对称 (圆柱 / 圆锥) 体. 用 $(r, \theta, z)$: $$x = r\cos\theta, \quad y = r\sin\theta, \quad z = z$$ Jacobian = $r$: $$\iiint_V f(x,y,z)\,dV = \iiint_{V'} f(r\cos\theta, r\sin\theta, z) \cdot r\,drd\theta dz$$

柱坐标 = 极坐标 + z 轴拉伸. 多 $r$ 因子跟二维极坐标同源 (面积微元 $dA = r\,drd\theta$, 再乘 $dz$ 得体积微元 $dV = r\,drd\theta dz$).

例题 7-2

求圆柱 x²+y² ≤ 1, 0 ≤ z ≤ 2 的体积. 柱坐标: 0 ≤ r ≤ 1, 0 ≤ $\theta$ ≤ 2π, 0 ≤ z ≤ 2. V = ∫₀^(2π) ∫_0¹ ∫_0² r dz dr dθ = 2π · (1/2) · 2 = 2π ✓ (πR²h = $\pi$·1·2)

例题 7-3

求圆锥 z = √(x²+y²) 跟 z = 1 围的体积. 柱坐标: z 从 r 到 1; r 从 0 到 1; $\theta$ 从 0 到 2π. V = ∫₀^(2π) ∫_0¹ ∫_r¹ r dz dr dθ = 2π ∫_0¹ r(1-r) dr = 2π · (1/2 - 1/3) = $\pi$/3.

知识点 7.5: 二重积分换元法 (一般化) (对齐书 §10.2 三)

二重积分换元公式

设变换 $T: (u,v) \to (x,y)$, $x=x(u,v), y=y(u,v)$ 在 $D'$ 上可逆 + 连续可微. Jacobian $$J = \frac{\partial(x,y)}{\partial(u,v)} = \det \begin{pmatrix} \partial x/\partial u & \partial x/\partial v \\ \partial y/\partial u & \partial y/\partial v \end{pmatrix}$$ 则 $$\iint_D f(x,y)\,dxdy = \iint_{D'} f(x(u,v), y(u,v)) \cdot |J|\,dudv$$

生活类比: 把方格地图扭曲变形 (鱼眼镜头), 每个小方格面积变了 — $|J|$ 是"变形比".

生活类比: 你把方格地图扭曲变形 (比如鱼眼镜头), 每个小方格面积变了 — |J| 就是"变形比".

极坐标特例: x = r $\cos\theta$, y = r $\sin\theta$. J = | $\cos\theta$ -r $\sin\theta$ $\sin\theta$ r $\cos\theta$ | = r(cos²θ + sin²θ) = r. ✓ (跟之前的 r 因子一致)

知识点 8: 球坐标

球坐标系

适合球对称 (球/半球) 体. 用 $(r, \varphi, \theta)$:

  • $r \ge 0$: 到原点距离
  • $0 \le \varphi \le \pi$: 跟 $z$ 轴正向夹角 (天顶角)
  • $0 \le \theta \le 2\pi$: $xy$ 平面投影跟 $x$ 轴夹角 (方位角)

坐标转换: $$x = r\sin\varphi \cos\theta, \quad y = r\sin\varphi \sin\theta, \quad z = r\cos\varphi$$

球坐标换元公式

$$\iiint_V f\,dV = \iiint_{V'} f(r\sin\varphi\cos\theta, r\sin\varphi\sin\theta, r\cos\varphi) \cdot r^2 \sin\varphi\,dr\,d\varphi\,d\theta$$ Jacobian = $r^2 \sin\varphi$.

例题 4

球体 x²+y²+z² ≤ R² 的体积 (用球坐标). V = ∫₀^(2π) ∫₀^$\pi$ ∫₀^R r² sinφ dr dφ dθ = (R³)/(3) · 2 · 2π = (4)/(3)$\pi$ R³ 经典!

知识点 8.7: 球坐标 vs 柱坐标的选择诀窍

核心: 看积分区域跟被积函数.

球坐标好用:

  • 区域含 √(x²+y²+z²) 这种球对称
  • 被积函数含 (x²+y²+z²)^n
  • 例: 求球或半球体积, 球内积分

柱坐标好用:

  • 区域含 √(x²+y²) (圆柱对称, 但 z 方向独立)
  • 被积函数含 (x²+y²)^n + z 项
  • 例: 圆柱体, 圆锥体, 旋转体

例题 8-6

(柱 vs 球 选择): ∭_V (x²+y²) dV, V 是球 x²+y²+z² ≤ R². 球坐标试: x²+y² = r² sin²φ. ∫_0^(2π) ∫_0^$\pi$ ∫_0^R r² sin²φ · r² sinφ dr dφ dθ = 2π · (R^5/5) · ∫_0^$\pi$ sin³φ dφ = 2π · R^5/5 · 4/3 = 8πR^5/15. ✓

(柱坐标也能算但要分段, 球坐标更直接)

10.4 重积分的应用

知识点 8.5: 三重积分使用极/柱/球坐标的"判别口诀"

直角坐标: 区域是长方体, 简单平面边界. 柱坐标 (r, $\theta$, z): 区域绕 z 轴对称 (圆柱/圆锥/旋转体). 球坐标 (r, $\varphi$, $\theta$): 区域球对称 (球/球壳/半球).

关键判别:

  • 边界含 x²+y² → 柱
  • 边界含 x²+y²+z² → 球
  • 边界含 z = √(x²+y²) (圆锥) → 柱 或 球
  • 没特殊对称 → 直角

例题 8-3

(柱坐标): ∭_V z dV, V 是圆柱 x²+y²$\le$4, 0$\le$z$\le$3. 柱坐标: 0$\le$r$\le$2, 0≤$\theta$≤2π, 0$\le$z$\le$3. = ∫_0^(2π)∫_0²$\int$_0³ z·r dz dr dθ = 2π · 2 · (9/2) = 18π.

知识点 9: 求体积

二重积分求体积

$z = f(x,y) \ge 0$ 在 $D$ 上的"曲顶柱体"体积: $$V = \iint_D f(x,y)\,d\sigma$$ $f$ 是曲面高度, 体积 = 曲面在 $D$ 上方与 $xy$ 平面之间的空间.

知识点 10: 求质量

质量公式

  • 平面薄板 (密度 $\mu(x,y)$): $m = \iint_D \mu\,d\sigma$
  • 空间体 (密度 $\mu(x,y,z)$): $m = \iiint_V \mu\,dV$

知识点 11: 求重心

平面薄板重心: x̄ = (∬ xμ dσ)/(∬ $\mu$ dσ), ȳ = (∬ yμ dσ)/(∬ $\mu$ dσ)

知识点 12: 求转动惯量

转动惯量 (Moment of Inertia)

平面薄板的转动惯量 (绕指定轴):

  • 绕 $x$ 轴: $I_x = \iint_D y^2 \mu\,d\sigma$
  • 绕 $y$ 轴: $I_y = \iint_D x^2 \mu\,d\sigma$
  • 绕原点 (极): $I_O = \iint_D (x^2 + y^2) \mu\,d\sigma = I_x + I_y$

核心思想: $I = \int (\text{到轴距离})^2 \cdot dm$.

转动惯量 = "转动难度"

质量分布越远离轴, $I$ 越大, 越难转动. 物理上 $L = I\omega$ (角动量), $\tau = I\alpha$ (扭矩驱动角加速度). 花样滑冰演员收腿 ($I$ 减小) 旋转加速.

例题 12

均匀薄圆盘 半径 R, 面密度 ρ, 求绕中心 (z 轴) 的转动惯量. I = ∬ (x²+y²) ρ dσ. 极坐标: = ∫₀^(2π) ∫_0^R r² · ρ · r drdθ = 2πρ · R⁴/4 = (πρR⁴)/2. 质量 M = πR²ρ. 所以 I = M R²/2. (这就是工科常用公式: 圆盘绕中心转动惯量 = (1/2) M R².)

知识点 12.5: 重积分应用的几何例题

例题 12-1

用二重积分求 y=x² 和 y=x 围的区域面积. 求交点: x²=x → x=0, 1. 区域 0$\le$x$\le$1, x²$\le$y$\le$x. S = ∬_D 1 dxdy = ∫_0¹$\int$_(x²)^x dy dx = ∫_0¹ (x-x²) dx = 1/6.

例题 12-2

(旋转体): 求 y=√x, y=x² 围的区域绕 x 轴旋转的体积. V = ∫_0¹ $\pi$(y_上²-y_下²) dx = $\pi$∫_0¹ (x-x⁴) dx = $\pi$·(1/2-1/5) = 3π/10.

例题 12-3

(重心): 半圆 x²+y²$\le$R², y$\ge$0, 求重心 ȳ. 对称: x̄=0. M = πR²/2. ∬ y dA: 极坐标 = ∫_0^$\pi$∫_0^R r $\sin\theta$·r dr dθ = (R³/3)·2 = 2R³/3. ȳ = (2R³/3)/(πR²/2) = 4R/(3π).

知识点 13: 引力计算 (重积分应用)

万有引力 (体积版)

质量 M 均匀分布在体 V, 对外一点 P 的引力大小: $$|\vec{F}| = G\iiint_V \frac{\rho\,dV}{r^2}$$ (方向沿质量元与 P 的连线)

牛顿壳定理

球对称质量分布对外部点的引力 = 把全部质量集中到球心后的引力. 这就是为啥地球表面附近, 我们近似把地球当成球心质点算重力. 牛顿当年证这条用了 20 多年.

例题 13

均匀球壳对外部一点 P 的引力 = ? (牛顿"壳定理") 结论: 球壳对外的引力 = 把全部质量集中到球心后的引力. (这就是为啥地球表面附近, 我们近似把地球当成球心质点算重力)

知识点 13.5: 重积分的概率应用 (现代加值)

二维联合概率密度

$f(x,y)$ 满足 $f \ge 0$ 且 $\iint_{\mathbb{R}^2} f\,dxdy = 1$ (归一化).

  • 事件概率: $P((X,Y) \in D) = \iint_D f(x,y)\,dxdy$
  • 边缘分布: $f_X(x) = \int_{-\infty}^\infty f(x,y)\,dy$
  • 期望: $E[g(X,Y)] = \iint g \cdot f\,dxdy$
  • 协方差: $\text{Cov}(X,Y) = \iint xy f\,dxdy - \mu_X \mu_Y$

概率 = 体积

联合密度 $f(x,y)$ 想象成 3D 曲面, $D$ 上的积分 = 曲面下方在 $D$ 上的体积, 数值 = 落在 $D$ 里的概率. 二维概率论本质上是二重积分.

例题 14-3

(二维正态分布积分): ∬_(R²) (1/(2π)) e^(-(x²+y²)/2) dxdy = 1 (验证归一化) 极坐标: = (1/(2π)) ∫_0^(2π) ∫_0^∞ e^(-r²/2) · r drdθ = (1/(2π)) · 2π · 1 = 1. ✓

AI 应用: 贝叶斯网络 / 概率图模型 / 高斯混合模型 (GMM) — 都涉及多维概率分布的积分.

知识点 14: 曲面面积公式

曲面 z = f(x,y) 在区域 D 上的面积: S = ∬_D √(1 + f_x² + f_y²) dxdy

例题 14

球面 x²+y²+z² = R² 在 z ≥ 0 部分的面积. z = √(R²-x²-y²), z_x = -x/z, z_y = -y/z. 1+z_x²+z_y² = 1 + (x²+y²)/z² = (x²+y²+z²)/z² = R²/(R²-x²-y²). √= R/√(R²-x²-y²). S = ∬_(D: x²+y²$\le$R²) R/√(R²-x²-y²) dxdy 极坐标: = ∫₀^(2π) ∫_0^R R r/√(R²-r²) drdθ = 2πR · [-√(R²-r²)]_0^R = 2πR · R = 2πR². ✓ (整个球表面 = 4πR², 上半球 = 一半 = 2πR² ✓ 验证)

知识点 13.5: 引力分量推导 (对齐书 §10.4 四)

问题: 一个均匀薄片 D, 面密度 $\mu$, 对薄片外一点 P(x₀,y₀,z₀) 的引力 $\vec{F}$.

思想: 把薄片切成无穷小, 每小块 dA 看作质点, 质量 dm = $\mu$ dA. 它对 P 的引力大小 = G·m·dm/r², 方向沿连线. 矢量积分.

F_x = G·m ∬_D (x-x₀)/r³ · $\mu$ dA F_y = G·m ∬_D (y-y₀)/r³ · $\mu$ dA F_z = G·m ∬_D (z-z₀)/r³ · $\mu$ dA

其中 r = √((x-x₀)² + (y-y₀)² + (z-z₀)²).

物理应用:

  • 月球-地球引力 (近似当点质量, 但实际地球非球对称要用积分)
  • 行星轨道扰动 (太阳系多体问题)
  • 卫星轨道设计

知识点 13.7: 二重积分用极坐标的标准化

口诀: "圆形/扇形/环形 → 极坐标; 长方形/平行四边形 → 直角坐标".

实用判据: 看 D 的边界方程

  • 含 x²+y² → 极坐标更顺
  • 含 x², y² 分开 → 直角坐标更顺
  • 含 (x-a)² + (y-b)² → 圆 — 平移 + 极坐标

例题 2-5

(圆环带): D = {1 ≤ x²+y² ≤ 4} 上算 ∬ ln(x²+y²) dxdy. 极坐标: 1 ≤ r ≤ 2, 0 ≤ $\theta$ ≤ 2π. = ∫_0^(2π) ∫_1² ln(r²) · r drdθ = 2π · ∫_1² 2r ln(r) dr. 设 u = ln r, dv = 2r dr: = 2π · [r² ln r]_1² - 2π · ∫_1² r dr = 2π · 4 ln 2 - 2π · [r²/2]_1² = 8π ln 2 - 3π.

知识点 15: 含参变量的积分

问题: 一个积分里, 除了积分变量 x 还有"参数" t. 比如: I(t) = ∫_a^b f(x, t) dx

这个 I 是 t 的函数 — 参数 t 变, 积分值跟着变.

生活类比: 一片区域内"风强度" f(x, t), x 是位置, t 是时间. 不同时间 t 算"整片区域总风量" 就是 I(t).

关键性质:

  1. 连续性: f 连续 → I(t) 连续
  2. 可微性 (Leibniz 公式): 在合理条件下可以"积分号下求导":

I'(t) = ∫_a^b ∂f/∂t dx

例题 12-2

I(t) = ∫_0¹ ln(1+tx)/(1+x²) dx, 求 I'(t). I'(t) = ∫_0¹ x/((1+tx)(1+x²)) dx. 进一步用部分分式可解出. (繁琐略)

应用: 微分方程 / 变换积分顺序的桥梁 / Feynman 积分技巧 (沿参数微分让难积分变易).

知识点 13.5: 蒙特卡洛积分原理

问题: n 维积分用网格法复杂度 O(N^n), n>5 计算上不可行 — AI/物理常要几百维.

蒙特卡洛 (MC): 用随机采样代替网格.

要算 I = ∫_D f(x) dx, D 体积 V. 在 D 上均匀取 N 个独立样本 x₁,...,x_N: Î = V · (1/N) ∑ f(xᵢ)

收敛速度: 误差 ~ σ/√N, 与维数无关 — MC 战胜网格法的关键.

例题 13-5

(用 MC 算 $\pi$): 单位正方形 [0,1]² 撒 N 点, 落在 x²+y²$\le$1 内的比例 ≈ $\pi$/4. N=10⁶ 时, $\pi$ ≈ 3.1416 ± 0.002.

重要性采样: 若 f 在某些区域取值大, 应多采那里. 引提议分布 q(x): I = ∫ [f/q] · q dx = E_q[f/q] Î = (1/N) ∑ f(xᵢ)/q(xᵢ), xᵢ ~ q

AI 关键应用:

  • 神经渲染 NeRF: 沿光线积分
  • 强化学习: 策略梯度 ∇J = E[∇log $\pi$ · A]
  • 变分推断: ELBO 中的期望
  • LLM PPO/DPO: advantage estimation
  • 扩散模型: score matching 中的高维期望

知识点 13.6: 高斯积分

高斯积分 (Gaussian Integral)

$$\int_{-\infty}^\infty e^{-x^2}\,dx = \sqrt{\pi}$$ 推导 (二维转极坐标): $$I^2 = \iint_{\mathbb{R}^2} e^{-(x^2+y^2)}\,dxdy = \int_0^{2\pi}\!\!\int_0^\infty e^{-r^2} r\,drd\theta = 2\pi \cdot \frac{1}{2} = \pi$$ ∴ $I = \sqrt{\pi}$.

一维推广: $\int_{-\infty}^\infty e^{-ax^2+bx}\,dx = \sqrt{\pi/a} \cdot e^{b^2/(4a)}$

n 维: $\int_{\mathbb{R}^n} e^{-\frac{1}{2} \vec{x}^T A \vec{x}}\,d\vec{x} = \frac{(2\pi)^{n/2}}{\sqrt{\det A}}$ ($A$ 正定).

一维做不出, 二维一招制胜

$e^{-x^2}$ 的一维不定积分没有初等表达! 但二维 $e^{-(x^2+y^2)} = e^{-r^2}$ 在极坐标下立刻可算. 把"算不了的一维"通过升维变成"易算的二维"是数学美技.

ML 应用:

  • 高斯过程: 协方差 Σ 正定, det(Σ) 在后验
  • 变分推断: 用高斯近似复杂后验
  • 扩散模型: 加噪 transition kernel 是高斯

知识点 13.7: Beta 函数 与 Gamma 函数

Gamma 函数

$$\Gamma(s) = \int_0^\infty x^{s-1} e^{-x}\,dx, \quad s > 0$$ 核心性质:

  • $\Gamma(s+1) = s \cdot \Gamma(s)$ (阶乘推广公式)
  • $\Gamma(n+1) = n!$ (整数情形)
  • $\Gamma(1/2) = \sqrt{\pi}$ (用高斯积分 + 换元 $u = \sqrt{x}$ 证)
  • 解析延拓: 定义于除 $0, -1, -2, \ldots$ 外整个复平面

Beta 函数

$$B(p, q) = \int_0^1 x^{p-1} (1-x)^{q-1}\,dx, \quad p, q > 0$$

桥: B(p, q) = Γ(p)Γ(q) / Γ(p+q).

推导草图: Γ(p)Γ(q) 用二元积分换元 (x=tu, y=t(1-u), t=x+y) 转 Beta.

例题 13-7

求 ∫_0^($\pi$/2) sin^4($\theta$) dθ. 换元 x = sin²($\theta$), 化为 B(5/2, 1/2)/2. = (1/2)·Γ(5/2)Γ(1/2)/Γ(3) = (1/2)·(3/4·√$\pi$)·√$\pi$/2 = 3π/16.

应用:

  • 统计: 卡方分布 χ²_k ~ Γ(k/2, 2)
  • 贝叶斯: Beta 是二项分布的共轭先验 (后验仍 Beta)
  • 物理: 第一类 Stirling 公式 Γ(n) ~ √(2π) n^(n-1/2) e^(-n)
  • 推荐系统: Thompson 采样用 Beta 分布建模点击率不确定性

知识点 13.8: Stirling 公式 — n! 的渐近估计

Stirling 公式

$$n! \sim \sqrt{2\pi n} \cdot \left(\frac{n}{e}\right)^n \quad (n \to \infty)$$ 精确版 (含修正项): $$n! = \sqrt{2\pi n} \left(\frac{n}{e}\right)^n \left(1 + \frac{1}{12n} + \frac{1}{288n^2} - \cdots\right)$$

Stirling 在统计/物理中的意义

$n!$ 增长极快 — 直接算 $100!$ 几乎不可能 ($\approx 10^{158}$). Stirling 把它变成乘幂形式, 取 $\ln$ 后 $\ln n! \approx n\ln n - n$, 统计力学 (玻尔兹曼熵) 和组合学 (排列计数) 处处用.

例题 13-8

估计 100!. ln(100!) ≈ (1/2)ln(200π) + 100·ln(100) - 100 ≈ 363.7 所以 100! ≈ 10^158.

应用:

  • 统计力学: 玻尔兹曼熵 S = k_B ln(N!/...) 化简
  • 信息论: 大数定律 + 信息量
  • 复杂度理论: 阶乘增长的算法 (TSP 暴力 = O(n!))
  • 香农定理: 编码界限

知识点 13.9: 多元积分换元雅可比详细推导

问题: 重积分换元 ∬ f(x,y) dxdy → ∬ f(x(u,v), y(u,v)) |J| du dv 中, |J| 从哪里来?

几何推导: 在 (u,v) 平面取小矩形 [u, u+du]×[v, v+dv]. 映射到 (x,y) 平面变成"准平行四边形":

  • 边 1: $\vec{r}$_u = (∂x/∂u, ∂y/∂u) du
  • 边 2: $\vec{r}$_v = (∂x/∂v, ∂y/∂v) dv

面积 = |$\vec{r}$_u × $\vec{r}$_v| = |∂x/∂u · ∂y/∂v - ∂x/∂v · ∂y/∂u| · du dv = |J| du dv.

即雅可比 = 局部"面积放大比".

例题 13-9

(极坐标): x = r $\cos\theta$, y = r $\sin\theta$. J = |∂x/∂r ∂x/∂$\theta$| = |$\cos\theta$ -r $\sin\theta$| = r cos²θ + r sin²θ = r. |∂y/∂r ∂y/∂$\theta$| |$\sin\theta$ r $\cos\theta$|

所以 dxdy = r dr dθ. 这就是极坐标多出 r 的原因.

例题 13-10

(球坐标): x = r sin $\varphi$ $\cos\theta$, y = r sin $\varphi$ $\sin\theta$, z = r cos $\varphi$. J = r² sin $\varphi$. dxdydz = r² sin $\varphi$ dr dφ dθ.

知识点 13.10: 概率密度变换 (机器学习的核心)

设随机变量 X 密度 p_X(x), 单调变换 Y = g(X). 则 Y 的密度: p_Y(y) = p_X(g⁻¹(y)) · |dg⁻¹(y)/dy| = p_X(x) / |g'(x)|

多元: p_Y(y) = p_X(x) / |det J|, 其中 J = ∂y/∂x.

应用:

  • 标准化流 (Normalizing Flows): 用一系列可逆变换 + 雅可比把简单分布变复杂
  • 变分自编码器 VAE: 重参数化 z = $\mu$+σ·ε, 用雅可比传梯度
  • GAN: 生成器 G 把 N(0,I) 变成数据分布
  • 扩散模型: 加噪逆过程的概率密度变换

例题 13-11

(Box-Muller): 从均匀分布生成高斯随机数. U₁, U₂ ~ Uniform(0,1). Z₁ = √(-2 ln U₁) cos(2π U₂), Z₂ = √(-2 ln U₁) sin(2π U₂). 计算雅可比可证 Z₁, Z₂ ~ N(0,1) 独立. 这是计算机生成正态随机数的经典方法.

知识点 13.11: 含参积分 (Feynman 求导技巧)

问题: 直接积分困难时, 引入参数将积分化为参数函数, 再对参数求导.

Leibniz 求导规则: d/dα [∫_a^b f(x, $\alpha$) dx] = ∫_a^b ∂f/∂$\alpha$ dx (积分限不依赖α)

例题 13-12

求 I = ∫_0^∞ (sin x)/x dx. 引入 I($\alpha$) = ∫_0^∞ e^(-αx) (sin x)/x dx, $\alpha$≥0. ∂I/∂$\alpha$ = -∫_0^∞ e^(-αx) sin x dx = -1/(α²+1) I($\alpha$) = -arctan $\alpha$ + C. $\alpha$→∞: $\vec{I}$0, 所以 C = $\pi$/2. I(0) = $\pi$/2. 狄利克雷积分 ✓

Feynman 技巧威力: 物理学家 Feynman 这套方法被誉为"积分黑魔法". 把一个难积分通过参数化变成 ODE 问题.

应用:

  • 物理: 路径积分计算
  • 信号: 滤波器频响推导
  • 统计: 矩生成函数求矩 E[X^n] = M^(n)(0)
  • 高维 ML: backprop 链式法则本质是含参求导

知识点 13.12: 反常重积分 与 收敛性判别

反常 (improper) 重积分: 积分区域无界或被积函数无界.

两种类型:

  1. 区域无界: ∬_(R²) f dA = lim_($\vec{D}$R²) ∬_D f dA
  2. 函数无界: ∬_(D\{P}) f dA, P 是奇点

收敛判别 (二维):

  • 在原点 $\vec{r}$0: 若 |f| ~ 1/r^$\alpha$, $\alpha$ < 2 则收敛 (用极坐标 r dr dθ)
  • 在无穷 $\vec{r}$∞: 若 |f| ~ 1/r^$\alpha$, $\alpha$ > 2 则收敛

例题 13-13

∬_(R²) e^(-(x²+y²)) dA = $\pi$ (前面高斯积分已证).

  • 区域无界但 e^(-r²) 在 $\vec{r}$∞ 急剧衰减 ⟹ 收敛.

例题 13-14

∬_(B(0,1)) 1/r dxdy. 极坐标 = ∫_0^(2π)∫_0^1 (1/r)·r dr dθ = 2π. 收敛 ($\alpha$=1<2) ✓

应用:

  • 量子: 波函数归一化 ∫|ψ|² = 1 必须收敛
  • 概率: pdf 积分 = 1 是反常积分
  • 信号: 总能量 ∫|f|² < ∞ 是 L² 空间条件
  • 神经网络: 损失函数在无穷处的衰减分析

知识点 13.13: 多重积分的 Fubini 定理详解

Fubini 定理

若 $f$ 在矩形 $[a,b]\times[c,d]$ 上 Riemann 可积, 则: $$\iint_{[a,b]\times[c,d]} f\,dxdy = \int_a^b\!\!\int_c^d f(x,y)\,dy\,dx = \int_c^d\!\!\int_a^b f(x,y)\,dx\,dy$$ 即积分次序可交换.

Tonelli 定理 (非负版)

若 $f \ge 0$ 且可测, 两种顺序积分总相等 (允许 $+\infty$).

可以交换积分次序.

反例 (无界函数, Fubini 失效): f(x,y) = (x-y)/(x+y)³ on (0,1]². ∫_0^1 [∫_0^1 f dy] dx = 1/2. ∫_0^1 [∫_0^1 f dx] dy = -1/2.

两边不等! 因为 |f| 在原点附近不可积 (∬|f| dxdy = ∞).

Tonelli 定理 (Fubini 的非负版本): 若 f ≥ 0 且可测, 则两种顺序积分总相等 (允许 ∞).

应用:

  • 概率论: 期望的累积形式 E[X] = ∫ P(X>t) dt (Layer Cake)
  • 偏微分方程: 弱解构造常用 Fubini
  • 信号: 卷积 ∫f(x-y)g(y)dy 频域分析
  • ML: 多变量损失函数边缘化

知识点 13.14: 雅可比与微分形式的等价

回顾: 重积分换元 dxdy = |J| du dv.

微分形式视角: dx ∧ dy = (∂x/∂u du + ∂x/∂v dv) ∧ (∂y/∂u du + ∂y/∂v dv) = (∂x/∂u · ∂y/∂v) du∧dv + (∂x/∂v · ∂y/∂u) dv∧du = (∂x/∂u · ∂y/∂v - ∂x/∂v · ∂y/∂u) du∧dv = det J · du∧dv

外积反对称 dv∧du = -du∧dv 自动产生雅可比行列式. 这是微分形式优雅之处.

知识点 13.15: 立方体的"双重广义体积"

n 维球体体积: V_n(R) = $\pi$^(n/2) R^n / Γ(n/2 + 1)

例:

  • n=1: 2R
  • n=2: πR²
  • n=3: (4/3)πR³
  • n=4: (1/2)π² R⁴
  • n=10: $\pi$^5 R^10 / 120

惊人现象: V_n(1) 在 n=5 达到峰值约 5.26, 然后下降, $\vec{n}$∞ 时 → 0. 高维单位球"几乎是空的".

应用:

  • "维数诅咒" (Curse of Dimensionality): 高维数据点几乎全在边界, ML 算法面对挑战
  • 蒙特卡洛积分: 高维球内采样效率随 n 指数下降
  • KNN 算法: 高维下"近邻"概念失效
