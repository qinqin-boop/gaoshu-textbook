## 8.3 曲面及其方程

平面方程是空间中"线性"的代数对象,曲面是其非线性推广:$F(x,y,z)=0$ 这种**隐式**方程刻画了球面、圆柱面、圆锥面、椭球面、双曲面、抛物面等无穷多种几何形态。本节系统给出:球面 / 旋转曲面 / 柱面 / 二次曲面五兄弟、参数化与截痕法、与 §9.7 切平面、§11.4 曲面积分的联结。这是计算机图形学(隐式曲面 / NURBS)、机器人 CAD、广义相对论时空几何的代数语言。

### 8.3.1 球面方程

::def
球心 $(x_0,y_0,z_0)$、半径 $R$ 的**球面**:
$$\boxed{(x-x_0)^2+(y-y_0)^2+(z-z_0)^2=R^2.}$$
展开得一般式 $x^2+y^2+z^2+Dx+Ey+Fz+G=0$。
::end

::thm
一般式 $x^2+y^2+z^2+Dx+Ey+Fz+G=0$ 是球面 $\iff$ $\tfrac{D^2+E^2+F^2}{4}-G>0$,其球心 $(-D/2,-E/2,-F/2)$,半径 $\sqrt{\frac{D^2+E^2+F^2}{4}-G}$。
::end

::proof
配方:
$$\Big(x+\tfrac{D}{2}\Big)^2+\Big(y+\tfrac{E}{2}\Big)^2+\Big(z+\tfrac{F}{2}\Big)^2=\frac{D^2+E^2+F^2}{4}-G.$$
右边 $>0$ 时是球面,$=0$ 是单点,$<0$ 是空集。
::end

::ex
**例 1** $x^2+y^2+z^2-2x+4y-6z+5=0$。配方 $(x-1)^2+(y+2)^2+(z-3)^2=1+4+9-5=9$,球心 $(1,-2,3)$,半径 $3$。
::end

### 8.3.2 旋转曲面

绕坐标轴旋转一条平面曲线生成的曲面叫**旋转面**(surface of revolution)。最常见的:绕 $z$ 轴。

::thm
设 $yOz$ 平面上曲线 $\Gamma:f(y,z)=0$,绕 $z$ 轴旋转一周生成的旋转面方程:
$$\boxed{f(\pm\sqrt{x^2+y^2},\,z)=0}\quad(\text{把 }y\text{ 替换为 }\pm\sqrt{x^2+y^2}).$$
::end

::proof
旋转面上一点 $P(x,y,z)$ 由曲线 $\Gamma$ 上某点 $(0,y_0,z_0)$ 绕 $z$ 轴旋转得到,且 $z=z_0$、$\sqrt{x^2+y^2}=|y_0|$。故 $y_0=\pm\sqrt{x^2+y^2}$,代入 $f(y_0,z_0)=0$ 即得方程。
::end

::ex
**例 2**(常见旋转面)
- 直线 $z=y\,\tan\alpha$($yOz$ 上,过原点,斜率 $\tan\alpha$)绕 $z$ 旋转 $\Rightarrow$ **圆锥面** $z^2=(x^2+y^2)\tan^2\alpha$;
- 抛物线 $z=y^2/(2p)$ 绕 $z$ 旋转 $\Rightarrow$ **抛物面** $z=(x^2+y^2)/(2p)$(碟形天线、伽利略反射);
- 圆 $(y-a)^2+z^2=r^2$($a>r>0$,在 $yOz$ 上不过 $z$ 轴)绕 $z$ 旋转 $\Rightarrow$ **环面**(轮胎面)$(\sqrt{x^2+y^2}-a)^2+z^2=r^2$。
::end

### 8.3.3 柱面

把一条平面曲线沿与该平面**不在该平面**的方向平移生成的曲面叫**柱面**。

::thm
设 $xOy$ 平面上曲线 $\Gamma:f(x,y)=0$ 沿 $z$ 轴方向平移,得**柱面方程** $f(x,y)=0$(不含 $z$)。

类似地,$f(y,z)=0$ 是沿 $x$ 轴的柱面,$f(x,z)=0$ 是沿 $y$ 轴的柱面。

**关键识别**:三元方程**缺少某个坐标变量** $\Leftrightarrow$ 是沿对应坐标轴的柱面。
::end

::ex
**例 3**(常见柱面)
- $x^2+y^2=R^2$:沿 $z$ 轴的**圆柱面**;
- $x^2/a^2+y^2/b^2=1$:沿 $z$ 轴的**椭圆柱面**;
- $y=x^2$:沿 $z$ 轴的**抛物柱面**;
- $y^2-z^2=1$:沿 $x$ 轴的**双曲柱面**。
::end

### 8.3.4 二次曲面五兄弟(标准型)

一般二次方程 $Ax^2+By^2+Cz^2+Dxy+Eyz+Fxz+\dots=0$ 经过坐标变换可分类为五种**标准型**(不考虑退化情形如柱面 / 锥面)。

| 类型 | 标准方程 | 几何特征 |
| --- | --- | --- |
| **椭球面** | $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}+\dfrac{z^2}{c^2}=1$ | 三轴半径 $a,b,c$;$a=b=c$ 退化球面 |
| **单叶双曲面** | $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}-\dfrac{z^2}{c^2}=1$ | "腰部" 椭圆,沿 $z$ 两侧扩开;一片连通(单叶) |
| **双叶双曲面** | $-\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}+\dfrac{z^2}{c^2}=1$ | 沿 $z$ 两片分离;每片像碗朝外 |
| **椭圆抛物面** | $z=\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}$ | 碗形,横截面椭圆;**$a=b$ 是旋转抛物面** |
| **双曲抛物面** | $z=\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}$ | 马鞍形;$x$ 方向开口向上,$y$ 方向开口向下 |

::intuition
五兄弟的**判别符号**:
- **同号 + 等于常数**:椭球(三正)或空集(三负);
- **二正一负 + 1**:单叶双曲面(腰部);
- **一正二负 + 1**(等价二正一负 + (-1)):双叶双曲面;
- **二平方等于一线性**:抛物面;同号 $\Rightarrow$ 椭圆,异号 $\Rightarrow$ 双曲。

代数判别用矩阵: $Q=\big(\text{quadratic form coefficients}\big)$ 的特征值符号,数学上叫 **Sylvester 惯性律**。
::end

::ex
**例 4**(单叶双曲面)$\dfrac{x^2}{4}+\dfrac{y^2}{9}-\dfrac{z^2}{1}=1$。
- $z=0$ 截痕:$x^2/4+y^2/9=1$(椭圆,半轴 $2,3$),"腰部"。
- $z=\pm 2$ 截痕:$x^2/4+y^2/9=5$(椭圆,半轴 $2\sqrt 5,3\sqrt 5$),更大。
- $x=0$ 截痕:$y^2/9-z^2=1$(双曲线)。
**特点**:不解开 $z$,绕轴拉伸放大。冷却塔、风电塔基底用单叶双曲面——既稳又省料。
::end

::ex
**例 5**(双曲抛物面)$z=x^2-y^2$ 的马鞍形几何:
- $z=0$:$x=\pm y$ 两条直线;
- $z=1$:$x^2-y^2=1$ 双曲线;
- $z=-1$:$y^2-x^2=1$ 双曲线(另一支)。

**奇特性质**:**含两族直母线**——$x+y=t,z=t(x-y)$ 与 $x-y=s,z=s(x+y)$ 各为一族直线,且每族任意两条不相交。这是为什么双曲抛物面能用**直钢梁**搭建(西班牙 Eduardo Torroja 等建筑师),实际工程意义远超几何美。
::end

### 8.3.5 截痕法画曲面

::intuition
**截痕法**:用 $x=$ 常、$y=$ 常、$z=$ 常的三族平面分别截曲面,得到三族**平面曲线**(交线),把这些曲线"堆叠"还原三维曲面。这是机械绘图、几何分析、Python matplotlib 等工具背后的核心思想。
::end

::ex
**例 6**(画椭球面)$\frac{x^2}{4}+\frac{y^2}{9}+z^2=1$。
- $z=c\in[-1,1]$ 截痕:$\frac{x^2}{4}+\frac{y^2}{9}=1-c^2$,椭圆,半轴 $2\sqrt{1-c^2},3\sqrt{1-c^2}$,$c=0$ 最大;
- $x=c$ 截痕:$\frac{y^2}{9}+z^2=1-c^2/4$,椭圆;
- $y=c$ 截痕:类似。

把横截椭圆从大到小堆叠就画出椭球面。
::end

### 8.3.6 应用速览

| 曲面 | 工程 / 科学应用 |
| --- | --- |
| **球面** | 行星轨道 / 卫星 / 镜头球面 / 多面体内切球 |
| **椭球面** | 行星实际形状(地球扁椭球)/ 螺旋 CT 重建 / PCA 协方差椭球 |
| **抛物面** | 抛物天线 / 反射望远镜 / 反射焦点 |
| **单叶双曲面** | 冷却塔 / 钢结构塔 / 直母线展开建造 |
| **双叶双曲面** | 双曲透镜 / GPS 多点定位区域 |
| **双曲抛物面** | 屋顶 / 薯片 / 直母线建筑 |
| **环面** | 轮胎 / 救生圈 / TOR 拓扑(物理标准模型 gauge field) |

::pitfall
**易错点**:
- 判断旋转面 / 柱面:**缺哪个变量就沿哪轴拉伸**(柱面)/ **被替换为 $\pm\sqrt{x^2+y^2}$ 的是另一坐标**(绕轴旋转);
- 二次曲面分类要看**所有系数符号**;只看几项有误判;
- 单叶 vs 双叶双曲面**只差一个符号**($+1$ vs $-1$),但几何完全不同(连通分量数);
- 截痕法画曲面要**多取几片**截面才看清楚整体形态;
- 实际工程中的曲面有时是**有界**的(冷却塔不无限延伸),要附加 $z$ 范围约束。
::end
