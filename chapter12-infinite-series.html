第十二章 无穷级数

对照: 同济大学《高等数学》第七版下册 第十二章
把"无限项相加"说清楚 — 看起来无穷, 居然能算出有限数.

12.0 整章在干啥?

1 + 1/2 + 1/4 + 1/8 + ... = 2 — 无穷多项相加居然等于 2?

这章讲:

  • 什么时候无穷项相加收敛 (等于有限数), 什么时候发散 (没结果或趋无穷)
  • 函数能不能写成幂级数 (多项式无穷展开), 比如 e^x = 1 + x + x²/2! + ...
  • 周期函数能不能写成傅里叶级数 (正弦余弦无穷展开)

对你的意义: 计算机算 sin, cos, e^x, log 全用幂级数 (CPU 没那魔法直接算). 信号处理/音频/图像压缩全用傅里叶. 概率统计的特征函数也是.

12.0.5 入门: 为啥"无限加" 居然能得到有限数?

直觉: 1 + 1 + 1 + ... = 无穷大. 这是对的.

但: 1 + 1/2 + 1/4 + 1/8 + ... = 2 也是对的! 怎么回事?

关键: "每一项越加越小" 且衰减够快, 总和可能收敛到有限数. 就像往杯子里先倒半杯水, 再倒半杯的半杯, 再再倒半杯... 永远倒不满第二杯 (杯子大小 = 1), 但能逼近 1.

这章学:

  1. 常数项级数: Σ a_n 何时收敛/发散
  2. 判别法: 比值/根值/比较/p-级数 等
  3. 幂级数: Σ a_n x^n, 像无穷次多项式
  4. 泰勒级数: 把任何"光滑函数" 展成幂级数
  5. 傅里叶级数: 把周期函数展成"无穷个 sin/cos 的和"

    6. 为啥重要:

    • CPU 算 sin/cos/exp 全靠泰勒级数前几项
    • MP3/JPEG 压缩用傅里叶级数原理
    • 量子力学波函数 = 本征函数级数
    • 神经网络无限宽极限 = 高斯过程级数

    直觉新认知: 复杂函数本质 = 简单函数的无穷叠加.

    12.1 常数项级数概念

    知识点 1: 定义

    级数 Σ_(n=1)^∞ u_n = u₁ + u₂ + u₃ + ...

    部分和 S_n = u₁ + ... + u_n.

    若 lim_(n→∞) S_n = S (有限数), 级数收敛, 和 = S. 否则发散.

    例题 1: 等比级数 Σ_(n=0)^∞ q^n (公比 |q| < 1). S_n = (1 - q^(n+1))/(1 - q) → (1)/(1-q). 收敛和 = 1/(1-q).

    例题 2: Σ 1 显然发散 (和趋无穷).

    例题 2-2 (馅饼故事): 1/2 + 1/4 + 1/8 + ... = (1/2)/(1-1/2) = 1. 故事: 想吃一整块饼, 先吃一半, 再吃剩下一半的一半 ... 累计吃了 1 块.

    例题 2-3 (telescoping/裂项): Σ_(n=1)^∞ 1/(n(n+1)). 裂项: 1/(n(n+1)) = 1/n - 1/(n+1). S_N = (1-1/2) + (1/2-1/3) + ... + (1/N - 1/(N+1)) = 1 - 1/(N+1) → 1.

    知识点 2: 收敛必要条件

    级数收敛 → lim u_n = 0.

    逆命题不一定! u_n → 0 不一定收敛. 反例: 调和级数 Σ 1/n, 1/n → 0 但发散.

    12.2 正项级数判别 (所有 u_n ≥ 0)

    知识点 3: 比较判别法

    若 0 ≤ u_n ≤ v_n:

    • Σ v_n 收敛 → Σ u_n 收敛
    • Σ u_n 发散 → Σ v_n 发散

    生活类比: 小数列被大数列盖住, 大的有限小的也有限.

    知识点 3.5: 比较判别法的反向应用 (对齐书 §12.2 进一步)

    正向: 知 Σ b_n 收敛 + 0≤a_n≤b_n → Σ a_n 也收敛. 反向: 知 Σ a_n 发散 + 0≤a_n≤b_n → Σ b_n 也发散.

    例题 3-1-1: Σ 1/(2^n - 1). 跟 Σ 1/2^n (等比, 收敛和 1) 比较: 1/(2^n-1) ≤ 2/2^n (n≥1). 2·Σ 1/2^n = 2 收敛 → Σ 1/(2^n-1) 收敛 (比较法).

    例题 3-1-2: Σ 1/(n + √n). 跟 Σ 1/(2n) 比较: 1/(n+√n) > 1/(2n) (n≥1). Σ 1/n 发散 → Σ 1/(2n) 发散 → Σ 1/(n+√n) 也发散.

    知识点 4: 比值判别法 (达朗贝尔)

    ρ = lim_(n→∞) (u_(n+1))/(u_n)

    • ρ < 1: 收敛
    • ρ > 1: 发散
    • ρ = 1: 失效

    例题 3: Σ (n)/(2^n). (u_(n+1))/(u_n) = ((n+1)/2^(n+1))/(n/2^n) = (n+1)/(2n) → 1/2 < 1 → 收敛.

    知识点 5: 根值判别法 (柯西)

    ρ = lim_(n→∞) sqrt[n]{u_n}, 判别同比值法.

    知识点 5.5: 极限判别法 (对齐书 §12.2 进一步)

    类比比值法, 但比的是 (u_n)/(1/n^p) — 用 p-级数当参照.

    若 lim u_n/(1/n^p) = c (0 < c < ∞):

    • p > 1: 都收敛
    • p ≤ 1: 都发散

    实操: 拿到一个复杂正项级数 Σ u_n, 想想 u_n 对大 n "像什么 1/n^p", 然后查 p-级数.

    例题 3-3-1: Σ (3n+1)/(n³+2n+5). 大 n 时: u_n ≈ 3n/n³ = 3/n². 用 p=2 参照, lim u_n / (1/n²) = 3 > 0. p=2 收敛 → 原级数收敛.

    例题 3-3-2: Σ 1/(2n - √n). 大 n 时: u_n ≈ 1/(2n) → 用 p=1 参照, lim = 1/2 > 0. p=1 (调和级数) 发散 → 原级数发散.

    知识点 6: p-级数 (重要参考)

    Σ_(n=1)^∞ (1)/(n^p):

    • p > 1 → 收敛
    • p ≤ 1 → 发散

    特别 Σ 1/n (p=1) 发散, Σ 1/n² (p=2) 收敛 (实际上 = π²/6, 著名).

    例题 3-2 (比较判别): 判断 Σ 1/(n²+1) 收敛性. 1/(n²+1) < 1/n², 而 Σ 1/n² 收敛 (p=2 > 1). 由比较法 → 收敛.

    例题 3-3 (比值判别): Σ n!/n^n 收敛性? u_(n+1)/u_n = (n+1)!/(n+1)^(n+1) · n^n/n! = (n+1) · n^n / (n+1)^(n+1) = (n/(n+1))^n = 1/(1+1/n)^n → 1/e < 1. 收敛.

    例题 3-4 (根值判别): Σ (1/(ln n))^n 收敛性 (n≥2). ⁿ√u_n = 1/ln n → 0 < 1. 收敛.

    知识点 6.5: 积分判别法

    若 f(x) 单调递减且 f(n) = u_n, 则 Σ u_n 跟 ∫_1^∞ f(x) dx 同时收敛或发散.

    例题 3-5: Σ 1/(n ln n) 收敛吗? 对应 ∫_2^∞ 1/(x ln x) dx, 令 u = ln x, du = dx/x → ∫ du/u → ln u → ∞. 发散.

    12.3 交错级数 (一正一负)

    知识点 6.5: p-级数完整证明 (对齐书 §12.2 一 进一步)

    定理: Σ 1/n^p 当 p > 1 收敛, p ≤ 1 发散.

    证明 (用积分判别法): 函数 f(x) = 1/x^p 在 [1, ∞) 单调减. ∫_1^∞ 1/x^p dx = lim_{N→∞} ∫_1^N 1/x^p dx

    • p ≠ 1: = lim [x^(1-p)/(1-p)]_1^N

    - p > 1: → -1/(1-p) = 1/(p-1) (有限) → 收敛 - p < 1: → ∞ (无限) → 发散

    • p = 1: ∫ 1/x dx = ln x → ln N → ∞ (发散)

    由积分判别法, 级数跟积分同收敛/同发散. ∎

    重要特例:

    • Σ 1/n (调和) 发散 (慢, 但发散)
    • Σ 1/n² 收敛 → π²/6 (Euler)
    • Σ 1/n^(1.001) 收敛 (即使 p 微大于 1)

    知识点 7: 莱布尼茨判别法

    Σ (-1)^(n-1) u_n (u_n > 0):

    • 若 u_n 单调递减且 u_n → 0 → 收敛

    例题 4: Σ (-1)^(n-1)/n = 1 - 1/2 + 1/3 - 1/4 + ... 满足条件 → 收敛 (实际 = ln 2).

    知识点 8: 绝对收敛 vs 条件收敛

    • Σ |u_n| 收敛 → 称绝对收敛. 此时 Σ u_n 也收敛.
    • Σ u_n 收敛但 Σ |u_n| 发散 → 条件收敛.

    例: Σ (-1)^(n-1)/n 条件收敛 (本身收敛但 Σ 1/n 发散).

    ---

    知识点 8.5: 莱布尼茨级数余项估计

    莱布尼茨判别法 收敛后, 余项估计: |R_n| = |S - S_n| ≤ |a_(n+1)|

    即取前 n 项作为近似, 误差不超过第 n+1 项的绝对值. 简单实用.

    例题 4-2: Σ (-1)^(n-1)/n² 取前 5 项作近似值, 估误差. S_5 = 1 - 1/4 + 1/9 - 1/16 + 1/25 ≈ 0.7836. |R_5| ≤ 1/36 ≈ 0.0278. 精确值 S = π²/12 ≈ 0.8225. 实际误差 ≈ 0.0389, 略大于估值但同量级.

    12.4 幂级数 (本章核心)

    知识点 8.5: 阿贝尔判别法 (对齐书 §12.4 进一步)

    阿贝尔定理: 若幂级数 Σ a_n x^n 在 x = x₀ ≠ 0 处收敛, 则在 |x| < |x₀| 上绝对收敛.

    几何意义: 幂级数有"收敛半径 R" — 在 (-R, R) 内绝对收敛, |x| > R 处发散. 边界 x = ±R 单独判.

    推论 (求收敛域 4 步):

    1. 用比值/根值法求 R
    2. 写出 |x| < R 即 (x₀-R, x₀+R) 区间
    3. 单独判 x = x₀ + R 和 x = x₀ - R 处的常数项级数
    4. 合并得到收敛域

      5. 知识点 9: 定义

      Σ a_n x^n = a₀ + a₁ x + a₂ x² + ... 含变量 x.

      收敛域: 所有让级数收敛的 x 值.

      知识点 10: 收敛半径 R

      ρ = lim |a_(n+1)/a_n|, R = 1/ρ (若 ρ = 0 取 R = ∞).

      • |x| < R: 绝对收敛
      • |x| > R: 发散
      • x = ± R: 单独判别

      例题 5: Σ x^n / n! 的收敛半径? a_n = 1/n!, |a_(n+1)/a_n| = 1/(n+1) → 0. R = ∞ → 对所有 x 收敛.

      例题 5-2: Σ n x^n 的收敛半径? a_n = n, |a_(n+1)/a_n| = (n+1)/n → 1. R = 1. 即 |x| < 1 收敛. x = ±1 时 Σ n(±1)^n 发散 (项不趋 0). 收敛域: (-1, 1).

      例题 5-3: Σ (x-1)^n / n² 的收敛半径 + 域? 中心在 x=1. a_n = 1/n², ρ = 1, R = 1. 即 |x-1| < 1 收敛, 0 < x < 2. x = 0: Σ (-1)^n/n² 绝对收敛. x = 2: Σ 1/n² 收敛. 收敛域: [0, 2] (含端点).

      知识点 10.5: 收敛半径 R 的两种求法 (对齐书 §12.3 二)

      比值法 (达朗贝尔): ρ = lim_(n→∞) |a_(n+1)/a_n|, R = 1/ρ

      • ρ = 0 → R = ∞ (对全体 x 收敛)
      • ρ = ∞ → R = 0 (仅 x=0 收敛)

      根值法 (柯西): ρ = lim_(n→∞) ⁿ√|a_n|, R = 1/ρ

      例题 5-3-1 (比值法): Σ n²x^n. a_n = n², |a_(n+1)/a_n| = (n+1)²/n² → 1. R = 1. 收敛域: |x| < 1.

      例题 5-3-2 (根值法): Σ (1/2^n) x^n. ⁿ√|a_n| = 1/2 → R = 2. 收敛域: |x| < 2.

      知识点 11: 几个重要幂级数展开

      (背下来, AI 数值计算用)

      e^x = Σ_(n=0)^∞ (x^n)/(n!) = 1 + x + (x²)/(2!) + (x³)/(3!) + ... (∀ x)

      sin x = Σ_(n=0)^∞ (-1)^n (x^(2n+1))/((2n+1)!) = x - (x³)/(3!) + (x^5)/(5!) - ... (∀ x)

      cos x = Σ_(n=0)^∞ (-1)^n (x^(2n))/((2n)!) = 1 - (x²)/(2!) + (x^4)/(4!) - ... (∀ x)

      ln(1+x) = Σ_(n=1)^∞ (-1)^(n-1) (x^n)/(n) = x - (x²)/(2) + (x³)/(3) - ... (-1 < x ≤ 1)

      (1)/(1-x) = Σ_(n=0)^∞ x^n = 1 + x + x² + x³ + ... (|x| < 1)

      CPU 这么算 sin(0.5): 取展开式前几项 0.5 - 0.5³/6 + 0.5^5/120 - ... 几项就精度够了.

      12.5 泰勒级数

      知识点 11.5: 幂级数的运算 (对齐书 §12.3 三)

      幂级数像无穷多项式, 有以下运算:

      1. 加减 (相同收敛半径下): Σ a_n x^n ± Σ b_n x^n = Σ (a_n ± b_n) x^n

      2. 乘法 (Cauchy 乘积): (Σ a_n x^n)(Σ b_n x^n) = Σ c_n x^n, 其中 c_n = Σ_{k=0}^n a_k b_{n-k}

      3. 逐项求导 (在收敛半径内): d/dx (Σ a_n x^n) = Σ n a_n x^(n-1)

      4. 逐项积分: ∫_0^x (Σ a_n t^n) dt = Σ (a_n/(n+1)) x^(n+1)

      例题 5-6 (推导 arctan x 展开): 1/(1+x²) = Σ (-x²)^n = Σ (-1)^n x^(2n) (|x|<1) 逐项积: ∫_0^x 1/(1+t²) dt = arctan x = Σ (-1)^n x^(2n+1)/(2n+1) 代 x=1 (极限存在): arctan 1 = π/4 = 1 - 1/3 + 1/5 - ... (莱布尼茨公式)

      知识点 12: 泰勒展开

      函数 f(x) 在 x₀ 处的泰勒级数: f(x) = Σ_(n=0)^∞ (f^((n))(x₀))/(n!) (x - x₀)^n

      x₀ = 0 时叫麦克劳林级数.

      思想: 任何"足够好"的函数都能用无穷多项多项式逼近. 取前几项即得近似公式 (上面的 sin, e^x 系列就是).

      例题 6: 求 f(x) = e^x 的麦克劳林展开. f^((n))(x) = e^x, f^((n))(0) = 1. e^x = Σ_(n=0)^∞ (1)/(n!) x^n. ✓

      例题 6-2 (估算 e^(0.1)): 取前 4 项: 1 + 0.1 + 0.005 + 0.0001667 ≈ 1.10517. (精确 1.10517091...)

      例题 6-3 (sin 30° 估算): π/6 ≈ 0.5236. sin(π/6) ≈ 0.5236 - 0.5236³/6 ≈ 0.5236 - 0.0239 = 0.4997 ≈ 0.5. ✓

      知识点 12.5: 由已知展开导出新展开 (实用技巧)

      思想: 从已知 e^x, sin x, cos x, 1/(1-x), ln(1+x) 展开式变换 (代换/求导/积分) 出新函数的展开. 比直接套泰勒公式快得多.

      例题 6-4: 求 1/(1+x²) 的展开. 1/(1-u) = Σ u^n. 令 u = -x²: 1/(1+x²) = 1 - x² + x⁴ - x⁶ + ... (|x|<1)

      例题 6-5 (莱布尼茨公式): 求 arctan x 的展开. arctan x = ∫_0^x 1/(1+t²) dt = Σ (-1)^n x^(2n+1)/(2n+1) = x - x³/3 + x⁵/5 - ... 代 x = 1 (条件收敛): π/4 = 1 - 1/3 + 1/5 - 1/7 + 1/9 - ... (这就是莱布尼茨求 π 的著名公式)

      例题 6-6: e^(x²) 的展开. e^u = Σ u^n/n!. 令 u = x²: e^(x²) = 1 + x² + x⁴/2! + x⁶/3! + ...

      例题 6-7 (sin x / x 展开): sin x / x = (x - x³/3! + x⁵/5! - ...)/x = 1 - x²/3! + x⁴/5! - ... 有用: 这就是为啥 lim_(x→0) sin x/x = 1 (展开式第一项就是 1).

      知识点 12.6: 余项估计 (展开有多准?)

      拉格朗日余项: 取展开前 n 项, 误差 R_n(x) ≤ M · |x-x₀|^(n+1) / (n+1)! 其中 M 是 f^(n+1) 在区间上的最大值.

      例题 6-8: 用 sin x ≈ x - x³/6 算 sin(0.5), 误差有多大? |R_3(0.5)| ≤ max|sin''''| · 0.5⁵/5! ≤ 1 · (0.03125)/120 ≈ 0.00026. 实际: x-x³/6 = 0.5 - 0.0208 = 0.4792, sin(0.5) = 0.4794, 误差 ≈ 0.0002 ✓.

      12.5 函数项级数的一致收敛性 (对齐书 §12.6)

      知识点 13: 函数项级数

      之前讲的是 Σ a_n (常数项) 或 Σ a_n x^n (幂级数). 更一般的是 Σ u_n(x), 每项是任意 x 的函数.

      知识点 13.5: 一致收敛 (核心概念)

      普通收敛 (pointwise): 在每个固定的 x, Σ u_n(x) → S(x) 收敛. 一致收敛 (uniform): 收敛速度跟 x 无关 — 不管 x 取啥, 加到 N 项后误差都小于 ε.

      为啥重要: 一致收敛允许"积分换序" / "求导换序" / "极限换序". 普通收敛不一定能.

      生活类比: 100 个学生考试. 普通收敛 = "每个人最终都及格" (但有人慢有人快). 一致收敛 = "全班同时及格" (整体进度齐).

      知识点 13.6: Weierstrass M 判别法

      如果存在 M_n 使 |u_n(x)| ≤ M_n 对所有 x 都成立, 且 Σ M_n 收敛, 则 Σ u_n(x) 一致收敛.

      应用: 判断幂级数在收敛半径内的子区间上一致收敛.

      ---

      知识点 12.7: 泰勒级数收敛条件 (对齐书 §12.4)

      f(x) 在 x₀ 处的泰勒级数收敛到 f(x) 的条件: 拉格朗日余项 → 0.

      R_n(x) = f^(n+1)(ξ)/(n+1)! · (x-x₀)^(n+1)

      当 R_n(x) → 0 (n → ∞), 泰勒级数才真"代表" 该函数.

      反例: f(x) = e^(-1/x²) (x ≠ 0), f(0) = 0. 在 x₀ = 0 处所有 f^(n)(0) = 0, 泰勒级数 = 0, 但 f 在邻域 ≠ 0.

      这说明: 函数无限可微 ≠ 泰勒展开收敛到函数本身. 工程实用中, e^x, sin x, cos x, ln(1+x), (1+x)^α 在它们的收敛半径内都收敛到原函数 (这是好情况).

      知识点 12.8: 用幂级数解微分方程 (对齐书 §12.5 二)

      思想: 把未知函数 y(x) 写成幂级数 y = Σ a_n x^n, 代入微分方程, 比较系数解出 a_n.

      例题 6-9: y' = y, y(0) = 1. 用幂级数法解. 设 y = Σ a_n x^n, 则 y' = Σ n·a_n x^(n-1). 代入: Σ n·a_n x^(n-1) = Σ a_n x^n → (n+1) a_(n+1) = a_n. 递推: a_(n+1) = a_n/(n+1), a₀ = 1. 所以 a_n = 1/n!, y = Σ x^n/n! = e^x. ✓

      幂级数法在工程/物理特殊微分方程 (贝塞尔 / 勒让德) 必备.

      知识点 12.9: 欧拉公式 (对齐书 §12.5 三)

      著名公式: e^(ix) = cos x + i sin x

      推导: 用 e^x, cos x, sin x 的麦克劳林展开. e^(ix) = Σ (ix)^n/n! = Σ i^n x^n/n! 分类 i^n 周期 4: i⁰=1, i¹=i, i²=-1, i³=-i, i⁴=1,... 偶数项 (i^(2k) = (-1)^k): x^(2k)/(2k)! · (-1)^k → cos x 展开 奇数项 (i^(2k+1) = i(-1)^k): i · x^(2k+1)/(2k+1)! · (-1)^k → i sin x 展开 合起来 e^(ix) = cos x + i sin x. ✓

      最美公式: 代 x = π: e^(iπ) + 1 = 0 五个常数 e, i, π, 1, 0 一起出现, 称为"数学最美公式".

      AI 应用: 离散傅里叶变换 (DFT) 用 e^(-2πikn/N), 这就是欧拉公式离散版.

      12.6 傅里叶级数 (对齐书 §12.7)

      知识点 12.10: 三角函数系正交性 (对齐书 §12.7 一)

      正交性: 在 [-π, π] 上, 函数系 {1, cos x, sin x, cos 2x, sin 2x, ..., cos nx, sin nx, ...} 两两的积分为 0 (除自己跟自己).

      验证 (举例): ∫_{-π}^π cos(mx) cos(nx) dx = 0 (m ≠ n) ∫_{-π}^π sin(mx) sin(nx) dx = 0 (m ≠ n) ∫_{-π}^π cos(mx) sin(nx) dx = 0 (所有 m, n) ∫_{-π}^π cos²(nx) dx = ∫_{-π}^π sin²(nx) dx = π (自己跟自己, n ≥ 1) ∫_{-π}^π 1·1 dx = 2π

      几何意义 (向量类比): 把每个函数看作"无穷维向量", "内积" 定义为 ∫f·g dx. 正交 = "向量两两垂直". 类比 R³ 的 (i→, j→, k→) 正交基.

      为啥重要: 傅里叶系数 a_n, b_n 由正交性推出:

      • a_n = (1/π) ∫_{-π}^π f(x) cos(nx) dx
      • b_n = (1/π) ∫_{-π}^π f(x) sin(nx) dx

      (类比: 向量 a→ 在 i→ 上的投影是 a→·i→ / |i→|² = a_x)

      知识点 12.12: 复数形式傅里叶级数推导

      实数形式: f(x) = (a₀/2) + Σ (a_n cos nx + b_n sin nx) 用欧拉公式 cos nx = (e^(inx)+e^(-inx))/2, sin nx = (e^(inx)-e^(-inx))/(2i):

      a_n cos nx + b_n sin nx = ((a_n - i b_n)/2) e^(inx) + ((a_n + i b_n)/2) e^(-inx)

      定义: c_n = (a_n - i b_n)/2 (n>0), c_{-n} = (a_n + i b_n)/2, c₀ = a₀/2. 复数形式: f(x) = Σ_{n=-∞}^∞ c_n e^(inx) c_n = (1/(2π)) ∫_{-π}^π f(x) e^(-inx) dx

      为啥复数形式好用:

      1. 表达更简洁 (一组系数 c_n, 不分 a, b)
      2. 直接对应离散傅里叶变换 (DFT) → FFT 算法基础
      3. 复指数 e^(inx) 周期性自然

        4. 知识点 13: 周期函数的傅里叶级数

        周期 2π 的函数 f(x) 展开成正弦/余弦无穷和: f(x) = (a₀)/(2) + Σ_(n=1)^∞ (a_n cos nx + b_n sin nx)

        系数: a_n = (1)/(π) ∫_(-π)^π f(x) cos nx dx b_n = (1)/(π) ∫_(-π)^π f(x) sin nx dx

        思想: 任何周期函数都能拆成一堆不同频率的正弦余弦叠加.

        生活类比: 钢琴弹一个音 = 基音 + 多个泛音叠加. 傅里叶就是这么拆.

        音频/图像压缩 (MP3/JPG): 用傅里叶变换转到频域, 去掉人耳/眼不敏感的高频, 数据量大幅减小.

        知识点 13.5: 狄利克雷收敛定理 (对齐书 §12.7 二)

        问题: 一个函数的傅里叶级数, 在什么条件下"真的等于"原函数?

        狄利克雷条件:

        1. f(x) 在 [-π, π] 上分段连续 (有限个间断点)
        2. f(x) 在 [-π, π] 上分段单调 (有限个极值)

          3. 满足时, 傅里叶级数:

          • 连续点处: 收敛到 f(x) 本身
          • 间断点处: 收敛到 (f(x⁺) + f(x⁻))/2 (左右极限平均值)

          经典反例 (吉布斯现象): 方波 f(x) = sgn(x). 傅里叶级数在 0 处"震荡"且峰值超出 9% 不能消除, 即使 N → ∞.

          意义:

          • 工程: 滤波器设计 / 信号重构必学
          • 解释为啥音频 MP3 在尖锐转换处会有"嗡嗡声"

          知识点 13.6: 正弦级数与余弦级数 (对齐书 §12.7 三)

          • 奇函数 f(x) = -f(-x): 傅里叶级数只剩 sin 项, 称正弦级数
          • 偶函数 f(x) = f(-x): 只剩 cos 项, 称余弦级数

          奇偶延拓技巧: 函数 f(x) 只定义在 [0, π]. 想展成傅里叶级数:

          • 奇延拓: 定义 f(-x) = -f(x) 扩到 [-π, π] → 得到正弦级数
          • 偶延拓: 定义 f(-x) = f(x) → 得到余弦级数

          应用: 偏微分方程边值问题 — 弦振动方程 / 热传导方程的解都用正弦/余弦级数.

          知识点 14: 周期 2L 函数的傅里叶级数

          更一般的, 周期 2L: f(x) = (a₀)/(2) + Σ_(n=1)^∞ (a_n cos(nπ x)/(L) + b_n sin(nπ x)/(L))

          a_n = (1)/(L) ∫_(-L)^L f(x) cos(nπ x)/(L) dx b_n = (1)/(L) ∫_(-L)^L f(x) sin(nπ x)/(L) dx

          ---

          知识点 14.5: 周期函数傅里叶级数公式速查 (对齐书 §12.8)

          周期 2L 函数 f(x): f(x) = (a₀)/2 + Σ_(n=1)^∞ (a_n cos(nπx/L) + b_n sin(nπx/L))

          系数: a_n = (1/L) ∫_{-L}^L f(x) cos(nπx/L) dx (n = 0, 1, 2,...) b_n = (1/L) ∫_{-L}^L f(x) sin(nπx/L) dx

          特例 L = π: 退化到周期 2π 公式.

          复数形式 (欧拉公式重写): f(x) = Σ_(n=-∞)^∞ c_n · e^(i·nπx/L) c_n = (1/(2L)) ∫_{-L}^L f(x) e^(-i·nπx/L) dx

          为啥用复数: 信号处理领域更优雅, FFT 算法原生支持. 实数版本可看作复数版本的"特例".

          知识点 14.7: 傅里叶级数 vs 傅里叶变换

          | 项 | 傅里叶级数 | 傅里叶变换 | |---|---|---| | 函数 | 周期函数 | 任意函数 | | 频谱 | 离散 (整数倍频率) | 连续 | | 形式 | Σ 求和 | ∫ 积分 | | 工程用途 | 音乐合成器 (周期波形) | 一般信号分析 |

          二者一脉相承: 周期 T → ∞ 时, 傅里叶级数过渡到傅里叶变换.

          知识点 14.9: 级数判敛的"决策树" 速查

          拿到一个级数 Σ a_n, 一步步判断:

          1. 先看必要条件: lim a_n → 0 吗?

          - 不趋 0 → 发散 (不用再判) - 趋 0 → 继续

          1. 正项级数 (a_n ≥ 0):

          - 像 1/n^p? → p-级数 (p>1 收敛, p≤1 发散) - 比值 a_(n+1)/a_n 有极限 ρ? → 比值判别 (ρ<1 收敛, ρ>1 发散, ρ=1 失效) - 是否单调? → 积分判别 - 跟已知级数比? → 比较法 / 极限判别

          1. 交错级数 (符号正负相间):

          - |a_n| 单调减且 → 0? → 莱布尼茨判别 → 收敛 - 进一步: Σ|a_n| 收敛吗? 收 → 绝对收敛 (强), 不收 → 条件收敛

          1. 任意级数:

          - 先用 Σ|a_n| 转成正项级数判 - Σ|a_n| 收 → 原级数绝对收敛

          知识点 14.10: 常见经典级数收敛性记忆表

          | 级数 | 收敛/发散 | 备注 | |---|---|---| | Σ 1/n (调和) | 发散 | 慢, 但确实发散 | | Σ 1/n² | 收敛 | 和 = π²/6 (巴塞尔问题) | | Σ 1/n³ | 收敛 | 阿佩里常数 | | Σ 1/n^p (p>1) | 收敛 | p-级数 | | Σ q^n (\|q\|<1) | 收敛 | 等比, 和 = 1/(1-q) | | Σ 1/n! | 收敛 | 和 = e - 1 | | Σ (-1)^(n-1)/n | 条件收敛 | 和 = ln 2 (莱布尼茨) | | Σ x^n/n! | 全 x 收敛 | 和 = e^x |

          知识点 14.8: 帕塞瓦尔定理 (能量守恒)

          核心: 信号的"时域能量" = "频域能量".

          具体: f(x) 周期 2π, 傅里叶系数 a_n, b_n. 则: (1/π) ∫_{-π}^π |f(x)|² dx = (a₀²)/2 + Σ_(n=1)^∞ (a_n² + b_n²)

          物理意义: 信号总能量 = 各频率分量能量之和.

          应用:

          • 信号处理: 压缩时丢掉小能量频率, 不影响主体
          • 通信: 计算信号强度
          • 量子力学: 波函数模平方积分 = 概率守恒

          例题 6-10: 周期 2π 矩形波 f(x) = 1 (0<x<π), -1 (-π<x<0). 求 ∫|f|² dx 跟级数对应. ∫_{-π}^π |f|² dx = 2π. f 的傅里叶级数: f(x) = (4/π) Σ sin((2k-1)x)/(2k-1). Σ b_(2k-1)² = (16/π²) Σ 1/(2k-1)² = (16/π²) · (π²/8) = 2. π · Σ b² = π · 2 = 2π ✓ (帕塞瓦尔验证)

          补充: 阿贝尔与狄利克雷判别法

          知识点 8.6: 阿贝尔判别法与狄利克雷判别法

          问题: 莱布尼茨判别法只能判交错级数. 一般"半正半负"或"振荡符号"如何判?

          阿贝尔判别法 (Abel): ∑ aₙbₙ 收敛, 若:

          1. ∑ bₙ 收敛
          2. {aₙ} 单调有界

            3. 狄利克雷判别法 (Dirichlet): ∑ aₙbₙ 收敛, 若:

            1. 部分和 |B_N| = |∑_(n=1)^N bₙ| 有界 (不需 ∑bₙ 收敛)
            2. {aₙ} 单调趋 0

              3. 例题 8-3 (用 Dirichlet): ∑ sin(nθ)/n, θ≠2kπ.

              • aₙ = 1/n 单调趋 0 ✓
              • bₙ = sin(nθ). 部分和 |∑sin(kθ)| ≤ 1/|sin(θ/2)| 有界 ✓

              ∴ 收敛.

              例题 8-4: ∑ cos(nθ)/√n, 0<θ<2π. 同理由 Dirichlet 收敛.

              这解释了为什么傅里叶级数对一般连续函数能收敛 — 系数虽不绝对收敛, 但 Dirichlet 给出条件收敛.

              知识点 8.7: 双重级数 (Double Series)

              定义: ∑∑ a_(m,n), m,n=1,2,...

              柯西定理: 若 ∑∑ |a_(m,n)| 收敛, 则按任意方式累加都得到同一极限. 此时可以

              • 按行累加: ∑_m (∑_n a_(m,n))
              • 按列累加: ∑_n (∑_m a_(m,n))
              • 沿反对角累加: ∑_(m+n=k) a_(m,n) 然后对 k 求和

              反例 (条件收敛时不可交换): 取 a_(m,n) 为 m=n 时 = 1, m=n+1 时 = -1, 其余 = 0. 按行: 全是 0+0+... = 0. 总和 0. 按列: 全是 1+(-1)+(-1)+... = 1 然后 0+... 总和 1. ❌ 不一致.

              这就是为什么"绝对收敛"在重排理论里至关重要 — 它是无穷加法保持"交换律 + 结合律"的充分条件.

              补充: 黎曼重排定理 + ζ 函数初步

              知识点 8.8: 黎曼重排定理 (Riemann Rearrangement)

              惊人结论: 条件收敛的级数, 通过重排可以收敛到任意实数 (甚至发散).

              核心思路: 设 ∑aₙ 条件收敛. 把项分成正项 ∑pₖ 和负项 ∑qₖ (相反数).

              • ∑aₙ 条件收敛 ⇒ ∑pₖ = +∞ 且 ∑qₖ = +∞ (否则要么 ∑|aₙ|<∞ 要么本身发散)

              构造任意目标 S:

              1. 取正项直到部分和首次 > S
              2. 加负项直到部分和首次 < S
              3. 加正项直到首次 > S
              4. ...

                5. 每次"压过"S 用掉一个 pₖ 或 qₖ; 因 aₙ→0, 每次"过头"幅度→0, 故部分和振荡收敛到 S.

                例题 8-5: 交错调和 ∑(-1)^(n+1)/n = ln 2. 重排 (1正+2负): 1 - 1/2 - 1/4 + 1/3 - 1/6 - 1/8 + ... = (1/2)ln 2. 重排改变了和!

                教训: 处理条件收敛级数 (傅里叶系数、量子振幅), 不能随便交换顺序. 绝对收敛是数值稳定的前提.

                知识点 8.9: 黎曼 ζ 函数 与素数

                定义: ζ(s) = ∑_(n=1)^∞ 1/n^s, Re(s) > 1.

                已知特殊值:

                • ζ(2) = π²/6 (Basel 问题)
                • ζ(4) = π⁴/90
                • ζ(2k) = (-1)^(k+1) B_(2k) (2π)^(2k) / (2 (2k)!) (Bernoulli)
                • ζ(-1) = -1/12 (解析延拓, 物理学中弦理论用)

                欧拉乘积: ζ(s) = ∏_p prime 1/(1 - p^(-s))

                证明速写: 1/(1-p^(-s)) = ∑_(k=0)^∞ p^(-ks). 乘起来按算术基本定理对应每个 n^(-s) 恰好一次.

                意义: 把"加" (左边求和) 等于"乘" (右边按素数). 这是解析数论的起点, 黎曼猜想 (RH) 就讨论 ζ 函数零点位置.

                应用:

                • 素数分布: 素数计数 π(x) ~ x/ln x 用 ζ 推
                • 物理: 卡西米尔效应 (真空能正规化) 用 ζ(-3) = 1/120
                • 密码学: 大整数因子分解的难度依赖素数分布
