## 11.1 第一类曲线积分(对弧长)

一元定积分 $\int_a^b f(x)\,dx$ 是在 $x$ 轴一段区间上对 $f$ 求积分。空间或平面里弯曲的"曲线" $L$ 不是直线段,但它仍是「一维」对象——所以理论上也可以在 $L$ 上"对 $f$ 求积分",只需把直线段上的 $dx$ 换成沿曲线的**弧长微元** $ds$。这就是**第一类曲线积分**(line integral with respect to arc length)。本节给出定义、参数化计算公式,以及与质量 / 形心 / 转动惯量的物理对应。

### 11.1.1 定义与几何意义

::def
设 $L$ 是 $\mathbb{R}^3$(或 $\mathbb{R}^2$)中分段光滑曲线,$f(x,y,z)$ 在 $L$ 上有定义。把 $L$ 分成 $n$ 段小弧 $\Delta L_i$,弧长 $\Delta s_i$,在 $\Delta L_i$ 上取点 $(\xi_i,\eta_i,\zeta_i)$,作和
$$\sigma=\sum_{i=1}^n f(\xi_i,\eta_i,\zeta_i)\Delta s_i.$$
若分段细化时 $\sigma$ 有极限,与分法、取点无关,则称之为 $f$ 沿 $L$ 的**第一类曲线积分**,记 $\displaystyle\int_L f(x,y,z)\,ds$。
::end

::intuition
**与第一类曲面积分类似**:$ds$ 是**正**的弧长元,**积分与曲线方向无关**(沿 $L$ 走过去和走回来积分值相同)。物理:$f=$ 线密度时 $\int f\,ds=$ 总质量。
::end

### 11.1.2 计算公式

::thm
(参数化计算)设 $L$ 由 $\vec{r}(t)=(x(t),y(t),z(t))$,$t\in[\alpha,\beta]$ 光滑参数化($\vec{r}'(t)\neq\vec{0}$)。则
$$ds=|\vec{r}'(t)|\,dt=\sqrt{x'(t)^2+y'(t)^2+z'(t)^2}\,dt,$$
$$\int_L f\,ds=\int_\alpha^\beta f(x(t),y(t),z(t))\,\sqrt{x'(t)^2+y'(t)^2+z'(t)^2}\,dt.$$
::end

::proof
弧长元的几何来源:$\vec{r}(t+dt)-\vec{r}(t)\approx\vec{r}'(t)\,dt$,长度 $|\vec{r}'(t)|\,dt$。所以小弧长 $\Delta s_i\approx|\vec{r}'(t_i^*)|\Delta t$,代入 Riemann 和取极限得公式。
::end

::ex
**例 1**(参数化算简单积分)$L$:$x=t,y=t^2$,$t\in[0,1]$。算 $\int_L y\,ds$。

**解**:$\vec{r}'=(1,2t)$,$|\vec{r}'|=\sqrt{1+4t^2}$。
$$\int_L y\,ds=\int_0^1 t^2\sqrt{1+4t^2}\,dt.$$
(具体值靠数值或代换 $u=2t$ 给出 $\approx 0.55$)
::end

**显式曲线情形**:$L:y=g(x)$,$x\in[a,b]$ 取 $t=x$,$x'(t)=1,y'(t)=g'(x)$:
$$ds=\sqrt{1+g'(x)^2}\,dx,\quad\int_L f\,ds=\int_a^b f(x,g(x))\sqrt{1+g'(x)^2}\,dx.$$

**极坐标情形**:$L:r=r(\theta)$,$\theta\in[\alpha,\beta]$,$x=r\cos\theta,y=r\sin\theta$:
$$ds=\sqrt{r^2+r'^2}\,d\theta.$$

### 11.1.3 物理 / 几何应用

| 几何 / 物理量 | 公式 |
| --- | --- |
| 曲线弧长 | $L=\int_L 1\,ds$ |
| 曲线总质量 | $M=\int_L\mu\,ds$($\mu=$ 线密度) |
| 形心坐标(均匀) | $\bar x=\frac{1}{L}\int_L x\,ds$,$\bar y$ 同理 |
| 关于轴的转动惯量 | $I=\int_L d^2\mu\,ds$($d=$ 到轴距离) |
| 曲线引力 | $\vec{F}=G\int_L\frac{\vec{x}-\vec{P}}{|\vec{x}-\vec{P}|^3}\mu\,ds$ |

::ex
**例 2**(均匀半圆形丝的质心)半圆 $L:x^2+y^2=R^2,y\ge 0$,均匀线密度 $\mu$。参数化 $x=R\cos t,y=R\sin t,t\in[0,\pi]$,$ds=R\,dt$。
$$L=R\pi,\quad M=\mu R\pi,$$
$$\bar y=\frac{1}{L}\int_L y\,ds=\frac{1}{R\pi}\!\int_0^\pi R\sin t\cdot R\,dt=\frac{R}{\pi}\cdot 2=\frac{2R}{\pi}.$$
比半圆**板**质心 $4R/(3\pi)\approx 0.42R$ 更"靠上",因为质量集中在弧上而不是面上。
::end

::pitfall
**易错点**:
- $ds$ **永远是正**;参数化时 $\sqrt{}$ 不要漏掉绝对值;
- 显式 $y=g(x)$ 的 $ds=\sqrt{1+g'^2}\,dx$ 是常用闭式,但记得 $y'$ 不是 $g(x)$ 而是 $g'(x)$;
- 极坐标 $ds=\sqrt{r^2+r'^2}\,d\theta$ 容易记成 $\sqrt{1+r'^2}$——**有 $r^2$ 项**;
- 第一类曲线积分**与方向无关**,所以 $\int_L=\int_{-L}$。这跟第二类(下节)最大区别。
::end

## 11.2 第二类曲线积分(对坐标)

第一类沿弧长积分,**方向无关**;但物理"做功"明显**有方向**——逆风走 $\vec{F}$ 给负功、顺风给正功。需要**带方向**的曲线积分,这就是**第二类曲线积分**或**对坐标的曲线积分**。它是 §11.3 Green 公式与 §11.6 Stokes 公式的核心对象,也是物理"功 / 环量 / 电磁感应电动势"的代数化。

### 11.2.1 定义

::def
设有向曲线 $L$ 从 $A$ 到 $B$,$\vec{F}=(P,Q,R)$ 在 $L$ 上有定义。把 $L$ 分成 $n$ 段有向弧,记 $i$ 段位移向量 $\Delta\vec{r}_i=(\Delta x_i,\Delta y_i,\Delta z_i)$,在该弧上取点 $(\xi_i,\eta_i,\zeta_i)$,作和
$$\sigma=\sum_{i=1}^n\vec{F}(\xi_i,\eta_i,\zeta_i)\cdot\Delta\vec{r}_i=\sum[P_i\Delta x_i+Q_i\Delta y_i+R_i\Delta z_i].$$
若细化极限存在且与分法 / 取点无关,称为**第二类曲线积分**,记
$$\int_L P\,dx+Q\,dy+R\,dz\quad\text{或}\quad\int_L\vec{F}\cdot d\vec{r}.$$
::end

::intuition
**物理意义**:沿曲线 $L$ 由力 $\vec{F}$ 做的总功 $W=\int_L\vec{F}\cdot d\vec{r}$。每小段位移 $d\vec{r}$,力的有效分量 $\vec{F}\cdot d\vec{r}$。**反向走** $\vec{r}_-$(从 $B$ 到 $A$)时所有 $d\vec{r}$ 反号,积分整体变号 $\int_{-L}=-\int_L$——这是和第一类的本质区别。
::end

### 11.2.2 计算公式

::thm
(参数化)设 $L$ 由 $\vec{r}(t)=(x(t),y(t),z(t))$,$t:\alpha\to\beta$ 给出(注意 $t$ 不一定单调增,反向时 $\alpha>\beta$),则
$$\int_L P\,dx+Q\,dy+R\,dz=\int_\alpha^\beta\big[P(x(t),y(t),z(t))x'(t)+Q\cdot y'(t)+R\cdot z'(t)\big]\,dt.$$
::end

::ex
**例 3**(算环量)$\vec{F}=(-y,x,0)$,$L$:单位圆 $x^2+y^2=1$,$z=0$,逆时针正向。

**解**:$x=\cos t,y=\sin t,t\in[0,2\pi]$,$dx=-\sin t\,dt,dy=\cos t\,dt$。
$$\int_L-y\,dx+x\,dy=\int_0^{2\pi}[\sin^2 t+\cos^2 t]\,dt=2\pi.$$
**物理**:旋转流场 $\vec{F}=(-y,x,0)$(切向逆时针)沿圆环一圈做的功 = 环量。
::end

### 11.2.3 两类曲线积分的转换

::thm
设 $L$ 由 $\vec{r}(t)$ 光滑参数化,**单位切向量** $\vec{\tau}=\vec{r}'(t)/|\vec{r}'(t)|=(\cos\alpha,\cos\beta,\cos\gamma)$(方向余弦)。则
$$\int_L\vec{F}\cdot d\vec{r}=\int_L(P\cos\alpha+Q\cos\beta+R\cos\gamma)\,ds=\int_L\vec{F}\cdot\vec{\tau}\,ds.$$
::end

::proof
$d\vec{r}=\vec{\tau}\,ds$(切向乘弧长元),代入即得。
::end

::intuition
$d\vec{r}=\vec{\tau}\,ds$ **就是 $d\vec{S}=\vec{n}\,dS$ 在曲面积分中的对应**:把"带方向的微元"分解成"方向 + 大小"。这种结构在所有微分形式 / 流形积分中反复出现。
::end

### 11.2.4 与 Green / Stokes 的衔接

如果 $L$ 是**平面闭曲线** $\partial D$,则 Green(§11.3)给出
$$\oint_{\partial D}P\,dx+Q\,dy=\iint_D(Q_x-P_y)\,dx\,dy.$$
如果 $L$ 是**空间闭曲线** $\partial\Sigma$,Stokes(§11.6)给出
$$\oint_{\partial\Sigma}\vec{F}\cdot d\vec{r}=\iint_\Sigma(\nabla\times\vec{F})\cdot d\vec{S}.$$
**第二类曲线积分的所有威力都通过这两条定理表达** —— 它把"沿曲线的积分"换算成"区域上的积分",计算便利的同时揭示几何 / 物理本质。

### 11.2.5 与第一类的对照

| 性质 | 第一类 $\int f\,ds$ | 第二类 $\int\vec{F}\cdot d\vec{r}$ |
| --- | --- | --- |
| 被积量 | 标量 $f$ | 向量 $\vec{F}$ |
| 微元 | $ds>0$ 弧长 | $d\vec{r}$ 带方向位移 |
| 方向相关 | **否**($\int_{-L}=\int_L$) | **是**($\int_{-L}=-\int_L$) |
| 物理意义 | 质量 / 弧长 / 形心 / 转动惯量 | 功 / 环量 / 电动势 / 磁通变化 |
| 应用定理 | (无对应"边界 - 内部"定理) | Green / Stokes |
| 闭曲线积分记号 | $\oint_L f\,ds$ | $\oint_L\vec{F}\cdot d\vec{r}$ |

::pitfall
**易错点**:
- 第二类**对方向敏感**,要看清正向:平面 Green 取**逆时针**;空间 Stokes 用**右手系**;Gauss 用**外法向**;
- 参数化时 $t:\alpha\to\beta$ **$\alpha,\beta$ 顺序对应曲线起点终点**,不要简单地按数值小到大;
- $d\vec{r}=\vec{\tau}\,ds$ 是连接两类积分的关键;**实操不一定要换算**,但理论上等价;
- **保守场**(梯度场)沿闭曲线 $\oint\vec{F}\cdot d\vec{r}=0$;判定靠 §11.6 Stokes $\nabla\times\vec{F}=\vec{0}$ + 单连通区域。
::end
