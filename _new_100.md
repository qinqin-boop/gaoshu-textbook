## 10.0 整章在干啥?

上册定积分 $\int_a^b f(x)\,dx$ 解决"曲线下面积"。本章把它升到**多维区域上的积分**:平面区域上的二重积分(曲面下体积 / 板的质量 / 联合概率)、空间立体上的三重积分(立体质量 / 转动惯量 / 引力 / 电场)。核心工具是 **Fubini 定理**(化累次积分)与**坐标变换**(极 / 柱 / 球,带雅可比因子)。这是物理 / 概率 / 工程"分布量"计算的统一语言。

### 10.0.1 全章地图

```
§10.1 概念 ─→ 二重积分定义 (Riemann 和极限) + 7 大性质
              │
§10.2 二重计算 ← Fubini (X-型 / Y-型 区域) + 极坐标 (雅可比 r)
              │       ↓ Gauss 积分 √π
              ↓
§10.3 三重积分 ← 直角 / 柱坐标 (r) / 球坐标 (ρ² sin φ)
              │       ↓ Newton 球壳定理
              ↓
§10.4 应用 ── 质量/质心/转动惯量/引力/概率 (六大物理量)
              │
§10.5 含参变量 ← Leibniz 微分 + Γ/Beta 函数 + Feynman 神技
```

### 10.0.2 三种核心积分对比

| 积分 | 几何对象 | 计算公式 | 应用 |
| --- | --- | --- | --- |
| 一元定积分(上册) | 区间 $[a,b]$ | $\int_a^b f\,dx$ | 面积 / 位移 |
| 二重积分 | 平面区域 $D$ | $\iint_D f\,d\sigma=\int\!\!\int f\,dx\,dy$(Fubini) | 体积 / 板质量 / 联合概率 |
| 三重积分 | 空间立体 $\Omega$ | $\iiint_\Omega f\,dV$(柱 / 球坐标) | 立体质量 / 转动惯量 / 引力 |

### 10.0.3 坐标选择口诀

| 区域几何特征 | 推荐坐标 | 雅可比因子 |
| --- | --- | --- |
| 长方体 / 折面体 | 直角 | $1$ |
| 圆盘 / 圆环 / 扇形 | 平面极坐标 | $r$ |
| 圆柱 / 圆锥 / 旋转体 | 柱坐标 | $r$ |
| 球 / 球壳 / 球扇 | 球坐标 | $\rho^2\sin\varphi$ |
| 椭球 / 仿射变换 | 仿球 / 仿射 | $\det A$ 或 $abc\,\rho^2\sin\varphi$ |

### 10.0.4 学这章的意义

- **物理**:质心 / 转动惯量 / 引力 / 电场都用本章公式;Newton 球壳定理是天体力学经典证明;
- **概率 / 统计**:联合密度 / 边缘分布 / 期望 / 协方差全是重积分;高维标准正态用极坐标算归一化;
- **工程**:有限元法 / 计算流体力学 / 应力张量积分;
- **微积分理论**:Fubini 定理 + 换元公式(雅可比)是分析学一阶工具,Lebesgue 测度论的入门;
- **AI**:贝叶斯方法的边缘化 / 重参数化 / 概率密度估计全用多维积分。
