## 12.0 整章在干啥?

「无穷多个数加起来」一开始是个**哲学难题**:Zeno 悖论里阿喀琉斯永远追不上乌龟,因为他每追一段乌龟又跑了一段,有无穷段——但有限时间内怎么能跨越"无穷"距离?微积分用「**部分和的极限**」给出答案:无穷加 = 有限步的极限,如果该极限存在则等于"无穷和"。本章系统讲数项级数(常数加)、函数项级数(变量加)、幂级数(多项式基)、Taylor 级数(光滑函数展开)、Fourier 级数(三角基),最后引出**信号处理 / 数值计算 / AI Diffusion / 量子力学**的级数语言。

### 12.0.1 全章地图

```
§12.1 常数项级数 (数加) ──── 必要条件 u_n→0 + Cauchy 准则
        │
        ├─ §12.2 正项级数 6 法判别 (比较/比值/根值/积分/Raabe)
        │       └─ p-级数 ∑1/n^p (p>1 收敛) 作参考
        │
        ├─ §12.3 含负项: 交错 (Leibniz) + 绝对/条件 (Riemann 重排)
        │

        │
        ├─ §12.5 Taylor 级数 (光滑函数 → 幂级数) + 3 余项 + 光滑非解析反例
        │
   §12.6 Fourier 级数 (周期函数 → 三角函数级数) + Dirichlet 收敛 + Parseval
```

### 12.0.2 五种级数对比

| 级数类型 | 一般形式 | 基函数 | 典型应用 |
| --- | --- | --- | --- |
| 数项级数 | $\sum u_n$($u_n\in\mathbb{R}$) | (无) | 极限求和、数值常数 $\pi^2/6,\ln 2,e$ |
| 函数项级数 | $\sum u_n(x)$ | 任意 | 函数逼近、求和函数 |
| 幂级数 | $\sum a_n(x-x_0)^n$ | 单项式 $(x-x_0)^n$ | 解析函数、生成函数、Newton 法 |
| Taylor 级数 | $\sum\frac{f^{(n)}(x_0)}{n!}(x-x_0)^n$ | 同上 | 光滑函数局部展开 |
| Fourier 级数 | $\frac{a_0}{2}+\sum(a_n\cos nx+b_n\sin nx)$ | $\sin nx,\cos nx$ | 周期函数 / 信号 / PDE 边值 |

### 12.0.3 三种"敛散性"模式

::intuition
学级数最容易搞混的地方:**"收敛"有几种**。

1. **逐点收敛**:$S_n(x)\to f(x)$ 对每个 $x$;
2. **绝对收敛**:$\sum|u_n|$ 也收敛;
3. **一致收敛**:$\sup_x|S_n(x)-f(x)|\to 0$,「均匀地」逼近;
4. **$L^2$ / 均方收敛**:$\int(f-S_n)^2\,dx\to 0$。

逻辑关系:**绝对收敛 ⇒ 收敛**;**一致收敛 ⇒ 逐点收敛**;但**逐点 ⇏ 一致**(Gibbs 现象 §12.6)。每种判别法都对应不同模式:Weierstrass M-判别给一致;Bessel 给 $L^2$;Leibniz 给逐点。
::end

### 12.0.4 与其它章的衔接

- **§9 微分**:Taylor 级数把光滑函数局部展开,Hessian 二阶判别(§9.8)、Newton 法(§9.10 补充)都基于一/二阶 Taylor;
- **§10 重积分**:积分判别(§12.2)、$\Gamma$ / $\beta$ 函数(§10.5)用级数与广义积分互推;
- **§11 三大公式**:Maxwell 方程组的解(电磁波)= Fourier 级数展开;
- **复变函数论**(上册之外):幂级数收敛区是**圆盘**;实数轴幂级数是它的截面;
- **概率 / AI**:Fourier 编码、Diffusion noise schedule、Word2Vec 词向量都用级数 / 谱方法。

### 12.0.5 入门:为什么"无限加"居然能得到有限数?

::intuition
回答 Zeno 悖论的关键:**和不是"加完",而是"逼近"**。

定义:$\sum u_n=S$ $\iff$ 部分和 $S_N=\sum_{n\le N}u_n\to S$。「无限加」从未真的"完成"过——我们只是说"无论你要多接近 $S$,从某个 $N$ 之后部分和总在那个距离内"。

阿喀琉斯每秒跑 10 米,乌龟每秒 1 米,起点差 9 米:
- 第 1 段:阿喀琉斯跑到乌龟原位用 0.9 秒,乌龟又前进 0.9 米;
- 第 2 段:再追 0.9 米用 0.09 秒,乌龟前进 0.09 米;
- $\dots$
总时间 $T=0.9+0.09+0.009+\dots=\frac{0.9}{1-0.1}=1$ 秒。**这是几何级数 $\sum 0.9\cdot 0.1^n=1$,有限!** Zeno 的"无穷段"在数学上和等于 $1$ 秒,在该秒后阿喀琉斯就追上了。**关键是「无穷个项和有限」并不矛盾**——只要项衰减够快。
::end

### 12.0.6 怎么读这章

1. **必背 4 个参考级数的敛散性**:几何 $r$、p-级数 $1/n^p$、调和 $1/n$、$e^x$ 展开;
2. **必会 6 法判别**:比较 / 极限比 / 比值 / 根值 / 积分 / Leibniz;
3. **理解收敛半径**:Cauchy-Hadamard 永真,比值版本需要极限存在;
4. **熟练 5 个 Maclaurin 展开**:$e^x,\sin x,\cos x,\ln(1+x),(1+x)^\alpha$;
5. **Fourier 系数公式记住**:$a_n=\frac{1}{\pi}\int f\cos nx$,$b_n=\frac{1}{\pi}\int f\sin nx$。
