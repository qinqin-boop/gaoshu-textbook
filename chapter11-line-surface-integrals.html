第十一章 曲线积分与曲面积分

对照: 同济大学《高等数学》第七版下册 第十一章
一元定积分在直线上, 重积分在区域上, 这章在曲线和曲面上.

11.0 整章在干啥?

之前积分都在直线段 (一元) 或平面区域/立体区域 (重积分). 现实问题:

  • 沿弯曲的钢丝丝求质量 → 第一类曲线积分
  • 力沿弯曲路径做的功 → 第二类曲线积分
  • 弯曲钢板的质量 → 第一类曲面积分
  • 流体穿过曲面的通量 → 第二类曲面积分

核心工具: 格林公式 / 高斯公式 / 斯托克斯公式 — 把曲线/曲面积分转化为简单的重积分.

对你的意义: 电磁学麦克斯韦方程组 / 流体动力学 / 计算机图形学的光照渲染 / Bezier 曲线长度 — 全用.

11.0.5 入门: 为啥从"区域上积分" 升级到"曲线/曲面上"?

中学/上册的 ∫f(x) dx 在线段上, 这章升级到"弯曲的线"和"弯曲的面" 上积分.

两类曲线积分:

  • 第一类: 沿曲线长度积. 不带方向. 例: 一根弯曲钢丝的质量 (密度不均).
  • 第二类: 沿曲线"坐标方向" 积. 带方向. 例: 力沿弯曲路径做的功.

两类曲面积分 (类比一下):

  • 第一类: 对曲面面积积. 例: 弯曲钢板的质量.
  • 第二类: 对曲面"坐标" 积. 例: 流体穿过曲面的通量.

为啥不直接用重积分?

  • 重积分是"区域上 (xy 平面/立体)"积. 弯曲线/弯曲面不是简单区域, 要新方法.
  • 用"参数化": 把曲线/曲面写成参数形式, 转化成参数空间上的简单积分.

这章核心:

  • 4 个新积分: 第一/二类曲线 + 第一/二类曲面
  • 3 大公式: 格林 / 高斯 / 斯托克斯 — 把曲线/曲面积分转化为重积分

学完你能描述麦克斯韦方程组 (电磁学终极公式) 的数学结构.

11.1 第一类曲线积分 (对弧长)

知识点 1: 定义

曲线 L 上, 密度 μ(x,y) 不均匀, 求质量. ∫_L μ(x,y) ds ds = 曲线上小弧长元.

生活类比: 一根弯曲钢丝, 每点密度不同, 总质量 = 切成小段, 每段密度×长度, 加起来.

知识点 2: 计算 — 参数法

曲线参数 x = x(t), y = y(t), t ∈ [α, β]: ∫_L f ds = ∫_α^β f(x(t), y(t)) √(x'(t)² + y'(t)²) dt √(x'² + y'²) dt = ds, 弧微元公式.

例题 1: 求圆周 x² + y² = 1 上 ∫ x² ds. 参数: x = cos t, y = sin t, t ∈ [0, 2π]. x'² + y'² = sin² t + cos² t = 1, ds = dt. ∫ = ∫₀^(2π) cos² t dt = π

例题 1-2: 一根直线段 L 从 (0,0) 到 (3,4), 线密度 ρ(x,y) = x+y, 求质量. 参数 x = 3t, y = 4t, t ∈ [0,1], ds = √(9+16) dt = 5dt. M = ∫_L (x+y) ds = ∫_0¹ (3t+4t) · 5 dt = 35 · 1/2 = 17.5.

例题 1-3: 求半圆 y = √(1-x²) (y≥0, x ∈ [-1,1]) 的弧长. 弧长 = ∫_L 1 ds. 参数: x = cos t, y = sin t, t ∈ [0, π], ds = dt. 弧长 = ∫_0^π dt = π. ✓ (半径 1 半圆的弧长 = πR = π)

知识点 1.5: 第一类曲线积分的物理应用 (对齐书 §11.1 一)

第一类曲线积分能算很多物理量, 都是"密度乘弧长" 的累计:

  1. 弧长: L = ∫_L 1 ds (密度 = 1)
  2. 质量: M = ∫_L μ(x,y,z) ds
  3. 质心:

x̄ = ∫_L x·μ ds / M, ȳ = ∫_L y·μ ds / M

  1. 转动惯量 (绕 z 轴): I_z = ∫_L (x²+y²) μ ds

    2. 例题 1-2 (质心): 一根均匀钢丝沿曲线 y = x² (0 ≤ x ≤ 1), 求质心 x̄. 参数: x = t, y = t², ds = √(1+4t²) dt. M = ∫_0¹ √(1+4t²) dt (设密度 1, 弧长本身). x̄·M = ∫_0¹ t·√(1+4t²) dt = (1/12)[(1+4t²)^(3/2)]_0¹ = (5√5 - 1)/12. M = (1/2)[t√(1+4t²) + (1/2) sinh^(-1)(2t)]_0¹ = √5/2 + ln(2+√5)/4. (具体值代入算)

    知识点 2.5: 第一类曲线积分性质

    • 方向无关: 沿 L 正向走或反向走, 积分值一样 (因为 ds > 0)
    • 可加性: ∫_(L₁+L₂) f ds = ∫_(L₁) + ∫_(L₂)
    • 线性: ∫ (af+bg) ds = a∫f + b∫g

    11.2 第二类曲线积分 (对坐标)

    知识点 2.7: 第一类曲线积分参数法详细 (对齐书 §11.1 二)

    情况 1: 一般参数 x=x(t), y=y(t), t ∈ [α, β] ds = √(x'(t)² + y'(t)²) dt ∫_L f ds = ∫_α^β f(x(t), y(t)) · √(x'² + y'²) dt

    情况 2: 直接 y = y(x), x ∈ [a, b] (相当于 x=x, y=y(x)) ds = √(1 + y'(x)²) dx ∫_L f ds = ∫_a^b f(x, y(x)) · √(1+y'²) dx

    情况 3: 极坐标 r = r(θ), θ ∈ [α, β] ds = √(r² + r'²) dθ ∫_L f ds = ∫_α^β f(r cosθ, r sinθ) · √(r²+r'²) dθ

    例题 1-3 (一般参数): 螺旋线 x = cos t, y = sin t, z = t (t ∈ [0, 2π]) 上 ∫(x²+y²+z²) ds. ds = √(sin²+cos²+1) dt = √2 dt. x²+y²+z² = 1 + t². ∫ = ∫_0^(2π) (1+t²) · √2 dt = √2 · [t + t³/3]_0^(2π) = √2 · (2π + 8π³/3).

    ---

    知识点 3: 定义 (力沿路径做功)

    变力 F→(x,y) = (P, Q) 沿曲线 L 做的功: W = ∫_L P dx + Q dy

    生活类比: 你推车, 力跟方向都变, 走弯曲路径, 总功 = F→ · dr→ 沿路径积分.

    第一类 vs 第二类区别:

    • 第一类: ds 是标量 (长度), 跟方向无关
    • 第二类: dx, dy 带方向 (反向走积分变号)

    知识点 4: 计算

    参数 x = x(t), y = y(t): ∫_L P dx + Q dy = ∫_α^β [P(x(t), y(t)) x'(t) + Q(x(t), y(t)) y'(t)] dt

    例题 2: ∫_L y dx + x dy, L 是从 (0,0) 到 (1,1) 的直线. 参数: x = t, y = t, t ∈ [0, 1], x' = y' = 1. = ∫_0¹ [t · 1 + t · 1] dt = ∫_0¹ 2t dt = 1

    例题 2-2 (功的实际计算): 力场 F→ = (-y, x), 沿单位圆逆时针走一圈做的功? 参数 x = cos t, y = sin t, t ∈ [0, 2π], dx = -sin t dt, dy = cos t dt. W = ∫_L P dx + Q dy = ∫_0^(2π) [(-sin t)(-sin t) + cos t · cos t] dt = ∫_0^(2π) (sin² t + cos² t) dt = ∫_0^(2π) 1 dt = 2π.

    例题 2-3 (路径相同, 端点不同, 结果不同): ∫_L (y dx - x dy), L 从 (0,0) 到 (1,1).

    • 路径 A (直线 y = x): 参数 x = t, y = t. = ∫_0¹ (t · 1 - t · 1) dt = 0.
    • 路径 B (折线先 (0,0)→(1,0) 再 (1,0)→(1,1)):

    段 1: y=0, dy=0, x: 0→1, ∫ 0 dx - x · 0 = 0. 段 2: x=1, dx=0, y: 0→1, ∫ y · 0 - 1 · dy = -1. 总 = -1. 结论: A 跟 B 端点相同但路径不同 → 积分不同 (说明这不是保守场, 不满足路径无关条件).

    11.3 格林公式 (本章核心 1)

    知识点 5: 格林公式

    联系第二类曲线积分跟二重积分.

    D 是平面闭区域, ∂ D 是它的边界 (取正方向: 逆时针): ∮_(∂ D) P dx + Q dy = ∬_D ((∂ Q)/(∂ x) - (∂ P)/(∂ y)) dxdy

    思想: 沿闭曲线积分 = 内部"旋度"的累加.

    例题 3: 用格林公式求 ∮_C (x+y) dx + (x-y) dy, C 是单位圆周逆时针. P = x+y, Q = x-y. Q_x = 1, P_y = 1, 被积函数 Q_x - P_y = 0. 所以 ∮ = 0.

    例题 3-2 (用格林公式算面积): 用 ∮ x dy 算单位圆面积. S = ∮_C x dy. 参数 x = cos t, y = sin t, dy = cos t dt. = ∫_0^(2π) cos t · cos t dt = ∫_0^(2π) cos² t dt = π. ✓

    面积公式: 任何由闭曲线 C 围的平面区域 D 的面积: S = (1/2) ∮_C (x dy - y dx) = ∮_C x dy = -∮_C y dx (三个公式都对, 选 C 上容易算的)

    例题 3-3: 用格林公式算 ∮_C 2xy dx + x² dy, C 是顺时针单位圆周. 逆时针时: P = 2xy, Q = x², Q_x = 2x, P_y = 2x, Q_x - P_y = 0 → ∮ = 0. 顺时针 = -(逆时针) = 0.

    知识点 5.5: 格林公式的推导 (对齐书 §11.3 一)

    直觉: 把闭区域 D 切成大量小矩形 ΔS, 每个小矩形里 (Q_x - P_y) ΔS 等于沿这个小矩形边界的 P dx + Q dy. 相邻小矩形共边方向相反相消, 加起来只剩最外圈 = ∮_(∂D).

    推论一: 用格林公式算面积

    S(D) = (1/2) ∮_(∂D) (x dy - y dx) = ∮ x dy = -∮ y dx

    (三个公式都对, 选 ∂D 上容易算的)

    例题 3-4: 用格林公式算椭圆 x²/a² + y²/b² = 1 的面积. 参数 x = a cos t, y = b sin t. dx = -a sin t dt, dy = b cos t dt. S = (1/2)∮(x dy - y dx) = (1/2)∫_0^(2π) [a cos t · b cos t - b sin t · (-a sin t)] dt = (1/2) · ab · ∫_0^(2π) (cos²t + sin²t) dt = (1/2) · ab · 2π = πab ✓

    应用价值: 不规则区域面积难算, 沿边界一圈积分往往容易.

    知识点 6: 平面上曲线积分与路径无关

    ∫_L P dx + Q dy 跟路径无关 ⟺ (∂ Q)/(∂ x) = (∂ P)/(∂ y) (单连通区域).

    ⟺ 存在原函数 u(x,y) 使得 du = P dx + Q dy.

    生活类比: "重力做功只跟起点终点高度差有关, 走啥路径无关" — 重力是保守力, 满足这个.

    11.4 第一类曲面积分 (对面积)

    知识点 6.5: 全微分原函数 (对齐书 §11.3 三)

    问题: P dx + Q dy 啥时候是某个函数 u(x,y) 的全微分?

    条件: ∂Q/∂x = ∂P/∂y (单连通区域) 此时存在 u 使 du = P dx + Q dy, u 称为原函数.

    求 u 的方法:

    1. 对 ∂u/∂x = P 关于 x 积分: u = ∫P dx + g(y)
    2. 求 ∂u/∂y, 跟 Q 比较, 解出 g(y)

      3. 例题 6-3: 验证 (2x+y) dx + (x+2y) dy 是全微分, 求原函数. P = 2x+y, Q = x+2y. ∂Q/∂x = 1 = ∂P/∂y ✓. u = ∫(2x+y) dx = x² + xy + g(y). ∂u/∂y = x + g'(y) = Q = x + 2y → g'(y) = 2y → g(y) = y². u = x² + xy + y². (验证: du = (2x+y)dx + (x+2y)dy ✓)

      物理意义: 保守力场存在势能函数. 重力/弹性力/库仑电力都是保守的, 总有"势能".

      知识点 7: 定义

      曲面 Σ 上密度 μ(x,y,z), 求质量: ∬_Σ μ(x,y,z) dS dS = 曲面上小面积元.

      知识点 8: 计算

      曲面 z = z(x,y), 投影到 xy 平面得 D_(xy): dS = √(1 + z_x² + z_y²) dxdy ∬_Σ f dS = ∬_(D_(xy)) f(x,y,z(x,y)) √(1+z_x²+z_y²) dxdy

      例题 4: ∬_Σ z dS, Σ 是 z = 1 - x² - y² 在 z ≥ 0 部分. z_x = -2x, z_y = -2y, √(1+4x²+4y²). 投影 D_(xy): x²+y² ≤ 1. ∬ (1-x²-y²)√(1+4(x²+y²)) dxdy 换极坐标 r² = x²+y²: = ∫₀^(2π)∫_0¹ (1-r²)√(1+4r²) · r drdθ (具体值代入算)

      ---

      知识点 8.5: 曲面面积公式推导 (对齐书 §11.4)

      问题: 曲面 z = f(x,y) 在区域 D 上的面积 A 怎么算?

      思想: 把 D 切小, 每个小区域 ΔS_xy 对应曲面上一片倾斜的小面积 ΔS. 倾斜板面积 = 平面投影面积 / |cos α|, 其中 α 是切平面跟 xy 面的夹角.

      cos α = 1 / √(1 + f_x² + f_y²)

      所以 ΔS = √(1 + f_x² + f_y²) · ΔS_xy.

      加和取极限: A = ∬_D √(1 + f_x² + f_y²) dxdy

      例题 4-2 (半球面积): 半球面 z = √(R² - x² - y²) (z ≥ 0) 的面积. f_x = -x/z, f_y = -y/z. 1 + f_x² + f_y² = 1 + (x²+y²)/z² = R²/(R²-x²-y²). A = ∬_(x²+y²≤R²) R/√(R²-x²-y²) dxdy. 极坐标: A = ∫_0^(2π) ∫_0^R R·r/√(R²-r²) drdθ = 2πR · [-√(R²-r²)]_0^R = 2πR · R = 2πR². ✓ (整个球面 4πR², 半个 2πR² ✓)

      ---

      11.5 第二类曲面积分 (对坐标)

      知识点 9: 定义 (流量/通量)

      流体的流速场 v→ = (P, Q, R), 流过有向曲面 Σ 的通量: ∬_Σ P dydz + Q dzdx + R dxdy

      生活类比: 河流流过一片网, 通量 = 单位时间穿过网的水量.

      知识点 10: 高斯公式 (本章核心 2)

      联系闭合曲面通量跟体积内散度.

      V 是空间闭区域, ∂ V 是它的边界曲面 (外侧): ∯_(∂ V) P dydz + Q dzdx + R dxdy = ∭_V ((∂ P)/(∂ x) + (∂ Q)/(∂ y) + (∂ R)/(∂ z)) dV

      通俗: 闭曲面流出去多少 = 内部"产出"多少 (散度).

      例题 5: ∯_Σ x dydz + y dzdx + z dxdy, Σ 是单位球外侧. P_x + Q_y + R_z = 1 + 1 + 1 = 3. = ∭_V 3 dV = 3 · (4)/(3)π = 4π 经典!

      例题 5-2 (求体积的高斯公式技巧): 用 ∯ z dxdy 算单位球体积. P = 0, Q = 0, R = z. P_x + Q_y + R_z = 1. ∯ z dxdy = ∭_V 1 dV = V (球体积). 所以 V = ∯ z dxdy. 等价地, V = (1/3) ∯ (x dydz + y dzdx + z dxdy). (代入单位球: V = (1/3) · 4π = 4π/3 ✓)

      例题 5-3: 求向量场 F→ = (x², y², z²) 通过单位立方体表面 (外侧) 的通量. div F = 2x + 2y + 2z. ∯ F· dS = ∭_V (2x+2y+2z) dV = 2 ∫_0¹∫_0¹∫_0¹ (x+y+z) dx dy dz = 2 · 3 · (1/2) = 3. (对称性: ∭ x dV = ∭ y dV = ∭ z dV = 1/2)

      ---

      11.6 斯托克斯公式 (本章核心 3)

      知识点 10.5: 高斯公式 vs 闭曲面 vs 开曲面 (对齐书 §11.6)

      高斯公式只对闭合曲面成立. 如果是开曲面 (没"包围立体"), 不能直接用.

      应对策略: 加一个"补面" 让它闭合, 算两次相减:

      • 设开曲面 Σ, 补一个 Σ' 让 Σ ∪ Σ' 闭合包围 V
      • 用高斯: ∯_(Σ∪Σ') = ∭_V div F dV
      • 反推: ∬_Σ = ∭ - ∬_(Σ')

      例题 5-4: ∬_Σ x dydz + y dzdx + z dxdy, Σ 是 z = 1-x²-y² (z ≥ 0) 上侧. 不是闭合. 补 Σ' = {z=0, x²+y²≤1} 下侧. Σ ∪ Σ' 包围立体 V (上方抛物面下方圆盘). div(P,Q,R) = 1+1+1 = 3. ∯ = 3·V体积 = 3·∫_0¹ π(1-r²)·... 算得 3π/2. Σ' 部分: 在 z=0 平面, dz=0, x, y 部分也不贡献. ∬_(Σ') = 0. 所以 ∬_Σ = 3π/2.

      知识点 11: 斯托克斯公式

      联系有向曲面的旋度 跟 边界曲线的环流量.

      Σ 是空间有向曲面, ∂ Σ 是它的边界 (取相应正方向): ∮_(∂ Σ) P dx + Q dy + R dz = ∬_Σ | dydz & dzdx & dxdy ∂/∂ x & ∂/∂ y & ∂/∂ z P & Q & R |

      通俗: 沿边界绕一圈的环流量 = 曲面上"旋度"累加. 格林公式是它的平面特例.

      知识点 12: 三大公式总结

      | 公式 | 联系 | |---|---| | 牛顿-莱布尼茨 (上册) | 区间积分 ↔ 端点值 | | 格林公式 (平面) | 闭曲线 ↔ 平面区域 | | 高斯公式 (空间) | 闭曲面 ↔ 立体体积 | | 斯托克斯公式 (空间) | 空间闭曲线 ↔ 曲面 |

      统一思想: 边界积分 = 内部某种"导数"的积分. 这是微积分基本定理的高维推广. 物理叫"边界 → 内部"对应关系.

      11.6 通量与散度 (对齐书 §11.6 三)

      知识点 12.5: 通量 (flux)

      定义: 向量场 F→ = (P, Q, R) 通过有向曲面 Σ 的通量: Φ = ∬_Σ F→ · dS→ = ∬_Σ P dydz + Q dzdx + R dxdy

      生活类比: 流体的速度场 F→, 通量 Φ = 单位时间穿过曲面 Σ 的流量 (体积).

      知识点 12.6: 散度 (divergence)

      定义: 标量场, 描述向量场在某点"产出"或"吸收"的强度: div F→ = ∂P/∂x + ∂Q/∂y + ∂R/∂z = ∇ · F→

      几何意义:

      • div F→ > 0: 该点是源 (向外发射)
      • div F→ < 0: 该点是汇 (向内吸收)
      • div F→ = 0: 该点既不源也不汇 (流入 = 流出, 称"无源场")

      生活类比: 一片湖, 在某点散度 > 0 说明这里有泉眼往外冒水; 散度 < 0 说明这里有漏洞往里吸水.

      高斯公式 (向量形式): ∯_(∂V) F→ · dS→ = ∭_V div F→ dV 通量 = 散度的体积积分. 闭曲面通量 = 内部源/汇 累计.

      11.7 环流量与旋度 (对齐书 §11.7 三)

      知识点 12.7: 环流量 (circulation)

      定义: 向量场 F→ 沿闭曲线 L 的环流量: Γ = ∮_L F→ · dr→ = ∮_L P dx + Q dy + R dz

      生活类比: 沿闭曲线"绕一圈" F→ 做的功. 涡漩中心环流量大.

      知识点 12.8: 旋度 (curl)

      定义: 向量场, 描述某点"旋转"强度: rot F→ = ∇ × F→ = | i→ j→ k→ ∂/∂x ∂/∂y ∂/∂z P Q R |

      几何意义: 在某点放一个小风车, 旋度方向 = 风车轴方向, 旋度大小 = 转速.

      • rot F→ = 0 处: 无旋场 (流体不打转)
      • |rot F→| 大: 涡漩中心

      生活类比: 龙卷风中心旋度极大, 平直气流旋度为 0.

      斯托克斯公式 (向量形式): ∮_(∂Σ) F→ · dr→ = ∬_Σ rot F→ · dS→ 环流量 = 旋度通过曲面 的通量. 闭曲线环流 = 内部旋转累计.

      物理意义对应:

      • 麦克斯韦电磁: ∇·E = ρ/ε₀ (高斯定律, 电场散度跟电荷密度) / ∇×B = μ₀J + ... (安培定律, 磁场旋度跟电流)
      • 流体: ∇·v→ = 0 (不可压缩) / ∇×v→ = 0 (无旋流)

      ---

      知识点 12.9: 三大公式统一思想 (Generalized Stokes 定理)

      牛顿-莱布尼茨 / 格林公式 / 高斯公式 / 斯托克斯公式 — 一个公式的不同维度形态:

      | 公式 | 维度 | 边界 | 内部 | |---|---|---|---| | 牛顿-莱布尼茨 | 1D | 端点 a, b | 区间 [a, b] | | 格林 | 2D | 闭曲线 ∂D | 平面区域 D | | 高斯 | 3D | 闭曲面 ∂V | 立体 V | | 斯托克斯 | 2D 嵌入 3D | 曲面边界 ∂Σ | 曲面 Σ |

      统一: ∫_(∂M) ω = ∫_M dω (广义斯托克斯公式, 微分流形上)

      意义: 这四个公式都是微积分基本定理的高维推广 — "积分跟原函数 (或反导数) 的端点值差" 这件事, 从 1D 推广到任意维.

      应用:

      • 流体动力学: Navier-Stokes 方程
      • 电磁学: Maxwell 方程组 (4 个方程都是这几个公式的物理形式)
      • 广义相对论: 黎曼几何积分
      • 拓扑学: 同调论

      知识点 12.10: 三大公式记忆口诀

      口诀 (用关系而不用公式):

      • "闭曲线绕一圈 = 平面区域内每点的"旋度" 累加" → 格林
      • "闭曲面通量 = 立体内每点的"散度" 累加" → 高斯
      • "空间闭曲线环量 = 曲面上每点的"旋度" 累加" → 斯托克斯
      • 关键 idea: 边界 = 内部 d 算子的积分

      第十一章 小结

      • 第一类曲线/曲面积分: 对长度/面积, 不带方向
      • 第二类曲线/曲面积分: 对坐标, 带方向 (反向变号)
      • 格林公式: 闭曲线积分 = 平面二重积分
      • 高斯公式: 闭曲面通量 = 立体三重积分
      • 斯托克斯公式: 空间闭曲线环量 = 曲面积分
      • 关键 idea: 边界积分 = 内部累加

      实际中: 麦克斯韦方程组就是高斯+斯托克斯写的; 流体的连续性方程也是高斯; 电磁感应是斯托克斯.

      知识点 12.11: 麦克斯韦方程组 — 微积分应用经典

      麦克斯韦电磁学 4 个方程, 都是高数下册的公式 — 这章学的就是它们的数学基础:

      1. 高斯定律 (电): ∮_(∂V) E→ · dS→ = Q/ε₀

      • 类: 高斯公式 (闭曲面通量 = 内部电荷)
      • 微分形式: ∇·E→ = ρ/ε₀ (电场散度 = 电荷密度)

      2. 高斯定律 (磁): ∮_(∂V) B→ · dS→ = 0

      • 闭曲面磁通量 = 0 (磁场没"源", 只有"环")
      • 微分形式: ∇·B→ = 0

      3. 法拉第电磁感应: ∮_L E→ · dr→ = -dΦ/dt

      • 类: 斯托克斯公式 (环量 = 旋度通量)
      • 微分形式: ∇×E→ = -∂B→/∂t

      4. 安培-麦克斯韦定律: ∮_L B→ · dr→ = μ₀(I + ε₀ dΦ_E/dt)

      • 类: 斯托克斯公式
      • 微分形式: ∇×B→ = μ₀(J→ + ε₀ ∂E→/∂t)

      学完高数下册你就能"看懂"麦克斯韦方程组的数学结构 — 即使物理细节不懂, 数学层面所有符号 (∇·, ∇×, 闭曲面/线积分) 都是这章教的.

      知识点 13: 曲线/曲面积分应用大全 (本章工程价值)

      1. 物理学经典:

      • 力做的功: W = ∫_L F→·dr→
      • 质量: M = ∫_L μ ds (一维) 或 ∬_Σ μ dS (二维)
      • 流体通量 Φ = ∬_Σ v→·dS→
      • 引力场强 / 电场强度
      • 电流: I = ∬_Σ J→·dS→

      2. 电磁学 (Maxwell 完整版):

      • 电场高斯定律: 给定电荷 → 推算电场分布
      • 磁场: 安培定律由电流推磁场
      • 法拉第感应: 磁通变化 → 电动势
      • 电磁波传播: Maxwell 方程 → 波动方程

      3. 流体力学 (Navier-Stokes):

      • 不可压缩: ∇·v→ = 0
      • 涡量: ω→ = ∇×v→
      • 边界条件: 闭曲面通量 = 流入流出之差

      4. 计算机图形 (现代渲染):

      • 表面积 → 决定阴影面积
      • 立体角积分 → 路径追踪
      • BRDF / BTDF → 全局光照

      5. 几何与拓扑:

      • 曲面面积 (机翼/船体 设计常用)
      • 体积公式 (高斯公式特例)
      • 计算流形曲率 (黎曼几何)

      6. 工程问题:

      • 飞机翼型升力: ∮ ρv→·dr→ 沿翼型周线
      • 船体推进力: 桨叶曲面通量
      • 风洞测试: 力和力矩都是曲线/曲面积分

      学完这章 = 你能"算" 复杂三维场景里的"总量" (功/通量/质量/势).
