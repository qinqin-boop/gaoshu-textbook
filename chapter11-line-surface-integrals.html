11.0 整章在干啥?

升级到弯曲对象上积分

之前积分都在直线段 (一元 $\int f\,dx$) 或平面/立体区域 (重积分). 现实问题:

  • 沿弯曲钢丝求质量 → 第一类曲线积分 $\int_L \mu\,ds$
  • 力沿弯曲路径做功 → 第二类曲线积分 $\int_L \vec{F}\cdot d\vec{r}$
  • 弯曲钢板质量 → 第一类曲面积分 $\iint_\Sigma \mu\,dS$
  • 流体穿过曲面通量 → 第二类曲面积分 $\iint_\Sigma \vec{F}\cdot d\vec{S}$

核心工具: Green / Gauss / Stokes 三大公式, 把曲线/曲面积分转回简单重积分.

对你的意义: 电磁学 Maxwell 方程组 / 流体动力学 N-S 方程 / 计算机图形学光照渲染 / Bezier 曲线长度 — 全靠这章.

11.0.5 入门: 为啥从"区域上积分" 升级到"曲线/曲面上"?

升级地图: 4 种积分 + 3 大公式

中学/上册的 $\int f(x)\,dx$ 在直线段上, 这章升级到"弯曲线/面"上积分.

4 种新积分: | 类型 | 几何对象 | 带方向? | 物理例子 | |---|---|---|---| | 第一类曲线 | 弧长 $ds$ | 不带 | 弯钢丝质量 | | 第二类曲线 | 坐标 $dx, dy$ | 带 | 力做功 | | 第一类曲面 | 面积 $dS$ | 不带 | 弯钢板质量 | | 第二类曲面 | 投影坐标 $dydz$ 等 | 带 | 流体通量 |

3 大公式 (把曲线/曲面积分转回重积分):

  • Green: 平面闭路 ↔ 平面区域
  • Gauss: 闭曲面通量 ↔ 立体内散度
  • Stokes: 曲面边界环流 ↔ 曲面旋度

学完你能描述麦克斯韦方程组 (电磁学终极公式) 的数学结构.

为啥不直接用重积分?

  • 重积分只能在"区域上 ($xy$ 平面 / 立体)"积. 弯曲线 / 弯曲面不是简单区域, 要新方法.
  • 参数化大法: 把曲线 / 曲面写成参数形式 $\vec{r}(t)$ 或 $\vec{r}(u, v)$, 转化成参数空间上的简单积分.
  • 3 大公式作用: 把"曲线 / 曲面上的积分" 转回"重积分" — 难变易.

学完能达到: 描述麦克斯韦方程组 (电磁学终极公式) 的数学结构, 这是大学微积分的真正顶峰.

11.1 第一类曲线积分 (对弧长)

知识点 1: 定义

第一类曲线积分 (对弧长)

曲线 L 上, 密度 $\mu(x,y)$ 不均匀, 求质量. $$\int_L \mu(x,y)\,ds$$ $ds$ = 曲线上小弧长元素.

弯钢丝直觉 — 一维"密度 × 长度累加"

一根弯曲钢丝, 每点密度不同, 总质量 = 切成无数小段 → 每段密度 $\mu$ × 弧长 $ds$ → 累加 → 极限.

跟一元 $\int_a^b f(x)\,dx$ 区别:

  • 直线段在 $x$ 轴: $ds = dx$
  • 弯曲路径: $ds = \sqrt{x'^2 + y'^2}\,dt$ (毕达哥拉斯算斜边)

为啥不分方向: $ds > 0$ 永远正 (长度本质非负), 反向走总质量不变.

知识点 2: 计算 — 参数法

参数法弧微元公式

曲线 $L$ 参数 $x = x(t), y = y(t)$, $t \in [\alpha, \beta]$: $$\int_L f(x,y)\,ds = \int_\alpha^\beta f(x(t), y(t)) \sqrt{x'(t)^2 + y'(t)^2}\,dt$$ 其中 $ds = \sqrt{x'^2 + y'^2}\,dt$ 是弧微元.

例题 1

求圆周 x² + y² = 1 上 ∫ x² ds. 参数: x = cos t, y = sin t, t ∈ [0, 2π]. x'² + y'² = sin² t + cos² t = 1, ds = dt. ∫ = ∫₀^(2π) cos² t dt = $\pi$

例题 1-2

一根直线段 L 从 (0,0) 到 (3,4), 线密度 ρ(x,y) = x+y, 求质量. 参数 x = 3t, y = 4t, t ∈ [0,1], ds = √(9+16) dt = 5dt. M = ∫_L (x+y) ds = ∫_0¹ (3t+4t) · 5 dt = 35 · 1/2 = 17.5.

例题 1-3

求半圆 y = √(1-x²) (y$\ge$0, x ∈ [-1,1]) 的弧长. 弧长 = ∫_L 1 ds. 参数: x = cos t, y = sin t, t ∈ [0, $\pi$], ds = dt. 弧长 = ∫_0^$\pi$ dt = $\pi$. ✓ (半径 1 半圆的弧长 = πR = $\pi$)

知识点 1.5: 第一类曲线积分的物理应用 (对齐书 §11.1 一)

第一类曲线积分 4 大物理量

都是"密度乘弧长"的累积:

  1. 弧长: $L = \int_L 1\,ds$ (密度 = 1)
  2. 质量: $M = \int_L \mu(x,y,z)\,ds$
  3. 质心: $\bar{x} = \int_L x \mu\,ds / M$, $\bar{y} = \int_L y \mu\,ds / M$
  4. 转动惯量 (绕 z 轴): $I_z = \int_L (x^2+y^2) \mu\,ds$

跟二重积分类比

| 二维平板 (ch10) | 弯曲线 (ch11) | |---|---| | 质量 = $\iint \mu\,d\sigma$ | 质量 = $\int_L \mu\,ds$ | | 重心 = $\iint x\mu\,d\sigma / M$ | 重心 = $\int_L x\mu\,ds / M$ | | 转惯 = $\iint (x^2+y^2)\mu\,d\sigma$ | 转惯 = $\int_L (x^2+y^2)\mu\,ds$ |

公式同形, 只是积分对象从"面积"变成"弧长". 物理 / 几何含义完全对应.

例题 1-2

(质心): 一根均匀钢丝沿曲线 y = x² (0 ≤ x ≤ 1), 求质心 x̄. 参数: x = t, y = t², ds = √(1+4t²) dt. M = ∫_0¹ √(1+4t²) dt (设密度 1, 弧长本身). x̄·M = ∫_0¹ t·√(1+4t²) dt = (1/12)[(1+4t²)^(3/2)]_0¹ = (5$\sqrt{5}$ - 1)/12. M = (1/2)[t√(1+4t²) + (1/2) sinh^(-1)(2t)]_0¹ = √5/2 + ln(2+√5)/4. (具体值代入算)

知识点 2.5: 第一类曲线积分性质

第一类曲线积分性质

  1. 方向无关: 沿 $L$ 正向或反向走, 积分值一样 (因为 $ds > 0$)
  2. 可加性: $\int_{L_1 + L_2} f\,ds = \int_{L_1} f\,ds + \int_{L_2} f\,ds$
  3. 线性: $\int_L (af + bg)\,ds = a\int_L f\,ds + b\int_L g\,ds$

跟普通定积分像

第一类曲线积分本质上"把弯曲路径当直线段算长度+加权", 跟一元 $\int_a^b f(x)\,dx$ 性质一致.

11.2 第二类曲线积分 (对坐标)

知识点 2.7: 第一类曲线积分参数法详细 (对齐书 §11.1 二)

三种参数化弧微元公式

1. 一般参数 $x = x(t), y = y(t), t \in [\alpha, \beta]$: $$ds = \sqrt{x'(t)^2 + y'(t)^2}\,dt$$

2. 直接 $y = y(x), x \in [a, b]$ (即 $x = x, y = y(x)$): $$ds = \sqrt{1 + y'(x)^2}\,dx$$

3. 极坐标 $r = r(\theta), \theta \in [\alpha, \beta]$: $$ds = \sqrt{r^2 + r'(\theta)^2}\,d\theta$$

选参数化的实战经验

  • 参数方程直接给: 用情况 1
  • 函数形式 $y = f(x)$: 用情况 2 (无需引参数)
  • 圆 / 心脏线 / 玫瑰线: 用情况 3 (极坐标天然)

例题 1-3

(一般参数): 螺旋线 x = cos t, y = sin t, z = t (t ∈ [0, 2π]) 上 ∫(x²+y²+z²) ds. ds = √(sin²+cos²+1) dt = √2 dt. x²+y²+z² = 1 + t². ∫ = ∫_0^(2π) (1+t²) · √2 dt = √2 · [t + t³/3]_0^(2π) = √2 · (2π + 8π³/3).

知识点 3: 定义 (力沿路径做功)

第二类曲线积分 (对坐标)

变力 $\vec{F}(x,y) = (P, Q)$ 沿曲线 L 做的功: $$W = \int_L \vec{F}\cdot d\vec{r} = \int_L P\,dx + Q\,dy$$

推车做功的直觉

你推车, 力跟方向都变, 走弯曲路径, 总功 = $\vec{F}\cdot d\vec{r}$ 沿路径累加. 这是"功 = 力 × 距离"的连续累加版.

第一类 vs 第二类核心区别

| 维度 | 第一类 (对弧长) | 第二类 (对坐标) | |---|---|---| | 微元 | $ds > 0$ (长度, 永远正) | $dx, dy$ (带正负) | | 方向相关 | 否 (反向积分不变) | 是 (反向变号) | | 被积量 | 标量 $f$ | 向量 $\vec{F}$ | | 物理 | 质量 / 长度积分 | 功 / 通量 | | 转换 | $\int_L P\,dx + Q\,dy = \int_L (P\cos\alpha + Q\cos\beta)\,ds$ ||

知识点 3.5: 第二类曲线积分跟方向相关 (对齐书 §11.2 一 进一步)

第二类必看方向

第二类曲线积分跟路径方向相关! 反向走积分变号: $$\int_{L^-} P\,dx + Q\,dy = -\int_L P\,dx + Q\,dy$$ 做题忘标方向 (逆时针/顺时针, 从 A 到 B 还是 B 到 A) → 符号错.

(L^- 表示 L 的反向)

物理意义: 力沿正方向做正功, 沿反方向做负功 (反向走"克服了"力).

例题 3-3

(验证): L 从 (0,0) 到 (1,0), $\vec{F}$ = (1, 0) (沿 x 轴恒力). 正方向: W = ∫_L 1·dx + 0·dy = ∫_0¹ 1 dx = 1. 反方向 (从 (1,0) 回 (0,0)): W = ∫_(L^-) = ∫_1⁰ 1 dx = -1. ✓

第一类对比: ∫_L f ds 不分方向 (因为 ds > 0 永远正).

知识点 4: 计算

第二类曲线积分参数法

曲线 $L$ 参数化 $x = x(t), y = y(t)$, $t \in [\alpha, \beta]$ (从 $\alpha$ 到 $\beta$ 即正向): $$\int_L P\,dx + Q\,dy = \int_\alpha^\beta \left[P(x(t), y(t)) \cdot x'(t) + Q(x(t), y(t)) \cdot y'(t)\right] dt$$ 关键: $dx \to x'(t)\,dt$, $dy \to y'(t)\,dt$ (微元用参数代换).

参数法的"代换"机制

$dx, dy$ 都是"位移微元", 在参数化下: $$d\vec{r} = (dx, dy) = (x'(t), y'(t))\,dt$$ 所以 $\vec{F}\cdot d\vec{r} = (P, Q)\cdot(x', y')\,dt = (Px' + Qy')\,dt$, 把曲线积分变成关于 $t$ 的一元定积分.

选参数化口诀:

  • 直线段 $A \to B$: 用 $\vec{r}(t) = A + t(B - A), t \in [0, 1]$
  • 圆 / 椭圆: 用三角函数 ($x = r\cos t, y = r\sin t$)
  • 折线: 分段写, 每段独立参数化, 结果相加

例题 2

∫_L y dx + x dy, L 是从 (0,0) 到 (1,1) 的直线. 参数: x = t, y = t, t ∈ [0, 1], x' = y' = 1. = ∫_0¹ [t · 1 + t · 1] dt = ∫_0¹ 2t dt = 1

例题 2-2

(功的实际计算): 力场 $\vec{F}$ = (-y, x), 沿单位圆逆时针走一圈做的功? 参数 x = cos t, y = sin t, t ∈ [0, 2π], dx = -sin t dt, dy = cos t dt. W = ∫_L P dx + Q dy = ∫_0^(2π) [(-sin t)(-sin t) + cos t · cos t] dt = ∫_0^(2π) (sin² t + cos² t) dt = ∫_0^(2π) 1 dt = 2π.

例题 2-3

(路径相同, 端点不同, 结果不同): ∫_L (y dx - x dy), L 从 (0,0) 到 (1,1).

  • 路径 A (直线 y = x): 参数 x = t, y = t. = ∫_0¹ (t · 1 - t · 1) dt = 0.
  • 路径 B (折线先 (0,0)→(1,0) 再 (1,0)→(1,1)):

段 1: y=0, dy=0, x: $\vec{0}$1, ∫ 0 dx - x · 0 = 0. 段 2: x=1, dx=0, y: $\vec{0}$1, ∫ y · 0 - 1 · dy = -1. 总 = -1. 结论: A 跟 B 端点相同但路径不同 → 积分不同 (说明这不是保守场, 不满足路径无关条件).

11.3 格林公式 · 平面流形上的微分与积分

第二类曲线积分把「沿曲线一圈的流量 / 功」算成一个标量。如果曲线恰好是一个封闭区域的边界,那么沿边界的环路积分应该跟区域内部的某个二重积分有联系——这一联系就是 Green 公式。它是 Gauss 公式(三维)与 Stokes 公式(任意维)在平面上的特例,也是经典「微分与积分互逆」(微积分基本定理 $\int_a^b f'=f(b)-f(a)$)在二维上的推广。

11.3.1 区域与边界的定向约定

设 $D\subset\mathbb{R}^2$ 是有界闭区域,其边界 $\partial D$ 是分段光滑的简单闭曲线(或若干条简单闭曲线的并)。约定 $\partial D$ 的正方向为:沿曲线行走时,$D$ 始终位于左手边。对单连通区域,正方向就是逆时针;对有洞的区域,外边界逆时针、内边界顺时针。记号 $\displaystyle\oint_{\partial D^+}P\,dx+Q\,dy$ 表示沿正向边界的环路积分。

11.3.2 Green 公式

(Green 公式)设 $D\subset\mathbb{R}^2$ 是有界闭区域,$\partial D$ 分段光滑,函数 $P(x,y),Q(x,y)$ 在 $D\cup\partial D$ 上有连续一阶偏导,则 $$\oint_{\partial D^+}\!P\,dx+Q\,dy=\iint_D\!\bigg(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\bigg)\,dx\,dy.$$

思路:先证 $D$ 为 Type I(简单 $y$-区域,即 $D=\{(x,y):a\le x\le b,\ \varphi_1(x)\le y\le\varphi_2(x)\}$)时分别有 $$\oint_{\partial D^+}\!P\,dx=-\iint_D\frac{\partial P}{\partial y}\,dx\,dy,\qquad\oint_{\partial D^+}\!Q\,dy=\iint_D\frac{\partial Q}{\partial x}\,dx\,dy.\qquad(*)$$ 两式相加即 Green。一般区域可剖分为有限个 Type I + Type II 区域,内部界线两次走方向相反,互相抵消,只剩外边界。

证第一式:对 Type I, $$\iint_D\frac{\partial P}{\partial y}\,dx\,dy=\int_a^b\!dx\int_{\varphi_1(x)}^{\varphi_2(x)}\frac{\partial P}{\partial y}\,dy=\int_a^b\!\big[P(x,\varphi_2(x))-P(x,\varphi_1(x))\big]\,dx.$$ 另一方面,$\partial D^+$ 由四段拼成:下界 $y=\varphi_1(x)$,$x$ 从 $a$ 到 $b$;右竖直段(若有);上界 $y=\varphi_2(x)$,$x$ 从 $b$ 到 $a$;左竖直段。竖直段上 $dx=0$ 不贡献。所以 $$\oint_{\partial D^+}\!P\,dx=\int_a^b P(x,\varphi_1(x))\,dx+\int_b^a P(x,\varphi_2(x))\,dx=\int_a^b\![P(x,\varphi_1)-P(x,\varphi_2)]\,dx,$$ 正好是上面二重积分的负号。第二式对 Type II 区域同理。证毕。

把 $D$ 切成无穷多小方格,每个方格四条边贡献的 $P\,dx+Q\,dy$ 中只有「净环流量」留下;相邻方格的公共边因方向相反互相抵消。这个净环流量的密度恰好是 $\partial_x Q-\partial_y P$(下面会看到它就是二维旋度)。Green 把「边界一圈的总环流」等于「内部旋度密度的二重积分」。

11.3.3 直接推论

把 $P=-y/2$,$Q=x/2$ 代入,$\partial_x Q-\partial_y P=1$,Green 公式给出用边界积分计算面积的公式:

(面积公式)平面区域 $D$ 的面积可由边界积分表出: $$S(D)=\frac12\oint_{\partial D^+}\!x\,dy-y\,dx=\oint_{\partial D^+}x\,dy=-\oint_{\partial D^+}y\,dx.$$

把 Green 写成向量形式:记 $\vec{F}=(P,Q,0)$,$\nabla\times\vec{F}=(0,0,\partial_x Q-\partial_y P)$,$d\vec{S}=\vec{k}\,dx\,dy$,则 $$\oint_{\partial D}\vec{F}\cdot d\vec{r}=\iint_D(\nabla\times\vec{F})\cdot d\vec{S},$$ 这就是 Stokes 公式的二维特例;Green 是平面 Stokes。把 $\vec{F}\to(Q,-P,0)$ 重整可写成「散度形式」$\oint\vec{F}\cdot\vec{n}\,ds=\iint\nabla\cdot\vec{F}\,dx\,dy$——这是 Gauss 公式的二维版。Green = 二维 Stokes = 二维 Gauss,只是观察角度不同。

11.3.4 例题

例 1(用 Green 算难积分)计算 $\displaystyle I=\oint_C\!(2xy-x^2)\,dx+(x+y^2)\,dy$,$C$ 是抛物线 $y=x^2$ 与 $y^2=x$ 围成区域的正向边界。

解:$P=2xy-x^2$,$Q=x+y^2$,$\partial_x Q-\partial_y P=1-2x$。区域 $D=\{0\le x\le 1,\ x^2\le y\le\sqrt x\}$。 $$I=\iint_D(1-2x)\,dx\,dy=\int_0^1\!\!dx\int_{x^2}^{\sqrt x}(1-2x)\,dy=\int_0^1(1-2x)(\sqrt x-x^2)\,dx.$$ 展开 $(1-2x)(\sqrt x-x^2)=\sqrt x-x^2-2x^{3/2}+2x^3$,积分: $$\int_0^1\!=\tfrac{2}{3}-\tfrac{1}{3}-\tfrac{4}{5}+\tfrac{1}{2}=\tfrac{20}{30}-\tfrac{10}{30}-\tfrac{24}{30}+\tfrac{15}{30}=\tfrac{1}{30}.$$

例 2(算面积)用边界积分求椭圆 $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$ 的面积。

解:参数化 $x=a\cos t$,$y=b\sin t$,$t\in[0,2\pi]$,正向。 $$S=\tfrac12\oint x\,dy-y\,dx=\tfrac12\!\int_0^{2\pi}\!(a\cos t\cdot b\cos t-b\sin t\cdot(-a\sin t))\,dt=\tfrac{ab}{2}\!\int_0^{2\pi}\!dt=\pi ab.$$

例 3(复连通的关键反例)取 $\displaystyle P=-\frac{y}{x^2+y^2},\;Q=\frac{x}{x^2+y^2}$,验证 $\partial_x Q=\partial_y P$ 但 $\oint_{|x|^2+|y|^2=1}\!P\,dx+Q\,dy=2\pi\neq 0$。

解:直接求偏导 $\partial_x Q=\partial_y P=\frac{y^2-x^2}{(x^2+y^2)^2}$。沿单位圆 $x=\cos t,y=\sin t$ 积分: $$\oint=\int_0^{2\pi}\!\!\bigg[-\sin t\cdot(-\sin t)+\cos t\cdot\cos t\bigg]dt=\int_0^{2\pi}\!1\,dt=2\pi.$$ 为什么没等于 $0$?——因为 $(P,Q)$ 在原点不连续可微,$D$ 含原点,Green 公式条件破坏。挖掉原点后 $D$ 变复连通,需修正:外边界 $-$ 内边界 $=$ 内部积分,与原点处的绕数关联。这是复变函数中留数定理和拓扑中de Rham 上同调的起点。

易错点:

  • 边界方向写反 $\Rightarrow$ 符号反掉。正向 = 区域在左。
  • $P,Q$ 在 $D$ 内有奇点(分母为零)时不能直接用 Green;要先挖去奇点小圆。
  • 二重积分被积函数是 $\partial_x Q-\partial_y P$,不是 $\partial_y Q-\partial_x P$;顺序记错符号反。
  • Green 要求 $P,Q$ 一阶偏导连续;只「存在」不够,要排除诸如 $\sqrt{x^2+y^2}$ 在原点这种点。

11.4 第一类曲面积分(对面积)

把第一类曲线积分 $\int_L f\,ds$(沿曲线对弧长积分)的思想升一维:对一张空间曲面 $\Sigma$ 算 $\iint_\Sigma f(x,y,z)\,dS$,几何意义是把 $f$ 当作曲面密度,$dS$ 是曲面上微元面积。这一节给出 $dS$ 的两种参数化公式——显式(曲面 $z=z(x,y)$)与一般参数($\vec{r}=\vec{r}(u,v)$),并解释为什么 $dS=\sqrt{1+z_x^2+z_y^2}\,dx\,dy$ 与 $|\vec{r}_u\times\vec{r}_v|\,du\,dv$ 都是同一回事。

11.4.1 定义与几何意义

设 $\Sigma$ 为光滑(或分片光滑)曲面,$f(x,y,z)$ 在 $\Sigma$ 上连续。把 $\Sigma$ 任意分成 $n$ 小片 $\Delta S_i$,在每片上任取 $(x_i,y_i,z_i)$,作和 $\sum_{i=1}^n f(x_i,y_i,z_i)\Delta S_i$。若细分时该和有极限且与分法、取点无关,则称之为 $f$ 在 $\Sigma$ 上的第一类曲面积分,记作 $$\iint_\Sigma f(x,y,z)\,dS.$$ $dS$ 是正的面积元素,与曲面的「正向 / 反向」无关。

跟第一类曲线积分 $\int f\,ds$ 一一对应:

  • $f=1$:曲面面积 $S=\iint_\Sigma dS$;
  • $f=\mu$(面密度):曲面质量 $M=\iint_\Sigma\mu\,dS$;
  • $f=x\mu/M$(归一化):质心 $x$ 坐标 $\bar x=\frac{1}{M}\iint_\Sigma x\mu\,dS$;
  • $f=(x^2+y^2)\mu$:绕 $z$ 轴转动惯量 $I_z=\iint_\Sigma(x^2+y^2)\mu\,dS$。

11.4.2 显式参数化 $z=z(x,y)$ 的面积元素

若曲面 $\Sigma$ 在区域 $D_{xy}\subset\mathbb{R}^2$ 上由 $z=z(x,y)$ 给出(光滑,$z_x,z_y$ 连续),则 $$dS=\sqrt{1+z_x^2+z_y^2}\,dx\,dy,$$ $$\iint_\Sigma f(x,y,z)\,dS=\iint_{D_{xy}}f\big(x,y,z(x,y)\big)\sqrt{1+z_x^2+z_y^2}\,dx\,dy.$$

几何法(切平面投影):在点 $(x,y,z(x,y))$ 处切平面法向 $\vec{n}=(-z_x,-z_y,1)$。切平面与 $xy$ 平面(法向 $\vec{k}=(0,0,1)$)的夹角 $\alpha$ 满足 $$\cos\alpha=\frac{\vec{n}\cdot\vec{k}}{|\vec{n}|\,|\vec{k}|}=\frac{1}{\sqrt{1+z_x^2+z_y^2}}.$$ $xy$ 平面上一个小矩形 $dx\,dy$,把它"提"到切平面变成一个倾斜矩形,投影长度不变但实际长度 $=\text{投影长度}/\cos\alpha$。所以倾斜小块面积 $$dS=\frac{dx\,dy}{\cos\alpha}=\sqrt{1+z_x^2+z_y^2}\,dx\,dy.\;\square$$

例 1(半球面积)上半球面 $z=\sqrt{R^2-x^2-y^2}$($z\ge 0$)。

解:$z_x=-x/z$,$z_y=-y/z$, $$1+z_x^2+z_y^2=1+\frac{x^2+y^2}{z^2}=\frac{R^2}{R^2-x^2-y^2}.$$ $$S=\iint_{x^2+y^2\le R^2}\!\frac{R}{\sqrt{R^2-x^2-y^2}}\,dx\,dy=\int_0^{2\pi}\!d\theta\!\int_0^R\!\frac{R\,r}{\sqrt{R^2-r^2}}\,dr=2\pi R\cdot\big[-\sqrt{R^2-r^2}\big]_0^R=2\pi R^2.$$ 整个球面 $4\pi R^2$。

11.4.3 一般参数化 $\vec{r}=\vec{r}(u,v)$ 的面积元素

若 $\Sigma$ 由 $\vec{r}(u,v)=(x(u,v),y(u,v),z(u,v))$,$(u,v)\in D$ 光滑参数化,且 $\vec{r}_u\times\vec{r}_v\neq\vec{0}$,则 $$dS=|\vec{r}_u\times\vec{r}_v|\,du\,dv,\quad\iint_\Sigma f\,dS=\iint_D f(\vec{r}(u,v))\,|\vec{r}_u\times\vec{r}_v|\,du\,dv.$$

$(u,v)$ 平面上 $du\,dv$ 的小矩形,经 $\vec{r}$ 映射后,顶点 $(u,v),(u+du,v),(u,v+dv)$ 分别映到 $\vec{r},\vec{r}+\vec{r}_u du,\vec{r}+\vec{r}_v dv$。这三点决定的小块近似是一个平行四边形,两条邻边 $\vec{r}_u du$ 与 $\vec{r}_v dv$。由向量积几何意义,平行四边形面积 $$dS=|\vec{r}_u du\times\vec{r}_v dv|=|\vec{r}_u\times\vec{r}_v|\,du\,dv.\;\square$$

$|\vec{r}_u\times\vec{r}_v|$ 是从 $(u,v)$ 平面到曲面的「面积放大因子」——和二维换元公式中的 $|J|$ 一样,只是这里 $\vec{r}_u\times\vec{r}_v$ 是三维空间向量,模就是平行四边形面积。显式版与参数版等价:对 $\vec{r}=(x,y,z(x,y))$,取 $u=x,v=y$,有 $\vec{r}_u=(1,0,z_x)$,$\vec{r}_v=(0,1,z_y)$,$\vec{r}_u\times\vec{r}_v=(-z_x,-z_y,1)$,模 $\sqrt{1+z_x^2+z_y^2}$,与显式公式合体。

例 2(球面参数化)球面 $\vec{r}(\varphi,\theta)=R(\sin\varphi\cos\theta,\sin\varphi\sin\theta,\cos\varphi)$,$\varphi\in[0,\pi]$,$\theta\in[0,2\pi]$。算 $dS$。

解: $$\vec{r}_\varphi=R(\cos\varphi\cos\theta,\cos\varphi\sin\theta,-\sin\varphi),\quad\vec{r}_\theta=R(-\sin\varphi\sin\theta,\sin\varphi\cos\theta,0).$$ $$\vec{r}_\varphi\times\vec{r}_\theta=R^2\sin\varphi\,(\sin\varphi\cos\theta,\sin\varphi\sin\theta,\cos\varphi)=R^2\sin\varphi\cdot\hat{r}.$$ $|\vec{r}_\varphi\times\vec{r}_\theta|=R^2\sin\varphi$,$dS=R^2\sin\varphi\,d\varphi\,d\theta$。球面面积 $=\int_0^{2\pi}\!d\theta\!\int_0^\pi R^2\sin\varphi\,d\varphi=2\pi R^2\cdot 2=4\pi R^2$。

例 3(对称巧算)$\Sigma$ 为球面 $x^2+y^2+z^2=R^2$,算 $\iint_\Sigma(x^2+y^2+z^2)\,dS$。

解:在 $\Sigma$ 上 $x^2+y^2+z^2=R^2$ 是常数,所以 $$\iint_\Sigma R^2\,dS=R^2\cdot 4\pi R^2=4\pi R^4.$$

例 4(锥面)$\Sigma$ 为锥面 $z=\sqrt{x^2+y^2}$,$0\le z\le 1$ 部分。算其侧面积。

解:$z_x=x/\sqrt{x^2+y^2}$,$z_y=y/\sqrt{x^2+y^2}$,$z_x^2+z_y^2=1$,$\sqrt{1+z_x^2+z_y^2}=\sqrt 2$。投影是 $x^2+y^2\le 1$ 的圆盘: $$S=\iint_{x^2+y^2\le 1}\sqrt 2\,dx\,dy=\sqrt 2\cdot\pi=\sqrt 2\,\pi.$$

易错点:

  • $dS$ 总取正值,与曲面定向无关;若曲面在 $D_{xy}$ 上不能写成单值函数(如整个球面),要分片或换参数;
  • 显式公式 $\sqrt{1+z_x^2+z_y^2}$ 要求 $z$ 可微;在曲面有"棱"或"尖"(如圆锥顶点)的点上公式失效,需排除测度零集;
  • 一般参数化中要保证 $\vec{r}_u\times\vec{r}_v\neq\vec{0}$,否则参数化退化;
  • 不要把第一类与第二类混淆:第一类 $\iint f\,dS$ 是标量积分,第二类 $\iint\vec{F}\cdot d\vec{S}$ 带方向(下一节)。

11.5 第二类曲面积分(对坐标)· Gauss 公式

第一类 $\iint f\,dS$ 是标量积分,面积元 $dS>0$ 与定向无关。物理上更常见的量——流体单位时间穿过曲面的流量、电场穿过曲面的电通量、磁通量——都依赖穿过的方向;把曲面换个方向(从前面看变成从后面看),通量整体反号。这就需要带方向的曲面积分:第二类曲面积分。本节给出它的定义、与第一类的转换公式、参数化算法,最后证明它的核心定理——Gauss 公式(散度定理)。

11.5.1 双侧曲面与定向

若曲面 $\Sigma$ 上每点存在连续的单位法向量场 $\vec{n}(P)$,称 $\Sigma$ 为双侧(可定向)曲面。选定 $\vec{n}$ 的连续场就给曲面定了向——另一选 $-\vec{n}$ 是反向。

典型双侧:球面 / 椭球面 / 圆柱侧面 / 任何光滑闭曲面; 典型单侧(不可定向):Möbius 带、Klein 瓶——这类曲面不存在第二类曲面积分。

闭曲面默认取外法向为正向;开曲面 $z=z(x,y)$ 形式默认取上侧(法向 $z$ 分量为正)。

设 $\Sigma$ 是定向的光滑曲面,$\vec{n}$ 是单位法向,$\vec{F}=(P,Q,R)$ 在 $\Sigma$ 上连续。第二类曲面积分(通量)定义为 $$\iint_\Sigma\vec{F}\cdot d\vec{S}:=\iint_\Sigma\vec{F}\cdot\vec{n}\,dS=\iint_\Sigma(P\cos\alpha+Q\cos\beta+R\cos\gamma)\,dS,$$ 其中 $\vec{n}=(\cos\alpha,\cos\beta,\cos\gamma)$。等价地写成"对坐标"积分: $$\iint_\Sigma\vec{F}\cdot d\vec{S}=\iint_\Sigma P\,dy\,dz+Q\,dz\,dx+R\,dx\,dy.$$

$d\vec{S}=\vec{n}\,dS$ 是带方向的面积元,长度为 $dS$,方向沿法向。$\vec{F}\cdot d\vec{S}=\vec{F}\cdot\vec{n}\,dS$ 是 $\vec{F}$ 在法向上的分量乘上面积——「单位时间内沿法向穿过此微元的"东西"有多少」。如果 $\vec{F}$ 是流速,$\iint\vec{F}\cdot d\vec{S}$ 就是单位时间穿过曲面的流量(体积/时间)。换侧 $\vec{n}\to -\vec{n}$,积分整体变号。

11.5.2 投影计算公式

(投影公式 / 对坐标公式)若 $\Sigma:z=z(x,y)$,$(x,y)\in D_{xy}$,取上侧,则 $$\iint_\Sigma R(x,y,z)\,dx\,dy=\iint_{D_{xy}}R(x,y,z(x,y))\,dx\,dy.$$ 取下侧时,等式右边换符号。同理可投影到 $yz$ / $zx$ 平面计算 $P\,dy\,dz$、$Q\,dz\,dx$。

上侧法向 $\vec{n}=(-z_x,-z_y,1)/\sqrt{1+z_x^2+z_y^2}$,$\cos\gamma=1/\sqrt{1+z_x^2+z_y^2}$。由两类转换公式: $$\iint_\Sigma R\,dx\,dy=\iint_\Sigma R\cos\gamma\,dS=\iint_{D_{xy}}R(x,y,z(x,y))\cdot\frac{1}{\sqrt{1+z_x^2+z_y^2}}\cdot\sqrt{1+z_x^2+z_y^2}\,dx\,dy=\iint_{D_{xy}}R\,dx\,dy.$$ 下侧 $\cos\gamma<0$,差一个负号。证毕。

正负号的物理理解:$dx\,dy$ 表示「向 $xy$ 平面正方向投影」,曲面上侧时投影同号、下侧反号。若曲面被"分片"后某些片是上侧、某些下侧(如整个球面),必须分片处理,符号分别取——这是最易错的点。

11.5.3 一般参数化与定向

若 $\Sigma:\vec{r}(u,v)$ 光滑参数化,取定向为 $\vec{n}=\vec{r}_u\times\vec{r}_v/|\vec{r}_u\times\vec{r}_v|$(即 $u$-曲线、$v$-曲线、$\vec{n}$ 构成右手系),则 $$\iint_\Sigma\vec{F}\cdot d\vec{S}=\iint_D\vec{F}(\vec{r}(u,v))\cdot(\vec{r}_u\times\vec{r}_v)\,du\,dv.$$

$d\vec{S}=\vec{n}\,dS=\vec{n}\cdot|\vec{r}_u\times\vec{r}_v|\,du\,dv=(\vec{r}_u\times\vec{r}_v)\,du\,dv$。代入定义即得。

例 1(球面通量)$\vec{F}=(x,y,z)$,$\Sigma$ 为单位球面外侧,求 $\iint_\Sigma\vec{F}\cdot d\vec{S}$。

解:外法向 $\vec{n}=(x,y,z)$(在单位球面上),$\vec{F}\cdot\vec{n}=x^2+y^2+z^2=1$。 $$\iint_\Sigma 1\,dS=4\pi.$$

11.5.4 Gauss 公式(散度定理)

向量场 $\vec{F}=(P,Q,R)$ 的散度: $$\operatorname{div}\vec{F}=\nabla\cdot\vec{F}=\frac{\partial P}{\partial x}+\frac{\partial Q}{\partial y}+\frac{\partial R}{\partial z}.$$ $\operatorname{div}\vec{F}>0$ 在源(向外发射),$<0$ 在汇(向内吸收),$=0$ 即无源场(局部流入 $=$ 流出)。

(Gauss 公式 / 散度定理)设 $V\subset\mathbb{R}^3$ 为有界闭区域,边界 $\partial V$ 是分片光滑闭曲面,取外法向。若 $P,Q,R$ 在 $\overline V$ 上有连续一阶偏导,则 $$\oiint_{\partial V}P\,dy\,dz+Q\,dz\,dx+R\,dx\,dy=\iiint_V\!\bigg(\frac{\partial P}{\partial x}+\frac{\partial Q}{\partial y}+\frac{\partial R}{\partial z}\bigg)dV.$$ 向量形式:$\displaystyle\oiint_{\partial V}\vec{F}\cdot d\vec{S}=\iiint_V\nabla\cdot\vec{F}\,dV$。

与 Green 同构的证明。只需证三个分量等式之一,例如 $$\oiint_{\partial V}R\,dx\,dy=\iiint_V\frac{\partial R}{\partial z}\,dV.\qquad(*)$$ 设 $V$ 是 Type I-z 区域(每条铅直线与 $V$ 交一段):$V=\{(x,y,z):(x,y)\in D_{xy},\ z_1(x,y)\le z\le z_2(x,y)\}$。 右边 $=\displaystyle\iint_{D_{xy}}\!\!\bigg(\!\int_{z_1}^{z_2}\!\frac{\partial R}{\partial z}dz\!\bigg)dx\,dy=\iint_{D_{xy}}\![R(x,y,z_2)-R(x,y,z_1)]\,dx\,dy$。

$\partial V$ 由上盖 $z=z_2$(上侧)、下底 $z=z_1$(下侧)、侧面(铅直,法向无 $z$ 分量,$dx\,dy$ 投影为 $0$)三部分组成。由投影公式: $$\oiint_{\partial V}\!R\,dx\,dy=\underbrace{\iint_{D_{xy}}\!R(x,y,z_2)\,dx\,dy}_{\text{上盖}}+\underbrace{(-1)\!\!\iint_{D_{xy}}\!R(x,y,z_1)\,dx\,dy}_{\text{下底反号}}+0.$$ 两侧相等,$(*)$ 成立。对 $P,Q$ 同理(把 $V$ 看作 Type I-x、Type I-y 区域)。一般 $V$ 可剖分,内界双向抵消,只剩外边界。证毕。

"边界通量 $=$ 内部散度积分" 是「微元守恒」的全局版:每个微小立方体盒子里"净流出 $=$ 内部源强度 $\times$ 体积",把所有盒子加起来,内部相邻面的通量正负相消,只剩下最外层的整个 $\partial V$。这就是为什么 Gauss 公式被看作三维版的 Newton-Leibniz——「内部 $\to$ 边界」。

11.5.5 例题

例 2(直接套散度)$\vec{F}=(x,y,z)$,$\Sigma$ 为单位球外侧。

解:$\nabla\cdot\vec{F}=1+1+1=3$, $$\iint_\Sigma\vec{F}\cdot d\vec{S}=\iiint_V 3\,dV=3\cdot\tfrac{4\pi}{3}=4\pi.$$ 跟例 1 直接算结果一致。

例 3(求体积的反向技巧)单位球体积 $V$ 可由 $\oiint z\,dx\,dy$ 算出。

解:取 $\vec{F}=(0,0,z)$,$\nabla\cdot\vec{F}=1$, $$\oiint_{\partial V}\!z\,dx\,dy=\iiint_V 1\,dV=V.$$ 对称版:$V=\tfrac13\oiint(x\,dy\,dz+y\,dz\,dx+z\,dx\,dy)$。

例 4(立方体通量)$\vec{F}=(x^2,y^2,z^2)$,$\Sigma$ 为单位立方体 $[0,1]^3$ 外表面。

解:$\nabla\cdot\vec{F}=2x+2y+2z$, $$\iint_\Sigma\vec{F}\cdot d\vec{S}=\iiint_{[0,1]^3}(2x+2y+2z)\,dV=2\cdot 3\cdot\tfrac12=3.$$

例 5(物理应用 — Gauss 电场定理)真空电场散度方程 $\nabla\cdot\vec{E}=\rho/\varepsilon_0$,任意闭曲面 $\Sigma=\partial V$: $$\Phi_E=\oiint_\Sigma\vec{E}\cdot d\vec{S}=\iiint_V\frac{\rho}{\varepsilon_0}\,dV=\frac{Q_{\text{enc}}}{\varepsilon_0}.$$ 结论:闭曲面电通量正比于内部电荷——Maxwell 方程组第一条的积分形式,中学物理里"高斯定理"的来源。同理由 $\nabla\cdot\vec{B}=0$ 得磁通量永远为零(没有磁单极子)。

11.5.6 与三大公式的统一

| 公式 | 边界 | 内部 | | --- | --- | --- | | Newton-Leibniz | 端点差 $f(b)-f(a)$ | $\int_a^b f'\,dx$ | | Green | 闭曲线环路 $\oint P\,dx+Q\,dy$ | $\iint(\partial_x Q-\partial_y P)\,d\sigma$ | | Gauss(散度定理) | 闭曲面通量 $\oiint\vec{F}\cdot d\vec{S}$ | $\iiint\nabla\cdot\vec{F}\,dV$ | | Stokes | 空间闭曲线环流 $\oint\vec{F}\cdot d\vec{r}$ | $\iint(\nabla\times\vec{F})\cdot d\vec{S}$ |

统一思想:$\int_{\partial M}\omega=\int_M d\omega$——这是广义 Stokes 公式(微分形式版本)。$\omega$ 是 $k$-形式,$d\omega$ 是它的外微分,定理把「边界上的 $k$-形式积分」等价于「内部 $(k+1)$-形式积分」。Newton-Leibniz、Green、Gauss、Stokes 都是这同一定理在不同维度的实现:

| 维度 | $\omega$ 阶 | $\int_{\partial M}\omega=\int_M d\omega$ | | --- | --- | --- | | $1$ | $0$-形式 $f$ | $f|_a^b=\int_a^b df$($\Rightarrow$ Newton-Leibniz) | | $2$ | $1$-形式 $P\,dx+Q\,dy$ | Green | | $3$ | $1$-形式 | Stokes | | $3$ | $2$-形式 $P\,dy\wedge dz+\dots$ | Gauss |

这是现代微分几何的中心定理之一,统一了 19 世纪以来发展的"边界 $\to$ 内部"对应,也是 Maxwell 方程组、规范场论、de Rham 上同调的代数核心。

易错点:

  • Gauss 公式要求 $\partial V$ 闭合且取外法向;开曲面用不了 Gauss,必须封口(补一片底面),封口面上单独算再减;
  • $P,Q,R$ 在 $V$ 内有奇点(如点源 $1/r$)时,公式失效;需挖去小球后用复连通版本(留数 / 立体角分析);
  • "右手系参数化"决定 $\vec{r}_u\times\vec{r}_v$ 指向外还是内,搞错正负号都反;
  • 第二类曲面积分对反向反号——这是它与第一类最根本的区别。

11.6 Stokes 公式

平面上边界环路 $=$ 内部旋度密度积分(Green);三维空间闭曲面通量 $=$ 内部散度积分(Gauss)。还差一种联系——空间曲线环流 $=$ 张在曲线上的任意曲面的旋度通量——这就是 Stokes 公式,Green 在三维曲面上的完整推广。它统一了所有"边界 $\leftrightarrow$ 内部"的对应,也是 19 世纪经典电磁学(法拉第电磁感应、Ampère 环路定律)与现代微分几何(de Rham 上同调、规范场论)共同的代数根。

11.6.1 旋度与方向匹配

向量场 $\vec{F}=(P,Q,R)$ 的旋度(curl)是新的向量场: $$\nabla\times\vec{F}=\begin{vmatrix}\vec{i}&\vec{j}&\vec{k}\\ \partial_x&\partial_y&\partial_z\\ P&Q&R\end{vmatrix}=\bigg(\frac{\partial R}{\partial y}-\frac{\partial Q}{\partial z},\;\frac{\partial P}{\partial z}-\frac{\partial R}{\partial x},\;\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\bigg).$$

在点 $P$ 处放一只"小风车":风车轴沿单位向量 $\vec{n}$,风车叶面就是与 $\vec{n}$ 垂直的圆盘。流场 $\vec{F}$ 推动叶片,风车转速等于 $(\nabla\times\vec{F})\cdot\vec{n}$——旋度向量在 $\vec{n}$ 方向上的投影。所以 $\nabla\times\vec{F}$ 本身的方向就是「转速最大的转轴方向」,模长是该方向的旋转密度。

(曲面定向与边界右手系)给定双侧光滑曲面 $\Sigma\subset\mathbb{R}^3$ 与单位法向 $\vec{n}$。边界 $\partial\Sigma$ 的正向由右手定则确定:拇指沿 $\vec{n}$ 指,四指弯曲方向就是 $\partial\Sigma^+$ 的走向。直观地说,"沿正向走时,$\Sigma$ 在你左侧"——与 Green 中平面区域左侧规则一致。

11.6.2 Stokes 公式

(Stokes 公式)设 $\Sigma$ 是 $\mathbb{R}^3$ 中分片光滑、双侧的有向曲面,边界 $\partial\Sigma$ 为分段光滑闭曲线,取右手系正向。若 $\vec{F}=(P,Q,R)$ 在含 $\Sigma$ 的开集上有连续一阶偏导,则 $$\oint_{\partial\Sigma}\vec{F}\cdot d\vec{r}=\iint_\Sigma(\nabla\times\vec{F})\cdot d\vec{S}.$$ 展开成分量形式: $$\oint_{\partial\Sigma}P\,dx+Q\,dy+R\,dz=\iint_\Sigma\bigg(\frac{\partial R}{\partial y}-\frac{\partial Q}{\partial z}\bigg)dy\,dz+\bigg(\frac{\partial P}{\partial z}-\frac{\partial R}{\partial x}\bigg)dz\,dx+\bigg(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\bigg)dx\,dy.$$

思路:只需对每个分量分别证,例如证明 $$\oint_{\partial\Sigma}P\,dx=\iint_\Sigma\frac{\partial P}{\partial z}\,dz\,dx-\frac{\partial P}{\partial y}\,dx\,dy.\qquad(*)$$ 对 $Q\,dy$、$R\,dz$ 同理(三分量轮换),合起来就是 Stokes。

证 $(*)$:设 $\Sigma:z=z(x,y)$,$(x,y)\in D_{xy}$,取上侧,则 $\partial\Sigma$ 在 $xy$ 平面上的投影 $\partial D_{xy}$ 取逆时针(右手系)。把 $\partial\Sigma$ 参数化时 $z=z(x(t),y(t))$,所以 $$\oint_{\partial\Sigma}\!P(x,y,z)\,dx=\oint_{\partial D_{xy}}\!P(x,y,z(x,y))\,dx.$$ 对右边用 Green 公式(取 $P^*=P(x,y,z(x,y))$,$Q^*=0$): $$\oint_{\partial D_{xy}}\!P^*\,dx=-\iint_{D_{xy}}\frac{\partial P^*}{\partial y}\,dx\,dy.$$ 对 $P^*$ 用复合函数链式法则:$\dfrac{\partial P^*}{\partial y}=\dfrac{\partial P}{\partial y}+\dfrac{\partial P}{\partial z}\cdot z_y$。代回: $$\oint_{\partial\Sigma}\!P\,dx=-\iint_{D_{xy}}\!\bigg(\frac{\partial P}{\partial y}+\frac{\partial P}{\partial z}z_y\bigg)dx\,dy.\qquad(\dagger)$$

另一方面,由两类曲面积分转换公式(上侧 $\cos\gamma=1/\sqrt{1+z_x^2+z_y^2}$,$\cos\beta=-z_y/\sqrt{\,\cdots\,}$): $$\iint_\Sigma\!\frac{\partial P}{\partial z}\,dz\,dx=\iint_\Sigma\!\frac{\partial P}{\partial z}\cos\beta\,dS=-\iint_{D_{xy}}\!\frac{\partial P}{\partial z}\,z_y\,dx\,dy,$$ $$\iint_\Sigma\!\frac{\partial P}{\partial y}\,dx\,dy=\iint_\Sigma\!\frac{\partial P}{\partial y}\cos\gamma\,dS=\iint_{D_{xy}}\!\frac{\partial P}{\partial y}\,dx\,dy.$$ 相减: $$\iint_\Sigma\!\bigg(\frac{\partial P}{\partial z}\,dz\,dx-\frac{\partial P}{\partial y}\,dx\,dy\bigg)=-\iint_{D_{xy}}\!\bigg(\frac{\partial P}{\partial z}z_y+\frac{\partial P}{\partial y}\bigg)dx\,dy.\qquad(\ddagger)$$ $(\dagger)$ 与 $(\ddagger)$ 右边相同,所以 $(*)$ 成立。

对 $Q\,dy$、$R\,dz$:对调字母 $(x,y,z)\to(y,z,x)\to(z,x,y)$ 重复上面推理,各得一式。三式相加,左边合并为 $\oint\vec{F}\cdot d\vec{r}$,右边合并为 $\iint(\nabla\times\vec{F})\cdot d\vec{S}$。证毕。

一般曲面:可剖分为有限片"$z=z(x,y)$ 型",内界双向走相消,只剩外边界,定理推广完成。

微元解释:把 $\Sigma$ 切成无穷多小平面四边形。每个小四边形上(假设大小 $dA$,法向 $\vec{n}$)绕边界的环流近似为 $(\nabla\times\vec{F})\cdot\vec{n}\,dA$——这是上面"小风车转速"的直接积分版。把所有微元的环流加起来,相邻边因方向相反相消,只剩最外圈 $\partial\Sigma$,正好得 Stokes。

11.6.3 旋度的不依赖坐标定义

旋度看似带坐标系,实际是纯几何对象。下面这个等式是它的内蕴定义:

(旋度作为环量密度极限)对 $\mathbb{R}^3$ 中点 $P_0$ 和单位向量 $\vec{n}$,设 $\Sigma_r$ 是过 $P_0$、法向 $\vec{n}$、面积 $\pi r^2$ 的小圆盘,则 $$(\nabla\times\vec{F})(P_0)\cdot\vec{n}=\lim_{r\to 0^+}\frac{1}{\pi r^2}\oint_{\partial\Sigma_r}\!\vec{F}\cdot d\vec{r}.$$

由 Stokes:$\oint_{\partial\Sigma_r}\vec{F}\cdot d\vec{r}=\iint_{\Sigma_r}(\nabla\times\vec{F})\cdot\vec{n}\,dS$。由于 $\nabla\times\vec{F}$ 连续,$\Sigma_r$ 极小时被积函数近似为常值 $(\nabla\times\vec{F})(P_0)\cdot\vec{n}$,从而积分近似 $(\nabla\times\vec{F})(P_0)\cdot\vec{n}\cdot\pi r^2$。除以 $\pi r^2$ 并令 $r\to 0^+$ 即得。证毕。

这条等式说明旋度的定义不依赖坐标系选择——把球面上的旋度搬到曲线坐标系,公式形式变(雅可比插入)但极限值不变。这就是为什么在 GR、规范场论中 $\nabla\times$ 推广为外微分 $d$。

11.6.4 物理意义:法拉第电磁感应

Maxwell 方程组的第三条: $$\nabla\times\vec{E}=-\frac{\partial\vec{B}}{\partial t}.$$ 对任意有向曲面 $\Sigma$ 两边求曲面积分,左边用 Stokes 化为边界环流: $$\oint_{\partial\Sigma}\!\vec{E}\cdot d\vec{r}=-\frac{d}{dt}\iint_\Sigma\!\vec{B}\cdot d\vec{S}=-\frac{d\Phi_B}{dt}.$$ 左端是闭合导线沿 $\partial\Sigma$ 一圈的电动势($\mathrm{emf}$),右端是穿过 $\Sigma$ 的磁通量随时间变化的负数。这就是法拉第电磁感应定律:磁通量变化产生电动势——发电机、变压器、感应炉的物理基础。Stokes 公式把 Maxwell 微分形式跟工程师每天用的"绕圈"积分量直接打通。

11.6.5 例题

例 1(验证 Stokes)$\vec{F}=(-y,x,0)$,$\partial\Sigma$ 是单位圆 $x^2+y^2=1$,$z=0$,正向。直接和用 Stokes 各算一次。

直接:参数 $x=\cos t,y=\sin t,t\in[0,2\pi]$, $$\oint=\int_0^{2\pi}(\sin^2 t+\cos^2 t)\,dt=2\pi.$$

用 Stokes:取 $\Sigma$ 为单位圆盘($z=0$,$\vec{n}=\vec{k}$ 上侧), $$\nabla\times\vec{F}=(0-0,\,0-0,\,1-(-1))=(0,0,2),$$ $$\iint_\Sigma(\nabla\times\vec{F})\cdot d\vec{S}=\iint_{D}2\,dA=2\cdot\pi=2\pi.\checkmark$$

例 2(用 Stokes 简化)$\vec{F}=(y,-x,z^2)$,$L$ 为三角形 $A(1,0,0)\to B(0,1,0)\to C(0,0,1)\to A$ 沿这个顺序走,求 $\oint_L\vec{F}\cdot d\vec{r}$。

解:计算 $\nabla\times\vec{F}=(0-0,\,0-0,\,-1-1)=(0,0,-2)$。取 $\Sigma$ 为三角形所在平面 $x+y+z=1$ 的内部三角形片,法向 $\vec{n}=(1,1,1)/\sqrt 3$(由顶点顺序右手系定向)。三角形面积 $S=\tfrac{\sqrt 3}{2}$(三边都是 $\sqrt 2$,等边)。 $$\oint=\iint_\Sigma(\nabla\times\vec{F})\cdot\vec{n}\,dS=(0,0,-2)\cdot\bigg(\frac{1}{\sqrt 3},\frac{1}{\sqrt 3},\frac{1}{\sqrt 3}\bigg)\cdot S=-\frac{2}{\sqrt 3}\cdot\frac{\sqrt 3}{2}=-1.$$

例 3(保守场判别)若 $\vec{F}$ 在单连通区域里满足 $\nabla\times\vec{F}=\vec{0}$,则任意闭曲线积分为零,从而 $\vec{F}$ 是保守场($\vec{F}=\nabla\phi$ 某势函数)。

解(思路):由 Stokes,任意 $\partial\Sigma$ 上环流 $=\iint_\Sigma\vec{0}\cdot d\vec{S}=0$。在单连通区域里,任两条同起终点的曲线之差是闭路,所以 $\vec{F}$ 沿路径积分路径无关;固定起点 $P_0$ 后定义 $\phi(P)=\int_{P_0}^P\vec{F}\cdot d\vec{r}$,$\nabla\phi=\vec{F}$。注意:单连通条件不可去掉——反例 $\vec{F}=(-y,x,0)/(x^2+y^2)$ 在 $\mathbb{R}^3\setminus\{z\text{-轴}\}$ 上 $\nabla\times\vec{F}=\vec{0}$,但绕 $z$-轴一圈环流为 $2\pi$ 不为零(空间非单连通)。

11.6.6 三大公式与微分形式视角

把第 11.5 节末尾的统一表再展开一点:每条「边界 $\leftrightarrow$ 内部」定理都是同一条广义 Stokes 公式 $$\int_{\partial M}\omega=\int_M d\omega$$ 在不同维度下的具体写法。$\omega$ 是 $k$-形式,$d\omega$ 是它的外微分($(k+1)$-形式),$M$ 是 $(k+1)$-维带边流形:

| $\dim M$ | $\omega$ 阶 | $d\omega$ 阶 | 公式 | 物理量 | | --- | --- | --- | --- | --- | | $1$ | $0$ | $1$ | $f\big|_a^b=\int_a^b df$ | Newton-Leibniz | | $2$ | $1$ | $2$ | Green | 平面环流 ↔ 旋度 | | $2$(嵌入 $\mathbb{R}^3$) | $1$ | $2$ | Stokes | 空间环流 ↔ 旋度通量 | | $3$ | $2$ | $3$ | Gauss | 通量 ↔ 散度 | | $n$ | $k$ | $k+1$ | 一般 Stokes | (任意维) |

外微分满足 $d^2=0$(连用两次外微分为零),对应于:$\nabla\cdot(\nabla\times\vec{F})=0$(旋度场无源)、$\nabla\times(\nabla\phi)=\vec{0}$(梯度场无旋)。这两条恒等式在物理上分别表达"磁场来自旋度故无磁单极"、"势场无环流故路径无关"——经典 Maxwell 理论与 de Rham 上同调的接口。

易错点:

  • $\partial\Sigma$ 的正向必须与 $\Sigma$ 的法向 $\vec{n}$ 用右手系匹配;搞反整体反号。
  • $\Sigma$ 必须双侧可定向;Möbius 带不可定向,Stokes 失效。
  • 应用「$\nabla\times\vec{F}=\vec{0}\Rightarrow$ 保守」需要区域单连通;复连通区域用反例 $(-y,x)/(x^2+y^2)$ 提醒;
  • Stokes 公式中曲面 $\Sigma$ 可任意选择,只要边界一致——这是「Stokes 与 $\Sigma$ 选择无关」性质,本质上由 $d^2=0$ 与封闭性保证。
  • $\vec{F}$ 在 $\Sigma$ 上要有连续一阶偏导;只「分量连续」不够。

11.7 环流量与旋度 (对齐书 §11.7 三)

知识点 12.7: 环流量 (circulation)

环流量

向量场 $\vec{F}$ 沿闭曲线 $L$ 的环流量: $$\Gamma = \oint_L \vec{F}\cdot d\vec{r} = \oint_L P\,dx + Q\,dy + R\,dz$$

环流量物理意义

沿闭曲线"绕一圈" $\vec{F}$ 做的总功. 涡旋中心附近环流量大; 均匀流环流量为 0; 保守力沿任意闭曲线环流量为 0.

知识点 12.8: 旋度 (curl)

旋度

向量场 $\vec{F} = (P, Q, R)$ 的旋度是向量场: $$\text{curl}\,\vec{F} = \nabla \times \vec{F} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ \partial/\partial x & \partial/\partial y & \partial/\partial z \\ P & Q & R \end{vmatrix}$$ 展开: $\left(\dfrac{\partial R}{\partial y} - \dfrac{\partial Q}{\partial z}, \dfrac{\partial P}{\partial z} - \dfrac{\partial R}{\partial x}, \dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y}\right)$

小风车几何

某点放小风车, 旋度方向 = 风车轴方向, 旋度大小 = 转速.

  • $\nabla\times\vec{F} = 0$: 无旋场 (流体不打转, 即保守场)
  • $|\nabla\times\vec{F}|$ 大: 涡旋中心

类比: 龙卷风中心旋度极大, 平直气流旋度为 0.

斯托克斯公式 (向量形式)

$$\oint_{\partial\Sigma} \vec{F} \cdot d\vec{r} = \iint_\Sigma (\nabla\times\vec{F}) \cdot d\vec{S}$$ 对应关系: 沿曲面边界的环流 = 曲面上旋度通量的累积. Green 公式是 $\Sigma$ 退化到平面的 2D 特例.

旋度 vs 环流 — 微观 vs 宏观

  • 旋度 $\nabla\times\vec{F}$: 点级微观"小风车转速" (向量)
  • 环流量 $\oint \vec{F}\cdot d\vec{r}$: 沿宏观闭曲线一圈的"总功"
  • Stokes 桥梁: 全曲面上"每点旋度"加起来 = 边界"总环流". 跟 Gauss "全立体散度 = 边界通量"是孪生兄弟.

环流量 = 旋度通过曲面 的通量. 闭曲线环流 = 内部旋转累计.

物理意义对应:

  • 麦克斯韦电磁: ∇·E = ρ/ε₀ (高斯定律, 电场散度跟电荷密度) / ∇×B = μ₀J + ... (安培定律, 磁场旋度跟电流)
  • 流体: ∇·$\vec{v}$ = 0 (不可压缩) / ∇×$\vec{v}$ = 0 (无旋流)

知识点 12.9: 三大公式统一思想 (Generalized Stokes 定理)

广义 Stokes 定理

$$\int_{\partial M} \omega = \int_M d\omega$$ 其中 $M$ 是流形, $\partial M$ 是其边界, $\omega$ 是微分形式, $d\omega$ 是外导数.

四个公式 = 一个公式的不同维度

| 公式 | 维度 | 边界 | 内部 | |---|---|---|---| | 牛顿-莱布尼茨 | 1D | 端点 $a, b$ | 区间 $[a, b]$ | | Green | 2D | 闭曲线 $\partial D$ | 平面区域 $D$ | | Gauss | 3D | 闭曲面 $\partial V$ | 立体 $V$ | | Stokes | 2D 嵌入 3D | 曲面边界 $\partial\Sigma$ | 曲面 $\Sigma$ |

核心: 都是微积分基本定理的高维推广 — "积分等于原函数在端点值的差", 从 1D 推广到任意维.

应用:

  • 流体动力学: Navier-Stokes 方程
  • 电磁学: Maxwell 方程组 (4 个方程都是这几个公式的物理形式)
  • 广义相对论: 黎曼几何积分
  • 拓扑学: 同调论

知识点 12.10: 三大公式记忆口诀

三大公式一句话记忆

  • Green: "平面闭曲线绕一圈 = 平面区域内每点的"旋度"累加"
  • Gauss: "闭曲面通量 = 立体内每点的"散度"累加"
  • Stokes: "空间闭曲线环流 = 曲面上每点的"旋度"累加"

关键 idea: 边界 = 内部 $d$ 算子的积分 — 这就是广义 Stokes 定理 $\int_{\partial M} \omega = \int_M d\omega$ 的统一思想.

知识点 12.1: 三种"导数算子"梳理

向量分析三大算子

| 算子 | 输入 → 输出 | 公式 | 物理意义 | |---|---|---|---| | 梯度 grad / $\nabla$ | 标量场 → 向量场 | $(\partial f/\partial x, \partial f/\partial y, \partial f/\partial z)$ | 函数最快上升方向 | | 散度 div / $\nabla\cdot$ | 向量场 → 标量场 | $\partial P/\partial x + \partial Q/\partial y + \partial R/\partial z$ | 体积"源"密度 (流出−流入) | | 旋度 curl / $\nabla\times$ | 向量场 → 向量场 | $(\partial R/\partial y - \partial Q/\partial z, \ldots)$ | 局部"涡旋"轴向量 |

两个零等式 (基本恒等式)

  1. 梯度场无旋: $\nabla \times (\nabla f) = 0$ — 势能场沿任意闭路环流为 0
  2. 旋度场无源: $\nabla \cdot (\nabla \times \vec{F}) = 0$ — "没有磁单极"

二者对应 de Rham 上同调中的 $d^2 = 0$, 是微分几何拓扑的核心.

知识点 12.2: 拉普拉斯算子 Δ = ∇·∇

拉普拉斯算子 (Laplacian)

$$\Delta f = \nabla \cdot (\nabla f) = \frac{\partial^2 f}{\partial x^2} + \frac{\partial^2 f}{\partial y^2} + \frac{\partial^2 f}{\partial z^2}$$ 即"梯度的散度" (二阶混合导数和).

满足 $\Delta f = 0$ 的函数称 调和函数 (Harmonic).

调和函数平均值性质

$\Delta f = 0$ 意味着该点值 = 周围球面平均值. 直观: 在调和函数中, 任何"凸起" 都会被周围平均"拉平". 这就是为什么 $\Delta T = 0$ 描述稳态热传导 (温度场达到平衡, 处处=邻域平均).

应用全表:

  • 静电势满足 Δφ = -ρ/ε₀ (Poisson 方程)
  • 真空: Δφ = 0 (Laplace 方程)
  • 稳态热传导: ΔT = 0
  • 重力势: Δφ = 4πGρ
  • 量子力学: -(ℏ²/2m) Δψ + Vψ = Eψ (薛定谔方程)
  • 计算机视觉边缘检测: Laplacian-of-Gaussian (LoG)

知识点 12.3: 流体力学的连续性方程

连续性方程 (质量守恒)

密度 $\rho$ 速度 $\vec{v}$ 的流体, 质量守恒律的微分形式: $$\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \vec{v}) = 0$$ 不可压缩 ($\rho$ 常数): $\nabla \cdot \vec{v} = 0$ (速度场无源).

Gauss 公式推导

$\oiint_{\partial V} \rho \vec{v} \cdot d\vec{S} = \iiint_V \nabla\cdot(\rho\vec{v})\,dV$ 表示流出质量.

等于内部质量减少率 $-\partial/\partial t \iiint_V \rho\,dV$.

二者相等 + 积分号交换 ⟹ 微分方程. 应用: CFD 模拟 / 空气动力学 / 天气预报 N-S 方程的基础.

知识点 12.4: 麦克斯韦方程组 (微分 + 积分形式)

Maxwell 方程组 4 条 (微分形式)

电磁场核心 4 方程, 用场论算子统一表达:

  1. 高斯电场: $\nabla \cdot \vec{E} = \rho/\epsilon_0$ (电荷是电场源)
  2. 高斯磁场: $\nabla \cdot \vec{B} = 0$ (无磁单极)
  3. 法拉第: $\nabla \times \vec{E} = -\partial \vec{B}/\partial t$ (变磁场生电场)
  4. 安培-Maxwell: $\nabla \times \vec{B} = \mu_0 \vec{J} + \mu_0 \epsilon_0 \partial \vec{E}/\partial t$ (电流 + 变电场生磁场)

Maxwell 方程组 (积分形式)

用 Gauss/Stokes 转化全局形式:

  1. $\oiint_{\partial V} \vec{E}\cdot d\vec{S} = Q_{\text{内}}/\epsilon_0$
  2. $\oiint_{\partial V} \vec{B}\cdot d\vec{S} = 0$
  3. $\oint_{\partial \Sigma} \vec{E}\cdot d\vec{r} = -d\Phi_B/dt$
  4. $\oint_{\partial \Sigma} \vec{B}\cdot d\vec{r} = \mu_0 I_{\text{内}} + \mu_0 \epsilon_0 d\Phi_E/dt$

微分 vs 积分形式选哪个

  • 微分: 偏微分方程, 解析推导 (波动方程从这推)
  • 积分: 直接对应实验测量 (电容器/磁通 等)
  • 二者等价, Gauss/Stokes 公式互通.

知识点 12.5: 电磁波 (波动方程推导)

电磁波动方程

真空 ($\rho = 0, J = 0$), 对法拉第方程取旋度 + Maxwell 方程组化简: $$\Delta \vec{E} - \mu_0 \epsilon_0 \frac{\partial^2 \vec{E}}{\partial t^2} = 0$$ 即 $\vec{E}$ 满足波动方程, 波速 $c = 1/\sqrt{\mu_0 \epsilon_0} \approx 3 \times 10^8$ m/s.

电磁学 + 光学的世纪大统一

1865 年 Maxwell 推出 $c = 1/\sqrt{\mu_0 \epsilon_0}$ 后, 跟实验测得光速 $c$ 完全一致 — 立刻断言: 光就是电磁波 (而不是单独某种东西). 这是 19 世纪物理学最大统一之一, 比爱因斯坦光速不变 (1905) 早 40 年.

后世 LED / 激光 / 无线电 / 5G 全是 $\vec{E} = E_0 \sin(kz - \omega t)$ 这种平面波解的工程实现.

例题 12-4

$\vec{E}$ = E₀ sin(kz - ωt) x̂→ (沿 z 传播, x 偏振). 代入波方程: -k²E + (ω²/c²)E = 0 ⇒ ω = ck (色散关系). LED/激光/无线电的基本构型.

知识点 12.6: Helmholtz 分解定理

Helmholtz 分解

在合适条件下, 任意向量场 $\vec{F}$ 可唯一分解为: $$\vec{F} = -\nabla \varphi + \nabla \times \vec{A}$$

  • $\varphi$ 是标量势 (scalar potential)
  • $\vec{A}$ 是向量势 (vector potential)
  • $-\nabla\varphi$ 部分: 无旋 ($\nabla \times (-\nabla \varphi) = 0$) — "源"部分
  • $\nabla \times \vec{A}$ 部分: 无散 ($\nabla \cdot (\nabla \times \vec{A}) = 0$) — "旋"部分

任何向量场 = 散场 + 旋场

$\vec{F}$ 永远拆成 "源/汇主导" 部分 + "涡旋主导" 部分. 例: 麦克斯韦方程:

  • $\vec{E} = -\nabla\varphi - \partial \vec{A}/\partial t$ (静态势 + 时变向量势)
  • $\vec{B} = \nabla \times \vec{A}$ (向量势的旋度)

气象/流体: 风场分解 = 辐散场 + 旋转场, 用于天气预报模型.

例题 12-6

(电磁场): 麦克斯韦方程中

  • 电场 $\vec{E}$ = -∇$\varphi$ - ∂$\vec{A}$/∂t (静态部分 + 时变部分)
  • 磁场 $\vec{B}$ = ∇×$\vec{A}$

$\varphi$ 和 $\vec{A}$ 一起叫"四维势" ($\varphi$, $\vec{A}$) — 相对论中是 4-向量.

应用:

  • 流体力学: 不可压缩流速分解 (势流 + 涡流)
  • 图像处理: 法线场分解 = 法向反射 + 切向流
  • GIS: 风场分解 (辐散场 + 旋转场), 用于气象预测

知识点 12.7: 微分形式 (Differential Forms) 入门

微分形式与广义 Stokes

微分形式 $\omega$ 是流形上"可积分对象". 经典对象 (函数/向量场) 都是其特例.

广义 Stokes 定理: $$\int_M d\omega = \int_{\partial M} \omega$$

  • $\omega$ 是 $(k-1)$ 次形式
  • $d\omega$ 是 $k$ 次形式 (外导数)
  • $M$ 是 $k$ 维流形, $\partial M$ 是其边界

高数三大公式 = 一个定理在不同维度

Green / Gauss / Stokes 全部是广义 Stokes 在 $k = 2, 3, 2$ 时的特例. 这就是为什么它们形式如此相似 — 因为本质上是同一个定理.

学完高数, 上微分几何课直接秒懂.

经典 4 大定理 = 同一广义 Stokes

| 经典定理 | 微分形式 | $k$ (流形维数) | |---|---|---| | Newton-Leibniz | $\int_{[a,b]} df = f(b) - f(a)$ | 1 | | Green | $\int_D d\omega = \int_{\partial D} \omega$ | 2 | | Stokes | $\int_S d\omega = \int_{\partial S} \omega$ | 2 | | Gauss | $\int_V d\omega = \int_{\partial V} \omega$ | 3 |

外导数 $d$ 的核心性质: $d \circ d = 0$ (二阶外导为零) — 这就是 $\nabla\times(\nabla f) = 0$ 和 $\nabla\cdot(\nabla\times\vec{F}) = 0$ 的统一表述.

微分形式的现代应用 4 条

  1. 广义相对论: 用张量场写场方程, 张量就是多线性微分形式
  2. 微分几何: 流形上微积分的 "通用语言"
  3. 代数拓扑: de Rham 上同调连接"流形几何"跟"代数拓扑"
  4. 数据科学: 拓扑数据分析 (TDA) 用持久同调衡量数据流形拓扑

知识点 12.8: 圆环面 (Torus) 面积体积 + 例题

圆环面 (donut shape): 大圆半径 R, 小圆半径 r, 用参数化: x = (R + r cos u) cos v y = (R + r cos u) sin v z = r sin u 其中 u, v ∈ [0, 2π].

面积: S = ∫_0^(2π)∫_0^(2π) |r_u × r_v| du dv = ∫_0^(2π)∫_0^(2π) r(R + r cos u) du dv = 4π²Rr.

体积 (圆环内部): V = 2π²R r²

Pappus 定理 (旋转体):

  • 圆环表面积 = 小圆周长 × 大圆周长 = (2πr)(2πR) = 4π²Rr ✓
  • 圆环体积 = 小圆面积 × 大圆周长 = (πr²)(2πR) = 2π²Rr² ✓

Pappus 定理一般化: 任意平面图形 (面积 A) 绕同平面外的轴旋转, 旋转体: V = A · (2π·重心到轴距离) S = L · (2π·重心到轴距离), L 是图形周长.

例题 12-8

矩形 [0,1]×[0,1] 绕 x = 2 旋转的体积. 重心 (1/2, 1/2). 到轴距离 = 3/2. A = 1. V = 1 · 2π · 3/2 = 3π.

应用:

  • 工程: 管道/轮胎/弹簧 = 圆环
  • 拓扑: 环面 χ = 0, "亏格 1"
  • 物理: 等离子体托卡马克 (Tokamak) 装置形状
  • AI: 数据可能分布在低维环面上 (流形假设)

知识点 12.9: 立体角 (Solid Angle) 概念

问题: 二维"角"用弧长 / 半径 = $\theta$ (弧度). 三维"立体角"如何定义?

定义: 立体角 $\Omega$ = 球面上面积 / r² (单位: 球面度 steradian sr).

全球面 $\Omega$ = 4π sr (即面积 4πr² / r²).

点电荷 Gauss 定理: 任意闭曲面包围单位点电荷, ∮∮ $\vec{E}$·dS→ = 1/ε₀. 本质: 4π·k_e (球面立体角积分).

半球面对中心的立体角. 半球面积 2πr². $\Omega$ = 2πr²/r² = 2π sr.

  • 雷达: 天线波束宽度

知识点 12.10: 流体力学 Navier-Stokes 方程 (千禧难题)

N-S 方程 (不可压缩流体): ∂$\vec{v}$/∂t + ($\vec{v}$·∇)$\vec{v}$ = -∇p/ρ + ν Δv→ + $\vec{f}$

各项含义:

  • ∂$\vec{v}$/∂t: 局部加速度
  • ($\vec{v}$·∇)$\vec{v}$: 对流加速度 (非线性核心)
  • -∇p/ρ: 压力梯度
  • ν Δv→: 粘性扩散 (ν = 运动黏度)
  • $\vec{f}$: 体力 (重力等)

连续性方程 (不可压缩): ∇·$\vec{v}$ = 0.

Clay 千禧难题: 三维 N-S 方程是否存在光滑全局解? 至今未解决, 100 万美元奖金.

应用:

  • 天气预报: ECMWF/GFS 数值天气模型基础方程
  • 飞机设计: CFD 仿真用 N-S
  • 心血管医学: 模拟血流
  • 推进 AI 物理引擎: 流体特效 (好莱坞 VFX)

知识点 12.11: 散度与 Laplacian 的源汇直觉

散度 ∇·$\vec{F}$ = 体积"源"密度.

  • ∇·$\vec{F}$ > 0: 该点是"源" (流出 > 流入)
  • ∇·$\vec{F}$ < 0: 该点是"汇" (流入 > 流出)
  • ∇·$\vec{F}$ = 0: 无源无汇 (不可压缩流, 静电场真空)

Laplacian Δf = "二阶散度" 通量平均.

几何直觉:

  • Δf > 0: 当地值低于周围平均 (温度场: 该点正在升温)
  • Δf < 0: 当地值高于周围平均 (该点正在降温)
  • Δf = 0: 调和函数 (稳态)

热传导方程: ∂T/∂t = $\alpha$ ΔT. 解释: 温度变化率 ∝ 局部 Laplacian. 哪里"低于周围平均"哪里就被加热.

例题 12-11

f = x² + y² + z². Δf = 2 + 2 + 2 = 6 > 0. 该函数在每点 (除原点) 都是"凹" — 等于 r², 越远越高.

统一图谱: | 物理方程 | 形式 | |---|---| | 热传导 | ∂T/∂t = $\alpha$ ΔT | | 波动 | ∂²u/∂t² = c² Δu | | Schrödinger | iℏ ∂ψ/∂t = -ℏ²/(2m) Δψ + Vψ | | Poisson | ΔΦ = -ρ/ε₀ (静态) |

全部都是 Laplacian 的变奏曲. 这就是高数下册场论的统一威力.

知识点 12.12: Maxwell 应力张量 (电磁场的"压力")

Poynting 向量 (能量流密度)

$$\vec{S} = \frac{1}{\mu_0} \vec{E} \times \vec{B}$$ 单位 W/m². 给出电磁场能量流方向和强度.

电磁能量密度: $u = \frac{1}{2}(\epsilon_0 E^2 + B^2/\mu_0)$.

Poynting 定理 (电磁能量守恒)

$$\frac{\partial u}{\partial t} + \nabla \cdot \vec{S} = -\vec{E} \cdot \vec{J}$$ 即"局部能量增长率 + 能量流出散度 = $-$电场对电流做的功". 是 Gauss 散度定理的电磁学版本.

太阳能 / 激光威力来源

$|\vec{S}| = c u$ — 光携带能量沿光线传播.

  • 太阳辐照: 1361 W/m² (in AU 处, 太阳常数)
  • 激光: 通过聚焦 $\vec{S}$ 极高 (聚焦镜把 W/m² 提到 GW/m²)

例题 12-12

电磁波 (前面推导过) 的 Poynting 向量沿传播方向, $|\vec{S}|$ = c·u. 光携带能量沿光线传播.

应用:

  • 太阳能板: 计算辐照度 (1361 W/m² at AU)
  • 微波炉: Poynting 向量在腔内分布决定加热均匀性
  • 激光: 高能密度光束

知识点 12.13: 流形上的积分初步

流形 (Manifold)

局部看像 $\mathbb{R}^n$ 的几何对象, 整体可以是弯曲的.

  • 球面 $S^2$: 局部看像 $\mathbb{R}^2$ (你站在地球某点感觉是平的)
  • Möbius 带: 不可定向流形
  • 环面 $T^2$: 像甜甜圈 (拓扑同欧拉数 0)
  • 高维空间: 数据流形假设 (ML)

流形上积分 3 步法

  1. 用局部坐标 (chart) 把流形碎片化
  2. 在每片上做欧氏积分 (Riemann/Lebesgue 都行)
  3. 用单位分解 (partition of unity) 把局部积分粘起来

广义相对论用伪 Riemann 流形, ML 用数据流形假设, 量子场论用规范丛.

例题 12-13

(球面积分): ∫_S² 1 dA = 4πR². 取参数化 (球坐标 $\varphi$, $\theta$), dA = R² sin $\varphi$ dφ dθ, 积出来 4πR² ✓.

关键概念:

  • 切丛 (Tangent Bundle): 流形上每点的切空间组合
  • 余切丛 / 微分形式
  • 流形上的体积形式

应用:

  • 广义相对论: 时空是 4 维伪 Riemann 流形
  • 机器学习: 流形假设 (manifold hypothesis) — 高维数据实际在低维流形上
  • 拓扑数据分析 (TDA)
  • 数据可视化: t-SNE/UMAP 假设数据在低维流形上展开
