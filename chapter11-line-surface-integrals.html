一元定积分在直线上, 重积分在区域上, 这章在曲线和曲面上.

11.0 整章在干啥?

之前积分都在直线段 (一元) 或平面区域/立体区域 (重积分). 现实问题:

  • 沿弯曲的钢丝丝求质量 → 第一类曲线积分
  • 力沿弯曲路径做的功 → 第二类曲线积分
  • 弯曲钢板的质量 → 第一类曲面积分
  • 流体穿过曲面的通量 → 第二类曲面积分

核心工具: 格林公式 / 高斯公式 / 斯托克斯公式 — 把曲线/曲面积分转化为简单的重积分.

对你的意义: 电磁学麦克斯韦方程组 / 流体动力学 / 计算机图形学的光照渲染 / Bezier 曲线长度 — 全用.

11.0.5 入门: 为啥从"区域上积分" 升级到"曲线/曲面上"?

升级地图: 4 种积分 + 3 大公式

中学/上册的 $\int f(x)\,dx$ 在直线段上, 这章升级到"弯曲线/面"上积分.

4 种新积分: | 类型 | 几何对象 | 带方向? | 物理例子 | |---|---|---|---| | 第一类曲线 | 弧长 $ds$ | 不带 | 弯钢丝质量 | | 第二类曲线 | 坐标 $dx, dy$ | 带 | 力做功 | | 第一类曲面 | 面积 $dS$ | 不带 | 弯钢板质量 | | 第二类曲面 | 投影坐标 $dydz$ 等 | 带 | 流体通量 |

3 大公式 (把曲线/曲面积分转回重积分):

  • Green: 平面闭路 ↔ 平面区域
  • Gauss: 闭曲面通量 ↔ 立体内散度
  • Stokes: 曲面边界环流 ↔ 曲面旋度

学完你能描述麦克斯韦方程组 (电磁学终极公式) 的数学结构.

为啥不直接用重积分?

  • 重积分是"区域上 (xy 平面/立体)"积. 弯曲线/弯曲面不是简单区域, 要新方法.
  • 用"参数化": 把曲线/曲面写成参数形式, 转化成参数空间上的简单积分.

这章核心:

  • 4 个新积分: 第一/二类曲线 + 第一/二类曲面
  • 3 大公式: 格林 / 高斯 / 斯托克斯 — 把曲线/曲面积分转化为重积分

11.1 第一类曲线积分 (对弧长)

知识点 1: 定义

第一类曲线积分 (对弧长)

曲线 L 上, 密度 $\mu(x,y)$ 不均匀, 求质量. $$\int_L \mu(x,y)\,ds$$ $ds$ = 曲线上小弧长元素.

生活类比: 一根弯曲钢丝, 每点密度不同, 总质量 = 切成小段, 每段密度×长度, 加起来.

知识点 2: 计算 — 参数法

参数法弧微元公式

曲线 $L$ 参数 $x = x(t), y = y(t)$, $t \in [\alpha, \beta]$: $$\int_L f(x,y)\,ds = \int_\alpha^\beta f(x(t), y(t)) \sqrt{x'(t)^2 + y'(t)^2}\,dt$$ 其中 $ds = \sqrt{x'^2 + y'^2}\,dt$ 是弧微元.

例题 1

求圆周 x² + y² = 1 上 ∫ x² ds. 参数: x = cos t, y = sin t, t ∈ [0, 2π]. x'² + y'² = sin² t + cos² t = 1, ds = dt. ∫ = ∫₀^(2π) cos² t dt = $\pi$

例题 1-2

一根直线段 L 从 (0,0) 到 (3,4), 线密度 ρ(x,y) = x+y, 求质量. 参数 x = 3t, y = 4t, t ∈ [0,1], ds = √(9+16) dt = 5dt. M = ∫_L (x+y) ds = ∫_0¹ (3t+4t) · 5 dt = 35 · 1/2 = 17.5.

例题 1-3

求半圆 y = √(1-x²) (y$\ge$0, x ∈ [-1,1]) 的弧长. 弧长 = ∫_L 1 ds. 参数: x = cos t, y = sin t, t ∈ [0, $\pi$], ds = dt. 弧长 = ∫_0^$\pi$ dt = $\pi$. ✓ (半径 1 半圆的弧长 = πR = $\pi$)

知识点 1.5: 第一类曲线积分的物理应用 (对齐书 §11.1 一)

第一类曲线积分能算很多物理量, 都是"密度乘弧长" 的累计:

  1. 弧长: L = ∫_L 1 ds (密度 = 1)
  2. 质量: M = ∫_L $\mu$(x,y,z) ds
  3. 质心:

x̄ = ∫_L x·$\mu$ ds / M, ȳ = ∫_L y·$\mu$ ds / M

  1. 转动惯量 (绕 z 轴): I_z = ∫_L (x²+y²) $\mu$ ds

例题 1-2

(质心): 一根均匀钢丝沿曲线 y = x² (0 ≤ x ≤ 1), 求质心 x̄. 参数: x = t, y = t², ds = √(1+4t²) dt. M = ∫_0¹ √(1+4t²) dt (设密度 1, 弧长本身). x̄·M = ∫_0¹ t·√(1+4t²) dt = (1/12)[(1+4t²)^(3/2)]_0¹ = (5$\sqrt{5}$ - 1)/12. M = (1/2)[t√(1+4t²) + (1/2) sinh^(-1)(2t)]_0¹ = √5/2 + ln(2+√5)/4. (具体值代入算)

知识点 2.5: 第一类曲线积分性质

第一类曲线积分性质

  1. 方向无关: 沿 $L$ 正向或反向走, 积分值一样 (因为 $ds > 0$)
  2. 可加性: $\int_{L_1 + L_2} f\,ds = \int_{L_1} f\,ds + \int_{L_2} f\,ds$
  3. 线性: $\int_L (af + bg)\,ds = a\int_L f\,ds + b\int_L g\,ds$

跟普通定积分像

第一类曲线积分本质上"把弯曲路径当直线段算长度+加权", 跟一元 $\int_a^b f(x)\,dx$ 性质一致.

11.2 第二类曲线积分 (对坐标)

知识点 2.7: 第一类曲线积分参数法详细 (对齐书 §11.1 二)

情况 1: 一般参数 x=x(t), y=y(t), t ∈ [$\alpha$, $\beta$] ds = √(x'(t)² + y'(t)²) dt ∫_L f ds = ∫_α^$\beta$ f(x(t), y(t)) · √(x'² + y'²) dt

情况 2: 直接 y = y(x), x ∈ [a, b] (相当于 x=x, y=y(x)) ds = √(1 + y'(x)²) dx ∫_L f ds = ∫_a^b f(x, y(x)) · √(1+y'²) dx

情况 3: 极坐标 r = r($\theta$), $\theta$ ∈ [$\alpha$, $\beta$] ds = √(r² + r'²) dθ ∫_L f ds = ∫_α^$\beta$ f(r $\cos\theta$, r $\sin\theta$) · √(r²+r'²) dθ

例题 1-3

(一般参数): 螺旋线 x = cos t, y = sin t, z = t (t ∈ [0, 2π]) 上 ∫(x²+y²+z²) ds. ds = √(sin²+cos²+1) dt = √2 dt. x²+y²+z² = 1 + t². ∫ = ∫_0^(2π) (1+t²) · √2 dt = √2 · [t + t³/3]_0^(2π) = √2 · (2π + 8π³/3).

知识点 3: 定义 (力沿路径做功)

第二类曲线积分 (对坐标)

变力 $\vec{F}(x,y) = (P, Q)$ 沿曲线 L 做的功: $$W = \int_L \vec{F}\cdot d\vec{r} = \int_L P\,dx + Q\,dy$$

生活类比: 你推车, 力跟方向都变, 走弯曲路径, 总功 = $\vec{F}$ · dr→ 沿路径积分.

第一类 vs 第二类区别:

  • 第一类: ds 是标量 (长度), 跟方向无关
  • 第二类: dx, dy 带方向 (反向走积分变号)

知识点 3.5: 第二类曲线积分跟方向相关 (对齐书 §11.2 一 进一步)

第二类必看方向

第二类曲线积分跟路径方向相关! 反向走积分变号: $$\int_{L^-} P\,dx + Q\,dy = -\int_L P\,dx + Q\,dy$$ 做题忘标方向 (逆时针/顺时针, 从 A 到 B 还是 B 到 A) → 符号错.

(L^- 表示 L 的反向)

物理意义: 力沿正方向做正功, 沿反方向做负功 (反向走"克服了"力).

例题 3-3

(验证): L 从 (0,0) 到 (1,0), $\vec{F}$ = (1, 0) (沿 x 轴恒力). 正方向: W = ∫_L 1·dx + 0·dy = ∫_0¹ 1 dx = 1. 反方向 (从 (1,0) 回 (0,0)): W = ∫_(L^-) = ∫_1⁰ 1 dx = -1. ✓

第一类对比: ∫_L f ds 不分方向 (因为 ds > 0 永远正).

知识点 4: 计算

第二类曲线积分参数法

曲线 $L$ 参数化 $x = x(t), y = y(t)$, $t \in [\alpha, \beta]$ (从 $\alpha$ 到 $\beta$ 即正向): $$\int_L P\,dx + Q\,dy = \int_\alpha^\beta \left[P(x(t), y(t)) x'(t) + Q(x(t), y(t)) y'(t)\right] dt$$ 关键: $dx \to x'(t) dt$, $dy \to y'(t) dt$.

例题 2

∫_L y dx + x dy, L 是从 (0,0) 到 (1,1) 的直线. 参数: x = t, y = t, t ∈ [0, 1], x' = y' = 1. = ∫_0¹ [t · 1 + t · 1] dt = ∫_0¹ 2t dt = 1

例题 2-2

(功的实际计算): 力场 $\vec{F}$ = (-y, x), 沿单位圆逆时针走一圈做的功? 参数 x = cos t, y = sin t, t ∈ [0, 2π], dx = -sin t dt, dy = cos t dt. W = ∫_L P dx + Q dy = ∫_0^(2π) [(-sin t)(-sin t) + cos t · cos t] dt = ∫_0^(2π) (sin² t + cos² t) dt = ∫_0^(2π) 1 dt = 2π.

例题 2-3

(路径相同, 端点不同, 结果不同): ∫_L (y dx - x dy), L 从 (0,0) 到 (1,1).

  • 路径 A (直线 y = x): 参数 x = t, y = t. = ∫_0¹ (t · 1 - t · 1) dt = 0.
  • 路径 B (折线先 (0,0)→(1,0) 再 (1,0)→(1,1)):

段 1: y=0, dy=0, x: $\vec{0}$1, ∫ 0 dx - x · 0 = 0. 段 2: x=1, dx=0, y: $\vec{0}$1, ∫ y · 0 - 1 · dy = -1. 总 = -1. 结论: A 跟 B 端点相同但路径不同 → 积分不同 (说明这不是保守场, 不满足路径无关条件).

11.3 格林公式 (本章核心 1)

知识点 4.5: 两类曲线积分之间的联系 (对齐书 §11.2 三)

两类曲线积分转换

第二类 $\int_L P\,dx + Q\,dy$ 可通过切向单位向量 $(\cos\alpha, \sin\alpha)$ 转换为第一类: $$\int_L (P, Q)\cdot d\vec{r} = \int_L (P\cos\alpha + Q\sin\alpha)\,ds$$

第一类 vs 第二类 区别

  • 第一类 (对弧长): $ds$ 是标量长度, 跟方向无关. 反向走积分不变号.
  • 第二类 (对坐标): $dx, dy$ 带方向. 反向走积分变号 (做负功).

记忆: 第一类 = "走多远 (距离)", 第二类 = "顺势走还是逆势走 (有向)".

用法: 当 P, Q 难直接积时, 把 (P cosα + Q sinα) 当成新被积函数, 用第一类公式.

例题 4-2

(转换): ∫_L 2x dy - y dx, L 是单位圆周逆时针. 参数: x=cos t, y=sin t. dx=-sin t dt, dy=cos t dt. = ∫_0^(2π) (2 cos t · cos t - sin t · (-sin t)) dt = ∫_0^(2π) (2cos²t + sin²t) dt = ∫_0^(2π) (1 + cos²t) dt = 2π + $\pi$ = 3π.

知识点 5: 格林公式

Green 公式

联系第二类曲线积分跟二重积分. D 是平面闭区域, $\partial D$ 是边界 (正方向 = 逆时针): $$\oint_{\partial D} P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dxdy$$

思想: 沿闭曲线积分 = 内部"旋度"的累加.

例题 3

用格林公式求 ∮_C (x+y) dx + (x-y) dy, C 是单位圆周逆时针. P = x+y, Q = x-y. Q_x = 1, P_y = 1, 被积函数 Q_x - P_y = 0. 所以 ∮ = 0.

例题 3-2

(用格林公式算面积): 用 ∮ x dy 算单位圆面积. S = ∮_C x dy. 参数 x = cos t, y = sin t, dy = cos t dt. = ∫_0^(2π) cos t · cos t dt = ∫_0^(2π) cos² t dt = $\pi$. ✓

面积公式: 任何由闭曲线 C 围的平面区域 D 的面积: S = (1/2) ∮_C (x dy - y dx) = ∮_C x dy = -∮_C y dx (三个公式都对, 选 C 上容易算的)

例题 3-3

用格林公式算 ∮_C 2xy dx + x² dy, C 是顺时针单位圆周. 逆时针时: P = 2xy, Q = x², Q_x = 2x, P_y = 2x, Q_x - P_y = 0 → ∮ = 0. 顺时针 = -(逆时针) = 0.

知识点 5.5: 格林公式的推导 (对齐书 §11.3 一)

Green 公式推导直觉

把闭区域 $D$ 切成大量小矩形 $\Delta S$, 每小矩形里 $(Q_x - P_y) \Delta S$ 等于沿小矩形边界的 $P\,dx + Q\,dy$. 相邻小矩形共边方向相反, 内部环量相消, 加起来只剩最外圈 = $\oint_{\partial D}$.

这是"局部 → 全局"的典型: 内部累加 = 边界积分.

Green 公式算面积

$$S(D) = \frac{1}{2}\oint_{\partial D} (x\,dy - y\,dx) = \oint_{\partial D} x\,dy = -\oint_{\partial D} y\,dx$$ 三个公式都对 (因为 $\iint_D 1\,dxdy = S$, 取 $P = -y/2, Q = x/2$ 或 $P=0, Q=x$ 或 $P=-y, Q=0$).

例题 3-4

用格林公式算椭圆 x²/a² + y²/b² = 1 的面积. 参数 x = a cos t, y = b sin t. dx = -a sin t dt, dy = b cos t dt. S = (1/2)∮(x dy - y dx) = (1/2)∫_0^(2π) [a cos t · b cos t - b sin t · (-a sin t)] dt = (1/2) · ab · ∫_0^(2π) (cos²t + sin²t) dt = (1/2) · ab · 2π = πab ✓

应用价值: 不规则区域面积难算, 沿边界一圈积分往往容易.

知识点 6: 平面上曲线积分与路径无关

路径无关判据 (单连通区域)

$\int_L P\,dx + Q\,dy$ 路径无关 ⟺ 在单连通区域内 $\dfrac{\partial Q}{\partial x} = \dfrac{\partial P}{\partial y}$ ⟺ 存在原函数 $u(x,y)$ 使 $du = P\,dx + Q\,dy$.

路径无关 = 保守力

"重力做功只跟起点终点高度差有关, 路径无关" — 重力是保守力, 满足该条件. 摩擦力做功跟路径长度有关, 不满足.

工程意义: 路径无关时, 起点 A → 终点 B 的功 = $u(B) - u(A)$, 不需要算整条路径.

知识点 6.7: 保守力场与势能函数

保守力场

力场 $\vec{F}$ 称为保守的, 当且仅当沿任意闭路径做功为零: $$\oint \vec{F}\cdot d\vec{r} = 0$$ 等价于存在势能函数 $U$ 使 $\vec{F} = -\nabla U$.

保守力场 3 等价 (单连通区域)

  1. $\oint \vec{F}\cdot d\vec{r} = 0$ 沿任意闭路径
  2. 存在 $U$ 使 $\vec{F} = -\nabla U$ (势能函数)
  3. $\nabla \times \vec{F} = 0$ (旋度为 0)

保守力场举例:

  • 重力: $\vec{F}$ = -mg $\vec{k}$, U = mgh
  • 弹簧弹力: F = -kx, U = (1/2)kx²
  • 库仑电力: $\vec{F}$ = -k Qq/r² r̂→, U = kQq/r
  • 万有引力: $\vec{F}$ = -GMm/r² r̂→, U = -GMm/r

非保守力:

  • 摩擦力 (依赖路径长度, 不依赖位移)
  • 磁场对运动电荷的洛伦兹力 (永远垂直速度, 不做功但场不保守)

判别 3 等价条件 (单连通区域):

  1. ∮ $\vec{F}$·dr→ = 0 (任意闭路径)
  2. 存在 U 使 $\vec{F}$ = -∇U
  3. ∇ × $\vec{F}$ = 0 (旋度为 0)

例题 6-4

$\vec{F}$ = (y, x, 0). 检查是否保守. ∇×$\vec{F}$ = (0, 0, ∂x/∂x - ∂y/∂y) = (0, 0, 0). 保守 ✓ 势能: ∂U/∂x = -y → U = -xy + g(y). ∂U/∂y = -x = -x + g'(y) → g'(y)=0. U = -xy. $\vec{F}$ = -∇U = (y, x, 0) ✓.

11.4 第一类曲面积分 (对面积)

知识点 6.5: 全微分原函数 (对齐书 §11.3 三)

全微分判别

设 $P, Q$ 在单连通区域 $D$ 上连续可微. 则下列等价:

  1. $P\,dx + Q\,dy$ 是某函数 $u(x,y)$ 的全微分 (即 $du = P\,dx + Q\,dy$)
  2. $\dfrac{\partial Q}{\partial x} = \dfrac{\partial P}{\partial y}$
  3. $\int_L P\,dx + Q\,dy$ 跟路径无关 (只跟起点终点有关)
  4. 沿任意闭路径 $\oint_C P\,dx + Q\,dy = 0$

跟物理保守力对应

$u$ 即"势能", $\vec{F}=(P,Q)$ 即保守力场. 重力/弹簧力/库仑力都满足这条件. 摩擦力不满足 (路径相关).

求 u 的方法:

  1. 对 ∂u/∂x = P 关于 x 积分: u = ∫P dx + g(y)
  2. 求 ∂u/∂y, 跟 Q 比较, 解出 g(y)

例题 6-3

验证 (2x+y) dx + (x+2y) dy 是全微分, 求原函数. P = 2x+y, Q = x+2y. ∂Q/∂x = 1 = ∂P/∂y ✓. u = ∫(2x+y) dx = x² + xy + g(y). ∂u/∂y = x + g'(y) = Q = x + 2y → g'(y) = 2y → g(y) = y². u = x² + xy + y². (验证: du = (2x+y)dx + (x+2y)dy ✓)

物理意义: 保守力场存在势能函数. 重力/弹性力/库仑电力都是保守的, 总有"势能".

知识点 7: 定义

第一类曲面积分 (对面积)

曲面 $\Sigma$ 上密度 $\mu(x,y,z)$, 求质量: $$\iint_\Sigma \mu(x,y,z)\,dS$$ $dS$ = 曲面上小面积元. 物理意义: $\mu$=密度则 = 总质量; $\mu=1$ 则 = 曲面面积.

知识点 7.5: 第二类曲面积分参数化 (对齐书 §11.5)

第二类 vs 第一类 曲面积分

  • 第一类 (对面积): $\iint_\Sigma f\,dS$, $dS > 0$, 方向无关
  • 第二类 (对坐标): $\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy$, 带方向 (上侧/下侧 / 内侧/外侧)

第二类换侧 (反向) → 积分变号.

第二类曲面积分投影公式

对于曲面 $\Sigma: z = z(x,y)$ 投影到 $xy$ 平面得 $D_{xy}$: $$\iint_\Sigma R(x,y,z)\,dxdy = \pm\iint_{D_{xy}} R(x, y, z(x,y))\,dxdy$$ 取 "+" 当 $\Sigma$ 是上侧 (法向量与 $z$ 轴正向夹锐角), "−" 当下侧.

(上侧+, 下侧-)

对 P dydz, Q dzdx 类似, 投到 yz, zx 平面.

例题 5-3

(经典): ∬_Σ z dxdy, Σ 是单位球面外侧. 上半: z=√(1-x²-y²), 投 D_xy: x²+y²$\le$1. ∬_(上侧) = ∬_D √(1-x²-y²) dxdy. 极坐标: = ∫_0^(2π)∫_0¹ √(1-r²)·r drdθ = 2π·1/3 = 2π/3. 下半: z=-√(1-x²-y²), 投影同, 下侧符号取负. ∬_(下侧) = -∬_D (-√(1-x²-y²)) dxdy = ∫_D √... dxdy = 2π/3. 合计: 4π/3.

知识点 8: 计算

曲面 z = z(x,y), 投影到 xy 平面得 D_(xy): dS = √(1 + z_x² + z_y²) dxdy ∬_Σ f dS = ∬_(D_(xy)) f(x,y,z(x,y)) √(1+z_x²+z_y²) dxdy

例题 4

∬_Σ z dS, Σ 是 z = 1 - x² - y² 在 z ≥ 0 部分. z_x = -2x, z_y = -2y, √(1+4x²+4y²). 投影 D_(xy): x²+y² ≤ 1. ∬ (1-x²-y²)√(1+4(x²+y²)) dxdy 换极坐标 r² = x²+y²: = ∫₀^(2π)∫_0¹ (1-r²)√(1+4r²) · r drdθ (具体值代入算)

知识点 8.5: 曲面面积公式推导 (对齐书 §11.4)

曲面面积公式

曲面 $z = f(x,y)$ 在区域 $D$ 上的面积: $$A = \iint_D \sqrt{1 + f_x^2 + f_y^2}\,dxdy$$

几何推导

切片小区域 $\Delta S_{xy}$ 对应曲面一片倾斜小面积 $\Delta S$. 倾斜板面积 = 平面投影 / $|\cos\alpha|$, 其中 $\alpha$ 是切平面跟 xy 面的夹角. $$\cos\alpha = \frac{1}{\sqrt{1 + f_x^2 + f_y^2}}$$ 所以 $\Delta S = \sqrt{1 + f_x^2 + f_y^2}\,\Delta S_{xy}$.

加和取极限: A = ∬_D √(1 + f_x² + f_y²) dxdy

例题 4-2

(半球面积): 半球面 z = √(R² - x² - y²) (z ≥ 0) 的面积. f_x = -x/z, f_y = -y/z. 1 + f_x² + f_y² = 1 + (x²+y²)/z² = R²/(R²-x²-y²). A = ∬_(x²+y²$\le$R²) R/√(R²-x²-y²) dxdy. 极坐标: A = ∫_0^(2π) ∫_0^R R·r/√(R²-r²) drdθ = 2πR · [-√(R²-r²)]_0^R = 2πR · R = 2πR². ✓ (整个球面 4πR², 半个 2πR² ✓)

11.5 第二类曲面积分 (对坐标)

知识点 8.7: 第一类曲面积分参数化 (对齐书 §11.4 二)

曲面积分两种参数化

直接参数 $z = z(x,y)$ (用 $xy$ 投影): $$dS = \sqrt{1 + z_x^2 + z_y^2}\,dxdy$$ $$\iint_\Sigma f\,dS = \iint_{D_{xy}} f(x, y, z(x,y)) \sqrt{1+z_x^2+z_y^2}\,dxdy$$

一般参数方程 $\vec{r}(u, v) = (x(u,v), y(u,v), z(u,v))$: $$dS = |\vec{r}_u \times \vec{r}_v|\,dudv$$

球面参数化速记

球面 $x^2+y^2+z^2 = R^2$ 用球坐标 ($r=R$ 固定, $\varphi, \theta$ 变): $$dS = R^2 \sin\varphi\,d\varphi\,d\theta$$ 故球面面积 $= \int_0^{2\pi}\!\!\int_0^\pi R^2 \sin\varphi\,d\varphi\,d\theta = 4\pi R^2$.

例题 4-3

(球面积分): ∬_Σ (x²+y²+z²) dS, Σ 是 x²+y²+z²=R² 球面. 在球面上 x²+y²+z² = R² 为常数. = R² · S(球) = R² · 4πR² = 4πR⁴.

知识点 9: 定义 (流量/通量)

第二类曲面积分 (对坐标 / 通量)

流体流速场 $\vec{v} = (P, Q, R)$, 通过有向曲面 $\Sigma$ 的通量 (flux): $$\iint_\Sigma \vec{v}\cdot d\vec{S} = \iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy$$ 单位时间内穿过曲面的流量.

生活类比: 河流流过一片网, 通量 = 单位时间穿过网的水量.

知识点 9.5: 第二类曲面积分跟第一类的转换 (对齐书 §11.5 三)

两类曲面积分转换

设 $\vec{n} = (\cos\alpha, \cos\beta, \cos\gamma)$ 是曲面 $\Sigma$ 单位外法向量: $$\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iint_\Sigma (P\cos\alpha + Q\cos\beta + R\cos\gamma)\,dS$$ 即 $\iint_\Sigma \vec{F}\cdot d\vec{S} = \iint_\Sigma \vec{F}\cdot \vec{n}\,dS$.

几何理解

$d\vec{S} = \vec{n}\,dS$ — 第二类的 $d\vec{S}$ 是带方向的面积元 (法向乘面积). 转换公式只是把方向显式拆出来.

例题 9-3

(转换示例): ∬_Σ x dydz + y dzdx + z dxdy, Σ 是球面 x²+y²+z²=R² 外侧. 外法向量: $\vec{n}$ = (x/R, y/R, z/R). (P, Q, R) = (x, y, z), 点乘 $\vec{n}$ = (x²+y²+z²)/R = R²/R = R. 转第一类: ∬_Σ R dS = R · 4πR² = 4πR³. 跟之前用高斯公式直接算得 4πR³ ✓.

知识点 10: 高斯公式 (本章核心 2)

Gauss 公式 (散度定理)

联系闭合曲面通量跟体积内散度. $V$ 是空间闭区域, $\partial V$ 是边界曲面 (取外侧): $$\oiint_{\partial V} \vec{F}\cdot d\vec{S} = \iiint_V \nabla\cdot\vec{F}\,dV$$

Gauss 直觉

闭曲面流出去多少 = 内部"产出"多少.

  • 真空电场: $\nabla\cdot\vec{E} = \rho/\epsilon_0$ (电荷 = 源)
  • 不可压缩流体: $\nabla\cdot\vec{v} = 0$ (无源汇)
  • 流出体积 = 内部散度积分

是"局部源累加 = 边界通量"的高维形式.

通俗: 闭曲面流出去多少 = 内部"产出"多少 (散度).

例题 5

∯_Σ x dydz + y dzdx + z dxdy, Σ 是单位球外侧. P_x + Q_y + R_z = 1 + 1 + 1 = 3. = ∭_V 3 dV = 3 · (4)/(3)$\pi$ = 4π 经典!

例题 5-2

(求体积的高斯公式技巧): 用 ∯ z dxdy 算单位球体积. P = 0, Q = 0, R = z. P_x + Q_y + R_z = 1. ∯ z dxdy = ∭_V 1 dV = V (球体积). 所以 V = ∯ z dxdy. 等价地, V = (1/3) ∯ (x dydz + y dzdx + z dxdy). (代入单位球: V = (1/3) · 4π = 4π/3 ✓)

例题 5-3

求向量场 $\vec{F}$ = (x², y², z²) 通过单位立方体表面 (外侧) 的通量. div F = 2x + 2y + 2z. ∯ F· dS = ∭_V (2x+2y+2z) dV = 2 ∫_0¹$\int$_0¹$\int$_0¹ (x+y+z) dx dy dz = 2 · 3 · (1/2) = 3. (对称性: ∭ x dV = ∭ y dV = ∭ z dV = 1/2)

11.6 斯托克斯公式 (本章核心 3)

知识点 10.5: 高斯公式 vs 闭曲面 vs 开曲面 (对齐书 §11.6)

Gauss 只对闭曲面有效

Gauss 公式要求曲面闭合 (有内外明确分隔, 取外侧). 开曲面 (没包围立体) 不能直接套.

补面法 (常用技巧)

开曲面 $\Sigma$ → 补一个 $\Sigma'$ 让 $\Sigma \cup \Sigma'$ 闭合包围 $V$:

  1. 用 Gauss: $\oiint_{\Sigma \cup \Sigma'} \vec{F}\cdot d\vec{S} = \iiint_V \nabla\cdot\vec{F}\,dV$
  2. 反推: $\iint_\Sigma \vec{F}\cdot d\vec{S} = \iiint_V \nabla\cdot\vec{F}\,dV - \iint_{\Sigma'} \vec{F}\cdot d\vec{S}$

(补面通常选平面, 因为平面上的曲面积分容易算.)

例题 5-4

∬_Σ x dydz + y dzdx + z dxdy, Σ 是 z = 1-x²-y² (z ≥ 0) 上侧. 不是闭合. 补 Σ' = {z=0, x²+y²$\le$1} 下侧. Σ ∪ Σ' 包围立体 V (上方抛物面下方圆盘). div(P,Q,R) = 1+1+1 = 3. ∯ = 3·V体积 = 3·∫_0¹ $\pi$(1-r²)·... 算得 3π/2. Σ' 部分: 在 z=0 平面, dz=0, x, y 部分也不贡献. ∬_(Σ') = 0. 所以 ∬_Σ = 3π/2.

知识点 11: 斯托克斯公式

Stokes 公式

联系有向曲面的旋度 跟 边界曲线的环流量. $\Sigma$ 是空间有向曲面, $\partial\Sigma$ 是边界 (右手法则取正向): $$\oint_{\partial\Sigma} \vec{F}\cdot d\vec{r} = \iint_\Sigma (\nabla\times\vec{F})\cdot d\vec{S}$$

右手法则定向

$\Sigma$ 法向量 $\vec{n}$ 跟 $\partial\Sigma$ 正向必须用右手法则匹配: 大拇指指 $\vec{n}$, 四指弯曲方向即 $\partial\Sigma$ 正向. 配错符号反.

Green 公式是 Stokes 公式取 $\Sigma$ 平面区域、$\vec{n} = \vec{k}$ (上侧) 的特例.

通俗: 沿边界绕一圈的环流量 = 曲面上"旋度"累加. 格林公式是它的平面特例.

知识点 11.5: 斯托克斯公式的物理意义详解

Stokes 物理意义

核心想象: 一片旋转的水面 (曲面 $\Sigma$), 边界是绕圈水流 ($\partial\Sigma$).

$$\oint_{\partial\Sigma} \vec{F}\cdot d\vec{r} = \iint_\Sigma (\nabla\times\vec{F})\cdot d\vec{S}$$

左边: 边界绕一圈的"环流量" (绕圈水流的总流速贡献). 右边: 曲面上每一点"旋度向量"穿过曲面的总通量 ("局部转速"的全局累加).

二者相等 = 局部旋转 (旋度) 累加 = 边界整体环流.

左边: 沿边界绕一圈做的功 (环量). 右边: 曲面内部每点"旋度" 的累加 (通量形式).

物理验证 (法拉第电磁感应): ∮_L $\vec{E}$·dr→ = - d/dt ∬_Σ $\vec{B}$·dS→

  • 左: 闭合导线电动势 (环量)
  • 右: 磁通量随时间变化的负数

这就是发电机原理 — 转动的磁铁让线圈周围 $\vec{B}$ 变, 引起 $\vec{E}$ 环量 (电压).

例题 11

$\vec{F}$ = (-y, x, 0) 沿单位圆周 x²+y²=1 (z=0) 一圈的环量. 直接算: ∮ -y dx + x dy. 参数 x=cos t, y=sin t. = ∫_0^(2π) (sin²t + cos²t) dt = 2π.

用斯托克斯验证: ∇×$\vec{F}$ = (∂(0)/∂y - ∂x/∂z, ∂(-y)/∂z - ∂(0)/∂x, ∂x/∂x - ∂(-y)/∂y) = (0, 0, 2). ∬_Σ 2 dS = 2 · $\pi$ (圆面积) = 2π ✓

知识点 12: 三大公式总结

| 公式 | 联系 | |---|---| | 牛顿-莱布尼茨 (上册) | 区间积分 ↔ 端点值 | | 格林公式 (平面) | 闭曲线 ↔ 平面区域 | | 高斯公式 (空间) | 闭曲面 ↔ 立体体积 | | 斯托克斯公式 (空间) | 空间闭曲线 ↔ 曲面 |

统一思想: 边界积分 = 内部某种"导数"的积分. 这是微积分基本定理的高维推广. 物理叫"边界 → 内部"对应关系.

11.6 通量与散度 (对齐书 §11.6 三)

知识点 12.5: 通量 (flux)

通量 (Flux)

向量场 $\vec{F} = (P, Q, R)$ 通过有向曲面 $\Sigma$ 的通量: $$\Phi = \iint_\Sigma \vec{F}\cdot d\vec{S} = \iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy$$

通量物理意义

流体速度场 $\vec{F}$, 通量 $\Phi$ = 单位时间穿过曲面 $\Sigma$ 的流量 (体积). 电场通量则关联电荷 (Gauss 定律: $\Phi_E = Q/\epsilon_0$).

知识点 12.6: 散度 (divergence)

散度

向量场 $\vec{F} = (P, Q, R)$ 的散度是标量场: $$\text{div}\,\vec{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z} = \nabla \cdot \vec{F}$$

散度几何/物理意义

  • $\text{div}\,\vec{F} > 0$: 源 (向外发射)
  • $\text{div}\,\vec{F} < 0$: 汇 (向内吸收)
  • $\text{div}\,\vec{F} = 0$: 无源场 (流入 = 流出)

类比: 湖某点散度 > 0 = 泉眼往外冒水; 散度 < 0 = 漏洞往里吸水.

高斯公式 (向量形式): ∯_(∂V) $\vec{F}$ · dS→ = ∭_V div $\vec{F}$ dV 通量 = 散度的体积积分. 闭曲面通量 = 内部源/汇 累计.

11.7 环流量与旋度 (对齐书 §11.7 三)

知识点 12.7: 环流量 (circulation)

环流量

向量场 $\vec{F}$ 沿闭曲线 $L$ 的环流量: $$\Gamma = \oint_L \vec{F}\cdot d\vec{r} = \oint_L P\,dx + Q\,dy + R\,dz$$

环流量物理意义

沿闭曲线"绕一圈" $\vec{F}$ 做的总功. 涡旋中心附近环流量大; 均匀流环流量为 0; 保守力沿任意闭曲线环流量为 0.

知识点 12.8: 旋度 (curl)

旋度

向量场 $\vec{F} = (P, Q, R)$ 的旋度是向量场: $$\text{curl}\,\vec{F} = \nabla \times \vec{F} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ \partial/\partial x & \partial/\partial y & \partial/\partial z \\ P & Q & R \end{vmatrix}$$ 展开: $\left(\dfrac{\partial R}{\partial y} - \dfrac{\partial Q}{\partial z}, \dfrac{\partial P}{\partial z} - \dfrac{\partial R}{\partial x}, \dfrac{\partial Q}{\partial x} - \dfrac{\partial P}{\partial y}\right)$

小风车几何

某点放小风车, 旋度方向 = 风车轴方向, 旋度大小 = 转速.

  • $\nabla\times\vec{F} = 0$: 无旋场 (流体不打转, 即保守场)
  • $|\nabla\times\vec{F}|$ 大: 涡旋中心

类比: 龙卷风中心旋度极大, 平直气流旋度为 0.

斯托克斯公式 (向量形式): ∮_(∂Σ) $\vec{F}$ · dr→ = ∬_Σ rot $\vec{F}$ · dS→ 环流量 = 旋度通过曲面 的通量. 闭曲线环流 = 内部旋转累计.

物理意义对应:

  • 麦克斯韦电磁: ∇·E = ρ/ε₀ (高斯定律, 电场散度跟电荷密度) / ∇×B = μ₀J + ... (安培定律, 磁场旋度跟电流)
  • 流体: ∇·$\vec{v}$ = 0 (不可压缩) / ∇×$\vec{v}$ = 0 (无旋流)

知识点 12.9: 三大公式统一思想 (Generalized Stokes 定理)

广义 Stokes 定理

$$\int_{\partial M} \omega = \int_M d\omega$$ 其中 $M$ 是流形, $\partial M$ 是其边界, $\omega$ 是微分形式, $d\omega$ 是外导数.

四个公式 = 一个公式的不同维度

| 公式 | 维度 | 边界 | 内部 | |---|---|---|---| | 牛顿-莱布尼茨 | 1D | 端点 $a, b$ | 区间 $[a, b]$ | | Green | 2D | 闭曲线 $\partial D$ | 平面区域 $D$ | | Gauss | 3D | 闭曲面 $\partial V$ | 立体 $V$ | | Stokes | 2D 嵌入 3D | 曲面边界 $\partial\Sigma$ | 曲面 $\Sigma$ |

核心: 都是微积分基本定理的高维推广 — "积分等于原函数在端点值的差", 从 1D 推广到任意维.

应用:

  • 流体动力学: Navier-Stokes 方程
  • 电磁学: Maxwell 方程组 (4 个方程都是这几个公式的物理形式)
  • 广义相对论: 黎曼几何积分
  • 拓扑学: 同调论

知识点 12.10: 三大公式记忆口诀

口诀 (用关系而不用公式):

  • "闭曲线绕一圈 = 平面区域内每点的"旋度" 累加" → 格林
  • "闭曲面通量 = 立体内每点的"散度" 累加" → 高斯
  • "空间闭曲线环量 = 曲面上每点的"旋度" 累加" → 斯托克斯
  • 关键 idea: 边界 = 内部 d 算子的积分

知识点 12.11: 麦克斯韦方程组 — 微积分应用经典

麦克斯韦电磁学 4 个方程, 都是高数下册的公式 — 这章学的就是它们的数学基础:

1. 高斯定律 (电): ∮_(∂V) $\vec{E}$ · dS→ = Q/ε₀

  • 类: 高斯公式 (闭曲面通量 = 内部电荷)
  • 微分形式: ∇·$\vec{E}$ = ρ/ε₀ (电场散度 = 电荷密度)

2. 高斯定律 (磁): ∮_(∂V) $\vec{B}$ · dS→ = 0

  • 闭曲面磁通量 = 0 (磁场没"源", 只有"环")
  • 微分形式: ∇·$\vec{B}$ = 0

3. 法拉第电磁感应: ∮_L $\vec{E}$ · dr→ = -dΦ/dt

  • 类: 斯托克斯公式 (环量 = 旋度通量)
  • 微分形式: ∇×$\vec{E}$ = -∂$\vec{B}$/∂t

4. 安培-麦克斯韦定律: ∮_L $\vec{B}$ · dr→ = μ₀(I + ε₀ dΦ_E/dt)

  • 类: 斯托克斯公式
  • 微分形式: ∇×$\vec{B}$ = μ₀($\vec{J}$ + ε₀ ∂$\vec{E}$/∂t)

补充: 场论 (Field Theory) 综合

知识点 12.1: 三种"导数算子"梳理

向量场分析有三个核心算子, 各对应一种物理直觉:

| 算子 | 输入 → 输出 | 公式 | 物理意义 | |---|---|---|---| | 梯度 grad / ∇ | 标量场 → 向量场 | (∂f/∂x, ∂f/∂y, ∂f/∂z) | 函数最快上升方向 | | 散度 div / ∇· | 向量场 → 标量场 | ∂P/∂x + ∂Q/∂y + ∂R/∂z | 体积"源"密度 (流出 - 流入) | | 旋度 curl / ∇× | 向量场 → 向量场 | (∂R/∂y - ∂Q/∂z, ...) | 局部"涡旋"轴向量 |

两个零等式 (基本拓扑):

  • ∇ × (∇f) = 0 — 梯度场无旋 (势能场无环流)
  • ∇ · (∇ × $\vec{F}$) = 0 — 旋度场无源 (没有"磁单极")

知识点 12.2: 拉普拉斯算子 Δ = ∇·∇

定义: Δf = ∂²f/∂x² + ∂²f/∂y² + ∂²f/∂z² = div(grad f)

Δf = 0 的解叫"调和函数", 物理含义: 该点的值 = 周围球面平均值 (平均值性质).

应用全表:

  • 静电势满足 Δφ = -ρ/ε₀ (Poisson 方程)
  • 真空: Δφ = 0 (Laplace 方程)
  • 稳态热传导: ΔT = 0
  • 重力势: Δφ = 4πGρ
  • 量子力学: -(ℏ²/2m) Δψ + Vψ = Eψ (薛定谔方程)
  • 计算机视觉边缘检测: Laplacian-of-Gaussian (LoG)

知识点 12.3: 流体力学的连续性方程

质量守恒律微分形式: ∂ρ/∂t + ∇·(ρv→) = 0

推导: 高斯定理 ∮∮ ρv→·dS→ = ∭ ∇·(ρv→) dV 是流出质量; 等于内部质量减少率 -∂/∂t ∭ ρ dV.

不可压缩 (ρ 常数): ∇·$\vec{v}$ = 0. 这就是 CFD 模拟、空气动力学的基础方程之一.

补充: 麦克斯韦四方程 + 电磁波

知识点 12.4: 麦克斯韦方程组 (微分 + 积分形式)

电磁场的核心 4 方程, 用场论算子统一表达:

微分形式 (本地):

  1. 高斯电场: ∇·$\vec{E}$ = ρ/ε₀ (电荷是电场源)
  2. 高斯磁场: ∇·$\vec{B}$ = 0 (无磁单极)
  3. 法拉第: ∇×$\vec{E}$ = -∂$\vec{B}$/∂t
  4. 安培-麦克斯韦: ∇×$\vec{B}$ = μ₀J→ + μ₀ε₀ ∂$\vec{E}$/∂t

积分形式 (全局, 用高斯/斯托克斯转化):

  1. ∮∮ $\vec{E}$·dS→ = Q_内/ε₀
  2. ∮∮ $\vec{B}$·dS→ = 0
  3. ∮ $\vec{E}$·dr→ = -dΦ_B/dt
  4. ∮ $\vec{B}$·dr→ = μ₀I_内 + μ₀ε₀ dΦ_E/dt

知识点 12.5: 电磁波 (波动方程推导)

真空 (ρ=0, J=0), 对法拉第方程取旋度: ∇×(∇×$\vec{E}$) = -∂(∇×$\vec{B}$)/∂t 左边 = ∇(∇·$\vec{E}$) - ΔE→ = -ΔE→. 右边 = -μ₀ε₀ ∂²E→/∂t². ∴ ΔE→ - μ₀ε₀ ∂²E→/∂t² = 0

这是波动方程, 波速 c = 1/√(μ₀ε₀) ≈ 3×10⁸ m/s. 光就是电磁波 — 麦克斯韦 1865 推出, 电磁学与光学大统一.

例题 12-4

$\vec{E}$ = E₀ sin(kz - ωt) x̂→ (沿 z 传播, x 偏振). 代入波方程: -k²E + (ω²/c²)E = 0 ⇒ ω = ck (色散关系). LED/激光/无线电的基本构型.

补充: Helmholtz 分解 + 微分形式入门

知识点 12.6: Helmholtz 分解定理

定理: 在合适条件下, 任意向量场 $\vec{F}$ 可唯一分解为: $\vec{F}$ = -∇$\varphi$ + ∇×$\vec{A}$

  • $\varphi$ 是标量势 (scalar potential)
  • $\vec{A}$ 是向量势 (vector potential)
  • -∇$\varphi$ 部分: 无旋 (∇×(-∇$\varphi$)=0) — "源"部分
  • ∇×$\vec{A}$ 部分: 无散 (∇·(∇×$\vec{A}$)=0) — "旋"部分

物理含义: 任何流场 = 纯散场 + 纯旋场.

例题 12-6

(电磁场): 麦克斯韦方程中

  • 电场 $\vec{E}$ = -∇$\varphi$ - ∂$\vec{A}$/∂t (静态部分 + 时变部分)
  • 磁场 $\vec{B}$ = ∇×$\vec{A}$

$\varphi$ 和 $\vec{A}$ 一起叫"四维势" ($\varphi$, $\vec{A}$) — 相对论中是 4-向量.

应用:

  • 流体力学: 不可压缩流速分解 (势流 + 涡流)
  • 图像处理: 法线场分解 = 法向反射 + 切向流
  • GIS: 风场分解 (辐散场 + 旋转场), 用于气象预测

知识点 12.7: 微分形式 (Differential Forms) 入门

统一观点: 高数下册末尾的 Green/Gauss/Stokes 三公式本质上是同一个定理(广义 Stokes), 用"微分形式"语言能写成: ∫_M dω = ∫_$\partial$M ω

其中:

  • ω 是 (k-1) 次形式
  • dω 是 k 次形式 (外导数)
  • M 是 k 维流形, ∂M 是其边界

对应表: | 经典 | 微分形式 | k | |---|---|---| | Newton-Leibniz | ∫_[a,b] df = f(b)-f(a) | 1 | | Green | ∫_D dω = ∫_$\partial$D ω | 2 | | Stokes | ∫_S dω = ∫_$\partial$S ω | 2 | | Gauss | ∫_V dω = ∫_$\partial$V ω | 3 |

外导数 d 的核心性质: d∘d = 0 (二阶外导为零). 这就是 "∇×(∇f)=0" 和 "∇·(∇×$\vec{F}$)=0" 的统一表述.

应用:

  • 现代物理: 广义相对论用张量, 张量= 多线性微分形式
  • 微分几何: 流形上微积分
  • 拓扑学: de Rham 上同调 = "几何拓扑桥"
  • 数据科学: 拓扑数据分析 (TDA) 用微分形式衡量数据流形拓扑

知识点 12.8: 圆环面 (Torus) 面积体积 + 例题

圆环面 (donut shape): 大圆半径 R, 小圆半径 r, 用参数化: x = (R + r cos u) cos v y = (R + r cos u) sin v z = r sin u 其中 u, v ∈ [0, 2π].

面积: S = ∫_0^(2π)∫_0^(2π) |r_u × r_v| du dv = ∫_0^(2π)∫_0^(2π) r(R + r cos u) du dv = 4π²Rr.

体积 (圆环内部): V = 2π²R r²

Pappus 定理 (旋转体):

  • 圆环表面积 = 小圆周长 × 大圆周长 = (2πr)(2πR) = 4π²Rr ✓
  • 圆环体积 = 小圆面积 × 大圆周长 = (πr²)(2πR) = 2π²Rr² ✓

Pappus 定理一般化: 任意平面图形 (面积 A) 绕同平面外的轴旋转, 旋转体: V = A · (2π·重心到轴距离) S = L · (2π·重心到轴距离), L 是图形周长.

例题 12-8

矩形 [0,1]×[0,1] 绕 x = 2 旋转的体积. 重心 (1/2, 1/2). 到轴距离 = 3/2. A = 1. V = 1 · 2π · 3/2 = 3π.

应用:

  • 工程: 管道/轮胎/弹簧 = 圆环
  • 拓扑: 环面 χ = 0, "亏格 1"
  • 物理: 等离子体托卡马克 (Tokamak) 装置形状
  • AI: 数据可能分布在低维环面上 (流形假设)

知识点 12.9: 立体角 (Solid Angle) 概念

知识点 12.10: 流体力学 Navier-Stokes 方程 (千禧难题)

N-S 方程 (不可压缩流体): ∂$\vec{v}$/∂t + ($\vec{v}$·∇)$\vec{v}$ = -∇p/ρ + ν Δv→ + $\vec{f}$

各项含义:

  • ∂$\vec{v}$/∂t: 局部加速度
  • ($\vec{v}$·∇)$\vec{v}$: 对流加速度 (非线性核心)
  • -∇p/ρ: 压力梯度
  • ν Δv→: 粘性扩散 (ν = 运动黏度)
  • $\vec{f}$: 体力 (重力等)

连续性方程 (不可压缩): ∇·$\vec{v}$ = 0.

Clay 千禧难题: 三维 N-S 方程是否存在光滑全局解? 至今未解决, 100 万美元奖金.

应用:

  • 天气预报: ECMWF/GFS 数值天气模型基础方程
  • 飞机设计: CFD 仿真用 N-S
  • 心血管医学: 模拟血流
  • 推进 AI 物理引擎: 流体特效 (好莱坞 VFX)

知识点 12.11: 散度与 Laplacian 的源汇直觉

散度 ∇·$\vec{F}$ = 体积"源"密度.

  • ∇·$\vec{F}$ > 0: 该点是"源" (流出 > 流入)
  • ∇·$\vec{F}$ < 0: 该点是"汇" (流入 > 流出)
  • ∇·$\vec{F}$ = 0: 无源无汇 (不可压缩流, 静电场真空)

Laplacian Δf = "二阶散度" 通量平均.

几何直觉:

  • Δf > 0: 当地值低于周围平均 (温度场: 该点正在升温)
  • Δf < 0: 当地值高于周围平均 (该点正在降温)
  • Δf = 0: 调和函数 (稳态)

热传导方程: ∂T/∂t = $\alpha$ ΔT. 解释: 温度变化率 ∝ 局部 Laplacian. 哪里"低于周围平均"哪里就被加热.

例题 12-11

f = x² + y² + z². Δf = 2 + 2 + 2 = 6 > 0. 该函数在每点 (除原点) 都是"凹" — 等于 r², 越远越高.

统一图谱: | 物理方程 | 形式 | |---|---| | 热传导 | ∂T/∂t = $\alpha$ ΔT | | 波动 | ∂²u/∂t² = c² Δu | | Schrödinger | iℏ ∂ψ/∂t = -ℏ²/(2m) Δψ + Vψ | | Poisson | ΔΦ = -ρ/ε₀ (静态) |

全部都是 Laplacian 的变奏曲. 这就是高数下册场论的统一威力.

知识点 12.12: Maxwell 应力张量 (电磁场的"压力")

电磁场的能量密度: u = (1/2)(ε₀ E² + B²/μ₀)

Poynting 向量 (能量流密度): $\vec{S}$ = (1/μ₀) $\vec{E}$ × $\vec{B}$ 单位: W/m²

能量守恒 (Poynting 定理): ∂u/∂t + ∇·$\vec{S}$ = -$\vec{E}$·$\vec{J}$

意思: 局部能量增长率 + 能量流出散度 = -电场对电流做的功.

例题 12-12

电磁波 (前面推导过) 的 Poynting 向量沿传播方向, $|\vec{S}|$ = c·u. 光携带能量沿光线传播.

应用:

  • 激光: 高能密度光束
